GRD3日誌---魚缸咖啡：高C/P值蛋糕與單品！

Jun 5, 2014

 

(這樣一份焦糖布蕾竟才NT.80，強大！)

按：魚缸咖啡已搬至三民路新址！

----------
魚缸珈琲(魚缸咖啡) Coffish
地址：新北市蘆洲區三民路78號(新址)
地址: 新北市蘆洲區永樂街50號
電話: (02)8282-8350
時間: 10:00~20:00 Fri~Wed
----------

五月初原本要去的下午茶是這一家，名字叫魚缸珈琲。

但由於當日店內發生狀況，記得是水箱有問題所以無水可用，

大夥只好臨時轉移陣地到板橋的Sweet as Hope。

 

一兩禮拜後不死心又跑到蘆洲，

而且幾天內還去了兩趟，

主要還是因為對味C/P值又高的好咖啡，

這篇寫的就是第二次回訪的經驗。

 

遠在蘆洲的魚缸，不賣魚不賣缸，

由余氏兄弟兩人擔任老闆，哥哥主要負責外場，

曾待過法樂琪與寒舍艾美的弟弟負責甜點與咖啡。

(店門口)

(店外的桌位)

(店內也有販賣一些咖啡器材)

(正在挑選的咖啡豆)

(左：Hachira N2，右：Geisha 2010)

(半成品狀態的甜點)


從第一杯焦糖拿鐵就開始驚奇。

糖漿無特殊之處，但豆子配得極好。

少見與奶泡的融合度，

當中不見絲毫突兀的咖啡焦苦，咖啡奶香與焦糖三樣均衡恰到好處。

以一杯單價才NT.90的義式咖啡而言，實在難得可貴。

(焦糖拿鐵)

(少見與奶泡的融合度)


余老闆(弟)個人偏好淺焙單品，

但由於客人接受度的關係所以店內供給則為中淺烘。

兩人聊了不少，

對愛店Coffee Sweet與季節香交換了不少心得，原來大家都熱衷此道呀。

先試他一杯Hachira N2，然後是一杯老闆的壓箱寶。

與前幾天在小家咖啡的單品杯測狀況有些類似，都有些烘焙拿捏不佳的問題。

另外大罐裝所引發出來的過重日曬豆發酵香氣更是讓人有些受不了。

老闆畢竟還年輕，相信假以時日一定能克服掉這些難關。

(單品)


講講主餐吧。

煙燻鮭魚凱薩沙拉與青醬薄切蜂蜜火腿帕尼尼意外地細心，原來都是余老闆(兄)親自製作。

立場不同，就算非餐飲本科系出來一樣也能美味。

帕尼尼外皮或許還不夠香，但配上這等級的風味拿鐵，套餐價NT.190真的很棒。

(青醬薄切蜂蜜火腿帕尼尼)

(煙燻鮭魚凱薩沙拉)

  

躺在玻璃櫃裡的甜點其貌不揚，

原來是半成品，客人選好後再由老闆(弟)加工。

焦糖布蕾本體使用到香草籽在成本上開銷不低。

上頭的燈籠果刻意先冷凍過減緩其酸味，配上味道不錯的巧克力裝飾。

或許外觀與細部上好比馬卡龍還有待加強，但一份賣NT.80，沒什麼好挑的。

(焦糖布蕾)


濕潤感極重的提拉米蘇相比下較無布蕾驚豔，暫先略過不提。

但兩道甜品使用到的巧克力味道細緻度不錯，讓人好奇起來。

一問之下，原來老闆使用的是Cocoa Barry。

在巧克力界堪稱是物美價廉的高級品，

價格不若Volrhona，但味道上完全不惶多讓。

 

大老遠跑到人生地不熟的蘆洲，

難得遇到一位咖啡知音，乘興而來，但也乘興而返！

(提拉米蘇)
