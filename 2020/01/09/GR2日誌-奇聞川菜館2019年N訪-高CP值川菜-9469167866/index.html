GR2日誌---奇聞川菜館2019年N訪：高C/P值川菜！

Jan 9, 2020

(青椒皮蛋)

 

------------
奇聞川菜館
地址：桃園市中壢區中豐北路5號二樓
電話：(03)280-4555
時間：11:00~14:00 17:00~20:30 Mon~Sat
------------

 

又一家桃園長輩帶路的好館子，聽說是幾位嫁來台灣的重慶妹子開的。

先從冒菜小攤做起，幾年後在A21環北站附近租下二樓正式開了餐廳。

首度前訪，純吃飯沒拍照，吃得很香。

二訪當然要帶相機，菜色上桌照例吆喝："相機先吃！"

 

(位在二樓的奇聞)

 

(平日中午客人較少)

 

(菜單)

 

用餐內容：
------------
口水雞(NT.200)
麻婆豆腐(NT.180)
土豆絲(NT.120)
青椒皮蛋(NT.120)
水煮牛(NT.320)
------------


二訪大夥兒吃得賓主盡歡。

口水雞肉質不算突出，但香料下得足下得夠本。

麻婆豆腐算喜歡的味型，鹹辣下飯。

土豆絲夠細夠脆，皮蛋刀工美，每次來這兩道都是必點。

水煮牛未達心中高標，有些可惜，畢竟是指標性的傳統川菜。

仔細回想應是香辣與調味沒搔到癢處，且肉缺嫩偏老。

 

(口水雞)

 

(麻婆豆腐)

 

(土豆絲)

 

(青椒皮蛋)

 

(刀工美)

 

(水煮牛)

 

(可惜肉過老了)

 

 

整體而言奇聞我去了N次，是一間吃得很舒服的家庭餐廳。

或許無法與北市頂尖又華麗的川菜相比，

但味道夠水準，氣氛家常而不拘束。

更重要的是，價位平易近人許多，

是每一次去桃園中壢找朋友最常列入考慮的覓食處之一。

 

(後來N訪預定的豬腳)

 

(幫家中長輩祝壽用的，好吃)

 

(酸菜魚很讚！)
