GRD日誌---28042008 Pierre Marcolini 試吃(Les Saveur du Monde)

Apr 28, 2008


四月甫拿到期望已久的Pierre Marcolini,

今天試吃這一盒"Les Saveur du Monde",

也就是世界各大產區的巧克力薄片(75% Cacao)

　包括:Equateur 75% Cacao, Madagascar 75% Cacao,

Java 75% Cacao, Ghana 75% Cacao, Venezuela Cacao 75%, Trinite Cacao 75%

還有Marcolini特調的混合款.

為此特地傍晚沒吃晚飯,只喝了幾杯拿鐵,

晚上教完課之後拿了一杯冷開水空腹開始品嘗.





















因為看過一些名人的Marcolini的心得,

所以決定將Marcolini特調擺在最後.



賞味心得:

1. Equateur 75% Cacao

現在回想起來,這一款真不應該最先品嘗,

這味道真是極為強烈呀,當巧克力在舌上開始溶化時,

那香味非常特別,但卻似曾相識,

　想了好久才想起這味道曾在以前吃的紅燒鰻魚罐頭裡出現過,

　嚴格說起來是些許焦炭味,微酸,有點類似咖啡裡的酸味,但不是很明顯.

跟後面的幾款相較之下,味道算是最有個性的.


2. Madagascar 75% Cacao

入口之後苦味較重些,有牛奶味,

　整體來說味道清新.


3. Java 75% Cacao

這一款很不錯,尾韻悠長,

　酸味一開始較為明顯,之後漸弱轉為些許苦味,

　入口過了一段時間我仍神遊在這虛幻的味覺世界裡.


4. Ghana 75% Cacao

甜味較為明顯,不過層次感較沒那麼重,

　吃完之後閉上眼睛好一陣子,微苦的尾韻才慢慢浮現.


5. Venezuela 75% Cacao

入口之後果香味四溢,酸味與甜味很均衡,

　很討喜又很讓人享受的味道.


6. Trinite 75% Cacao

入口時感到詫異,味道是這幾款中最淡的,

細細品嘗之後有些許核果味,層次感較不易感受出來,

　令人開心的是尾韻微甘呢.




最後重頭戲:

Pierre Marcolini 特調 75% Cacao

有趣的組合--->幾乎上述幾款出現過的味道都出現了.

不過很奇妙的是,期待的尾韻並沒有出來,有點虎頭蛇尾的感覺(笑)

但是真的得說,在吃完上述所有口味之後再試吃Marcolini的特調,

突然間,這些仍然存留在記憶中的味道又重新浮現真是很棒很棒的感覺,

　很過癮.





　結論:

　這真是一場十分鐘的味覺饗宴呀!
