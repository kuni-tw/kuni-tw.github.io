GRD日誌---01042010 SO MISO的創意日本料理

Apr 7, 2010





(友點的生魚片套餐)






















----------------------------------------------------------
SO MISO
地址：北市仁愛路四段112巷3弄2號
電話：(02)2325-5812
----------------------------------------------------------










愚人節當天中午，決定去鄰近的SO MISO大快朵頤一番，

　定了位置後，與友一同前往，

　離平常光顧的Pekoe不遠，真是沒想到。















　　(SO MISO店門口)




















　　(店內舒適宜人的裝潢與氣氛)


















　　(師傅的寶貝，從日本訂製而來的刀具)




















　　(吧檯)




















　　(自製的生薑)


























店內燈光舒適，無過暗過亮之問題，

氣氛宜人，不會過於拘謹又不會吵鬧，

　整體來說，清潔與質感兼具。

　就座後，小弟點了1000的中午套餐，

　據師傅說是晚間1200的套餐，移到中午減價成1000，
　
　感覺起來滿划算的。


　




　
　點餐時，與前頭的師傅聊了一番，

得知他有不少從日本訂製來的刀子，上面還刻著他的名字呢。

　中間聊到小弟對於台灣中高價位的日式料理向來頗為失望，

　或許在材料上可以趕得上日本的部分程度，

　但是在心智與心態上，仍有一段不小的差距，

　而這才是日本料理的精髓所在呀！

　




　

　光是個握壽司，至今在台灣沒吃過一次覺得好吃的，

　要不就飯太硬，太冷或米不對，

　要不就握太緊，太鬆或飯太多，

　要不就醋太少，太多還是太甜。

　殊不知，好吃的握壽司並不是材料，而是技術與心態先決呀。

　最討厭的就是這些所謂的高檔師傅傲慢的態度，

　一點也沒有日本懷石的精神。


　
　



　
　
　本想趁著評價直入雲霄的藤永SAN還在台北之時過去的，

　可惜那一年還願吃素，等一年後藤永SAN竟然回日本了。

　師傅一聽到這裡便說："那好，這一次我捏給你吃吃看。"

不然今天的中午套餐本來是沒有含握壽司的。







隨著前菜三小碟開始，SOMISO的創意逐漸慢慢釋出，

　比如章魚配上蛋黃醋，友人一開始還吃不出蛋黃醋，

　而我直覺確定是蛋黃醋，只是醋香不濃，所以還真難判定。

　師傅說是一種配上酒醋的蛋黃醋，醋有特地找過，

　這份用心的確吃得出來。














　　(前菜)













　隨後的生魚片，配上現磨的山葵，

　新鮮度夠，漂亮的色澤，

　美中不足的是切得過厚，以至於影響到了口感，

　有點可惜。













　　(生魚片)






























　山藥泥鮭魚卵，一小搓鮭魚卵置放在有如豆腐般的山藥泥上，

　口感十足，不過個人倒覺得不需要如此大費周章放在這麼寬的盤子上，

　以至於山藥泥層過淺，吃起來較沒那麼過癮。

　要是放在小點的容器內，應該也可以達到很棒的視覺效果，

　但口感更會升級好幾倍。













　　(山藥鮭魚卵)





























終於，師傅特別為我做的握壽司登場，

　原來是創意握壽司呀。

　創意加上一堆新IDEA的四貫握壽司，充滿著驚奇，

　但是個人最CARE的還是師傅的用心程度，

　在吃之前，師傅特地跟我說，今天的飯偏硬，

　所以應該還是無法令我滿足才對。

　"師傅您錯了，您這樣謙虛的態度才是我所欣賞的呀。"

　入口後，發現飯量似乎偏多，米硬了點，

　但米的硬度絕對不輸給之前在其他高價位名店所吃過的。

　光這一份用心，就讓這一份壽司大大升級了好多，

　光這一份壽司，小弟就覺得今天的來訪值回票價了。
　














　　(創意握壽司)






























　後來依序來了烤魚與烤雞肉，

　烤魚上置著細心擺置的辣椒絲，雞肉則是放在蒲葉上一起烤，

　兩者在口感還是味覺上皆無讓我特別驚艷的地方，

　甚至該說偏乾。

　
　










　　(烤魚與烤雞肉)





























　
　
　
　最後上桌的味增湯使用的是少見的紅味增，

　味道較濃郁，感覺還滿不錯的。














　　(味增湯)



























　甜點則是"皮蛋豆腐"，使用的是黑芝麻做成的膏狀豆腐，

　溫度似乎可以再冰一點，會更爽口。

　
　










　　(SO MISO的甜點：皮蛋豆腐)


























　
　
　付了帳時才又想起，這豐盛的一餐雖然沒有到這麼滿意，

　但才要價一千，感覺並不是那麼的貴，

　加上師傅用心照顧客人的態度，

　這錢花得很值得呀。

　












　　(飯後的冰紅茶)
