GRD3日誌---馬可波羅餐廳2012年首訪：頂級義式燉飯

Jun 13, 2012

(義式墨魚汁燉飯襯龍蝦及起士醬)

 

------------
馬可波羅餐廳
地址：台北市大安區敦化南路二段201號38樓
電話：(02)2378-8888
------------



自歐洲回國後至今過了好多年，

這期間為重溫曾經美好的Risotto回憶找遍台北各大義式餐廳，可惜無功而返。

唯一最接近的一次卻是在分子料理鼎恩餐廳所吃到的。

這可能也跟當天主廚Daniel Negreira本人的細心有關，

但實際上來說還是有一段微妙的小差距，主要是在Ｑ度與富有彈性的口感上。

在Tutto Bello，La Giara，La Festa，Osteria等多家名店品嘗後，

答案雖不盡相同但似乎都透露出一個關鍵：米種。

當今義大利最著名的米共有Arborio，Carnaroli與Vialone Nano三種。

其中的大顆粒的Carnaroli久煮不爛及口感彈牙的特性最受義式主廚們的歡迎。

講究點收割後再經一年以上的低溫熟成，但價格也高昂。

藉有耐心的拌煮與收汁，最後粒粒分明透出有如寶石般的光澤實在美麗極了。

 

當然義式燉飯並非一定得使用來自義大利產區的Risotto不可，

這就跟我曾吃過一次某日本板長使用美國米捏的握壽司一樣，美味到讓人感動。

食材的運用本來就有其彈性，最終目標還是在客人的感受上。

廚師再怎麼強調他的正統與高超技藝，客人不領情也是沒有用啊。

 

(馬可波羅餐廳內)

 

這晚我來到新主廚剛上任一個禮拜的馬可波羅餐廳。

位在遠企38樓，一邊品嘗美食一邊眺望台北夜景實在太棒了。

看著菜單選擇單點包括：

爐烤鵪鶉鑲鴨肝及冬栗捲佐紅酒洋梨醬汁

經典番茄起士沙拉

牛肉及雞肉清湯襯義式火腿餃

千層麵佐澳洲和牛肉丸及煙燻起士醬

義式墨魚汁燉飯襯龍蝦及起士醬

 

(窗外夜景)

 

首先上桌是南瓜泥開胃小點與餐前麵包，皆無限供應。

表現普通，不過麵包的油醋醬裡使用到陳年級的Balsamico，感覺還不錯。

 

 

(開胃小點)

 

(餐前麵包)

 

沒多久服務生端來爐烤鵪鶉鑲鴨肝與番茄起士沙拉。

前者在外觀上少了點用心與質感，但實地品嚐令人大吃一驚。

有別於樂沐陳嵐舒對於鵪鶉那令人嘆為觀止的華麗詮釋，

馬可波羅端的是食材基本功紮實的運用。

同時霸氣展現鵪鶉的野味與具有嚼勁及鮮嫩JUICY的口感。

在台北使用Mozzarella來做番茄起士沙拉的義式餐廳比比皆是，

而使用到Burrata的可就不多了。

講好玩點是混種Mozzarella，

實際上是一種鮮奶油與Mozzarella的混合體，口感潤滑奶味豐富。

由於保存不易，比較建議趁當日新鮮享用完畢。

原產地在Apulia的Burrata在早期沒有冰箱的時代是為了奶油的保存而製作的，

這也是他名字的由來。

除此外配上中等厚度的牛番茄切片與優質橄欖油的清香，

成功完成開胃前餐該有的任務。

 

(爐烤鵪鶉鑲鴨肝及冬栗捲佐紅酒洋梨醬汁)

 

(經典番茄起士沙拉)

 

此時新任大廚畢歐羅親自過來打聲招呼順便聊了一會，

得知小弟對於Risotto的偏好便拿出他使用的一年熟成Carnaroli，令人感到相當期待。

 

(新任大廚畢歐羅)

 

(一年熟成的Carnaroli)

 

接下來是千層麵佐澳洲和牛肉丸及煙燻起士醬，還有牛肉及雞肉清湯襯義式火腿餃。

前者嚐起來沒什麼特別吸引人，後者淺嚐後不禁嚇了一大跳。

濃郁肉味卻又清澈無比看得出花了不少功夫，

但喝起來卻有種精緻台式的林東芳清燉湯頭的錯覺。

跟王經理反映後他也不禁笑出來：

"其實主廚也很懊惱這一點，因為真的很多人抱怨過。"(我也笑了)

 

(千層麵佐澳洲和牛肉丸及煙燻起士醬)

 

(牛肉及雞肉清湯襯義式火腿餃)

 

終於到了令人屏息的Risotto時間。

看著服務生小心翼翼將義式墨魚汁燉飯襯龍蝦及起士醬端來。

唯美又深邃漆黑的米粒飽滿渾圓同時充滿深度的絕美光澤，剎那間將我所有魂魄都吸了過去。

圍繞中間的波士頓龍蝦好像海皇君般臨天下，入口後滿口的墨魚鮮味登時綻放。

黑漆漆的Carnaroli彈性雖然未臻完美但應該是墨魚醬汁的特性導致，

每一粒Risotto都吸飽沉甸甸又濃稠的鮮味，

突然間龍蝦海皇在一旁成了毫不起眼的配角太可憐啦。

 

(義式墨魚汁燉飯襯龍蝦及起士醬)

 

(漆黑的色澤有如寶石一般)

 

最後加點的提拉米蘇是現場製作的，

淋上現磨的Espresso噱頭十足，藉咖啡焦苦讓甜味更為豐富，

稱得上是錦上添花。

 

(現做的提拉米蘇)

 

 

(看起來還滿美的)
