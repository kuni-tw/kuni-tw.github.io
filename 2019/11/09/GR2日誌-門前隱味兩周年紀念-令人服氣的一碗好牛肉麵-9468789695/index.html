GR2日誌---門前隱味兩周年紀念：令人服氣的一碗好牛肉麵

Nov 9, 2019

(可能是這幾年最讓我激賞的一碗牛肉麵)

 

---------------
門前隱味
地址：新北市板橋區四川路一段87巷2弄12-2號
訂位網址：https://www.wumamii.com.tw/
---------------

 

這是一篇難以起筆的飲食紀錄，

因為打從一開始訂位到後來實際真正成行中間發生了太多插曲。

這些插曲大多是負面的，從起初訂位就困難重重。

或說不只困難，光點入店家官網就直接寫明"位子已滿"。

想取得訂位資格得先成為"熟客"，那就是得先網購他們家的燉煮牛肉。

一包NT.880，價格不斐，令人卻步。

重點是就算得到網購之後的熟客身分，

去實體店面吃麵依舊要排上好幾個月，甚至超過一年。

 

 

身邊部分饕友之間戲稱根本飢餓行銷，

此時真要感謝某號稱大熟客的粉絲幫忙喬位子。

聽說是偶然冒出的空位，時段雖不方便，但我是一個"為了吃我能忍"的人。

只是沒料到一兩個月後，店家寫訊息來告知更改了我的時間。

並非沒遇過超難訂的餐廳，但擅自更改客人訂位時間這還是首次碰到。

粉絲臨行前千交代萬交代：

"請別抱太大期待，不過就一碗兩百台幣出頭的家常牛肉麵嘛。"

"咦，才兩百出頭？"

"是...是阿。"

 

(昏暗巷弄間一處小店)

 

(真的相當小)

 

(裡頭僅五個座位)

 

先跳到文中：

當我嚐了第一口湯吃了第一口肉之後，原本的不爽霎那間一掃而空。

這大概是為何我提早動筆的緣故之一吧，因為很喜歡。

 

再跳回來：

晚間提早開車抵達板橋某巷弄就皺眉頭，好不容易找著車位竟然在遠處。

循址走進暗弄一處小店堪稱燈火闌珊，難怪取之"門前隱味"。

首次入訪，進門就是一股強烈的紅燒。

店內客人尚未到齊，氣氛凝重，我大氣也不敢喘一口。

隨人員陸續就坐，聽著他們愉悅交談，才驚覺清一色都是熟客。

有些固定幾個月到訪，甚至有仁兄兩禮拜前才來過，當然都是訂位。

雙邊的互動無自吹自擂也無歌功頌德，較像是偶爾碰面的好友。

老闆問了麵條與辣度後，先端上小菜與熱茶。

切薄的豆干乍看平凡無奇，入口香，是純粹的美味。

 

(老闆)

 

(附贈的小菜)

 

(豆干是簡單美味)

 

(熱茶)

 

(兩種不同的麵條)

 

(原來每個碗盤與杯子都不是一樣的)

 

(開始製作)

 

(甩麵很用力)

 

(用大杓篩牛肉湯)

 

今天的主角超乎預期。

我極少帶批判性的眼光去品嚐一碗牛肉麵，好比兩年前的安迪城堡。

沒辦法，兩家都難訂又那麼多門檻。

先試原味湯頭，深度的番茄，肉香以及讓我困惑的濃郁膠質。

加辣後亦不俗，一不小心喝光一半，馬上心知不妙。

又不好意思問老闆，只得細語叨擾鄰桌："這裡可以加湯嗎？"

"可以啊，跟大叔講就可以了。"

湯裡的紅白蘿蔔滷得"超~透~"，花腱大塊份量多，重點是"非常好吃"。

硬要挑骨頭的話，或許麵條稍軟了些，但絕未至干擾。

 

(顏色很漂亮)

 

(隔壁桌的)

 

(值得一嚐的辣醬)

 

(肉塊厚實量多與好吃)

 

(因剛好是萬聖節，被客人拱要戴面罩)

 

(還真戴了！)

 

第二次的湯頭我非常推薦，老闆刻意不過篩，湯中更濃稠更膠質。

仔細一翻，竟巧遇一小片"豬皮"，前述的困惑在此獲得了解答。

開心的一餐當中，老闆與大家聊得熱絡，

我才知今天剛好是"門前隱味兩周年紀念日"，實在太巧。

 

(第二次的湯頭，看這膠質！)

 

後語：

其實向來不太接受這類"搞剛"的訂位方式或門檻。

好比以前的RAW或前面提過的安迪城堡，

我每次都是跟饕友的團，有多的位子才湊。

就算餐廳真的超厲害，要我自己訂最多也就吃那麼一次就夠了。

我理解為何有不少人批評門前的行銷，

但完食後，我更理解為何他們家的熟客如此忠心。

超狹小店面只擠得下五人，一共六時段，難怪說一天限量30碗。

 

一碗非常優質的牛肉麵賣NT.225，

附小菜與熱茶，還可免費續湯與加麵。

若非訂位要等很久，不然我老實說，我也很樂意當常光顧的熟客。

是一家極少數純靠美味就完全扳回起頭壞印象的神奇小店，我服了。
