GR2日誌---Wildwood餐廳2019年N訪：語重心長

Dec 7, 2019

(少見的丁骨羊排)

 

----------------
WILDWOOD
地址：台北市信義區松壽路9號(新光三越A9館4F)
電話：(02)2722-8886
時間：11:30~15:00 16:00~21:30 Mon~Sun
----------------


寫文的速度永遠跟不上吃的，

吃的又跟不上未造訪的，總之是一種無解循環，只能將就。

Wildwood今年三月才去過，但就是沒機會寫。

七月時發生了Toyz小牛排咬不爛事件，

引發餐飲與顧客群兩邊幾乎完全對立的口水戰。

此時網路上風聲鶴唳，發言得謹慎，以免不小心得罪人。

過了十月風波逐漸平息，聽好友說Wildwood有新菜單，馬上相約前去。

 

(四樓門口)

Wildwood從初訪至今幾乎是從一而終的穩定。

三月時的印象猶在，紅甘因辣椒活躍，鮪魚泥因泰國米而酥脆。

綠蘆筍挺有意思，藉著紹興讓我多貪幾杯。

主餐們熟度無失誤，但醬汁才是最關鍵的亮點。

今天亦不惶多讓，雖然是午間，但朋友帶來好酒還是要開喝。

威石東木杉白葡萄特釀N.15，招牌的荔枝香與如沐春風的清爽果酸。

"欸，感謝好酒。"

"再怎麼好的酒若沒知音人能分享，也是枉然阿。"

(感動ing)

 

(威石東木杉白葡萄特釀N.15)

 

羽衣甘藍先拌過橄欖油再烤，入口酥鬆，

唯一可惜是不易吃，不論使用湯匙或叉子都不好取食。

澳洲Carpaccio有著細緻麻香，

玉米湯帶出違和的羅勒與薑，海鮮湯的甲殼味稍嫌平淡。

廣島牡蠣熟度沒話說，

來自美國的蘆筍Juicy又大根，與戳破的半熟蛋一起喫。

2014的Palazzo della Torre夠稱職，勻稱的風乾葡萄甜。

少見的T-Bone羊排，聽說乾熟七天，但效果不算明顯。

帶骨肋眼嚼勁不像肋眼，反倒接近紐約客。

兩塊肉熟度都選三分，正常表現，忽然想起Toyz事件。

餐飲界狂批網紅的拍片手法，尤其是把肉吐出來這個橋段。

也有人對於Toyz選的小牛排熟度不以為然："怎麼會選三分熟呢？"

反方們則對餐廳業者的態度開罵：

"不願聽取負面評論就算了，還說要告客人，這什麼餐廳阿！"

臨時問了某外場經理："結果有起訴了嗎？"

"欸，老闆最後還是選擇不提告..."

其他八卦容我在此先消音，爭議性的話題到此也該結束了。

唯一一道較失誤的大概就是撒索雞。

可能為了視覺效果與話題，將整隻雞腿雞腳與雞爪不分離一起上桌。

此雞是台北這陣子詢問度極高的雞種，遠自法國，雲林畜養。

明顯及濃郁的黃油是一大特徵，一口咬下，juicy且油脂豐富。

可惜部分肉質過乾柴，應是熟度沒抓好，

幸好一旁的沙嗲醬負責味覺上的補強。

 

(撒索雞腿)
