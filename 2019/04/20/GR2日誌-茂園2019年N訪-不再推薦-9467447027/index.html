GR2日誌---茂園2019年N訪：不再推薦

Apr 20, 2019

(2019在茂園吃到的白斬雞)

 

---------------
茂園餐廳
地址：台北市中山區長安東路二段185號
電話：(02)27528587
時間：11:30~14:00 17:30~20:30
---------------

 

這幾年每當朋友請我推薦台菜，

我腦海最先蹦出的前幾家一定有"茂園"兩字。

我喜歡他們家看似家常但實際有深度的味道，

連一些看似尋常不過的菜色好比白斬雞，也能整治不俗。


本日用餐內容：
---------------
白斬雞1/4(NT.220)
鹹冬瓜蒸午魚(NT.680)
椒鹽小捲(NT.200)
白菜滷(NT.380)
---------------

 

(門口)

 

(鮮魚櫃)

 

(這裡的菜單大多沒寫價格)

 

(帳單)

 

幾年沒來，趁旅歐歸國的學生邀老師吃飯，師徒倆相約中午。

期待的白斬雞最先上桌卻也最失落，

外觀尤為明顯，晶瑩帶透的皮已不復見，更遑論皮凍。

2009首訪吃他們家白斬雞簡直驚為天人，

剖面能見厚實又充滿膠質的皮凍。

當年入口雞皮Q，肉質彈，皮凍鮮美。

如今取而代之是肉汁缺，肉質乾，雞皮無趣。

 

(白斬雞)

 

(翻出舊照來比較)

 

(當年美妙的肉凍剖面)

 

以鹹冬瓜熱蒸的鮮魚向來堪稱茂園的招牌。

今天挑的是午魚，熟度沒失誤，但味道已無當年驚豔。

小卷鹹酥與香，學生難得多吃了幾塊。

白菜滷湯頭美，只是價格三級跳，

2009年吃的時候只要一百台票，如今這樣一份是NT.380。

 

(鹹冬瓜蒸午魚)

 

(小卷)

 

(白菜滷)

 

其實向來不太跟餐廳反映負面意見，因為大多沒什麼用。

但茂園是多年來的愛店，結帳時還是跟老闆娘提了白斬雞。

"喔，其實這是正常的狀況啦。"

"恩，好。"

望著結帳金額，中午這一餐花了近1500，有點肉痛。
