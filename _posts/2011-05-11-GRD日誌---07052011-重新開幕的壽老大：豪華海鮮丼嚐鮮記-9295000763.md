    壽老大<br/>
    新北市永和區大新街4號<br/>
    電話：(02)8923-8185<br/>
    時間：11:40-14:00 17:40-21:00 (週一休)<br/>
-----------------------------------------------------<br/>
<br/>
<br/>
<br/>
    <br/>
　有好一陣子沒去樂華夜市錢櫃入口對面的壽老大了，<br/>
    <br/>
　這陣子碰巧吃了幾家知名的散壽司，趁著壽老大重新開幕順便回去嚐嚐，<br/>
    <br/>
　傍晚與友走進熟悉的店門就座後直接點一份豪華海鮮丼，NT.450附湯，<br/>
    <br/>
　另外加點一份和風軟絲沙拉，NT.100<br/>
<br/>
<br/>
<br/>
<br/>
　(壽老大店門口)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191482.jpg" target="_blank"></a><br/>

<br/>
<br/>
<br/>
<br/>
　環顧一下嶄新的內部裝潢，廚房部分變得較之前來得大許多，<br/>
    <br/>
　整體質感與座位舒適度皆有顯著的提昇，令人感到耳目一新。<br/>
   <br/>
　沒多久軟絲沙拉與味噌湯上桌，前者生菜新鮮度頗佳口感清脆和風醬清爽溫和，<br/>
　<br/>
　後者味道頗濃郁表現不差。<br/>
<br/>
<br/>

<br/>
　終於輪到本日重頭戲豪華海鮮丼了，乍看下有些小失望，<br/>
    <br/>
　因用料不足少了干貝所以老闆娘請師傅給多點其他魚料，<br/>
    <br/>
　但份量與魚料豐富度仍比想像中弱些，不過還是大勝最近吃的幾家散壽司名店。<br/>
    <br/>
　使用水準不差的現磨山葵清爽但偏嗆，<br/>
    <br/>
　魚料部分共計有鮭魚，鮪魚，紅干，蔥花鮪魚泥，甜蝦，鮭魚卵與玉子燒，<br/>
    <br/>
　新鮮度略勝同區具有指標性意義的橋壽司，<br/>
    <br/>
　尤其是那兩尾甜蝦相當肥大與濃郁的蝦膏為整體表現加分不少呢，<br/>
    <br/>
　但若單以C/P值來論的話魚料用量與豐富度仍舊不敵後者。　<br/>
