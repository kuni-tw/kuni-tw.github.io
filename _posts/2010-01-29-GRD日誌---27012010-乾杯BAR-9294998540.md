（不開放電話訂位）<br/>
地址：台北市敦化南路一段 169 巷 5 號<br/>

  27號下午特地跑到忠孝敦南附近，<br/>
<br/>
　為的就是剛開幕不到一個禮拜正夯的Dazzling Cafe，<br/>
<br/>
  從敦化的Haagen Dazs一旁巷口走進去後，<br/>
<br/>
　赫然發現一直想去的乾杯BAR，剛好五點，店開始營業，<br/>
<br/>
  臨時便改了主意，先在這裡吃頓晚飯，<br/>
<br/>
　然後再去Dazzling Cafe吃甜點。<br/>
<br/>
<br/>
<br/>
<br/>

　進了店內，裡頭不大，只有吧檯可坐，<br/>
<br/>
  但位子還算頗寬敞，問清楚點餐方式後，<br/>
<br/>
　先單點一份烤綜合蔬菜組合(NT.150)，<br/>
　<br/>
　乾杯風義式生牛肉(NT.120)，一杯水蜜桃可爾必思氣水(NT.60)<br/>
<br/>
  後來則又加點了一盤肉與一盤少見的豬舌。<br/>

    (水蜜桃可爾必思氣水)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　在這裡若不想自己親自動手來烤，<br/>
<br/>
　可以請服務生現場代烤，這倒是很不錯的服務選項，<br/>
<br/>
　假如約了朋友來聚會，酒酣耳熱之際，<br/>
<br/>
　其實還滿難顧及到燒烤的品質，<br/>
<br/>
　而且現場的服務生在燒烤上的功力絕對夠專業。<br/>
<br/>
　老實說，請他們代烤雖然少了很多樂趣，<br/>
<br/>
　不過至少比我自己烤得美味多啦，<br/>
<br/>
　因為葛格可沒練過(笑)。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先說說烤綜合蔬菜組合吧，<br/>
<br/>
　裡頭有幾大片山藥切片，兩大支切半的筊白筍，<br/>
<br/>
　還有幾小片青椒與新鮮香菇。<br/>
<br/>
  經服務生烤過後，令人吃驚的是平日最討厭吃的香菇，<br/>
<br/>
　竟然可以變得如此軟嫩與美味。<br/>
<br/>
　山藥又熱又脆，筊白筍又香又嫩，<br/>
<br/>
　真好吃呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

　醃過的乾杯風義式生牛肉外層經烤過後，外熟內生，<br/>
<br/>
　淋上店家特製醬料，肉質口感非常柔嫩，<br/>
<br/>
　一口接一口，真的沒兩三下就吃光，<br/>
<br/>
　一樣也是很好吃。<br/>

  配上冰涼又微香的水蜜桃可爾必思氣水，<br/>
<br/>
　實在是太享受了。<br/>
<br/>
　隨後又叫了一盤肉與一盤豬舌，<br/>
<br/>
　看著服務生一邊烤我一邊吃，<br/>
<br/>
　真的，要不是待會還想留胃口吃別的，<br/>
<br/>
　不然我還想再待個一兩個鐘頭呢。<br/>
