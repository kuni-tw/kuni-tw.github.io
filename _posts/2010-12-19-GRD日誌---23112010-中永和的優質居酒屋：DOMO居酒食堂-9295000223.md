-------------------------------------------------------------------<br/>
　DOMO居酒食堂　<br/>
   地址：台北縣中和市中安街178巷13弄15號　(近永安市場捷運站)<br/>
   電話：(02)29226180<br/>
   官網：<a href="http://domo2012.pixnet.net/blog">http://domo2012.pixnet.net/blog</a><br/>
-------------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
    <br/>
　記得有一次去永安市場站附近的四號公園慢跑時，<br/>
    <br/>
　看到巷道裡有一家中永和地帶罕見的日式居酒屋，<br/>
　<br/>
　不禁感到相當好奇，心想著總要找一天來吃吃看。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(DOMO居酒食堂店外一景)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113953.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　禮拜三晚上學生臨時調課，九點多與友相約前去。<br/>
    <br/>
　進了店內，偏暗的光線擺設一般，且客人不多。<br/>
    <br/>
　看了菜單著實吃了一驚，相當多樣化的菜色，<br/>
　<br/>
　兩人便先點幾道試試水溫：<br/>
    <br/>
　桂花釀聖女，NT.60<br/>
    <br/>
　青蔥牛肉捲，NT.90<br/>
    <br/>
　日式生牛肉，NT.200<br/>
    <br/>
　明太子山藥，NT.120<br/>
    <br/>
　味噌醬圓鱈，NT.280<br/>
    <br/>
　魚子烏龍麵，NT.140<br/>
    <br/>
　最後再點壺店家特製的私釀梅酒，NT.120<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(牆上的菜單)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113954.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(菜單)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113955.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(店內一景)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113956.jpg" target="_blank"></a><br/>

<br/>
<br/>
  首先端上來的是店家招待的毛豆，不過因個人不大吃這種東西所以沒碰，<br/>
    <br/>
　朋友倒是吃得津津有味。<br/>
    <br/>
　然後是私釀梅酒，比想像中要來得好喝，酒精濃度頗高，<br/>
    <br/>
　順口且梅香濃郁，與後來上桌的餐點皆相當搭。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(店家招待的毛豆)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113957.jpg" target="_blank"></a><br/>
<br/>
<br/>

<br/>
<br/>
<br/>
　(私釀梅酒)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113958.jpg" target="_blank"></a><br/>

<br/>

　(非常好喝的梅酒)<br/>

<br/>

<br/>
<br/>
    <br/>
　桂花釀聖女令人驚艷，是自家醃釀的剝皮聖女番茄，<br/>
    <br/>
　桂花香味芬芳宜人且入味得宜，很棒。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(桂花釀聖女)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113960.jpg" target="_blank"></a></span> <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    <br/>
　青蔥牛肉捲口感一流，裡頭的香蔥很嫩而且滲出很自然的甜味<br/>
    <br/>
　Domo至此已讓我有不虛此行的感受了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(青蔥牛肉捲)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113961.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(青蔥牛肉捲口感一流)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113962.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　日式生牛肉更是一絕，上面一顆黃澄澄的新鮮蛋黃提供誘人的視覺效果，<br/>
    <br/>
　與切得很薄的生牛肉片共同創造出極為享受的鮮嫩肉質與順滑口感，<br/>
    <br/>
　真是沒想到呀。<br/>
<br/>
  <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(日式生牛肉)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113963.jpg" target="_blank"></a></span><br/>
<br/>
  <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　明太子山藥香脆，海風濃郁的明太子配上山藥切片，<br/>
    <br/>
　口感清爽，不過卻沒什麼太大的印象，這是第一道較為普通的菜色。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(明太子山藥)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113964.jpg" target="_blank"></a></span> <br/>

<br/>
<br/>
<br/>
<br/>
<br/>
    <br/>
　味噌醬圓鱈肉質很棒，師傅烤魚的實力由此可見不容小覷，<br/>
    <br/>
　不過有點過鹹，殊為可惜。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(味噌醬圓鱈)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113965.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(味噌醬圓鱈剖面)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483113966.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
　最後臨時加點的魚子烏龍麵實在是太好吃了。<br/>
    <br/>
　個人向來較不愛吃炒麵類的東西，<br/>
    <br/>
　沒想到DOMO居酒食堂的烏龍麵麵條竟然可以如此彈Ｑ。<br/>
    <br/>
　趁著微醺之際，繼續啜飲著甘美而又冷冽的私釀梅酒，<br/>
    <br/>
　配著精緻的居酒屋美食與美好的氣氛，心裡不禁大嚷著：<br/>
    <br/>
　"日本人實在是太懂得享受了呀！"<br/>
