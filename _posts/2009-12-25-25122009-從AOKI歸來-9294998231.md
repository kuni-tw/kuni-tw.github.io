<br/>
<br/>
<br/>


  剛從AOKI的夢想裡歸來。<br/>
<br/>
<br/>
　那份感動雖然不若最近胡椒餅的，<br/>
<br/>
　花了不少錢，但一切值得。<br/>
<br/>
　<br/>
　我花錢品嘗美食，向來不是為了"吃飽"，<br/>
<br/>
　更不想狂妄自大地排擠與攻擊他人，也不想膚淺地證明自己有多麼地博學多聞，<br/>
<br/>
  我只是為了增廣見聞與享受藝術而已，就如同音樂與美術一般，<br/>
<br/>
　人，不懂藝術是不會有真正的幸福的。<br/>
<br/>
<br/>
　<br/>
　何謂藝術？<br/>
<br/>
　可信手捎來的，平淡無奇或雜亂無章的，日常生活裡，<br/>
<br/>
　除了井然有序外，凡事皆有可能成為藝術的一份子。<br/>
<br/>
　藝術亦可是一種人文思想的表現，比如文學。<br/>
<br/>
　文藝復興時代後，人文價值逐漸凌駕於一切，<br/>
<br/>
　人類脫離了中古世紀的苦難後，藝術終於開花結果。<br/>
<br/>
　只可惜，不少心術不正之人學習藝術卻心懷不軌，<br/>
<br/>
　不懂欣賞藝術卻為了掩飾自己的不足而開始狂妄起來。<br/>
<br/>
　<br/>
<br/>
<br/>
