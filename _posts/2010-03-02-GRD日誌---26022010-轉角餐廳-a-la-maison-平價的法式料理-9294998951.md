----------------------------------------------<br/>
   轉角廚房(a la maison)<br/>
   地址：台北市大安區師大路92巷46號1樓<br/>
   電話：(02)2369-6188<br/>
----------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  26這一天下午，臨時想找一家餐廳果腹，<br/>
<br/>
　在網路上的美食分享裡挑了轉角廚房(a la maison)，<br/>
<br/>
　以價位來看，稱得上是相當平價的法式料理餐廳。<br/>
<br/>
<br/>
　從羅斯福路與師大路交叉口轉進去後，沒多久便看到92巷，<br/>
<br/>
　地理位置相當好找，走進店裡，仔細觀察一番，<br/>
<br/>
　店內空間不大，座位不多，不過氣氛滿具親和力的。<br/>
<br/>
　看了菜單點了最近剛推出的德國豬腳套餐，要價NT.550，<br/>
<br/>
  附餐前湯(我選南瓜濃湯)，沙拉，飲料(選冰咖啡)以及甜點(選鬆餅)<br/>
<br/>
　朋友點的則是香烤梅花豬套餐，要價NT.350。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(轉角廚房店內，店外，菜單等隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　首先上桌的是南瓜濃湯，顏色金黃的南瓜濃湯上置著幾粒烤麵包丁，<br/>
<br/>
　味道異常地濃郁與香甜。<br/>
<br/>
　然後沙拉登場，沙拉上立著一顆三分熟的白雞蛋，<br/>
<br/>
　相當漂亮的女服務生先示範如何使用刀具切殼食用，<br/>
<br/>
　然後好奇心的小弟也如法炮製一番，<br/>
<br/>
　然後使用一旁所附的烤麵包，上面灑著parmesan起司，<br/>
<br/>
　沾著裡頭半熟蛋黃一起入口，這讓享受沙拉多了一份樂趣。<br/>
<br/>
　沙拉口感一般，淋著事先選好的酒醋，夠清爽。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(南瓜濃湯隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(服務生示範如何切開三分熟水煮蛋)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(沙拉隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(小弟也來如法泡製一番吧)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　用完餐前湯與沙拉後，女服務生適時的來收盤子，<br/>
<br/>
　主餐沒多久便上桌，仔細一看，頗具份量的德國烤豬腳，<br/>
<br/>
　旁有紅蘿蔔，玉米筍，南瓜泥，洋蔥，<br/>
<br/>
　綠花椰菜與一小碟黃芥末。<br/>
<br/>
　紅蘿蔔顏色似乎有點淺，推測是蒸過，<br/>
<br/>
　配菜部分味道較清淡，沒什麼特別的印象。<br/>
<br/>
　不過豬腳部分相當美味，<br/>
<br/>
　諾大的一整根豬腳，表皮烤得酥脆，<br/>
<br/>
　裡頭肉質厚，膠質多，頗有咬勁，<br/>
<br/>
　一大口一大口咬得實在是相當過癮，<br/>
<br/>
　令人不禁回憶起當年在歐洲的陳年往事呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (德國豬腳隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　朋友的梅花豬也是一絕，<br/>
<br/>
　分量實在是無法形容的大與厚的烤梅花豬排，<br/>
<br/>
　一大口咬下，肥瘦分明與肉汁四溢的口感令人驚異，<br/>
<br/>
　可惜冷了後口感稍稍變得較硬。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (肥瘦適中又厚實的烤梅花豬排隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　主餐空盤收走後，服務生先詢問我冰咖啡偏好較濃還是淡，<br/>
<br/>
　我選前者，咖啡上桌後，先細嚐一口，<br/>
<br/>
　香甜濃郁的咖啡味，不禁大吃一驚，<br/>
<br/>
　這絕對是在餐廳內少見的水準咖啡。<br/>
<br/>
　<br/>
<br/>
　<br/>
　探聽之下，原來轉角使用的是雀巢Nespresso的義式濃縮咖啡系統，<br/>
<br/>
　這因為之前就喝過學生家中的，<br/>
<br/>
　好處是使用簡便與清理容易，缺點是成本較高。<br/>
<br/>
　這對於熱愛咖啡的小弟來說，絕對是大大地加分呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(轉角的冰咖啡與雀巢義式濃縮咖啡系統)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最後，小弟的鬆餅上桌，現場烤出來的原味鬆餅，<br/>
<br/>
　口感香甜，搭配鮮奶油，成為一場演出的完美句點。<br/>
<br/>
　整體而言，以梅花豬套餐C/P值最高，<br/>
<br/>
　德國烤豬腳雖然表現較佳，不過畢竟要價NT.550，<br/>
<br/>
　與較不常見的烤梅花豬相較之下，<br/>
<br/>
　如此的份量與肥瘦適中的厚實口感，<br/>
<br/>
　搭配細心準備的附餐，甜點與咖啡，<br/>
<br/>
　NT.350花得實在令人心甘情願呀。<br/>
