　這一次稱得上是大豐收，同時入手了不少很讚的星巴克收藏品，<br/>

　除了每一年必敗的日本櫻花隨行杯系列，<br/>

    <br/>

    <br/>

<br/>
<br/>
<br/>
<br/>
　趁著最近天氣回暖帶著心愛的杯子們到處走走，<br/>
    <br/>

    <br/>

    <br/>
　奶泡較一般摩卡多許多，取而代之是更濃郁的可可香，<br/>
    <br/>
　咖啡的焦苦變得微乎其微，喝起來較像是有"微量"咖啡元素的熱可可。<br/>
    <br/>
　除此之外同時點兩杯可得到一本限量新LOGO筆記本，<br/>
    <br/>
　每一家門市數量相當有限，共有白與黃兩種封皮顏色。(現已兌換光嚕)<br/>

------------------------------------沉痛的後語--------------------------------<br/>
    <br/>
　最後想了好久決心還是要為日本嚴重的災情說幾句沉痛的話：<br/>
    <br/>
　跟我們非常近的鄰國這一次竟然得遭受大自然如此無情的侵害，<br/>
    <br/>
　第一時間內先盡微薄之力捐款，同時也強烈地感受到人世間的無常，<br/>
    <br/>
　幾天以來心情異常地沉重，心裡無時無刻都會掛念著那些無辜的災民，<br/>
    <br/>
　在此除追悼那些不幸罹難者外，並願其他所有災民們能早日回歸正常生活，<br/>
    <br/>
　更期許自己從此後更能懂得珍惜身邊所有的一切，並再也不視為理所當然。<br/>
