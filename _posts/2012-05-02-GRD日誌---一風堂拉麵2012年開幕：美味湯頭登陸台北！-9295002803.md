<p><span> (中午的一風堂)<br/></span></p>
<p><span>-------</span><br/><span>一風堂拉麵</span><br/><span>地址：台北市中山北路一段85號</span><br/><span>電話：(02)2562-9222</span><br/><span>-------</span></p>
<p><span><span>台北最近這一波日式拉麵風潮愈演愈烈。</span></span></p>
<p><span>從去年的屯京點燃第一道烽火開始，持續延燒到現在仍絲毫未減其威力。</span></p>
<p><span>今天的一風堂試賣會再度成功在台北美食圈裡引爆新的戰局。</span></p>
<p><span>(門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255956.jpg" target="_blank"></a><br/></span></p>
<p><span>五月三號正式開幕營運的一風堂在日本名聲響亮。</span></p>
<p><span>其老闆河原成美拿過多次拉麵王稱號，絕對是天王等級。</span></p>
<p><span>這次由乾杯集團引進而來，</span></p>
<p><span>開幕與試賣日期老早傳遍全台，</span><span>試賣會當天小弟更是親身體驗一場排隊震撼教育。</span></p>
<p><span>(排隊人潮)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255957.jpg" target="_blank"></a><br/></span></p>
<p><span>早上十一點到達中山北路門口現場時才驚覺來遲了。</span></p>
<p><span>門口人聲鼎沸，聚滿了一排又一排的人潮。</span></p>
<p><span>隊伍從門外右側開始繞了足足有三圈，實在令人嘆為觀止。</span></p>
<p><span>十二點整時，隊伍尾端站著一位服務生婉拒後面到來的客人。</span></p>
<p><span>一點時服務生親切地走出來，發給烈陽下客人廚房裡自製的餅乾。</span></p>
<p><span>我們一點半時成功入座，店內十足日式木造裝潢。</span></p>
<p><span>小弟向來討厭排隊，</span></p>
<p><span>中間還曾與朋友輪班跑到隔壁的知多家吃點東西，</span><span>點餐時已沒什麼餓意。</span></p>
<p><span>(服務生發廚房自製的餅乾)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255959.jpg" target="_blank"></a><br/></span></p>
<p><span>(Nice)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255960.jpg" target="_blank"></a><br/></span></p>
<p><span>(開門的可是乾杯敦南店的店長)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255961.jpg" target="_blank"></a><br/></span></p>
<p><span>點了服務生推薦的和風叉燒小刈包(NT.60)，</span></p>
<p><span>一口酥炸牛舌(NT.140)，</span></p>
<p><span>辣味翠玉溏心蛋(NT.55)與赤丸新味拉麵(NT.230)</span></p>
<p><span>價位中等比想像中還要低，</span></p>
<p><span>服務生細心先詢問過我們希望的麵條硬度及上菜順序。</span></p>
<p><span>(十足日式木造風)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255962.jpg" target="_blank"></a><br/></span></p>
<p><span>(桌上擺設)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255963.jpg" target="_blank"></a></span></p>
<p><span>沒多久叉燒小刈包，炸牛舌與翠玉溏心蛋上桌。</span></p>
<p><span>小刈包裡的叉燒入味，肉質緊實，但沒什麼特別的。</span></p>
<p><span>炸牛舌好吃，外頭炸得偏老，但肉質處理佳，造就很有彈性的口感。</span></p>
<p><span>溏心蛋熟度令人驚艷，墊底的辣醬與底下白菜葉塔的清爽。</span></p>
<p><span>整體來說非常，非常好吃。</span></p>
<p><span>(朋友的GRD3與叉燒小刈包)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255964.jpg" target="_blank"></a><br/></span></p>
<p><span>(酥炸牛舌)　</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255965.jpg" target="_blank"></a><br/></span></p>
<p><span>一下子朋友與我的拉麵同時上桌，</span></p>
<p><span>每碗湯頭各嚐一口，</span><span>然後我就知道今天冗長的排隊值得了。</span></p>
<p><span>(白丸原味拉麵)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255967.jpg" target="_blank"></a><br/></span></p>
<p><span>(辣肉味噌拉麵)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255968.jpg" target="_blank"></a></span></p>
<p><span>一風堂絕妙的博多湯頭令人讚嘆，</span></p>
<p><span>赤丸新味拉麵同時有著偏花月嵐的焦蒜香與屯京平衡的濃郁豚骨。</span></p>
<p><span>朋友的辣肉味噌讓我的胃口大開，</span></p>
<p><span>但麵條部分令人困惑，朋友要求的中等與我的偏硬口感吃起來並無差別。</span></p>
<p><span>個人向來不愛細麵，但一風堂的細麵好吃，</span><span><strong>讓我忘掉排隊的不滿。</strong></span></p>
<p><span>(我的赤丸新味拉麵)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255969.jpg" target="_blank"></a><br/></span></p>
<p><span>(有著花月嵐的焦蒜香與屯京的濃郁豚骨)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255970.jpg" target="_blank"></a><br/></span></p>

<p><span>小弟這裡預祝一風堂大成功！</span></p>
