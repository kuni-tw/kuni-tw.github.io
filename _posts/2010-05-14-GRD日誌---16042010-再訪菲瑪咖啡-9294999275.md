-------------------------------------------------------<br/>
    菲瑪咖啡<br/>
    地址：台北縣永和市仁愛路300號<br/>
    電話：02-86608663<br/>
-------------------------------------------------------<br/>
<br/>
  距離上一次拜訪菲瑪已經是兩年多前了，<br/>
<br/>
<div><a href="http://www.wretch.cc/blog/kuni/11423285" target="_blank">(詳情請用滑鼠點閱：菲瑪咖啡初訪)</a></div><br/>
<br/>
　不過因為似乎不對味，後來也就沒再去了，<br/>
<br/>
　直到前一陣子，很巧合的，不少朋友紛紛又重新提到了這家咖啡，<br/>
<br/>
　咖啡界的好友更是給了相當高的評價，<br/>
<br/>
　便挑了一天傍晚，好不容易教完了整天的課，<br/>
<br/>
　拖著老邁年高的身子，重新前去拜訪。<br/>
　<br/>
　當天運氣相當不好，雨天路滑，自以為不用特別去查住址的小弟，<br/>
<br/>
　沒想到竟然在永和的仁愛公園那附近繞了大半天，<br/>
<br/>
　好不容易找到了咖啡廳，全身也濕了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　進了店裡，裝潢擺設依舊雅緻且燈光宜人，<br/>
<br/>
　不過似乎多了一台巨大的烘豆機，一問之下，<br/>
<br/>
　原來是老闆前陣子剛入手來自日本的機器。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　先點了一杯沒喝過的黑糖卡布，沒多久就端上桌，<br/>
<br/>
　上面一層晶瑩剔透深咖啡色的黑糖薄膜，<br/>
<br/>
　視覺上相當有質感，淺飲一口，才發現菲瑪的咖啡真的變了：<br/>
<br/>
　一股說不出來的香氣從口腔裡散了出來，<br/>
<br/>
　真的是想了半天仍說不出個所以然的香氣，<br/>
<br/>
　硬要說，可能有點類似核果類的感覺吧，<br/>
<br/>
　只記得也曾在某家咖啡廳聞過類似的味道。<br/>
<br/>
<br/>
<br/>
<br/>
　　(黑糖卡布)                                                                                <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　第二杯則是之前點過的摩卡，果然，<br/>
<br/>
　啜飲一口後，一樣有類似的香氣散了出來，<br/>
　<br/>
　咖啡味道與牛奶緊密的結合在一起，<br/>
<br/>
　至此不禁跑去老闆詢問了一番，<br/>
<br/>
　老闆一聽之下馬上要我聞聞看他的配方豆子，<br/>
<br/>
　打開玻璃蓋後一聞驚人，就是這個味道。<br/>
<br/>
　老闆跟小弟解說這是日曬豆的香氣，<br/>
<br/>
　繼續閒聊下去，也終於了解，<br/>
<br/>
　為何菲瑪的咖啡與兩年前的有如此顯著不同的口感了。<br/>
<br/>
　主要原因在於烘豆機，那一台來自日本的烘豆機，<br/>
<br/>
　老闆另外也更改了不少烘豆子的程序與方式。<br/>
<br/>
　兩杯喝了下來，整體表現均比兩年前好很多，<br/>
<br/>
　小弟口袋裡的好喝義式咖啡廳又多了一家啦。<br/>
　<br/>
<br/>
　<br/>
<br/>
　　(摩卡)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(老闆借我聞一下豆子)<br/>
<br/>
<br/>

<br/>

　　(菲瑪裡的氣氛依舊如此吸引人呀)<br/>
