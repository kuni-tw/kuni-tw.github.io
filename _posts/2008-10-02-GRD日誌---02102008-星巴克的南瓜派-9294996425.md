<br/>
<br/>
<br/>
<br/>
　　(呈美麗金黃色色澤的星巴克南瓜派)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　星巴克的南瓜塔在九月剛推出之時，<br/>
<br/>
　就不經意地擄獲了我的視線．<br/>
<br/>
　前陣子工作之餘沒事會找一家門市小憩的我，今天依然不例外，<br/>
<br/>
　點了一杯熱摩卡...呃，沒錯，是熱的，畢竟最近天氣也漸漸轉涼了，<br/>
<br/>
　大啖秋季螃蟹的時刻已到嚕，喔，這是題外話，先略過吧．<br/>
<br/>
　另外突然才想起中餐未吃，反正剛好要減肥，<br/>
<br/>
　一小份65元南瓜塔能成什麼氣候，不過才406大卡而已呀^^<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (南瓜塔俯瞰圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (近距離表層寫真)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  小巧的南瓜塔以酥派為底，黃澄的南瓜為內餡，<br/>
<br/>
　上面呈著金黃色色澤的漩渦，看起來很吸引人，<br/>
<br/>
　令人不自覺地有著食指大動的慾望，<br/>
<br/>
　切開後，正要拍剖面寫真，沒想到還有大自然的小生物跑來想合照留念，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(正要拍剖面寫真趕緊跑來拍照留念的小生物)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　裡頭還可看到刻意放進去一兩片的帶皮果肉，更增加視覺上的效果．<br/>
<br/>
　整體充滿著濃郁的南瓜香，真是好吃極了！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(美麗的剖面圖！)<br/>
