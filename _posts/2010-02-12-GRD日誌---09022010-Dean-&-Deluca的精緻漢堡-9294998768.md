<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  Dean &amp; Deluca一直以來是我最喜歡逛的美食地盤之一，<br/>
<br/>
  這裡也是世界著名的高檔食材集散地。<br/>
<br/>
　要買平常買不到的高檔食材嗎？來Dean &amp; Deluca尋尋寶吧。<br/>
<br/>




　Dean &amp; Deluca雖屬高檔食材超市，，<br/>
<br/>                                         
　不過裡頭一樣有熟食吧檯提供一些簡餐類，，<br/>
<br/>
　這一次趁下午空檔過去吃一頓，<br/>
<br/>
　明亮的空間裡，無拘無束地，享受著精緻的美食，<br/>
<br/>
　這實在是太享受啦。<br/>

<br/>

　　(與Ed Hardy合作，印上LOGO的餐巾紙)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
　點了從菜單上照片看起來就很搶眼的E套餐：<br/>
<br/>
　迷你漢堡*2+墨西哥牛肉捲，附上一瓶可口可樂，要價300<br/>
<br/>
<br/>
<br/>
  沒多久後，套餐上桌，<br/>
<br/>
　那兩粒小漢堡看起來真的很可愛，<br/>
<br/>
　微烤過的漢堡包上滿滿漂亮的芝麻，泛出油亮光澤，<br/>
<br/>
　一小碟沙拉，對切的牛肉捲，顏色相當漂亮的薯條。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先先吃一口沙拉，清脆的口感，<br/>
<br/>
　沙拉醬竟然不會油膩，反而有種清爽的感覺。<br/>
<br/>
　薯條皮酥脆，裡頭口感鬆軟，好吃。<br/>
<br/>
　牛肉捲更是驚人，雖然其中一半肉質較乾筋較多，<br/>
<br/>
　但整體說來，牛肉味十足，香氣令人食欲大開。<br/>
<br/>
　關鍵的小漢堡表現也相當搶眼，<br/>
<br/>
　漢堡部分麵包外皮微酥內鬆軟。<br/>
<br/>
　漢堡肉部分，肉多汁，肉味濃厚。<br/>
<br/>
　生菜新鮮口感好，而且顏色很漂亮，<br/>
<br/>
　跟漢堡搭在一起看起來令人垂涎欲滴，<br/>
<br/>
　三百元的漢堡套餐竟然可以如此地精緻迷人呀。<br/>
