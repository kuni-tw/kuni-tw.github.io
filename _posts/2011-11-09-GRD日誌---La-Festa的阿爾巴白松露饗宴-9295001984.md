<p><span>-----<br/>La Festa<br/><span>地址：台北市敬業四路168號2樓(維多麗亞酒店)</span><br/><span>電話：(02)6602-5671</span><br/>-----<br/></span></p>
<p><span>離初訪已過好一段時間。</span></p>
<p><span>當時特地跟餐廳講等十月大廚Igor Macchia再度訪台時一定要為我保留位置，</span></p>
<p><span>尤其十月正值世界頂級的阿爾巴白松露盛產期與一年一度的松露博覽會，</span></p>
<p><span>光得知Igor Macchia會特地挑好貨來台這消息就令人心癢難熬不已。</span><br/><br/><span>(Igor Macchia本人)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212640.jpg" target="_blank"></a></span></p>
<p><span>時間寫著十月二十二號，</span></p>
<p><span>這是為老饕熟客們與Igor Macchia敲定的相聚日期，</span></p>
<p><span>這次為搭配阿爾巴白松露所敲定的最終菜單如下：</span></p>
<p><span>----------</span><br/><span>香煎鱸魚佐馬鈴薯泥襯生蠔汁及新鮮白松露；</span></p>
<p><span>舒肥帝王蛋佐牛肝菌醬汁，香煎鴨肝及新鮮白松露；</span></p>
<p><span>白松露牧師帽餃；</span></p>
<p><span>手工細扁麵佐鼠尾草奶油醬汁及新鮮白松露；</span></p>
<p><span>炭烤和牛佐焗烤馬鈴薯及新鮮白松露；</span></p>
<p><span>白松露布丁襯白松露香草冰淇淋與橄欖油。</span><br/><span>----------</span><br/><br/><span>每道料理皆將由Igor Macchia毫不手軟刨下大量的新鮮阿爾巴白松露，</span></p>
<p><span>另依不同存放時間產生的香氣濃淡差異來搭配菜色，</span></p>
<p><span>每位熟客總量至少分配到超過20公克以上。</span></p>
<p><span>另外搭配的酒單也相當驚人平均價位皆不低，</span></p>
<p><span>包括：</span></p>
<p><span>----------</span><br/><span>Pio Cesare Gavi (DOCG)</span><br/><span>Pio Cesare Barbaresco (DOCG)</span><br/><span>Pio Cesare Barolo (DOCG)</span><br/><span>Pio Cesare Barbaresco Il Bricco (DOCG)</span><br/><span>Pio Cesare Barolo Ornato (DOCG)</span><br/><span><span>----------</span></span><br/><span>曾探過幾家高級餐廳包括松露之家阿爾巴白松露的行情，</span></p>
<p><span>今年因產量稀少被炒至天價，在台北用餐行情是加點一克七百台票，</span></p>
<p><span>這次一口氣要吃下二三十克光想像就覺得很奢侈。</span></p>
<p><span>(為La Festa當晚一般客人準備的白松露)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212641.jpg" target="_blank"></a><br/> </span></p>
<p><span>晚間微涼到達維多麗亞酒店時已近七點，</span></p>
<p><span>走進餐廳馬上見到Igor Macchia本人，</span></p>
<p><span>合照後在服務生帶領下走進深處的低調包廂，</span></p>
<p><span>門前赫然發現今晚的米白色主角正在閃亮發光，</span></p>
<p><span>躺在金色銅盤上的阿爾巴白松露看起來是如此地高貴。</span></p>
<p><span>(今晚閃閃發亮的主角：阿爾巴白松露，當晚要全部吃光)</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212647.jpg" target="_blank"></a></span></p>
<p><span>餐會開始前Igor Macchia本人親自上前與大夥們致意與解說餐點，</span></p>
<p><span>同時間侍酒師開始斟上第一支白酒Pio Cesare Gavi，</span></p>
<p><span>香氣淡雅好入口。</span></p>
<p><span>然後Igor走到桌旁開始他今晚最重要的工作，</span></p>
<p><span>首先為香煎鱸魚刨下滿滿白松露。</span></p>
<p><span>端上桌時那濃郁無比的松露香氣至今仍令人難以置信，</span></p>
<p><span>跟以前所經歷過的完全無法相提並論：</span></p>
<p><span>悠長又霸氣的蜂蜜榛果與松露香等混雜在一起，</span></p>
<p><span>渾厚至極無法言喻。</span><br/> <br/><span>(第一支與唯一的白酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212648.jpg" target="_blank"></a><br/></span></p>

<p><span>(香煎鱸魚佐馬鈴薯泥襯生蠔汁及新鮮白松露)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212651.jpg" target="_blank"></a><br/></span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212652.jpg" target="_blank"></a></span></p>
<p><span>接下來侍酒師為我們斟上Barbaresco，一款令人驚艷的義大利紅酒，</span></p>
<p><span>隨著時間香氣輪廓愈來愈濃郁明顯，單喝C/P值高，</span></p>
<p><span>後來又多續兩杯(當晚酒無限量供應，只能說最後喝到快翻天)　</span></p>
<p><span>這次的低溫水波蛋首度讓人感受到Igor Macchia在與不在場的差異，</span></p>
<p><span>上次的蛋黃與今天相比明顯過熟許多，</span></p>
<p><span>半熟蛋配上出土時間較新的阿爾巴白松露真是永遠也吃不膩呀。</span></p>
<p><span>(第一支紅酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212653.jpg" target="_blank"></a><br/></span></p>
<p><span>(舒肥帝王蛋佐牛肝菌醬汁，香煎鴨肝及新鮮白松露)</span></p>
<p><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212654.jpg" target="_blank"></a></span></p>
<p><span>搭配第三道料理牧師帽餃的是義大利紅酒Barolo，</span></p>
<p><span>與前者相反單喝較沒層次，但搭配餐點後甜味與香氣變得完整。</span></p>
<p><span>牧師帽餃是義大利皮蒙區(Piedmont)的傳統菜色，類似於我們常吃的餃子，</span></p>
<p><span>以前在歐洲時常買回家水煮飽餐一頓，內餡有如泥狀口感很軟，</span></p>
<p><span>滿滿現刨的阿爾巴白松露隨著熱氣電梯緩緩上升再度發出光芒。</span></p>
<p><span>(白松露牧師帽餃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212656.jpg" target="_blank"></a><br/></span></p>
<p><span>(第二支紅酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212657.jpg" target="_blank"></a><br/> </span></p>
<p><span>第四道的義大利麵搭配的是Barbaresco Il Bricco，</span></p>
<p><span>從這道開始搭配的紅酒似乎有高了一兩個等級的感覺，</span></p>
<p><span>不論在香氣還是口感上皆更有層次與深度。</span></p>
<p><span>義大利奶油手工細麵刻意簡單未經多餘調味繼續讓白松露處在高昂亢奮的心情中，</span></p>
<p><span>為接下來的重頭戲做最後一次的鋪陳。</span></p>
<p><span>(手工細扁麵佐鼠尾草奶油醬汁及新鮮白松露)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212658.jpg" target="_blank"></a><br/></span></p>
<p><span>(四支紅酒排排站)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212659.jpg" target="_blank"></a><br/></span></p>
<p><span>(好美的色澤呀)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212660.jpg" target="_blank"></a></span></p>
<p><span>終於炙燒澳洲和牛菲力上桌了，搭配的是Barolo Ornato，</span></p>
<p><span>與上款一樣很棒酒體強餘韻夠長，單喝表現非常搶眼。</span></p>
<p><span>菲力肉質無話可說軟嫩到極致不愧是跟隨牛排教父多年的好團隊，</span></p>
<p><span>這次Igor Macchia改刨存放多一兩個禮拜的阿爾巴白松露，</span></p>
<p><span>顏色較深香氣更濃厚所以較壓得住牛排的香氣。</span></p>
<p><span>不過吃到這時候也實在是太撐了。</span></p>
<p><span>(義大利製的牛排刀)</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212663.jpg" target="_blank"></a><br/></span></p>
<p><span>(菲力肉質無話可說軟嫩到極致)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212664.jpg" target="_blank"></a><br/></span></p>
<p><span>(這就是高水準的牛排)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212665.jpg" target="_blank"></a></span></p>
<p><span>甜點是白松露香草冰淇淋佐橄欖油與白松露布丁，</span></p>
<p><span>冰淇淋上再度刨下大量的阿爾巴白松露，</span></p>
<p><span>以清爽的方式適切地讓前幾道料理的白松露香氣繼續停留在口腔與鼻腔裡。</span></p>
<p><span>藉著濃厚無比的香氣特性Igor將阿爾巴白松露與帝王雞蛋存放在盒裡一天，</span></p>
<p><span>雞蛋隔著蛋殼仍能吸收松露香氣然後第二天取出做成烤焦糖布丁，</span></p>
<p><span>那入口時隨著布丁化開而同時融出的松露香氣實在令人嘆為觀止。</span></p>
<p><span>(白松露香草冰淇淋佐橄欖油)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212666.jpg" target="_blank"></a><br/></span></p>
<p><span>(以清爽的方式適切地讓前幾道料理的白松露香氣繼續停留在口腔與鼻腔裡)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212667.jpg" target="_blank"></a></span></p>
<p><span>(白松露布丁)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212668.jpg" target="_blank"></a></span></p>
<p><span>隨著廳內大夥氣氛逐漸炒熱起來乃至欲罷不能，</span></p>
<p><span>紛紛嚷著要Igor Macchia再做一道安可料理，</span></p>
<p><span>他老兄跑回廚房隔沒多久熱呼呼的一道披薩瞬間上桌，</span></p>
<p><span>令大夥拍案叫絕為當晚畫下最完美的句點。</span></p>
<p><span>(安可曲是Igor現做的特製披薩)</span></p>

<p><span>走出維多麗亞大門後強烈地感受自己全身上下簡直比白松露雞蛋還要香，</span></p>
<p><span>心想大概十年內暫時沒有再碰松露料理的勇氣了。</span><br/><span><br/></span></p>
