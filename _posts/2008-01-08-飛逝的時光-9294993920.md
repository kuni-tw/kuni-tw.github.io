 標題  飛逝的時光<br/>
 時間  Thu Jan  3 04:05:24 2008<br/>
───────────────────────────────────────<br/>
<br/>
<br/>
<br/>
　時光飛逝,當小風跟我提起土魠魚老闆娘問起我時,<br/>
<br/>
  我才驚覺時間的腳步竟然如此的急促.<br/>
<br/>
  網誌上的新年新希望選項還未蒙塵,興奮還未消逝,這樣就要收起了嗎?<br/>
<br/>
<br/>
  我又再度的浪費了不少時間,這我單價最高昂的財產;<br/>
<br/>
  如此惡劣的我.<br/>
<br/>
  一月四號似乎是個終結,只是似乎而已,我不太確定我到底該怎麼想,<br/>
<br/>
　因為我不能違背誓言,也是驕傲所在.<br/>
<br/>
  誓言不是便宜貨,說的算,就算是王法也得藐視,沒有二句話.<br/>
<br/>
<br/>
<br/>
　傍晚,吃著漸漸快要成為歷史的食物,雖然鹹了點,但是還滿適合這凝重的氣氛,<br/>
<br/>
  至少鹹不是淚水造成,而是滿懷的幸福所致.<br/>
<br/>
  晚上想著父母親在佈滿著音符的回程路上,享受閒宜的時光,<br/>
<br/>
  光是腦海裡繞過這一幕,就可以讓我安心的睡著.<br/>
<br/>
  雖然有一點類似俗氣沒品味的世俗觀,又有一種過年趕熱鬧的盲目,<br/>
<br/>
  但是只要庸人自擾,何需報與人知?<br/>
<br/>
<br/>
  在我心中有不少的隔層,有長滿蜘蛛網的,也有些透光又充滿著溫暖,<br/>
<br/>
  部分格局頗大不過仍未數位化,或是漸漸可悲地被遺忘的;<br/>
<br/>
　印象中,只要不是財務吃緊,基本上都有重新裝潢以及加大的機會.<br/>
<br/>
<br/>
  突然想起,人們的誤解諸如偏袒某事物看法常常得歸咎於自身的缺陷美,<br/>
<br/>
  雖然不妨侵犯到我,但是自掃門前雪同樣地會招來困擾,<br/>
<br/>
　這也是人際關係之中彼此牽動的大問題之一:<br/>
<br/>
  如何取得不被影響的平衡,也就是各取所需;<br/>
<br/>
　只是常常無法事後抽根煙講些負責任的話,畢竟人不是像煙一樣那麼好打發.<br/>
<br/>
<br/>
  我喜歡當個正面的人事時地物,當個光明磊落的,<br/>
<br/>
  永不畏懼自我批判自我嘲弄,只在乎超乎於一般世俗成見的教義上的堅持;<br/>
<br/>
  自我只是一種頹廢的藉口,無法自由,更進一步的會矇蔽自由,<br/>
<br/>
  使得自我與客觀皆無法處於和平共處的狀態.<br/>
<br/>
<br/>
  矛盾吧,人類本身不啻就是一種違背理想的矛盾嗎?<br/>
<br/>
