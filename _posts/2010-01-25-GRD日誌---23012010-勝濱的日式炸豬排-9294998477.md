<br/>
  禮拜六晚上，當課程提早結束後，<br/>
<br/>
　快十點之時來到公館汀洲路上漫遊，<br/>
<br/>
　這個時間點要找點新鮮又好吃的似乎不大容易呀。<br/>
<br/>
  不一會兒看到一間日式炸豬排店，<br/>
<br/>
　裡面燈光還亮著，似乎還有幾桌客人，眼睛為之一亮，<br/>
<br/>
　開了門詢問了一下服務生，從服務生的口音來看，應該是外地人，<br/>
<br/>
　得知還可以點餐後，便進去找了位置坐下。<br/>
<br/>
　本來想點蔥花蘿蔔泥腰內豬排套餐，<br/>
<br/>
　被告知腰內肉已賣完，便改點蔥花蘿蔔泥里肌肉豬排套餐，<br/>
<br/>
　菜單上還將豬排重量寫了出來，約130公克，整套餐要價：NT.240<br/>
<br/>
　然後我驚訝的發現，甜點竟然只有不同口味的冰淇淋可選，<br/>
<br/>
　個人硬著頭皮選了草莓口味，但在這冷天實在不是很想吃這個呀。<br/>
<br/>
<br/>
<br/>
  不一會兒，附餐與白飯等紛紛被端上桌，包括：<br/>
<br/>
　1.細切高麗菜絲，切得相當細，不過還沒有上一次的勝博殿來得細，<br/>
<br/>
　　淋上店家的柚子醬料，喔，這醬料的味道我反而比較喜歡，<br/>
<br/>
　　恩，相當可口的細切高麗菜絲呀。<br/>
<br/>
　2.一小份漬物，相當普通。<br/>
<br/>
　3.味增湯，一樣也是相當普通，味道實在沒什麼層次，<br/>
<br/>
　　說到不錯的好配菜，就不得不提起上一次才吃過的新井壽司了，<br/>
<br/>
    詳情請參閱：<a href="http://www.wretch.cc/blog/kuni/11593570" target="_blank">中和的新井壽司(請用滑鼠點選)</a><br/>
<br/>
<br/>
  4.白飯，口感普通，煮得稍為乾了點，有可能是因為快打烊的關係，<br/>
　　<br/>
　　但可吃得出來米沒有勝博殿來得好。<br/>
<br/>
　5.與勝博殿一樣的芝麻豬排沾醬，味道差不多，<br/>
<br/>
　　一樣要客人自行先將芝麻磨細，然後倒入店家自製沾醬。<br/>
<br/>
<br/>

　沒多久後，里肌肉豬排主餐登場，<br/>
<br/>
　上面鋪著厚厚的蘿蔔泥與蔥花，顏色很漂亮，<br/>
<br/>
　試探性的咬了一口沒沾醬的豬排配上蘿蔔泥，<br/>
<br/>
　恩，原來蘿蔔泥的重點在於增加清爽度與減低油膩感呀，<br/>
<br/>
　相當推薦的一種吃法，但是豬排肉相當偏乾硬，<br/>
<br/>
　幾乎沒什麼肉汁與肉味，個人相當不喜歡。<br/>

　結帳前，冰冷的甜點登場：草莓冰淇淋一球，<br/>
<br/>
　很普通的顏色，很人工的草莓香，<br/>
<br/>
　用湯匙舀了一兩口試嚐看看，恩，<br/>
<br/>
　不出所料，相當普通，當下詢問了一下服務生，<br/>
<br/>
　得知這家的服務生與老闆原來都是泰國人呀，<br/>
<br/>
　這冰淇淋也是泰國的，呃...突然間有點小錯愕，<br/>
