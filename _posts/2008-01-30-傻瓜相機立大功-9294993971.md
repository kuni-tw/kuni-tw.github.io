傻瓜相機立功 學生打敗攝影玩家<br/>
周麗蘭／大埤報導<br/>
<br/>
　小兵立大功▲大成商工3年級學生鍾明潔以手上的卡西歐7百萬畫素傻瓜相機，參加大埤<br/>
之美攝影比賽，擊敗眾多「大炮」拿下銀牌獎，令行家跌破眼鏡。（周麗蘭攝）<br/>
  (恕刪)<br/>
----------------------------------------------------------------------------------<br/>
  這真是一篇振奮人心的新聞呢!<br/>
<br/>
<br/>
  不過可能會被某些心眼狹窄的人士錯誤解讀,或是大作文章.<br/>
<br/>
  器材重不重要呢?當然重要.<br/>
<br/>
  心態重不重要?當然也重要!<br/>
<br/>
  兩者其實缺一不可.<br/>
<br/>
<br/>
  我們可以很確定的是,若有好心態卻沒有好器材一樣可以有好作品,<br/>
<br/>
  但是器材再怎麼好,心態不對,作品一樣可以很爛.<br/>
<br/>
<br/>
<br/>
　好器材為何重要?<br/>
<br/>
  因為攝影就是一種藝術價值的表現方式之一,<br/>
<br/>
　而器材本身就是扮演著傳遞作者訊息的媒介角色.<br/>
<br/>
<br/>
  器材愈好,作者就愈能將心中所想要的表達在作品之中,如此而已.<br/>
<br/>
<br/>
  這一篇新聞並不是要告訴我們,好器材並不重要,<br/>
<br/>
  而是說作者的心態比較重要,<br/>
<br/>
　但是這並不大表爛器材勝好器材唷^^<br/>
<br/>
<br/>
  阿...真期待某天進入單眼的領域呀(存錢存錢)<br/>
