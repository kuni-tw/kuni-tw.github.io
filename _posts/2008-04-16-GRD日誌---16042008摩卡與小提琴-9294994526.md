<br/>
<br/>
<br/>
<br/>
<br/>
  一杯香醇的摩卡,若是沒有了上面那一層迷人的鮮奶油便失去了它的韻味了,<br/>
<br/>
  有好一陣子點摩卡都會要求去掉上面的鮮奶油,<br/>
<br/>
  因為想藉此減少熱量.<br/>
<br/>
  但是有一次夥伴忘了去掉鮮奶油,我當下也不想給他們添麻煩還是喝了下去,<br/>
<br/>
  入喉一瞬間才驚覺原來這就是摩卡呀,那鮮奶油的微甜完全中和了摩卡本身些許的苦味,<br/>
<br/>
  真是美妙至極呢!<br/>
<br/>
<br/>
  從此之後我再也沒有計較那多餘的十幾大卡了...<br/>
