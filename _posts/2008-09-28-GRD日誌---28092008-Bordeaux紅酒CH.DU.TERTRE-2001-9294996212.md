<br/>
<br/>
<br/>
  (終於又找到一支好酒嚕)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
酒名：法國紅酒 CH.DU.TERTRE 2001<br/>
<br/>
<br/>
產地：Bordeaux<br/>
<br/>
<br/>
購買處：COSTCO<br/>
<br/>
<br/>
<br/>
  颱風天沒事溜到COSTCO看酒，挑了幾支紅酒，<br/>
<br/>
　回到家首先開的是這一支：CHATEAU.DU.TERETRE 2001<br/>
<br/>
  等級是Grand Cru，原料：<br/>
<br/>
　40%的CAB.SAU.<br/>
<br/>
  35%的MER. 5%的PV.<br/>
<br/>
  20@的CAB.FRANC.<br/>
<br/>
  使用開瓶器開瓶之後，香味柔順，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(使用已久的開瓶器，超好用！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(切開標籤後露出軟木塞頭)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(軟木塞慢慢升起...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一股宜人的桶味散了出來，馬上試喝了一口，味道不錯，<br/>
<br/>
　便與友慢慢品嘗，單寧細緻中竟然慢慢溢出甜味，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (中等厚實的酒體，香味四溢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這種甜味與加州Napa區的不太一樣，非常細膩，感覺比較像是"引"出來的，<br/>
<br/>
　就像某些甜味得靠其他味來引出來，比如鹹味，使甜更明顯，<br/>
<br/>
　但甜味本身是不變的．<br/>
<br/>
　這甜應該說是從單寧出來的吧，令人感覺很舒服，<br/>
<br/>
　愈喝愈順口，酒體中等厚實，顏色頗深，<br/>
<br/>
　喝到中途時鼻腔裡充滿著起司味與桶味，令人一直停不下來呀...<br/>
<br/>
　<br/>
