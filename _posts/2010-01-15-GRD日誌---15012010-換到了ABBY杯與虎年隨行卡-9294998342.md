　　(6oz.的星巴克黑色ABBY杯)<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　今天傍晚帶著只剩一格的好友卡與水源門市卡去了水源門市，<br/>
<br/>
　趁著虎年隨行卡剛發行時還有不少張，趕快先換為先，<br/>
<br/>
　這一次虎年限量商品有礙於已為了日本商品花了太多錢，<br/>
<br/>
　所以除了幫朋友買的幾只馬克杯與幾個磁鐵夾以外，<br/>
<br/>
　就都沒有買了。<br/>
<br/>
<br/>
<br/>
　前陣子好友卡用得超快，有鑑於最後兩格實在是不想用，<br/>
<br/>
　因為不想吃東西，所以趁每一次快用完時，<br/>
<br/>
　趕緊跟網友換全新沒蓋過的好友卡，來來去也換了四五張。<br/>
<br/>
　雖是如此，卻還是兌換到三張新的隨行卡，<br/>
<br/>
　後來全部送給好朋友們。<br/>
<br/>
<br/>
<br/>
<br/>
　這一次換到了剛發行的星巴克虎年隨行卡，<br/>
<br/>
　視覺上充滿喜氣洋洋的氣氛，預告著今年的好采頭，<br/>
<br/>
　實在是滿討喜的呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

  ( 詳情請見：<a href="http://www.wretch.cc/blog/kuni/11586953" target="_blank">星巴克公館水源門市開幕(請使用滑鼠點選)</a> )<br/>
<br/>
  門市夥伴送給當天來訪客人的集點卡片，<br/>
<br/>
　消費滿六次送一只6oz.的ABBY馬克杯，顏色共有兩種：綠與黑。<br/>
