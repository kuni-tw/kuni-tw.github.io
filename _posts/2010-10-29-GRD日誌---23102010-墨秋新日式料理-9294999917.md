<p><span>(墨秋的牡丹蝦刺身盛合)<br/><br/> <br/> ------------------------------------------------<br/> 墨秋新日式料理<br/> 地址：台北縣中和市秀朗路三段146號<br/> 電話：(02)2942-5668<br/> 時間：1130-2000<br/> ------------------------------------------------<br/><br/> <br/>一直想去中和市的墨秋新日式料理品嚐看看，<br/></span></p>
<p><span>這天晚上總算如願，與友沿著景平路再從秀朗路口進去，<br/></span></p>
<p><span>一下子便找著了。<br/> <br/>店外裝潢似乎有點老舊，並不是很顯眼，<br/></span></p>
<p><span>店內則好些，不過個人仍嫌有點雜亂，<br/></span></p>
<p><span>不是很喜歡。<br/> <br/>就座後與友一同點了NT.1600的雙人精緻套餐，<br/></span></p>
<p><span>裡頭菜色包括：<br/></span></p>
<p><span>1.海鮮野菜沙拉<br/></span></p>
<p><span>2.牡丹蝦刺身盛合<br/></span></p>
<p><span>3.芙蓉海鮮豆腐<br/></span></p>
<p><span>4.蒲葉牛肉味噌燒(或法式肝醬羊排)<br/></span></p>
<p><span>5.鹽烤海大蝦<br/></span></p>
<p><span>6.星鰻一本握壽司<br/></span></p>
<p><span>7.蛤蠣汁<br/></span></p>
<p><span>8.甜點<br/></span></p>
<p><span>另外加點一份蘆筍手捲，NT.40。<br/> <br/>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098520.jpg" target="_blank"></a><br/> <br/>(店內裝潢)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098521.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098522.jpg" target="_blank"></a><br/> <br/> <br/>墨秋的價位在中永和的日式料理界裡算得上是相當高，<br/></span></p>
<p><span>而且還非常偏遠，處在中和的邊陲地帶。<br/></span></p>
<p><span>點好餐後不禁開始好奇了起來，<br/></span></p>
<p><span>究竟墨秋身兼主廚的年輕老闆是基於什麼樣的技術與自信，<br/></span></p>
<p><span>讓他願意開出這等級的價位與菜單。<br/><br/>沒多久後海鮮野菜沙拉首先上桌，相當大的一份，<br/></span></p>
<p><span>裡頭有燻鮭，熟蝦，蝦卵與各種鮮蔬水果，<br/></span></p>
<p><span>料足新鮮度夠，沾醬調味得宜清爽，比想像中來得好吃許多，<br/></span></p>
<p><span>使人更加期待接下來的餐點。<br/> <br/>(海鮮野菜沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098523.jpg" target="_blank"></a><br/><br/>然後是我愛吃的蘆筍手捲，可能是被食堂屋的粗大蘆筍寵慣了，<br/></span></p>
<p><span>墨秋的其實並不差，蘆筍口感夠脆又清爽，<br/></span></p>
<p><span>不過仍舊與食堂屋的水準有一小段距離。<br/> <br/>(蘆筍手捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098524.jpg" target="_blank"></a><br/> <br/>隨後服務生端來一盆生魚片，<br/></span></p>
<p><span>上頭兩隻份量頗大的牡丹蝦，擺盤簡單俐落，<br/></span></p>
<p><span>首先試吃三種生魚片：鮭魚鮪魚與紅魽。<br/></span></p>
<p><span>溫度控制得剛剛好，魚肉似乎老闆有細心挑過，<br/></span></p>
<p><span>尤其是紅魽，油脂豐厚細嫩，絕對是這幾年來吃過的前三名。<br/></span></p>
<p><span>牡丹蝦新鮮，沒什麼腥味，SIZE也夠大，<br/></span></p>
<p><span>吃起來也很過癮。<br/></span></p>
<p><span>(牡丹蝦刺身盛合)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098525.jpg" target="_blank"></a><br/> <br/>(墨秋的生魚片水準相當優)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098526.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098527.jpg" target="_blank"></a><br/> <br/>此時老闆請我們一道南瓜手捲，<br/></span></p>
<p><span>裡頭有炸得酥脆的南瓜絲，口感有點介於餅乾與洋芋片，<br/></span></p>
<p><span>多了自然的南瓜甜味，我喜歡。<br/></span></p>
<p><span>(南瓜手捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098528.jpg" target="_blank"></a><br/><br/>接下來是芙蓉海鮮豆腐與鹽烤海大蝦(一人一隻)，<br/></span></p>
<p><span>前者有點像日本的揚出豆腐，只是味道更為精緻與鮮甜。<br/></span></p>
<p><span>後者則是烤大明蝦，與前面出過的牡丹蝦一樣，<br/></span></p>
<p><span>SIZE都很大，烤過的大明蝦肉質緊實有彈性，<br/></span></p>
<p><span>雖然有點過乾少了點肉汁，但還是很好吃。<br/> <br/>(芙蓉海鮮豆腐)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098529.jpg" target="_blank"></a><br/><br/>(鹽烤海大蝦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098531.jpg" target="_blank"></a><br/> <br/>(剝殼後看起來好好吃唷)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098532.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>星鰻握也相當令人驚艷，雖然舍利握功普通，</span></p>
<p><span>但魚身肥嫩無比，品質優，且醬汁調配得宜，<br/></span></p>
<p><span>實在令人難忘。<br/> <br/>(星鰻一本握壽司)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098533.jpg" target="_blank"></a><br/> <br/> <br/>法式肝醬羊排與蒲葉牛肉味噌燒這兩道熟食則令人驚異，<br/></span></p>
<p><span>前者沒有羊肉常見的死硬，熟度拿捏恰到好處，<br/></span></p>
<p><span>油花分布夠，增添了不少肥嫩的口感。<br/></span></p>
<p><span>後者鮮嫩的牛肉佈滿在散出微微清香的蒲葉上，<br/></span></p>
<p><span>藉著甜甜而又濃郁無比的味噌，將美味提升至升天的程度。<br/> <br/>(法式肝醬羊排)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098534.jpg" target="_blank"><br/> </a><br/>(蒲葉牛肉味噌燒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098536.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>此時熱騰騰的蛤蠣湯登場，我一看不禁瞪大了雙眼：<br/></span></p>
<p><span>"好大的蛤蠣呀！"<br/></span></p>
<p><span>一碗香噴噴的清湯碗底躺著一個巨大的蛤蠣，<br/></span></p>
<p><span>湯頭裡充滿著濃濃的蛤蠣鮮味，<br/></span></p>
<p><span>咬著如此少見的厚實蛤蠣肉感覺實在是太爽啦！<br/> <br/>(大蛤蠣湯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098537.jpg" target="_blank"></a><br/> <br/>(近看的很驚人)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098538.jpg" target="_blank"></a><br/><br/>(厚實的肉質咬起來超爽)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098539.jpg" target="_blank"></a><br/> <br/>最後在甜點上桌前不禁好奇點了一貫鮪魚赤身握，<br/></span></p>
<p><span>這一次倒是有點小失望，舍利握的力道算中間偏緊，<br/></span></p>
<p><span>但夾起來還是會散掉，當然也有可能是飯煮得不夠黏的關係，<br/></span></p>
<p><span>整體來看還算不錯，絕對有北市裡中價位以上的店家水準。<br/> <br/>(鮪魚赤身握)</span></p>
