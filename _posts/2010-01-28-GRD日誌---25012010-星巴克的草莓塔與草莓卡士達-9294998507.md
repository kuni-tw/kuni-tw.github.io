<br/>
<br/>
<br/>
<br/>
    (星巴克的草莓塔)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  趁著這一次虎年的相關產品新發售，<br/>
<br/>
　星巴克順勢也推出了草莓塔與草莓卡士達等糕點，<br/>
<br/>
　不巧小弟前陣子才剛在貝氏邏輯法式甜點吃過高水準的草莓塔，<br/>
<br/>
　<a href="http://www.wretch.cc/blog/kuni/11593845" target="_blank">(詳情請用滑鼠點選---&gt;個人相當喜愛的：貝氏邏輯法式烘培坊的味覺饗宴)</a><br/>
<br/>
　便不禁對著星巴克的相同產品抱著觀望態度，<br/>
<br/>
　不過25號那一天碰巧得在公館水源門市談事情，便順便各點了一份，<br/>
<br/>
　沒想到竟然有令人驚豔的表現。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　草莓塔，顏色較為普通，先咬了那一小口塔皮與草莓，<br/>
<br/>
　草莓的甜味相當明顯，微酸中帶著相當高的甜度，<br/>
<br/>
　真是令人吃驚。<br/>
<br/>
　吃過不少草莓類的糕點，最常令我感到不夠滿意的大部分都是草莓部分，<br/>
<br/>
　很少能在這一類糕點裡吃得到又甜又好吃的草莓果實，<br/>
<br/>
　或它與蛋糕本體融合得相當漂亮的甜味。<br/>
<br/>
　過酸是最常見的問題，使得草莓果實無法完整地融合在糕點內。<br/>
<br/>
　以連鎖店來說，星巴克的草莓塔雖然要價貴了些：NT.80<br/>
<br/>
  但是能如此統一保持草莓水準真的是實屬不易呀(吃了三次，水準皆一樣)<br/>
<br/>
　雖是如此，還是得說星巴克的草莓塔仍與貝氏邏輯的有一段不小的差距。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(草莓塔隨拍 by GRD2)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　草莓卡士達則令我感到相當不滿意(NT.80)，<br/>
<br/>
　味道淡多了，海綿蛋糕本體口感質感皆不錯，<br/>
<br/>
  但草莓味不明顯，裡層餡的果肉過於稀少，<br/>
<br/>
　味道少了特色，真是可惜呀。<br/>
<br/>
