<br/>
<br/>
<br/>
  (可憐的小金輪軸培林...真是慘不忍睹)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這一個月以來，愛車小金RV250前輪震動的感覺稍稍有了變化，<br/>
<br/>
　從一開始以為只是外殼在以往激烈的騎乘之下產生過多的震動，<br/>
<br/>
　到平雙那一次試車，騎乘當中有一次聽到一聲小爆音，<br/>
<br/>
　然後車子就變得稍笨了點！<br/>
<br/>
<br/>
<br/>
　最明顯的莫過於起步加速突然慢了點，這一點讓我非常無法適應，<br/>
<br/>
　然後就是明明後避震根本沒壓到底，卻有時候有類似的感覺，<br/>
<br/>
　便某晚回家時，下了車，搖搖前叉，心想該不會是輪軸的培林吧，<br/>
<br/>
　不對呀，培林才換過幾個月而已，不可能！<br/>
<br/>
　又搖了一下，猜測是珠碗，反正只要問題能找到就好，<br/>
<br/>
　就怕換了什麼東西問題依舊存在呀！<br/>
<br/>
　去了最常去的老店家，請阿林師父檢查，<br/>
<br/>
　他也推測是培林，後來拆了前輪後也證實了，<br/>
<br/>
　真是不可思議，看來車子大概是太重了！<br/>
<br/>
<br/>
<br/>
　RV250車雖貴，但是用料真的滿便宜的，<br/>
<br/>
　換個輪軸培林才三百．<br/>
<br/>
　換完後，整車加速好順暢，看著時速表靜靜的遽升，<br/>
<br/>
　心情也跟著好轉起來了呢！<br/>
<br/>
<br/>
