<br/>
<br/>
  國際級甜點名師Aoki之前在台北的臨時櫃撤走後，<br/>
<br/>
　這一次在Bellavita裡重新開幕，實在是一項令人開心的大好消息，<br/>
<br/>
　開幕沒幾天後，十二月二十五號，趁著下午小空檔之餘，臨時衝到Bellavita，<br/>
<br/>
　看著整層B2裡不少漂亮而又精緻的餐廳與美食專賣櫃，不禁心癢難耐，<br/>
<br/>
　只可惜小弟胃袋不夠大，反正來日方長不必急於一時呀。<br/>

<br/>
　Aoki並未弄得像樓上的SALON DE THE一樣，而是使用開放式空間，<br/>
<br/>
　這當然是有利有弊。<br/>
<br/>
　運氣好的小弟，有幸在現場先遇到Hasmore集團的總經理以及二代大老闆，<br/>
<br/>
　與他們深談了一會兒，聊的都是關於經營與美食部分，<br/>
<br/>
　個人相當佩服那老闆的想法與魄力。<br/>
<br/>
　Hasmore旗下的餐廳可不少，其中較有名的包括雙聖以及Ruth's Chris(茹絲葵)，<br/>
<br/>
  老闆人也相當低調，要是沒有介紹的話，從外在根本看不出來，<br/>
<br/>
　與我聊天時異常地謙虛，但面對員工時卻是相當嚴格與仔細。<br/>
<br/>
　<br/>
<br/>
　感想：<br/>
<br/>
　整體說來用餐環境非常棒，光線較暗但氣氛足，<br/>
<br/>
　使用的都是高檔餐具，服務上也滿得體的。<br/>
<br/>
<br/>
<br/>
　甜點部分則讓我感受到不少美好的味覺體驗，<br/>
<br/>
　這些味覺上的體驗若單獨只以好不好吃來評斷，<br/>
<br/>
　那實在是太荒廢了廚師在背後所作的一切努力了。<br/>
<br/>
<br/>
<br/>
  首先是Verrine Bamboo，一種冰淇淋與其他甜點口感混搭的全新口感，<br/>
<br/>
　上面那一層可可薄片脆餅，酥脆又香，<br/>
<br/>
　裡頭則有各種匪夷所思的餡料搭在一塊，每一口都是驚奇，<br/>
<br/>
　不過個人沒有相當喜愛就是了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Verrine Bamboo)<br/>
<br/>
<br/>
<br/>
<br/>

<br/>

<br/>
　MACARON部分，一顆單點是85，我們一共點了三顆:<br/>
<br/>
  玄米茶，焦糖鹽以及檸檬，另外還有三顆是老闆請的：<br/>
　<br/>
　焦糖鹽，巧克力與柚子。<br/>
<br/>
　Aoki的馬卡龍口感質感一流，比我在台灣吃過的幾乎都還要再細緻多了，<br/>
<br/>
　那又細又微酥的外皮真令人驚訝，內餡鬆軟層次分明，<br/>
<br/>
  幾乎所有的口味都能吃出它該有的特色與個性，<br/>
<br/>
　後味悠長，不會過重，而且香味與味道皆平衡。<br/>
<br/>
　可惜的是，整體吃下來，重複性的味道卻不少，<br/>
<br/>
　這部分待會再談。<br/>
<br/>
<br/>
<br/>

　<br/>
　蛋糕精選裡有六種小經典蛋糕，每一次皆不同，<br/>
<br/>
　這一次的是：檸檬塔，鹽味焦糖塔，紗綾，綠竹，<br/>
<br/>
　巧克力樂園與巧克力榛果。<br/>
<br/>
　這些小蛋糕其實都還滿不錯的，口感皆有一定水準，<br/>
<br/>
　不過在品嘗當中卻發現，部分味道與前述的馬卡龍是一樣的，<br/>
<br/>
　比如巧克力，焦糖鹽與檸檬的部分，<br/>
<br/>
　這樣較容易給人有點膩的感覺，與經理反應過後，<br/>
<br/>
　他說這部分倒是沒想到，之後會重新思考看看。<br/>

　<br/>
　總結：<br/>
<br/>
　這真的是一家國際級的甜點專賣店，馬卡龍口感部分令人讚賞，<br/>
<br/>
　味道處理得相當具有技術與經驗。<br/>
<br/>
　最為可惜的大概就是味道重複之現象了吧，<br/>
　<br/>
　反而無法顯出各類甜品的獨特風味呀。<br/>
