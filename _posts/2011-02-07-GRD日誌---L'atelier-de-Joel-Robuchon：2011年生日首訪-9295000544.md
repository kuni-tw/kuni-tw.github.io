<p><span> ------------</span><br/><span> L'atelier de Joel Robuchon</span><br/><span> 地址：台北市松仁路28號Bellavita五樓</span><br/><span> 電話：(02)8729-2628</span><br/><span> ------------</span></p>
<p> </p>
<p><span>今年的生日剛好與除夕撞期，所以提早一天慶祝。</span></p>
<p><span>決定去老早就想一嚐究竟的L'atelier de Joel Robuchon。</span><span><br/></span></p>
<p><span>Joel Robuchon這位美食界的大前輩於1945年4月7日在法國出生。</span></p>
<p><span>十五歲入行，三十六歲時先創Jamin餐廳，隔年即摘下米其林一星。</span></p>
<p><span>不到四十歲成為歷年來最短時間摘下三星者，非常厲害。</span></p>
<p><span>這位大人物後來曾有一段時間突然急流勇退，改主持美食節目。</span></p>
<p><span>2003年重新復出，轟動當時的美食界。</span></p>
<p><span>首先在巴黎與東京各開設一家"L'atelier de Joel Robuchon"。</span></p>
<p><span>之後陸續開了其他類型的餐廳，同時間摘下不少顆星星。</span></p>
<p><span>最後在2009年11月正式引進台灣，在知名的貴婦百貨BELLAVITA設立新據點。</span></p>
<p> </p>
<p><span>(以下是閒逛Bellavita時的隨拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160790.jpg" target="_blank"></a><br/><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160791.jpg" target="_blank"></a><br/><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160792.jpg" target="_blank"></a><br/><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160793.jpg" target="_blank"></a><br/><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160794.jpg" target="_blank"></a><br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160795.jpg" target="_blank"></a><br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160796.jpg" target="_blank"></a><br/><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160797.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>訂的是七點鐘的位子，提早到達先在各層樓逛逛。</span></p>
<p><span>時間到了才搭電扶梯上去五樓，</span></p>
<p><span>餐廳裡非常昏暗，在服務生引領下坐吧檯前。</span></p>
<p><span>後頭有幾桌影視圈的名人，</span></p>
<p><span>點了NT.2080的套餐，前菜選北義菠菜起司米餅佐帕馬火腿。</span></p>
<p><span>主餐是松露義式寬麵搭配野兔肉，</span></p>
<p><span>甜點加價選珍珠糖球佐松露淡奶慕斯及蜜濱梅Armagnac冰沙</span><span>。</span></p>
<p><span>另單點魚子醬與海膽佐白花椰菜鮮奶油(NT.1680)，</span></p>
<p><span>與法國Speciale Gillardeau生蠔(NT.320)，</span><span>還有配酒。</span></p>
<p> </p>
<p><span>(吧檯前香料玻璃櫃隨拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160798.jpg" target="_blank"></a><br/> <br/> <br/></span><span>以前歐洲窮學生時期曾狠下心來吃過幾次這種高檔餐廳。</span></p>
<p><span>為了簡單一盤松露烘蛋曾花上好幾百元，</span></p>
<p><span>也曾為一盤高檔的Risotto用掉了十幾天的伙食費。</span></p>
<p><span>當時雖窮，但還是願意勒緊褲帶滿足無止境的好奇心。</span></p>
<p><span>如今點餐可以不用這麼顧忌預算，突然間感觸良多了起來。</span></p>
<p><span>服務生先端上一籃麵包，由三樓的SALON DE THE製作。</span></p>
<p><span>剛烤過的麵包溫度恰到好處，道地的歐洲麵包口感較韌。</span></p>
<p><span>經咀嚼後麥香逐漸融進口腔裡，是再熟悉不過的味道。</span></p>
<p><br/><br/><span>(麵包籃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160799.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/></span><span>套餐的開胃小點是芥藍花蒸蛋與加點的法國Speciale Gillardeau生蠔。</span><span><br/></span></p>
<p><span>一小只玻璃杯裡斟滿洋蔥泡沫，上面撒點辣椒絲。</span></p>
<p><span>底下是口感柔順細緻的蒸蛋與芥藍花香，</span></p>
<p><span>生蠔部分新鮮度無可挑剔，沒任何腥味，</span><span>口感滑潤到一個極致。</span></p>
<p><br/><span>(開胃小點：芥藍花蒸蛋)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160800.jpg" target="_blank"></a><br/> <br/>(法國Speciale Gillardeau生蠔)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160801.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/></span><span>接下來是魚子醬與海膽佐白花椰菜鮮奶油。</span></p>
<p><span>是一道讓人有點失望的料理，</span></p>
<p><span>嚴格來說口感不錯，但魚子醬不甚明顯，</span></p>
<p><span>與海膽特有的濃郁海洋鮮味搭一起並沒有產生互補。</span></p>
<p><span>兩者特色似乎互相被抵消掉，可惜。</span></p>
<p> </p>
<p><span>(魚子醬與海膽佐白花椰菜鮮奶油)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160802.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/>(魚子醬與海膽佐白花椰菜鮮奶油剖面微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160803.jpg" target="_blank"><br/> </a><br/></span></p>
<p><span>此時是套餐前菜北義菠菜起司米餅佐帕馬火腿。</span></p>
<p><span>起司米餅火候拿捏很好，煎得微焦，</span><span>配合薄切帕馬火腿還算不錯。</span></p>
<p><span>   </span></p>
<p><span>(北義菠菜起司米餅佐帕馬火腿)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160804.jpg" target="_blank"></a>   </span><br/> <br/> <br/><span>主餐選松露義式寬麵搭配野兔肉，</span><span>份量大，可惜表現令人失望。</span></p>
<p><span>野兔肉切碎混在義式寬麵裡，口感偏油，</span><span>吃不到一半就過膩。</span></p>
<p><span>松露香味完全沒有出來，給人不知所云的感覺。</span></p>
<p><br/> <br/><span>(松露義式寬麵搭配野兔肉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160805.jpg" target="_blank"></a><br/></span></p>
<p><span><br/>(微距拍一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160806.jpg" target="_blank"></a></span></p>
<p> </p>
<p><span>加價點的甜點沒令人失望。</span></p>
<p><span>金黃色的球狀物在乾冰霧中，</span><span>帶給人夢幻的效果，也是這裡的招牌作。</span></p>
<p><span>使用吹製法將糖衣吹成一個球體，裡面塞滿內餡。</span></p>
<p><span>全名叫珍珠糖球佐松露淡奶慕斯及蜜濱梅Armagnac冰沙。</span></p>
<p><span>使用湯匙將糖衣球體敲碎後，一起拌著內餡享用。</span></p>
<p><span>松露香氣很明顯，配合微酸微甜的冰沙與慕斯。</span></p>
<p><span>表現非常搶眼，又令人無比著迷。</span></p>
<p> </p>
<p><span>(珍珠糖球佐松露淡奶慕斯及蜜濱梅Armagnac冰沙)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160807.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/>(令人為之屏息的一幕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160808.jpg" target="_blank"></a></span></p>
<p> </p>
<p><span>侍酒師表現尚可。</span></p>
<p><span>前餐配Domaine Michelot的Meursault，</span><span>甜與酸相當平衡果香足。</span></p>
<p><span>主餐配Santenay 1. Cru的Clos Tavannes，偏澀，</span><span>與野兔肉寬麵不搭。</span></p>
<p><span>沒想到他會法文，兩人就用法文溝通。</span></p>
<p><span>後來換了一瓶</span><span>2006的Hahn Estates，澀味較少，可惜仍舊不是很喜歡。</span></p>
<p><span>他看我使用GRD，不禁問了我一些攝影上的問題，</span><span>最後給了他一些建議。</span></p>
<p><span>大概是非常開心吧，</span><span>臨走前打包了一盒馬卡龍與甜點送給我。</span></p>
