<br/>
　從小在歐洲長大與生活十幾年，剛回國時，<br/>
<br/>
　原本還吃不慣台灣偏日系的麵包糕點，<br/><br/><span>　不過</span>在這幾年間慢慢地也接受了。<br/><span> </span><br/><span>　</span>當然內心當中，仍對於那較韌的麵皮深深地懷念著。<br/><span> </span><br/><span> </span><br/>
　這次得知法國百年麵包老店(Fondee en 1889)在台北仁愛圓環開業，<br/>
<br/>
　地點在前陣子剛撤走的Energie專櫃：仁愛路四段107號<br/>
<br/>
　便抽空跑去買了巧克力鬆糕，閃電泡芙還有鄉村三明治。<br/>
<br/>
<br/>
<br/>
<br/>
　巧克力鬆糕，一片175，不過甜，鬆軟間帶點微Ｑ，<br/>
<br/>
　與日系軟嫩不太一樣．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(閃電泡芙，NT.85，口感對國人來說較討喜，巧克力質感突出)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　鄉村三明治，一條185，麵包正如我所望，麵皮較韌，<br/>
<br/>
　咀嚼間麥香逐漸擴散，用料實在，<br/>
<br/>
　新鮮度佳(大概也很難不新鮮，畢竟才第一天開幕)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　整體說來相當貴，本來還想買其他糕點類，<br/>
<br/>
　但忘了帶錢包，只能使用零錢，<br/>
<br/>
　礙於預算，只好下一次再來光顧了！<br/>
