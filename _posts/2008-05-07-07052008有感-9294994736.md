<div class="article-content-inner" id="article-content-inner">
　最近身邊集結了不少大事,不論是私人或者是公眾的,<br/>
<br/>
  私人或是弦外的,理智的或是感性的,不論是哪種我都得說,<br/>
<br/>
  這都是我畢生所想要的:<br/>
<br/>
  人生四大味的酸甜苦辣,不同程度上的微調之後,<br/>
<br/>
  各以煎煮炒炸蒸烤燻炙醃漬風乾的形式呈現出來.<br/>
<br/>
<br/>
<br/> <br/>
<br/>
<br/>
<br/>
  有時候我特別喜歡風乾過的味道,有種不可預測的驚奇性,或說,<br/>
<br/>
  味道經層層抽絲剝繭的處理之後還能令人為之一亮,真是過癮極了,<br/>
<br/>
  亦或是那種藏和在初味底下另有一些難以形容的野性,<br/>
<br/>
  一種無法以人為控制的陳年因子.<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/> <br/>
　就因為這樣,每每吃燴飯時我從不將燴汁與飯一起攪拌,調酒時亦是如此.<br/>
<br/>
　我認為,均勻常常不是美味的開始,而是一種聽天由命,人云亦云又傀儡式的遊戲.<br/>
<br/>
　假使,將自己假想為長途火車上的旅客,盡頭未知,以不同的時速過站,<br/>
<br/>
  不同的天候狀況停站,途中來往的旅客從一開始就就座,沒過多久部分座位空了,<br/>
<br/>
  不同的臉孔替補上來,甚至連查票員也分老中青,路途遙遠,<br/>
<br/>
  以致於連廁所都不知道重建了幾次.<br/>
<br/>
　偶爾我跳開了車廂往外頭竄,車站裡人山人海我渾然不知身在何處,<br/>
<br/>
  能見度低到五指都看不清;如此的不定性,不可預知性就這樣子深深地吸引住我的味覺.<br/>
<br/>
<br/>
<br/> <br/>
<br/>
<br/>
<br/> <br/>
<br/>
  然後,慢慢地我腳也騰空了...<br/>
<br/>
<br/>
<br/> <br/>
<br/>
  究竟會飄到哪去呢?<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
