<br/>
<br/>
<br/>
<br/>
　　(三洋芋罐的一塊錢錢幣隨拍，感謝小風九月四號熱情贊助)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  友人與店長大人每天都會上我的網誌看我的新作品，偶爾留個言，<br/>
<br/>
　或是找機會給我技術性與構圖上的建議．<br/>
<br/>
　通常正面的評語我都會忘，負面的我卻一定銘記在心．<br/>
<br/>
<br/>
<br/>
　記得好像是從拉拉山的水蜜桃開始，我拍照的方式已有了顯著性的改變，<br/>
<br/>
　首先是"眼睛"．<br/>
<br/>
<br/>
　以前我會想辦法拍出"很棒"的東西，眼睛會先預設立場．<br/>
<br/>
　看到令人感動的東西，甚至不感動的，都會以偏向"誤打誤撞"的方式，<br/>
<br/>
　嘗試將之捕捉下來，回家後在螢幕上觀賞，<br/>
<br/>
　篩選然後調整部分作品的顏色與銳化，過程極為冗長，<br/>
<br/>
  這樣出來的照片：<br/>
<br/>
　一，極缺乏連貫性與故事性．<br/>
<br/>
　二，因為部分作品要在電腦上調整對比等等，使得發色不自然，<br/>
<br/>
　　　所以看久也不舒服．<br/>
<br/>
　三，永遠無法與自然發色的作品相比！<br/>
<br/>
<br/>
<br/>
　一直以來我都會去點閱以及觀賞那些大高手的作品，並針對優點以及缺點分析，<br/>
<br/>
　比如Kenny大的震撼性構圖，讓別人可以從照片中看到平常看不到的世界，<br/>
<br/>
　明明生活中不會去注意到的小點滴，卻會在他照片裡無限放大．<br/>
<br/>
　3000大的故事性，令人不得不佩服他願意花那麼多時間，<br/>
<br/>
　將腦海裡的想法全部付諸實行．<br/>
<br/>
　或是凱納那富有衝擊性的構圖，這也是相當困難的方式．<br/>
<br/>
　畢竟衝突性高的作品一般人較不易去接受，<br/>
<br/>
　但作品缺乏衝突性反而也會抑制創作者本身的成長！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　就在水蜜桃那篇，我第一次嘗試將眼睛裡所真實看到的付諸實現，<br/>
<br/>
　這使我走出了一大步，至少我知道我拍到了什麼，<br/>
<br/>
　不需要靠電腦調整顏色對比與銳化，<br/>
<br/>
　以自然發色的方式所產生出來的震撼力絕對不是後製後可以比擬的．<br/>
<br/>
　從此之後，我嘗試將照片拍得自然與舒服，<br/>
<br/>
　至少現階段是如此，而且會持續一段時間，這也是我當初買GRD2的目的．<br/>
<br/>
<br/>
<br/>
　我想要適當的融合衝突性與大眾性，挑戰視覺．<br/>
<br/>
　以食物來說，不論拍得多藝術，多麼有衝突性，只要看起來不好吃，<br/>
<br/>
　那就不是我要的．<br/>
　<br/>
　以GRD2來說，發色看起來很暖和，最棒的拍攝距離在三公尺以內，<br/>
<br/>
　其實還滿有利於食物的拍照，至少較勝GRD一代的冷性發色．<br/>
<br/>
<br/>
<br/>
<br/>
　我知道藝術的極限可以誇張到什麼地步，以前求學時看Wien Modern時記憶猶新，<br/>
<br/>
　連刮玻璃的聲音都可以稱之為藝術．<br/>
　<br/>
  但我並不想走到那個地步，這樣與我的人生規劃，<br/>
<br/>
　也與我父母親所灌輸給我的完全不符．<br/>
<br/>
　<br/>
<br/>
　人生在世，許多東西講得如何冠冕堂皇其實只是自欺欺人而已，<br/>
<br/>
　發自內心去質疑自己的認知時，常常會發現那些不過是選擇而已：<br/>
<br/>
　一種"不得已屈於現狀"的選擇，甚至可說是無力的屈服．<br/>
<br/>
　<br/>
　比如，你沒能力做到前面高手所做得到的，<br/>
<br/>
　就認為你只是忠於自我，這就是自我欺騙，長久之下你只會無止境的退化．<br/>
<br/>
　這當然沒什麼不好．<br/>
<br/>
<br/>
<br/>
  試問，渾渾噩噩的過一生有什麼不好的？^^<br/>
<br/>
