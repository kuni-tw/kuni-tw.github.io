   曼咖啡 Famonn Cafe<br/>
   地址：台北市光復北路9號<br/>
   電話：(02)2763-0383<br/>
   時間：07:00~22:00<br/>
----------------------------------------------------------------</span></p>
<span>
<p><br/>
<br/>
<br/>
<br/>
 </p>
<p>　八九月時咖啡界的好友曾跟我提起王品集團旗下這家即將開幕的品牌咖啡館，</p>
<p>　裡頭負責管控咖啡的莊宏彰曾在咖啡界前輩譚大哥開的立裴米緹待過好一段時間，</p>
<p>　因此我們在十月開幕後找了天中午過去探探。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(櫃台前)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212596.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
　曼咖啡開在原丹堤咖啡光復北店舊址所以感覺很面熟，</p>
<p>　營業方式較類似一般常見的連鎖咖啡，部分似乎有參考到星巴克的行銷策略，</p>
<p>　走進店內可以看到自有品牌的隨行杯等商品整齊排列在櫃子上，</p>
<p>　整體裝潢以淺藍為主走明亮淡雅風格，整體性與完成度高，</p>
<p>　但嚴格來說質感上仍與星巴克有一段不小的差距。</p>
<p><br/>
<br/>
<br/>
<br/>
<br/>
　(價位看板)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212597.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(自有品牌隨行杯等商品)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212598.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
</span>　<br/>
　點一杯熱巧克力摩卡(中杯NT.75)與哈密瓜塔(NT.95)</p>
<p>　取餐後因裡頭滿座所以坐在外頭開放區。</p>
<p>　淺嘗一口咖啡不禁愣住，濃郁的可可香裡透出適度的深烘咖啡微苦，</p>
<p>　以這價位能做到如此水準實在相當難得。<br/>
<br/>
<br/>
<br/>
<br/>
　(滿座的店內)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212599.jpg" target="_blank"></a><br/>

<br/>
<br/>
<br/>

　(開放區)<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
<br/>
　糕點部分比想像中來得好，</p>
<p>　哈密瓜的甜與芬芳和塔皮偏軟但細緻的口感搭配得宜令人驚訝，</p>
<p>　不過這價位偏高與中價位的咖啡有段不小的落差，</p>
<p>　偏貴的星巴克草莓塔也才八十台票，王品可能得好好重新思考一下呢。<br/>
