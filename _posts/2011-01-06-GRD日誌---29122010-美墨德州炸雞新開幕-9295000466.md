<br/><br/><br/><br/><br/><br/><br/>　(美式快餐店的店外裝潢)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　上一次來到師大夜市找朋友時在路口看到一家新開張的炸雞店，<br/>																			    <br/>　店外的裝潢相當地美式風格，色調鮮明，不禁好奇了起來。<br/>																			    <br/>　回到工作室後在網路上查了一下，原來是最近剛開幕的美墨德州炸雞。<br/>																			    <br/>　以前在國外時就相當喜愛Tex-Mex風味的食物了，<br/>																			    <br/>　看到美墨那兩個字當然會心動，便找了天晚上又跑回師大路。<br/>																			    <br/>　走進店裡在櫃檯點餐時，得知開幕期間全面九折，<br/>																			    <br/>　至於這項優惠到什麼時候截止則不清楚，<br/>																			    <br/>　服務生打趣的說這就要看老闆心情而定啦。<br/><br/><br/><br/><br/><br/><br/><br/>　(一樓的點餐櫃檯)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/></span><br/><br/><br/>																			    <br/>　點了三塊雞套餐，NT.139 附小配餐與中等SIZE的飲料。<br/>																			    <br/>　加一點錢升級為玉米濃湯與香辣墨西哥飯，<br/>																			    <br/>　另外單點了魚排與雞柳。<br/>																			    <br/>　不過看了半天帳單還是看不大懂，<br/>　<br/>　只知道全部打九折後變178，還滿便宜的。<br/>													  <br/><br/><br/><br/><br/><br/><br/>　(看不懂的帳單)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　二樓坐位區明亮且寬敞無比，就座後不一會兒餐點便全數上齊。<br/>																			    <br/>　首先嚐一口香辣墨西哥飯，可惜不香也不辣，頗為無味。<br/>																			    <br/>　然後是玉米濃湯，這就比較特別一點了，<br/>																			    <br/>　奶油香較濃些，玉米粒放得挺多的，整體味道來說尚可。<br/><br/><br/><br/><br/><br/><br/><br/>　(二樓用餐區寬敞而明亮)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/>　(香辣墨西哥飯)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/><br/>　(玉米濃湯)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(裡頭的玉米粒份量很多)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/></span><br/><br/><br/><br/><br/><br/><br/>　<br/>　會特地來到美墨最主要的目地當然就是炸雞了，<br/>																			    <br/>　這裡炸雞的炸衣偏厚，有點類似像餅乾的口感，<br/>																			    <br/>　不過卻完全沒有過油的問題。<br/>																			    <br/>　裡面的肉質滑嫩且充滿著肉汁，好吃。<br/>																			    <br/>　馬上啃完三大塊，回味無窮。<br/>																			    <br/>　雞柳與魚排也比現得可圈可點，<br/>																			    <br/>　足見美默德州炸雞在油炸上的技術相當有一套。<br/><br/><br/><br/><br/><br/><br/>　(三塊炸雞)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/>　<br/>　(雞柳與魚排)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/>　(炸衣裡的雞肉看起來多麼地鮮嫩多汁呀)<br/><br/><br/></span><span></span><br/></span><br/><br/><br/><br/><br/><br/><br/>
