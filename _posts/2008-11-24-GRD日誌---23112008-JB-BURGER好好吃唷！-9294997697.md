<br/>
<br/>
<br/>
    (JB的雙層牛肉漢堡好好吃呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  前幾天與爸媽初次在太卷共進午餐時，<br/>
  <br/>
  意外發現偶有耳聞的JB BURGER就在旁邊，<br/>
<br/>
　那時候就心想，當找一天來拜訪拜訪，<br/>
<br/>
　只是沒料到居然就在兩天後．<br/>
<br/>
<br/>
<br/>
　禮拜日中午空檔，騎單車練練體力，<br/>
<br/>
　騎到大安區時，心想JB BURGER就在附近，<br/>
<br/>
　便騎過去，這一次可不比前兩天的太卷，<br/>
<br/>
　一下子就找到啦．<br/>
<br/>
※地址：台北市仁愛路四段345巷15弄2號<br/>
  電話：02-27753027<br/>
<br/>
<br/>
  店內空間雖小，卻沒有擁擠的感覺，<br/>
<br/>
　點了雙層牛肉漢堡，單點120，<br/>
<br/>
　套餐則185，附飲料無限供應，薯條或是薯塊或是沙拉．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (JB BURGER的套餐飲料是無限自取的唷^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
 (無聊時用GRD2自拍一張！)<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
　JB BURGER的漢堡肉煎得相當嫩，<br/>
<br/>
　而最讓我開心的莫過於漢堡裡所夾的生菜相當豐富，<br/>
<br/>
　與充滿肉汁的漢堡一口咬下，相當清爽，一點也不膩．<br/>
<br/>
　好吃的東西有時候是很難去分析它為什麼好吃的，<br/>
<br/>
　而JB BURGER給我的感覺較類似加州燒烤堡，<br/>
<br/>
　簡單的說，就是好吃而已．　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(JB BURGER的雙層牛肉漢堡真的很好吃！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
(裡頭放的生菜很豐富，使得充滿肉汁的漢堡顯得清爽)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　薯塊部分表現得中規中矩，<br/>
<br/>
　不過那附上的沾醬我很喜歡，<br/>
<br/>
　兩者配合在一起，<br/>
<br/>
　會令人不自覺地一口接一口地無法停止呢！<br/>
<br/>
　<br/>
　<br/>
<br/>
<br/>
　　(JB BURGER的薯塊)<br/>
