<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
      (饌王牛肉麵)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　個人相當喜歡吃牛肉麵，理想中的牛肉麵肉質要煮得透與細，<br/>
<br/>
　湯頭要熬煮夠久，紅燒要夠味，麵條則Ｑ中帶軟，<br/>
　<br/>
　可惜的是吃了那麼多所謂的名店，至今仍未找到令我完全滿意的！<br/>
<br/>
<br/>
<br/>
　這一次吃的是饌王．<br/>
<br/>
　記得上一次台北牛肉麵節時，就曾吃過一次這一家號稱是人氣牛肉麵店，<br/>
<br/>
　印象中價格偏貴，卻也沒有不好吃，不過味道幾乎也快忘了，<br/>
<br/>
　禮拜六教課便趁午休時間抽空過去．<br/>
<br/>
<br/>
<br/>
　<br/>
　店家位置就在忠孝SOGO對面的二樓，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(店內裝潢一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　上了樓看了看菜單，裡頭有番茄，紅燒，麻辣等等口味的牛肉麵可供顧客選擇，<br/>
<br/>
　而個人則挑了他們的招牌牛肉麵"饌王牛肉麵"，<br/>
<br/>
　使用四十八小時至七十二小時熬成的甜美高湯裡有牛筋與腱心肉，<br/>
<br/>
　而另外還有一小份滷蹄花，這樣的組合要價280，<br/>
<br/>
　個人覺得滿貴的．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(菜單)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不一會兒服務生端了上來，拍完照馬上嚐了熱呼呼的湯頭，<br/>
<br/>
　真的只能以甜美來形容，那甜卻不是從調味料上來的那種甜，<br/>
<br/>
　而是從長時間細心慢火熬煮肉骨出來的，魯味不重，本身偏清淡，<br/>
<br/>
　肉質煮得恰恰好，配上ＱＱ的蹄花，<br/>
<br/>
　加上甜美湯頭可以無限次加湯，吃飽後感覺還滿舒服的，我喜歡！<br/>
<br/>
<br/>
<br/>
   (午後靠著窗吃牛肉麵別有一番風味)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　<br/>
   (招牌的饌王牛肉麵裡有牛筋與腱心肉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(魯蹄花)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(牛肉麵附的調味碟)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不過小碗就要價280的饌王對於一般人來說還真是偏貴，<br/>
<br/>
　但是湯頭真的值得一嚐，下一次若有機會再來光臨，<br/>
<br/>
　我看我還是點小碗的番茄牛肉麵就好，<br/>
<br/>
　一碗150，而且湯頭也可以無限次加呢．<br/>
　<br/>
