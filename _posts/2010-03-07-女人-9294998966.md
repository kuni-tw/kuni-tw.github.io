　不應該單獨只用女人為標題才對，<br/>
<br/>
　因為男女皆同，只是要寫的只跟女生有關而已。<br/>
<br/>
<br/>
　不管年紀多大，你永遠控制或是改變對方不了什麼，<br/>
<br/>
　對方也是一個值得你尊重的生命個體，就如同你需要被尊重是一樣的道理。<br/>
<br/>
　只可惜大多數人都不懂拿捏，總是寬以律己嚴以待人。<br/>
<br/>
<br/>
　很多人因過度自信而不願被改變這無可厚非，<br/>
<br/>
　但何為過度自信？這其實並不重要。<br/>
<br/>
　重要的是，你得學習如何尊重他人，<br/>
<br/>
<br/>
　因為<br/>
<br/>　"只有真正的適度尊重對方，你才能真正得到他的心。"<br/>
<br/>
　"你絕對無法靠改變對方變成自己所想要的樣子，而得到對方的。"<br/>
