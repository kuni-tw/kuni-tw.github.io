--------------------------------------------------<br/>
   Danny &amp; Company<br/>
   地址：台北市四維路52巷33號<br/>
   電話：(02)2705-9911<br/>
--------------------------------------------------<br/>
    <br/>
    <br/>
    <br/>
    <br/>
  台灣牛排界有個神人，網路上號稱牛排教主，<br/>
    <br/>
　他就是鄧有癸先生。<br/>
    <br/>
　之前他待過的幾家店，每一家因此都紅了起來，如此可見一般。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    <br/>
  去年則受維多麗亞酒店邀約，擔任168 Steak House的顧問，<br/>
    <br/>
　再度成功地將其打造成為牛排界的新巨星，至今仍一位難求。<br/>
    <br/>
    <br/>
    <br/>
　一個多月前，Danny &amp; Company開幕了，<br/>
    <br/>
　Danny是鄧先生的英文名字，他召集這些年來一直跟在身旁的老班底們，<br/>
    <br/>
　與一些金主們合股，嘗試重新打造新一家牛排界的傳奇。<br/>
    <br/>
　與168定位有點區隔的Danny &amp; Company，<br/>
    <br/>
  前者較專注於牛排，而後者則嘗試除牛排外再融入其他的元素與風格。<br/>
<br/>
    <br/>
<br/>
<br/>
<br/>
<br/>
　小弟前晚臨時想起隔日中午有空檔，心血來潮打了通電話過去，<br/>
    <br/>
　接電話的那一端不知道是誰，雖已客滿，<br/>
    <br/>
　但仍舊異常地客氣，告知中午每一桌只接一組客人，<br/>
    <br/>
　因為他們不會去預設客人幾點離開，<br/>
    <br/>
　不過也委婉地請小弟留下電話，以便上午他們確認名單若有空位時，<br/>
    <br/>
　可以電話通知。<br/>
    <br/>
　如此有禮貌又貼心的口氣馬上帶給小弟相當不錯的印象，<br/>
    <br/>
　隔日上午十一點半運氣相當好，接到電話，<br/>
    <br/>
　得知有空位，馬上與友人約好，於十二點四十五分時抵達現場。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　餐廳內光線偏暗，看著右面酒櫃上整排的紅酒，<br/>
    <br/>
　一瞥眼間不禁嚇了一跳，第一個映入眼簾的，<br/>
    <br/>
　就是在紅酒拍賣場上的膜拜紅酒貴客之一：screaming Eagle，<br/>
    <br/>
　其他的包括法國的波爾多五大酒莊，甚至連DRC都有。<br/>
    <br/>
　隨口問一下價位後，更是吃了一驚，<br/>
    <br/>
　好比小弟曾買過Napa Valley的Duckhorn，<br/>
    <br/>
　曾在酒商買過幾次，一瓶Cabernet Sauvignon約三千多，<br/>
    <br/>
　也曾買一瓶送給爸媽過，挑嘴的他們當時相當喜歡呢。<br/>
　<br/>
　但在Danny這裡點Merlot款，竟然不到三千，<br/>
    <br/>
　真是令人驚異的價格，光這一點就讓小弟有近期內擇期再訪的盤算嚕。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　Danny &amp; Company中午的商業午餐價格較晚上實惠許多，<br/>
    <br/>
　小弟點的是香煎鴨肝鵪鶉，前菜選築地直送鮮干貝，<br/>
    <br/>
　前湯選了鮮拌香草沙拉，另外附飲料，小弟選的是冰摩卡，<br/>
    <br/>
　甜點則選舒芙蕾佐紅酒凍，整套餐要價NT.1500<br/>
<br/>
<br/>
<br/>
  (中午套餐菜單)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  沒多久後，服務生先來詢問我們要喝有氣泡的礦泉水還是沒有的，<br/>
    <br/>
　小弟當然是選擇有氣泡的啦。<br/>
    <br/>
　突然想起以前在歐洲時，從來就沒喝過無氣泡的礦泉水，<br/>
　<br/>
　而氣泡式礦泉水通常會依氣泡程度分三種等級，<br/>
    <br/>
　以德文來說就是mild, medium und prickelnd。<br/>
    <br/>
　雖然喝氣泡式的已成習慣，有時候也很愛喝蘇打水，<br/>
    <br/>
　不過小時候還是不免好奇地問過長輩，為什麼不喝沒氣泡的，<br/>
    <br/>
　他們說，因為有氣泡的喝起來較不易口渴。<br/>
<br/>
<br/>
<br/>
<br/>
　　(好美的氣泡呀)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721552.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
<br/>
    <br/>
　Danny &amp; Company裡的礦泉水則來自義大利，<br/>
    <br/>
　服務生沒事就會來斟滿水杯，這一點還滿貼心的。<br/>
    <br/>
　<br/>
    <br/>
　然後端上搭配松露奶油的麵包，麵包烤得金黃，微溫，<br/>
    <br/>
　質地還滿細的，奶油裡摻的是黑松露，<br/>
    <br/>
　香味不甚明顯。<br/>
    <br/>
　將麵包先撕開，奶油塗抹上後，<br/>
    <br/>
　藉著麵包的微溫，奶油逐漸融化，<br/>
    <br/>
　此時松露的香氣漸漸散了出來。       <br/>
<br/>
<br/>
<br/>
　(附上的小麵包)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721553.jpg" target="_blank"></a><br/>
<br/>
<br/>
　<br/>
　(松露奶油)<br/>
</span><br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721554.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
　這裡使用的刀叉都是日本製的18-8不鏽鋼款，<br/>
    <br/>
　數字部分是指含鉻18%與含鎳8%的合金，<br/>
    <br/>
　外表美觀，材質也較穩定。<br/>
<br/>
 <br/>
<br/>
　(18-8不鏽鋼材質刀叉)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721555.jpg" target="_blank"></a></span>    <br/>
<br/>
<br/>
    <br/>
　<br/>
　沒多久後小弟的築地直送鮮干貝上桌，<br/>
    <br/>
　外表微炙過的生干貝，火侯很厲害，<br/>
    <br/>
　散著點焦香，口感不錯。<br/>
<br/>
<br/>
<br/>
　(築地直送鮮干貝)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721556.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
　鮮拌香草沙拉乍看之下不是很突出，<br/>
    <br/>
　上頭淋著以Basamico調製而成的沾醬，<br/>
    <br/>
　細膩的微酸，甜與香氣，然後是軟嫩異常的無花果切片，<br/>
    <br/>
　完整性相當高的一道前菜。<br/>
<br/>
<br/>
<br/>
  (鮮拌香草沙拉)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721557.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
　友點的主菜是燒烤蒜香牛排，上桌前服務生擺好牛排刀，<br/>
    <br/>
　當然，不用想太多，一定是頂級的Laguiole牛排刀，<br/>
    <br/>
　每一把都是經由專業的老師傅手工打造，細緻度驚人。<br/>
    <br/>
　牛排上桌後，友人與我不禁輕嘆一口氣，<br/>
    <br/>
　三分熟的斷面呈美麗又粉嫩的粉紅色，<br/>
    <br/>
　馬上用鋒利的Laguiole橫劃一刀，俐落地切下一小塊，<br/>
    <br/>
　難以形容的口感，散著純粹的微微肉香，<br/>
    <br/>
　不過似乎沒有預期來得如此美妙。<br/>
<br/>
<br/>
  (頂級的Laguiole牛排刀)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721559.jpg" target="_blank"></a></span><br/>

<br/>
<br/>
  (牛排的調味料，中間是頂級的鹽之花)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721558.jpg" target="_blank"></a><br/>
<br/>

<br/>
　(燒烤蒜香牛排)<br/>
</span><span><br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721560.jpg" target="_blank"></a><br/>

　(看這驚人的斷面秀)<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721561.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
</span><br/>
　接下來是小弟的香煎鴨肝鵪鶉，<br/>
    <br/>
  擺得既精緻且誘人，上頭一塊煎得酥香的鴨肝，   <br/>
    <br/>
　壓在鵪鶉肉與Risotto上，看起來如此地令人垂涎欲滴。<br/>
    <br/>
　先切了一小塊鴨肝在嘴裡，然後我差點尖叫起來，<br/>
    <br/>
　怎麼可以如此的軟嫩又入口即化？<br/>
    <br/>
　這根本是無比的罪惡深淵呀。<br/>
    <br/>
　Risotto部分也令人激賞，使用的是正統義大利Risotto的米，<br/>
    <br/>
　熟度剛好，微微米心的口感，<br/>
    <br/>
　可惜少了點以前在歐洲頂級餐廳吃過的Ｑ度與彈牙感，<br/>
    <br/>
　不過在台灣的牛排店裡，竟然可以吃到這種水準的Risotto，<br/>
    <br/>
　已經夠讓小弟懾服啦。<br/>
    <br/>
　可惜的是鵪鶉肉部分，好吃歸好吃，<br/>
<br/>
　不過口感似乎跟煎得較好的雞腿排沒什麼兩樣，<br/>
    <br/>
  這點令小弟深感困惑。<br/>
    <br/>
　友曾吃過美味許多的鵪鶉肉，便切一塊給他嚐一口，<br/>
    <br/>
　他也不是很喜歡，他跟我說，<br/>
    <br/>
　好吃的鵪鶉肉野味較重些，不過這部分小弟就不清楚啦。<br/>
<br/>

　後來服務生端上兩款甜點還有一小盤茶點，<br/>
    <br/>
　友點的是巧克力三部曲，我則是舒芙蕾佐紅酒凍，<br/>
    <br/>
  與前面的主餐相較之下，甜點部分實在是呈現極大的反差，<br/>
    <br/>
  只有巧克力部分表現稍好些，不過仍與前面主餐似乎不是很合，<br/>
    <br/>
　每一款只嚐了一兩口便作罷，<br/>
    <br/>
　而冰摩卡的部分則差強人意，咖啡味夠，不過有點過苦。<br/>
