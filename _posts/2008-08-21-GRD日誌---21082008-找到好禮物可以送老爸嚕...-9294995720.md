  因為老爸屬兔，這一次看到這幾款星巴克當天到貨的中秋節限量款馬克杯，<br/>
<br/>
　突然有股衝動想買下來送老爸，便請某熟識的夥伴朋友買下來，畢竟可以打折，<br/>
<br/>
　突然這麼多兔子聚在一塊，相信平常眼光還滿高的老爸應該也是會愣住的吧，<br/>
<br/>
　這樣我就很開心了．<br/>
<br/>
<br/>
　這五款分別有：<br/>
　<br/>
　1.黃色秋意馬克杯(突然想到某政治人物XD)<br/>
<br/>
  2.紅色秋意馬克杯<br/>
<br/>
　3.慶中秋馬克杯<br/>
<br/>
　4.浪漫中秋馬克杯<br/>
<br/>
  5.月光玉兔馬克杯<br/>
<br/>
　當中，黃色款秋意馬克杯大概是我最喜歡的吧，<br/>
<br/>
　顏色晶瑩剔透，加一隻紅澄澄的小兔兔在握把上頭，<br/>
<br/>
　真是美極了呢^^　<br/>
