<br/>
<br/>
　顛沛流離個四十八小時後,喪鐘敲起,<br/>
<br/>
  它一聲一聲的敲,我一眨一眨的閤.<br/>
<br/>
　手裡托著冒著泡的攝氏六七度的冰飲,搖晃著端詳一番,<br/>
<br/>
  捨不得將之抽離.<br/>
<br/>
  若將之比喻為浮生若夢,那現在將是回歸現實的一刻.<br/>
<br/>
  猶豫了半餉考慮了一番,最後選擇將夢作得圓滿.<br/>
<br/>
<br/>
<br/>
  我左手搭在臉龐,對望著十九吋的小框框,<br/>
<br/>
  將夢境轉化為真實並標上了句點.<br/>
<br/>
