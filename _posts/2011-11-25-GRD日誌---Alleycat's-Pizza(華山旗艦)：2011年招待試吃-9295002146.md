<p><span>(十二吋馬亞隆那披薩)<br/> <br/><br/> <br/> -------------------------------------------------------------<br/>Allycat's Pizza(華山旗艦店)<br/>電話：(02)2395-6006<br/>地址：台北市八德路一段一號<br/>時間：Mon~Thu 11:00~24:00<br/> 　　　Fri~Sat 11:00~02:00 <br/> 　　　Sun 11:00~24:00<br/> -------------------------------------------------------------</span></p>
<p><span>(華山園區的餐廳門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219568.jpg" target="_blank"></a></span><br/> <br/> <br/><span>很久沒有接受餐廳試吃的邀請活動了，</span></p>
<p><span>這次由Taipei Walker主辦給人感覺比較安心所以便欣然同意。</span></p>
<p><span>目的地是華山市場附近的Alley Cat's Pizza，</span></p>
<p><span>找了一天中午與店家約好便著輕裝前去。</span><br/> <br/><span>(Alleycat's精釀啤酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219569.jpg" target="_blank"></a><br/> <br/>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219570.jpg" target="_blank"></a><br/> <br/></span><span>華山園區有點類似信義區的四四南村，但少了些眷村多了點藝術味，</span></p>
<p><span>矮房水泥建築外觀的Alleycat's華山旗艦店內走的是歐式酒館風，</span></p>
<p><span>就座後馬上眼尖發現前桌常登媒體版面的幾位明星正在享用午餐。</span></p>
<p><span>依店家推薦的點餐內容包括Alleycat's精釀啤酒(NT.220)</span></p>
<p><span>瑞可達起司和鯷魚香蒜橄欖油烤麵包(NT.150)與十二吋馬亞隆那披薩(NT.480)</span></p>
<p><span>(瑞可達起司和鯷魚香蒜橄欖油烤麵包)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219571.jpg" target="_blank"></a><br/> <br/>(微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219572.jpg" target="_blank"></a><br/> </span></p>
<p><span><span>沒多久服務生先端上啤酒，由知名英國酒廠Amber Ale特別為Alleycat's所釀製，</span></span></p>
<p><span>配方來自於2007年威爾斯的啤酒冠軍Golden Valley Ale(2005年曾進英國決賽)</span></p>
<p><span>口感清爽苦味中帶有深度並充滿令人感到愉悅的濃郁花果芬芳，</span></p>
<p><span>最特別的是隔了一段時間後升至室溫味道依舊很棒，可見其耐喝程度。</span><br/> <br/><span>(餐廳內深處的樓中樓與吧台區)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219573.jpg" target="_blank"></a><br/> <br/></span><span>(有著專屬酒標的Alleycat's精釀啤酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219574.jpg" target="_blank"></a><br/> <br/></span><span>接下來是香蒜橄欖油烤麵包與馬亞隆那披薩，前者起司醬塗得很厚，</span></p>
<p><span>配上鯷魚與乾酪條是一道完成度很高的美味。</span></p>
<p><span>後者就叫人驚訝了，薄脆的餅皮中散出難得的窯爐炭烤焦香，</span></p>
<p><span>上頭擺滿義式香腸，臘肉腸，辛香肉腸及Salami，全然地義式重口味風格，</span></p>
<p><span>深淺交錯的橘紅顏色非常討喜象徵著南義大利的熱情陽光。</span></p>
<p><span>馬上跑至窯爐一旁觀察到底是什麼樣的爐子才烤得出這樣的風味，</span></p>
<p><span>回應我的是一位非常資深的師傅，據說分店的師傅們都是由他訓練出來的，</span></p>
<p><span>石磚打造而成的烤爐上共有八處不同溫度的窯口，</span></p>
<p><span>披薩們像似洗三溫暖般由師傅從最高溫依經驗一層一層慢慢移到低溫烘烤室，</span></p>
<p><span>這樣使餅皮變得更脆也成功避免了過焦。</span><br/> <br/><span>(大師傅詳盡地解說烘烤披薩的過程)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219575.jpg" target="_blank"></a><br/> <br/>(十二吋馬亞隆那披薩)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219576.jpg" target="_blank"></a><br/><br/>(深淺交錯的橘紅顏色象徵著南義大利的熱情陽光)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219577.jpg" target="_blank"></a><br/> <br/>(窗外)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219578.jpg" target="_blank"></a><br/> <br/>(one of the real Alleycats)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363219579.jpg" target="_blank"></a><br/> <br/></span><span>最後的義式冰淇淋Gelato是店經理額外招待的，我選了抹茶，巧克力與草莓森林，</span></p>
<p><span>令人懷念已久的歐式冰淇淋如今竟在披薩店裡重現，</span></p>
<p><span>清爽簡單的風味與高罪惡感的美式風格形成明顯的對比。</span><br/> <br/><span>(門外的Gelato義式冰淇淋餐櫃)</span></p>
