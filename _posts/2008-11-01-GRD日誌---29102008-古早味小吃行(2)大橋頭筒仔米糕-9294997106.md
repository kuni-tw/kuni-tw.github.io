  (大橋頭筒仔米糕蒸得恰到好處，Ｑ中散出油香)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  十月二十九號古早味美食行第一站阿華鯊魚煙剛結束，  <br/>
　(詳情請見：<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11473764" target="_blank">台灣古早味小吃行(一)　阿華鯊魚煙</a>)<br/>
<br/>
　馬上又衝到第二站：鄰近的大橋頭筒仔米糕，<br/>
<br/>
　位在北市延平北路三段41號，近台北橋，<br/>
<br/>
　抵達之時，大概因已過了中午人潮時段，<br/>
<br/>
　店內冷清．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(大橋頭筒仔米糕店門口)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　就座了後，因為事先已探聽過了，<br/>
<br/>
　點了豬肝湯與筒仔米糕．<br/>
<br/>
　豬肝湯要價35，湯頭不是很清，有點濁，<br/>
<br/>
　不過切得薄薄的肝片鮮嫩可口，味道則尚可．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(豬肝湯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (豬肝片鮮嫩可口)<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　再來則是重頭戲筒仔米糕了，要價30，<br/>
<br/>
　似乎小貴了點，有分肥肉或瘦肉兩種，<br/>
<br/>
　我選擇前者，以五花肉包裹米糕．<br/>
<br/>
　做法與大多數店家相同，在筒模底下鋪上肉片，滷蛋與魷魚，<br/>
<br/>
　再放置香菇，肉絲與蝦米拌炒過的糯米，然後放進蒸籠炊，<br/>
<br/>
　而用料與炊蒸的技術才是筒仔米糕好吃的關鍵．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(重頭戲筒仔米糕出場嚕！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　大橋頭的米糕蒸得恰到好處，Ｑ中散出油香，<br/>
<br/>
　果然好滋味呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(大橋頭的筒仔米糕，果然好滋味呀！)                                                  <br/>
