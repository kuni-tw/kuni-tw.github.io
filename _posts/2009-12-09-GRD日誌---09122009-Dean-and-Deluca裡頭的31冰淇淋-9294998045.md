<br/>
　下午從南港返回市中心時，路過微風，沒事就會在微風底下挑酒的我，<br/>
<br/>
　恰巧有點閒，停好了車便一路沿著電扶梯逛下去。<br/>
<br/>
　走到許久沒去的Dean &amp; Deluca，<br/>
<br/>
　才驚訝的發現不知什麼時候，知名的31冰淇淋專賣櫃已悄悄地進駐D&amp;D了，<br/>
<br/>
　沒吃過31冰淇淋的我馬上找了明亮的位置坐下。<br/>
<br/>
<br/>
<br/>
<br/>
　　(Dean &amp; Deluca裡明亮的空間)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Dean &amp; Deluca裡的31冰淇淋專櫃)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　點了香蕉巧克力布朗尼可麗餅，要價120，<br/>
<br/>
　與紙杯裝的大球假日禮讚，要價105。<br/>
<br/>
<br/>
　先說說感想吧，冰淇淋配上可麗餅倒是第一次嘗試，<br/>
<br/>
　視覺上看起來貌不驚人，咬了一口也頗為失望，<br/>
<br/>
　法式可麗餅的綿軟口感配上滑嫩的冰淇淋，<br/>
<br/>
　感覺上不太搭。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(個人不大喜歡冰淇淋配上可麗餅的口感)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　假日禮讚冰淇淋裡混著酸甜的覆盆子，開心果還有棉花糖粒，<br/>
　<br/>
　味道新奇，配上棉花糖的柔軟口感，<br/>
<br/>
　算得上是相當創新的口味。<br/>
<br/>
<br/>
<br/>
<br/>
　　(玻璃櫃裡一堆沒吃過的新口味)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(假日禮讚冰淇淋特寫)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不過，可能心臟不夠強吧，小弟我還是偏好較為傳統一點的冰淇淋路線，<br/>
<br/>
　或是在歐洲常吃的義式冰淇淋，<br/>
<br/>
　太過新奇的口味大概會讓我心臟麻痺吧。(笑)<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(走出微風馬上能感受到耶誕節的氣氛正悄悄到來)<br/>
