<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
  從得知伯朗有設咖啡館之後(應該是1998開始吧)<br/>
<br/>
　一直沒機會進去品嘗過他們的咖啡．<br/>
<br/>
  今天陰錯陽差經過，加上最近想多多嘗試不同的咖啡館，<br/>
<br/>
　就想機不可失便走進去找了位置坐下．<br/>
<br/>
　看了價目表之後挑了個最感興趣的楓糖咖啡，<br/>
<br/>
　點了大杯SIZE(價位NT130)，拿到一張令人詫異的長方電子盒便找位置就座．<br/>
<br/>
　經服務人員解說方得知那電子盒會在飲料做好之後震動(與手機震動雷同)　<br/>
<br/>
　客人便可親自去櫃檯領取，真是滿有趣的服務項目呢！<br/>
<br/>
　<br/>
　拿到我的楓糖咖啡時不禁愣了一下，從外觀來說相當討喜，<br/>
<br/>
　幾片大小片核桃片穿梭在綿密的奶泡與楓糖漿上，香味四溢，<br/>
<br/>
　唔，入口偏甜，可惡，早知道就不要求多壓幾下楓糖漿了，真是自作孽呀！<br/>
<br/>
　不過楓糖漿那迷人的獨特風味，配合著咖啡些許的微苦與奶香，<br/>
<br/>
　深深吸引著我這一隻螞蟻呀！<br/>
<br/>
<br/>
  雖然還沒試過其他款咖啡，大概等以後吧，<br/>
<br/>
　但是以連鎖咖啡來說，伯朗這一款楓糖咖啡水準做的真不錯！<br/>
<br/>
