<br/>
<br/>
<br/>
<br/>
<br/>
  (下午不爭氣的又跑去咖啡意識報到所攝)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  剛看完慈禧,心裡紛亂了好一下子.<br/>
<br/>
  我心想,是怎麼了?<br/>
<br/>
　這現實到殘忍般地步的劇情裡刻劃出來的難道不是實情嗎,<br/>
<br/>
  為何我卻有種看不下去的感受?<br/>
<br/>
<br/>
<br/>
　其實從小被灌以世俗觀念的我們,不常常也有類似的體驗?<br/>
<br/>
  尤其在看到人性最現實的一面時,亦或說是骯髒且邪惡的,<br/>
<br/>
  或敗壞風俗的,常常有種不忍心甚至說是罪惡感或厭惡感也不為過.<br/>
<br/>
<br/>
<br/>
  道德是否為必須的?<br/>
<br/>
<br/>
  人類沒有道德是否就與禽獸無異呢?<br/>
<br/>
<br/>
  這些話是我們被灌輸的還是我們自己想出來的?<br/>
<br/>
<br/>
<br/>
  想了想,尤其第三個問題給了我不少震撼性的啟發.<br/>
<br/>
　假如第三題答案為前者,那這應該是人類史上最大的謊言之一了吧.<br/>
<br/>
  試問,到底是誰,在最當初為了個人私慾而嘗試操弄人性,<br/>
<br/>
  這幾千年以來傳到其他既得利益者之後繼續無情的使用並廣為流傳至今天?<br/>
<br/>
<br/>
<br/>
<br/>
  男女之間那純純的情愛,就在這劇情裡的衝擊之下被打的稀巴爛,<br/>
<br/>
  過了不久入口成了我今晚可口無比的夜宵.<br/>
<br/>
<br/>
