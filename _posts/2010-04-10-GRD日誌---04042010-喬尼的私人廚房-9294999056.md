<br/>
  這一天特地與友人選到遠在新店的"喬尼私人廚房"談事情，<br/>
<br/>
  據說這家的主廚乃老闆遠從義大利網羅過來的，<br/>
<br/>
　可惜抵達餐廳時，才得知那位義大利主廚已離開另謀高就了。<br/>

<br/>
  點了一道前菜"義大利卡帕契歐生牛肉"，要價NT.300，<br/>
<br/>
  青醬雞肉蝴蝶麵，要價NT.300，<br/>
<br/>
  加NT.150成套餐，附湯，飲料與甜點，<br/>
<br/>
　聽說之前是120，看來是漲價了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(菜單)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先來的是我選的是主廚特製湯，<br/>
<br/>
　友選的是義大利牛肉湯。<br/>
<br/>
　牛肉湯湯頭濃郁，肉塊質地細嫩，<br/>
<br/>
　主廚特製湯是磨菇濃湯，倒是喝不大出來有什麼特別的，<br/>
<br/>
　不過兩者皆非常可口。<br/>
<br/>
<br/>
<br/>
　麵包則較令小弟狐疑了，外觀相當普通的兩小片麵包，<br/>
<br/>
　信手從籃子裡拿來咬了一口，經細細咀嚼後，<br/>
<br/>
　口感就如同一般的麵包，沒什麼特別之處。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(湯與麵包)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  義大利卡帕契歐生牛肉，相當漂亮的鮮紅色薄切生牛肉片，<br/>
<br/>
　上面鋪著Parmesan片，與淋著店家特製的沾醬，<br/>
<br/>
　口感極為軟嫩，配上濃郁風味的起司片與美味的沾醬，<br/>
<br/>
　真是令人心曠神怡。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(義大利卡帕契歐生牛肉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後，重頭戲青醬雞肉蝴蝶麵終於登場了，<br/>
<br/>
　染上青醬的蝴蝶麵，顏色有如碧玉一般，晶瑩剔透，<br/>
<br/>
  真是美極了。<br/>
　<br/>
　上桌時小弟不禁愣了一下，份量真的是不多，<br/>
<br/>
　尤其是前陣子才吃剛過永和知名又俗夠大碗的叉子義大利麵，<br/>
<br/>
　甚至與之前在台灣所吃過的任何一家義大利餐廳相比，<br/>
<br/>
　喬尼的份量真的不算多。<br/>
<br/>
　舀了一口麵，入口後，細心咀嚼一番，<br/>
<br/>
　非常濃郁的青醬香氣直撲鼻而來，<br/>
<br/>
　麵本身彈牙且Ｑ，好吃得不得了，<br/>
　　<br/>
　唯一美中不足的是，以我的食量來說，<br/>
<br/>
　不餓的狀態下大概連三盤都吃不夠吧。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
     (青醬雞肉蝴蝶麵)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
　友點的是義大利馬鈴薯麵疙瘩，Gnocchi，<br/>
<br/>
  這曾是我在歐洲時最愛的一道料理之一了，<br/>
<br/>
　以前自己沒事會去超市買回來自己下鍋來享受，<br/>
<br/>
　沒想到隔了好幾年今天在台灣竟然又吃到了，<br/>
<br/>
　原來Gnocchi的中文是馬鈴薯麵疙瘩呀。<br/>
<br/>
　上面灑著松子的Gnocchi，煮得很嫩，<br/>
<br/>
　濃稠至極的醬汁佈滿整盤的麵疙瘩，　<br/>
<br/>
　每一口都是人間美味。<br/>
<br/>
　或許前面的濃湯與麵包令人相當失望，<br/>
<br/>
　但這兩道主餐完完全全地擄獲我的胃，扳回一城。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (義大利馬鈴薯麵疙瘩)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
　與老闆聊天之際，老闆順便請了我一杯茴香cocktail，<br/>
<br/>
  聊到我最愛吃的Risotto時(義式燉飯)，<br/>
<br/>
　我不經意提到沒在台灣吃到使用正統Risotto米做的燉飯，<br/>
<br/>
　每一次都是失望的經驗，兩人也分享了不少對於這一道料理的心得。<br/>
<br/>
　很可惜這裡也沒有進這種米，老闆解釋說，因為成本太高，<br/>
<br/>
　所以使用的是泰國香米(德文是Jasminreis)<br/>
<br/>
　但他仍希望招待我吃吃看，請了服務生端了一大盤出來，<br/>
<br/>
　是菜單上的義大利紅酒培根燉飯，使用番紅花提味，<br/>
<br/>
　混著紅酒的香氣濃郁至極，舀了一口滿滿的燉飯入口，<br/>
<br/>
　培根與牛肝菌口感令人心醉，米煮得較乾，口感也不錯，<br/>
<br/>
　好吃是好吃，只可惜個人還是偏好義大利的Risotto米呀，<br/>
<br/>
　飽滿渾圓又大顆，煮得好甚至會有彈牙的錯覺，堪稱人間美味。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(義大利紅酒培根燉飯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　最後老闆知道我喜歡咖啡，特地親自弄了一杯Espresso，<br/>
<br/>
  細心做出來的咖啡畢竟是不一樣，香氣宜人，<br/>
　<br/>
　微酸微苦微澀，配上兩道甜點：<br/>
<br/>
　裡頭摻放著自製手指餅乾的提拉米蘇，<br/>
<br/>
　以及老闆親自弄的焦糖蘋果塔，兩者味道皆不錯，<br/>
<br/>
　我好喜歡呀。<br/>
