N.Y. Bagels Cafe 京站店<br/>
<br/>
台北市承德路一段一號27櫃(Y3出口)<br/>
Tel: 2552-6880<br/>
營業時間<br/>
週日 ～ 週四 7:00am ～ 11:00pm<br/>

------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　二月三號中午與許久不見的友人相約在京站見面，<br/>
<br/>
　臨時決定在裡頭新開的的NY BAGELS用餐，<br/>
<br/>
　因客滿，便先登記，等了約三十分鐘後回到餐廳裡就座。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

    (N.Y. Bagels Cafe 京站店)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
　點了煙燻鮭魚片三明治，有九種不同口味的BAGEL可以選，要價169，<br/>
<br/>
  搭配B套餐附炸薯塊與任一飲料，加價120，<br/>
<br/>
  個人選擇了較搭配的全麥BAGEL，飲料則是熱榛果拿鐵。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　等餐時間仔細觀察了一下周遭環境，<br/>
<br/>
　與信義門市差不多質感的裝潢等級，燈光與氣氛皆不錯，<br/>
<br/>
　不過個人還是較喜歡常去的仁愛門市，那裡較有親切感。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

　沒多久後飲料先登場，還不錯的拉花，<br/>
<br/>
　不過口感可就不行了，NY BAGEL的熱咖啡類還是一樣，<br/>
<br/>
　唉，我一直都不是很喜歡。<br/>

　　(熱榛果拿鐵)<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
  終於，煙燻鮭魚片上桌，夾著鮭魚片與番茄的全麥bagel對切，<br/>
<br/>
　一旁還有相當大份的炸薯塊，十足的美式餐點。<br/>
<br/>
　NY BAGEL的炸薯塊向來為其特色之一，<br/>
<br/>
  不削皮又大塊的薯塊將之炸得金黃，外酥內軟，<br/>
<br/>
　不過這一次炸得似乎有點過久，真可惜呀。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　煙燻鮭魚BAGEL的部分，bagel口感緊實，<br/>
<br/>
　不過鮭魚似乎放得有點過少，番茄切得較厚，<br/>
<br/>
　幾乎蓋住了鮭魚的味道，希望只是一時的失誤呀，<br/>
<br/>
　以後不要再發生了，不然這價位還真是滿貴的耶。<br/>
