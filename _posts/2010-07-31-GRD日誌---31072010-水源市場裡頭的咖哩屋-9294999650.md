<br/><br/><br/>    (咖哩屋的雞塊咖哩飯)<br/><br/><br/><br/><br/><br/><br/>-----------------------------------------------------<br/>   咖哩屋<br/>   地址：臺大羅斯福路的斜對面，水源市場35號<br/>   周一店休<br/>-----------------------------------------------------<br/>																			    <br/>																			    <br/>																			    <br/>																			    <br/>　自從永和的逸香咖哩結束營業後，<br/>																			    <br/>　有好一陣子沒吃到過低於一百元以下，又不錯吃的咖哩飯了。<br/><br/><br/><br/>  (逸香的平價美味咖哩飯已成追憶...)<br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　個人對於咖哩的偏好源自於小時候在祖母家的回憶，<br/>																			    <br/>　每逢假日到祖母家洗洗溫泉，她老人家心血來潮時，<br/>																			    <br/>　總是會弄一大鍋的咖哩雞給我吃，<br/>																			    <br/>　不知道怎麼的，極怕吃辣的我，<br/>																			    <br/>　每一次都會吃得香汗淋漓，但總是欲罷不能。<br/><br/><br/><br/><br/><br/>　一直到國外開始自己下廚後，<br/>																			    <br/>　偶爾懷念起以前的味道時，就會買一些日本的咖哩塊回來煮，<br/>																			    <br/>　加一堆的雞腿肉，馬鈴薯與紅蘿蔔，配上大同電鍋與義大利米，<br/>																			    <br/>　照樣吃得盡興無比。<br/><br/><br/><br/><br/>　這次有幸聽到網友們不經意透露出在公館的水源市場裡，<br/>																			    <br/>　有家平價的咖哩店，下午教課空檔中二話不說又衝了去，<br/>																			    <br/>　這也是我第一次走進水源市場裡，<br/>																			    <br/>　擁擠的裡頭，有賣菜的也有不少小吃店，<br/>																			    <br/>　找了許久繞了幾圈終於在另一頭找著了咖哩屋。<br/><br/><br/><br/><br/>  (水源市場側門口)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/>　(咖哩屋)<br/><br/></span><span><br/><br/><br/><br/></span><br/><br/><br/>　一間非常小的店面，看起來相當平凡無奇，<br/>																			    <br/>　友點了一盤老闆娘推薦的雞塊咖哩飯，一份要價NT.70<br/>																			    <br/>　我則是一份豬排咖哩飯，也是要價NT.70<br/>																			    <br/>　從價位上來說相當平價。<br/><br/><br/><br/><br/>　(菜單)<br/><br/><span><br/><br/><br/><br/><br/></span><br/>　首先上桌的是供無限續的味增湯，裡頭有幾塊豆腐，<br/>																			    <br/>　味道還可以，調味適中。<br/><br/><br/><br/>　(無限續的味增湯)<br/><br/><span><br/><br/><br/></span><br/><br/><br/>　然後是友的雞塊咖哩飯，初上桌時不禁愣了一下，<br/>																			    <br/>　幾塊有點類似獅子頭的雞塊圍繞在份量不小的咖哩飯旁，<br/>																			    <br/>　上頭一片荷包蛋，醬汁中還有幾塊馬鈴薯與紅蘿蔔。<br/>																			    <br/>　接著我的豬排咖哩也上桌了，<br/>																			    <br/>　與前者不同的是，薄薄的炸豬排取代了雞塊。<br/>																			    <br/>　馬上先偷嚐了一下友的雞塊，口感比預期的還要嫩上許多，<br/>																			    <br/>　又切得如此厚實大塊，感覺C/P值不錯。<br/>																			    <br/>　然後試咬了一口自個兒點的炸豬排，可惜切得真是太薄了，<br/>																			    <br/>　口感沒有雞塊來得好，咬起來也較不過癮些。<br/><br/><br/><br/>　(雞塊咖哩飯)<br/><br/><span><br/><br/><br/><br/><br/>　(豬排咖哩飯)<br/><br/></span><span><br/><br/><br/><br/>　(豬排咖哩飯)<br/><br/></span><span><br/><br/><br/><br/><br/>　(有夠薄的豬排)<br/><br/></span><span><br/><br/><br/><br/><br/></span><br/><br/>　咖哩醬部分偏小弟愛好的日式，較甜些，<br/>																			    <br/>　推測是咖哩塊煮出來的，詢問了一下老闆娘也證實了我的想法，<br/>																			    <br/>　可惜辣度無法依個人喜好調整，<br/>																			    <br/>　不然我會希望再辣一點。<br/><br/><br/><br/><br/><br/>　最後單點一份炸蝦，一份要價NT.40<br/>																			    <br/>　炸得金黃色的炸蝦，一份共有兩隻，<br/>																			    <br/>　看起來頗為可口，可惜口感不足，<br/>
