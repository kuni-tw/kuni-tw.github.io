　(友剛入手的RV250噴射版)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　愛車小金(RV250化油版)入手也幾年了，<br/>
    <br/>
　當中因當兵等等，曾有段時間沒什麼在騎，<br/>
    <br/>
　不過至今破了五萬公里後，引擎仍舊保持在最佳狀態，<br/>
    <br/>
　這當然也是因為平常有注意定時保養以及更換消耗品的關係，<br/>
    <br/>
　另外個人騎乘時也不喜歡拉轉。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (小弟的愛車小金：RV250化油版)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721500.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
　身邊朋友似乎也受到影響，更換大車的愈來愈多，<br/>
    <br/>
　且清一色都是RV系列的。<br/>
    <br/>
　比如最近有位朋友想請我幫他找舊款的噴射RV250，<br/>
    <br/>
　當然一方面也因小弟較不推剛推出的EVO新款，<br/>
    <br/>
　另外順便看能不能談到好價錢，<br/>
    <br/>
　沒想到最後竟然能以"辦到好現金價十四萬五"拿到，<br/>
<br/>
　比我當初買化油版的還要便宜！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　新車到手後，基於信任，小弟暫時幫訓車與保管，<br/>
    <br/>
　為期一個多月的時間，順便測試與比較兩台車的差異。<br/>
    <br/>
　說實在的，化油版本小弟至少已騎過七八台，<br/>
    <br/>
　當中包括某大羊板板主的中度改裝RV250，甚至是大改過的，<br/>
    <br/>
  可惜就是沒騎過噴射版的，這一次有此難得機會，<br/>
    <br/>
　順便看看，當初沒多花台幣一萬購買噴射版的遺憾到底有多少。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先說說裝備上的差異吧，<br/>
    <br/>
　雖然是舊版的，但以前令人稱羨的魚眼遠燈配上近燈HID噴射版規格，<br/>
    <br/>
　如今卻與EVO新款一樣，礙於新法規的關係，也少了HID。<br/>
　<br/>
　近燈亮度雖然還滿夠的，但在半夜時卻完全無法HID相比。<br/>
    <br/>
　除此之外，也取消了後照鏡上的方向燈，<br/>
    <br/>
　另外遙控器也沒了，所以無法遙控開啟坐墊以及啟動警報與斷電裝置，<br/>
    <br/>
　猜測除了成本上之考量外，大概就是維修較方便吧。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(少了HID的噴射版車頭)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721496.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(後照鏡上也沒方向燈啦)<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721497.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
　但令人欣喜的是，儀錶板右側的小型置物盒與龍骨上的置物架，<br/>
    <br/>
　皆改善了開啟裝置，從原本的按壓後自動彈起，<br/>
    <br/>
　改成較不易失靈的手動開啟，個人超愛的，<br/>
    <br/>
　有機會將小金也來換換看，規格似乎相同。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　　(RV250的後避震與卡鉗)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721501.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
  左手把下方多了個自動開啟坐墊鈕，舊版RV250的遙控器雖然較炫，<br/>
    <br/>
　不過似乎滿耗電的，小弟至今也換了三四次電池，<br/>
    <br/>
　最後乾脆不換了，直接從行李箱旁插鑰匙開啟，<br/>
    <br/>
　現在有了這電鈕，問題應該也會少了許多吧。<br/>
<br/>
<br/>
<br/>
<br/>
　(手把)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721503.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　行李箱部分沒什麼太大變化，不過底層突出了一大塊，<br/>
    <br/>
　推測是置放噴射電腦的地方，所以實際置物空間變少了。<br/>
<br/>
<br/>
　接下來當然要說說實地的騎乘感受了。<br/>
    <br/>
　訓車到了一階段後，找了一天晚上將車子拉到了尾速，<br/>
    <br/>
　路雖不夠長，不過化油版在那裏還是可輕易達到表速140km/h以上，<br/>
    <br/>
　噴射版一樣也是140km/h以上，不過中尾段加速較緩慢，<br/>
    <br/>
　所以到達尾速的時間點較化油版晚許多。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(斗大的RV250 EFI字樣)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721498.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　噴射版起步拉轉很快，一下子衝到6000rpm，<br/>
    <br/>
  化油版拉轉較慢，同樣油門之下，足足少了約一千轉左右。<br/>
    <br/>
　另外以五千轉來作定速測試時，<br/>
    <br/>
　化油版表速是88km/h，噴射版本是76km/h。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(儀錶板)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721502.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
  噴射版的起步異常地輕盈，在市區裡走走停停騎起來非常地輕鬆，<br/>
    <br/>
　比化油版好騎太多了，小小油門就可以有動力隨傳隨到的感覺。<br/>
    <br/>
　不過在大灌油門之下，化油版拉轉雖然較慢，<br/>
    <br/>
　但是貼背感較明顯，加速也較快一點。<br/>
    <br/>
　過時速八十後，兩車差距更大。<br/>
    <br/>
　不過，化油版騎到8000轉的極限時，<br/>
    <br/>
　油路會不大順，而噴射版卻完全無此困擾，<br/>
    <br/>
　在尾速上似乎有更好的延伸性。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　簡單的說，發展噴射版早已是時勢所趨了，<br/>
    <br/>
　噴射版不但解決了化油版無法即時順應天候的問題，<br/>
    <br/>
　而且還非常省油。<br/>
    <br/>
　這一段時間裡幾次拉轉加上測尾速，照裡說應該要相當耗油才對，<br/>
    <br/>
　但整體的公升油耗還是比化油版的多了足足三公里多，<br/>
    <br/>
  實在是驚人的差距呢。<br/>
    <br/>
　但化油版本那手動的獨特操控感，<br/>
    <br/>
　對於小弟來說，仍然還是魅力無窮呀。(下台一鞠躬)<br/>
<br/>
<br/>
