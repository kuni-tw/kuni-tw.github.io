<br/>
<br/>
    (大廚漢堡的肉相當juicy)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
------------------------------------------------------<br/>
  大廚漢堡<br/>
  臺北市大安區仁愛路四段122巷50號<br/>
  (02)27555062<br/>
------------------------------------------------------<br/>
<br/>
<br/>
<br/>
  禮拜六中午教課空檔，信步走至附近的大廚漢堡，<br/>
<br/>
　這是一家在網路上已耳聞多時的漢堡店，<br/>
<br/>
　不過褒貶不一，但管別人怎麼講，親口嚐嚐才是正道呀。<br/>
<br/>
<br/>
　店家內裝潢非常簡陋，質感相當不好，<br/>
<br/>
　偏向一般早餐店的氣氛，<br/>
<br/>
　與以前曾吃過的所有美式餐廳或漢堡專賣店相較之下，<br/>
<br/>
　實在是相當不討喜。<br/>
<br/>
<br/>
　也罷，俗語說得好，既來之則安之，<br/>
<br/>
　便點了牛肉漢堡套餐，要價200，附薯條與飲料，<br/>
<br/>
　飲料是冷飲喝到飽或南瓜濃湯二選一。<br/>
<br/>
　老闆娘詢問牛肉部分七八分熟是否可以接受，<br/>
<br/>
　心下還滿高興的，沒想到在這裡還可以要求牛肉熟度呢。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (大廚漢堡的套餐選擇)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　沒多久主餐便登場，<br/>
<br/>
　切成兩半的主漢堡上插著一個大炸洋蔥圈與一塊雞塊，<br/>
<br/>
　盤子裡還有偏淺色調的薯條，<br/>
　<br/>
　薯條部分口感有點特別，個人也不知道該如何精確的形容，<br/>
<br/>
　與之前在其他店家所吃過的進口薯條似乎不大一樣，<br/>
<br/>
　推測也有可能是油炸方式不同，總之口感還不差。<br/>
<br/>
　南瓜濃湯味道香甜，以附餐來說，個人還滿喜歡的。<br/>
<br/>
　牛肉漢堡上插著那一塊雞塊，視覺上還滿奇異的，<br/>
<br/>
　一大口咬下，類似拖鞋麵包的外型，口感卻大不相同，<br/>
<br/>
　鬆軟的麵包挾著相當JUICY的漢堡肉，肉味十足，<br/>
<br/>
　不過總覺得怪怪的，就是說不大出來。<br/>
<br/>
　硬是要形容的話，應該說：<br/>
<br/>
　在大安區裡一家類似連鎖早餐店的裝潢裡，<br/>
<br/>
  咬著插著雞塊的拖鞋麵包牛肉漢堡，喝著濃郁的南瓜濃湯，<br/>
<br/>
  感覺真是有點奇特呀。(笑)<br/>
