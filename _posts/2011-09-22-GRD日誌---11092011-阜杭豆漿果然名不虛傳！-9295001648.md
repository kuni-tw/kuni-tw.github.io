<p><span>(阜杭的薄餅蛋夾油條)<br/><br/> -------------------------------------------------------<br/>阜杭豆漿<br/>電話：(02)2392-2175 (02)2392-4053<br/>地址：台北市忠孝東路一段108號(華山市場2樓)<br/>時間：05:30~12:30  每月第二跟第四個星期一公休<br/> -------------------------------------------------------</span></p>
<p><span>(從華山市場一樓的門外開始排起)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212492.jpg" target="_blank"></a></span></p>
<p> </p>
<p><span>位於華山市場二樓美食街的阜杭豆漿創立於1958年，名氣相當響亮早已耳聞許久，</span></p>
<p><span>可惜除非教課不然平常小弟是一隻天塌下來也不願早起的夜貓子，</span></p>
<p><span>這一天不知道為何特別早起，迷糊中猛然想起懸念已久的阜杭當下立刻啟程。</span><br/> <br/><span>(進入裡頭的樓梯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212493.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(漸漸移動至二樓)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212494.jpg" target="_blank"></a><br/> <br/>(終於快到櫃台了)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212495.jpg" target="_blank"></a></span><br/> </p>
<p><span>抵達華山市場時已過十一點沒想到一樓門外仍見大排長龍，</span></p>
<p><span>早有事先做心理建設的小弟當然乖乖排隊，</span></p>
<p><span>從一樓門外開始漸漸往門內移動然後順著樓梯擠上二樓，</span></p>
<p><span>最後沿著玄關終於進入內用大廳一時間只見人滿為患，</span></p>
<p><span>不過說實在的從開始排隊經點餐到取餐竟不到半小時，</span></p>
<p><span>阜杭內場人員的速度與用心實在令人佩服。</span><br/> <br/><span>(大廳裡人山人海)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212496.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>(透過明亮清晰的大玻璃可以看到廚房全部)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212497.jpg" target="_blank"></a><br/> <br/></span><span>(製作厚燒餅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212498.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>點了鹹豆漿(NT.25)與冰豆漿(NT.22)各一碗，蘿蔔絲餅(NT.20)，</span></p>
<p><span>之前早聽說阜杭厚燒餅的大名可惜個人向來不愛吃這種東西，</span></p>
<p><span>最後想了想點了薄餅蛋夾油條(NT.50)算是一種妥協吧。</span></p>
<p> </p>
<p><span>(有如速食餐廳的櫃檯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212499.jpg" target="_blank"></a></span></p>
<p> </p>
<p><span>有如麥當勞等速食餐廳一般，檯前點餐到了檯尾取餐與付錢實在相當新穎，</span></p>
<p><span>走到有如百貨地下美食街的大廳找了地方就坐後終於要開始嚐鮮了。</span></p>
<p><span>只能說阜杭果然名不虛傳：甜豆漿口感比一般早餐店喝到的還要濃郁，</span></p>
<p><span>偏甜但很好喝早知道就多點一碗。</span></p>
<p><span>鹹豆漿部分個人向來偏好永和世界豆漿大王的沒想到阜杭的也很有特色，</span></p>
<p><span>根本就是熱的鹹豆花，調味較清淡是少見得以"好吃"來形容的豆漿。</span><br/><br/><span>(很棒的冰豆漿)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212503.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(好吃的鹹豆花，阿是鹹豆漿才對)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212500.jpg" target="_blank"></a><br/> <br/>(裡頭的油條)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212501.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>(看起來就很好吃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212502.jpg" target="_blank"></a><br/> <br/> <br/></span><span>接下來是薄餅蛋夾油條與蘿蔔絲餅，</span></p>
<p><span>前者比想像中的好吃，博燒餅很香配上軟嫩的煎蛋與微酥的油條堪稱平價美食，</span></p>
<p><span>後者也很特別，裡頭的餡料有點像似切絲的蘿蔔乾，這個倒是不大習慣。</span></p>
<p><span>(薄餅蛋夾油條)</span></p>
