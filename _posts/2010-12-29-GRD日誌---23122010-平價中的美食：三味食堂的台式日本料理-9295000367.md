<br/>　第一次吃到三味食堂應該是七八年前的事情了，那時剛回台灣不久，<br/>																			    <br/>　一位從音樂界跳槽至日本料理界的好友說要推薦我一家好吃的日式料理，<br/>																			    <br/>　兩人便搭乘捷運到了西門町，然後直接步行至貴陽街，<br/>																			    <br/>　那時候的三味食堂還沒有這幾年來如此的高人氣，<br/>																			    <br/>　兩人一下子便找著了空位坐下來點餐。<br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　熱愛日本料理的小弟當時才剛吃過幾家老字號的台式日本料理，<br/>																			    <br/>　比如位在西門町中知名度相當高的美觀園。<br/>																			    <br/>　不過真正會令我印象深刻的大概只有三味食堂了，<br/>																			    <br/>　尤其是那令人嘆為觀止的巨大份量。<br/>																			    <br/>　記得有一次跟兩位好兄弟一同約去大快朵頤，<br/>																			    <br/>　不小心點了大份的綜合生魚片(當時還有大小份之分)，<br/>																			    <br/>　端來時我們倒呼了一口寒氣，第一次看到用碗公裝的生魚片，<br/>																			    <br/>　吃到最後時至少還剩下一半，三人面面相覷了好一會兒，<br/>																			    <br/>　最後終於有人提議了："猜拳吧，輸的吃一片！"<br/>																			    <br/>　就這樣總算把整缽的生魚片幹完，<br/>																			    <br/>　之後整整一個禮拜都可以感覺到肚子裡有魚兒不停地在游泳。<br/><br/><br/><br/><br/><br/>　(桌上隨拍)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　大概有幾年沒再回去吃過了吧，這一天心血來潮，<br/>																			    <br/>　為了避開人潮特地挑了下午五點鐘過去，<br/>																			    <br/>　果然有位置，就座後迅速地點餐：<br/>																			    <br/>　半份手握壽司，NT.120(超擔心吃不下的)<br/>																			    <br/>　海膽手捲，NT.120<br/>																			    <br/>　鮭魚卵手捲，NT.120<br/>																			    <br/>　揚出豆腐，NT.80<br/>																			    <br/>　雞肉唐揚，NT.120<br/>																			    <br/>　以及後來加點的干貝串燒，NT.100<br/><br/><br/><br/><br/><br/><br/><br/><br/>　(隨客人取用的免費味噌湯)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　點完後照例自行取用免費喝到飽的味噌湯，熱騰騰又調味適中，<br/>																			    <br/>　個人很喜歡。<br/>																			    <br/>　沒多久後半份的手握壽司首先被端了上來，<br/>																			    <br/>　果不其然地還是很大份：<br/>																			    <br/>　壽司魚料仍舊切得又大又厚，甚至稱之為生魚排也不為過，<br/> <br/>　堪稱三味食堂的招牌作之一。<br/><br/><br/><br/><br/><br/><br/>　(半份的經典三味食堂手握壽司)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/>　(其實個人還滿愛的)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/></span><br/><br/>　然後登場的是海膽與鮭魚卵手捲，<br/>
