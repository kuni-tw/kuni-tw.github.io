<p><span><strong><br/> <br/> <br/> <br/> </strong><span>----------------------------------------------------</span><br/><span>極味棧(舊)</span><br/><span>地址：新北市永和區竹林路77號</span><br/><span>電話：(02)8925-8648</span><br/><span> ----------------------------------------------------</span></span></p>
<p><span> </span></p>
<p><span>一直想來這家風評不錯的精緻小吃店許久，</span></p>
<p><span>這晚本想去叉子新址嗑一餐晚飯沒想到門口人滿為患，</span></p>
<p><span>回程路經竹林路時赫然發現路邊一塊大招牌上寫著鮮明的三字：極味棧，</span></p>
<p><span>心下一驚馬上鄰近找了地方停車然後走進店門。</span><br/> <br/> <br/> <br/><span>(極味棧店門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198073.jpg" target="_blank"></a></span><br/> <br/> <br/> <br/><span>內乾淨明亮客人很多好不容易找著座位，</span></p>
<p><span>門旁立著一個大玻璃櫃，裡頭擺著不少小菜乍看之下頗為可口，</span></p>
<p><span>看完菜單後點了：</span></p>
<p><span>燙青菜(NT.30，選空心菜)，牛肉捲餅(NT.80)，</span></p>
<p><span>牛雙極(NT.100)，紅油抄手(NT.45)，怪味雞(NT.70)</span><br/> <br/><br/><span>(小菜玻璃櫃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198074.jpg" target="_blank"></a><br/> <br/> <br/>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198075.jpg" target="_blank"></a><br/> <br/> <br/> <br/>(遇到同好了，老闆也玩重機呢^^)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198076.jpg" target="_blank"></a><br/> <br/> <br/>(店內乾淨明亮)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198077.jpg" target="_blank"></a><br/> <br/> <br/></span><span>首先上桌的是燙空心菜與牛肉捲餅，</span></p>
<p><span>前者燙得剛好沒過熟，後者還不錯但沒什麼讓人印象深刻之處。</span><br/> <br/> <br/><span>(燙空心菜)</span></p>
<p><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198078.jpg" target="_blank"></a></span></p>
<p><span> <br/> <br/>(牛肉捲餅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198079.jpg" target="_blank"></a><br/> <br/> <br/>(微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198080.jpg" target="_blank"></a><br/> <br/> <span>然後是牛雙極與紅油抄手，所謂牛雙極就是指牛肉與牛肚，</span></span></p>
<p><span>牛肚切得稍厚口感偏硬所以較有嚼勁，</span></p>
<p><span>麵部分口感又Ｑ又彈牙實在深得我心。</span></p>
<p><span>紅油抄手麻而不辣香味迷人，與同位在雙和地區的濠記食坊有分庭抗禮之勢，</span><br/><span><a href="http://kuni.pixnet.net/blog/post/294998642-grd%E6%97%A5%E8%AA%8C---02022010-%E6%BF%A0%E8%A8%98%E9%A3%9F%E5%9D%8A%E7%9A%84%E7%B4%85%E6%B2%B9%E6%8A%84%E6%89%8B" target="_blank">(請參閱濠記食坊一行，用滑鼠點選此處)</a></span></p>
<p><span>後者辣味較為明顯且種類較多，不過極味棧的麻與香亦不惶多讓。</span><br/> <br/> <br/><span>(牛雙極)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198081.jpg" target="_blank"></a><br/> <br/> <br/>(紅油抄手)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198082.jpg" target="_blank"></a><br/> <br/> <br/>(開放式的廚房)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198085.jpg" target="_blank"></a><br/> <br/> </span></p>
<p><span><span>最後的怪味雞堪稱本日最讚的美味，怪味雞向來是小弟我最愛吃的料理之一，</span></span></p>
<p><span>在大台北地區吃過不知道多少次，只要菜單上有列幾乎必點，</span></p>
<p><span>極味棧的怪味雞使用的是去骨大雞腿，處理手法相當高明，</span></p>
<p><span>肉質極為鮮嫩，雞皮看似肥厚但實地吃起來卻異常地爽口，</span></p>
<p><span>配上上頭一大坨的配料同時吃得出微麻微辣微香微鹹微甜與微酸，令人拍案叫絕，</span><br/> <br/><span>吃完後心情舒爽至極，付完帳走出店門心底不禁暗喊一聲：</span></p>
<p><span>"實在是太好吃啦！"</span></p>
<p><br/><span>(怪味雞)</span></p>
