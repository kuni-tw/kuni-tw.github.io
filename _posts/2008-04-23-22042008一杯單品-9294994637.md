<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  一杯微酸的肯亞AA-PLUS,溫醺,被拉長的中韻,直到味覺開始轉弱時,<br/>
<br/>
  漸漸轉為細緻的單寧;這婉轉而又綿密的過程深深的陶醉了我.<br/>
<br/>
　就這樣,為此我可以耗上一兩小時.<br/>
<br/>
  將運轉中的時事暫且擱置在一旁,沉浸在花香與音符跳動的世界裡,<br/>
<br/>
  看著阿波羅管轄所在的地盤忽暗忽明,閃爍不定,然後我寫一封很誠摯的信給他.<br/>
<br/>
  窗格上忽見一隻穿戴著金色盔甲的信鴿正襟危坐的,不時地張望一下屋內;<br/>
<br/>
  結尾部分我放了幾行歌誦的短詩句,附上幾朵新鮮壓製好的的鳶尾花.<br/>
<br/>
  良久,玻璃門緩緩開啟,十年前曾失落了兩禮拜的天使走了進來,<br/>
<br/>
　她詫異的看著我,誘惑的上唇如同小蜥蜴尾巴般地微微顫了一下,<br/>
<br/>
　然後恍如隔世,兩人不甘寂寞的手握著手在道德與罪惡裡神遊,<br/>
<br/>
  她說了什麼我聽得很清楚,現在卻寫不出來,我像被詛咒似的竟下不了筆.<br/>
<br/>
  後來我置身在維也納金色大廳裡中的唯一一張黃金圓桌,整個大廳裡只有我一個人,<br/>
<br/>
  我走了過去拉開一張華麗的巴洛克風格木雕椅子.<br/>
<br/>
  桌上擺了一瓶斜放的零三年份的NAPA與一只三分之一滿的酒杯,<br/>
<br/>
  杯緣上竟然有我的手印,看著空酒瓶,我舌尖上的味覺開始漸漸回溯,<br/>
<br/>
　突然聽見一聲急促的鈴響,我接起了手機,<br/>
<br/>
　有如例行公事般的回了幾個制式化的虛字,然後才猛然回神,<br/>
<br/>
  剎那間我手裡仍握著當初那乳百色的英國骨瓷.<br/>
<br/>
