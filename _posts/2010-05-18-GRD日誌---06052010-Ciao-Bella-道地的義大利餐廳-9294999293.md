Bella義大利餐廳<br/>   地址：台北市信義區松仁路252號1樓<br/>   電話：(02)2758-4696<br/>--------------------------------------------------<br/>																			    <br/>																			    <br/>																			    <br/>																			    <br/>																			    <br/>  朋友知道我一直在找尋正統的義式燉飯，<br/>																			    <br/>　所以沒事便會幫我多留意一番，<br/>																			    <br/>　這一次他特地帶了張名片給我，神秘兮兮的對我笑著：<br/>																			    <br/>　這家你一定要去吃吃看。<br/> <br/><br/><br/>　先在網路上查詢了一番，原來兼主廚的老闆是義大利人，<br/>																			    <br/>　似乎讓我多了點希望，找了天中午大老遠到了信義區，<br/>																			    <br/>　相當不好找的地理位置，繞了好些時候終於看見了店家。<br/>																			    <br/>　進了去，相當擁擠的店內坐滿客人，<br/>																			    <br/>　等了好些時候才有空位出來。<br/> <br/><br/><br/><br/><br/>   <br/><br/>　　(店門口)<br/><br/><br/><span></span>																			 <br/><br/><br/><br/>　　(桌上的義大利食材籃子)<br/><br/><br/><span></span><br/><br/> <br/><br/><br/><br/>																			   <br/>　點餐時不禁問了一下老闆娘所使用的Risotto米種，<br/>																			    <br/>　老闆娘的回答讓我大吃一驚，是來自義大利的正統米種，<br/>																			    <br/>　回到台灣後就再也沒有吃過使用這種米做的義式燉飯了，<br/>																			    <br/>　這一次哪能輕易放過，點了一盤磨菇燉飯，要價NT.300，<br/>																			    <br/>  另外朋友點的是傳統南方燉番茄義大利麵商業午餐，要價NT.180，<br/>																			    <br/>  附本日湯與紅茶，然後加點烤甜椒冷盤，要價NT.80，<br/><br/>  與義式手工提拉米蘇，要價NT.200。<br/><br/><br/><br/><br/><br/>　　(菜單)<br/>																			    <br/><span><br/></span><span><br/></span><span></span> <br/><br/><br/><br/><br/><br/>　　(紅茶)<br/><br/><br/><span></span> <br/><br/><br/><br/><br/>	 (一下子喝光光後隨拍杯底)<br/><br/><br/><span></span><br/><br/>  沒多久後，本日湯上桌，淺嚐一口，<br/>																			    <br/>　味道滲出微微酸味，像似好久沒嚐到的熟悉味道，<br/>																			    <br/>　大概十幾年前了吧，記得以前住在寄宿學校時常常喝到的前湯中，<br/>																			    <br/>　就有這味道。<br/>																			    <br/>　此時不禁愣了一下，突然間，<br/>																			    <br/>　那些似乎早已被遺忘的青春期回憶再度浮現了出來。<br/> <br/>												  <br/><br/><br/><br/>　　(本日湯)<br/><br/><span><br/><br/><br/><br/><br/><br/></span><br/>　沒多久後，烤甜椒冷盤上桌，<br/>																			    <br/>　甜椒本身口感軟嫩又多汁，明亮的酸甜感，<br/>																			    <br/>　相當開胃，不過倒是沒什麼特別之處。<br/> <br/><br/><br/><br/>　　(烤甜椒冷盤)<br/><br/><span></span> <br/><br/><br/><br/><br/><br/><br/>																		    <br/>　終於，期待已久的義式燉飯終於上場，<br/>																			    <br/>　裡頭放了不少牛肝菌蘑菇，<br/>																			    <br/>　小弟迫不及待的用湯匙舀了一口香噴噴的燉飯，<br/>																			    <br/>　細細咀嚼之下，果然與回憶中的特有口感一樣，<br/>																			    <br/>　飽滿的米粒裡嚐得到米心，對於我們亞洲人來說，<br/>																			    <br/>　這吃起來倒較像是飯沒煮熟。<br/>																			    <br/>　不過不知道是不是技術的關係，<br/>																			    <br/>　米粒較沒有粒粒分明的光澤感，也少了特有的Q度，<br/>												  <br/>　另外調味部分似乎有點平淡。<br/><br/><br/><br/>　　(磨菇燉飯)<br/><br/>																			    <br/><span><br/><span></span><br/></span> <br/>																			    <br/><br/><br/><br/><br/><br/>　朋友的義大利麵部分表現似乎也差強人意，<br/>																			    <br/>　麵條給人稍乾硬的感覺，肉醬部分倒是頗具風味。<br/> <br/><br/><br/><br/>　　(傳統南方燉番茄義大利麵)<br/><br/> <br/><span></span>   <br/><br/><br/><br/> <br/>																			    <br/>　最令人難以忘懷的莫過於這裡價位相當高的限量提拉米蘇，<br/>																			    <br/>　外表毫不起眼的提拉米蘇，小小一塊竟然要價NT.200。<br/>																			    <br/>  但實地品嚐結果卻令人驚豔無比，<br/>																			    <br/>　入口後無比的滑膩柔順的口感，<br/>																			    <br/>　手指餅乾部分柔軟到一個極致，<br/>																			    <br/>　散出濃濃甜香，酒香與起司香，<br/>																			    <br/>　一吃下去瞬間感到幸福呀。<br/><br/><br/><br/><br/>
