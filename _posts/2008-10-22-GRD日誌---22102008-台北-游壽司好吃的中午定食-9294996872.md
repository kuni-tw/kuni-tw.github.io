<br/>
<br/>
  (游壽司最推薦的海膽軍艦壽司，看起來真美！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　早在好幾個月前，網路上知名的游壽司，<br/>
<br/>
　就悄悄地被列在我的美食待吃名單之中了．<br/>
<br/>
　尤其是他們的中午定食，便宜之下仍舊料好實在．<br/>
<br/>
<br/>
<br/>
　今年六月十八在葵元和食裡那吃到非常實惠的日式商業午餐至今記憶猶新，<br/>
<br/>
<br/>
  好不容易，前陣子送回原廠保修的GRD2終於回家，便臨時訂了位，<br/>
<br/>
　中午便飆去永康商圈的麗水街，好不容易找到外觀相當不起眼的游壽司，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (附的熱茶)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(真漂亮的芥茉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　坐在阿布拉師傅前面，點了中午的綜合壽司定食，要價380，<br/>
<br/>
　裡頭有沙拉，味噌湯，四貫握壽司與一大份散壽司．<br/>
<br/>
　首先上的是沙拉，淋上特調和風醬頗具風味，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(可口的沙拉，真開胃呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後是大份量的散壽司，可口的醋飯上擺滿工整的鮪魚，鮭魚，鮭魚卵，白旗魚，<br/>
<br/>
　海鱺與玉子燒切丁，好吃得不得了，好吃到光現在寫心得口水就開始分泌了，<br/>
<br/>
　可見有多麼歐依稀呀！　<br/>
<br/>
<br/>
　<br/>
<br/>
　　(散壽司)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(散壽司裡頭料多實在唷！)　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後四貫握壽司分別是黃雞，鮪魚，甜蝦與穴子魚，<br/>
<br/>
　然後真的得說，已經很久沒吃到好吃的握壽司的我，<br/>
<br/>
　吃到如此用心捏的握壽司真的非常感動．<br/>
<br/>
　上一次在蓮波葉料亭那不愉快的經驗如今在游壽司這裡慢慢褪去．<br/>
<br/>
<br/>
　雖然壽司料品質一般，但是調味上卻頗具巧思，<br/>
<br/>
　比如穴子魚上置放一小撮現磨的芥茉一起入口，<br/>
<br/>
　甜美的穴子魚料在不會死嗆的芥茉陪襯之下相得益彰，<br/>
<br/>
　芥茉的香味更顯得芬芳與清新．<br/>
<br/>
　雖然醋飯捏製的形狀不大正統，但是卻無損於味道本身，<br/>
<br/>
　尤其是飯煮得不會太硬，光這點我就很滿意了．<br/>
　<br/>
　以這樣的價位能吃到如此用心的握壽司，真令人感動呢．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(黃雞)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (鮪魚)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
　　(甜蝦)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(穴子魚)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(味道很棒呢！)<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　味噌湯可不是隨便放切剩的魚料就充數的唷，<br/>
<br/>
　料多味美，魚料的使用似乎有按照比例，另外還有切丁的嫩豆腐與杏鮑菇，<br/>
<br/>
　調味倒沒什麼令人驚豔之處，不過還是很可口呢．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(中規中矩的味噌湯)<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　吃完後，單點一份他們知名的海膽軍艦壽司，<br/>
<br/>
　等阿布拉端上來之後，我簡直不敢相信自己的眼睛，<br/>
<br/>
　非常漂亮的色澤，潤度與完整的形狀與顆粒，<br/>
<br/>
　可惜的是海膽味道似乎被過重的海苔味蓋掉，<br/>
<br/>
　這大概是唯一美中不足的地方吧．<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
    (值得推薦的海膽軍艦壽司，看起來好棒呀！)                                                                            <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(阿布拉師傅)<br/>
