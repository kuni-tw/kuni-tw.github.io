<br/>
　(星巴克第一款造型隨行卡：桐花造型卡)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　早在三四月時就聽到風聲：<br/>
    <br/>
　台灣星巴克即將推出第一張非四方形的造型隨行卡，<br/>
　<br/>
　如此振奮人心的消息一直到了五月六號當天，<br/>
    <br/>
　終於開了花，結了果。<br/>
    <br/>
　之前一直被討論許久的實用性問題，包括如何插卡等，<br/>
    <br/>
　也總算有了答案，桐花造型隨行卡只有左側採花瓣造型，<br/>
    <br/>
　右側維持一貫的四方以利插卡機讀取資料。<br/>
    <br/>
　實品本身以黃白綠來做三種基色，另外綠中還有深淺之別，<br/>
    <br/>
　卡上滿滿的桐花花朵圖騰，洋溢著溫暖奔放的好心情。　<br/>
<br/>
<br/>
<br/>
<br/>
　　(桐花造型隨行卡)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376702699.jpg" target="_blank"></a></span><span><br/>
<br/>
<br/>
</span> <br/>
<br/>
<br/>
　　(號碼還不錯呢)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376702689.jpg" target="_blank"></a></span>   <br/>
<br/>
　到了店裡，順便買了一杯新推出的香草密斯朵，<br/>
    <br/>
　早上十一點前點大杯只要NT.65，但卻少了一貫的星巴克客製化作風，<br/>
    <br/>
　不能加減糖量或是冰量等等，甚是遺憾。<br/>
    <br/>
　以個人平常喝咖啡的習慣，<br/>
    <br/>
　早上的特價冰香草蜜斯朵裡頭冰塊似乎有點兒過多，<br/>
    <br/>
　味道也似乎有點兒過淡。<br/>
    <br/>
<br/>
<br/>
(香草密斯朵)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376702687.jpg" target="_blank"></a><br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376702688.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
　而另外同時推出的桐花野餐提袋，<br/>
    <br/>
　其實也可以稱之為桐花野餐"墊"，<br/>
    <br/>
   先將提袋兩旁的拉鍊拉開，然後解開內裡的鈕扣，<br/>
    <br/>
　拆開後攤平變成為現成的野餐墊子。<br/>
    <br/>
　黝黑的素色外表下，除了中性又好搭配穿著，<br/>
    <br/>
　另外又隱藏著如此實用的設計，<br/>
    <br/>
　實在令小弟愛不釋手，第二天下午馬上拎著出門教課。<br/>
<br/>
<br/>
<br/>
<br/>
  (黝黑樸素的桐花野餐提袋)<br/>
<br/>
