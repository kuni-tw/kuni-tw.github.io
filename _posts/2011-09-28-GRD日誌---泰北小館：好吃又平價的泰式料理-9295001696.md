<p><span>(超級下飯的椒麻雞)<br/> <br/> -----------------------------------------------------------<br/>泰北小館<br/>地址：新北市中和區中和路454號<br/>電話：(02)8923-1689<br/>時間：平日11:30~14:00 17:00~21:30<br/> 假日：11:30~14:30 17:00~21:30<br/> -----------------------------------------------------------</span></p>
<p><span>吃過不少家泰式料理但可惜除少數一兩家外很少會遇到讓我覺得想再回訪的，</span></p>
<p><span>這一次經朋友推薦特地來到永安市場捷運站附近的泰北小館，</span></p>
<p><span>一家據說由某泰北華僑家族經營的泰國餐廳。</span><br/><br/><span>就座後看著菜單點了月亮蝦餅(NT.180)，椒麻雞(NT.150)，青醬魚(NT.200)</span></p>
<p><span>打拋牛肉(NT.130)，蝦醬空心菜(NT.100)，泰式豆腐煲(NT.180)</span></p>
<p><span>用泰國香米下去煮的白飯吃到飽算人頭一人才NT.20。</span></p>
<p> </p>
<p><span>(錫製冷水壺)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212517.jpg" target="_blank"></a><br/> <br/> </span></p>
<p><span>沒多久服務生便端來冷水壺與飯鍋，兩者皆為錫製品上頭刻滿花紋，</span></p>
<p><span>使用泰國香米煮的白飯無限量供應以人頭來算真的很便宜，</span></p>
<p><span>卻少了香米特有的香氣且口感偏乾推測是米質問題。</span></p>
<p><br/><span>(無限量供應的泰國香米飯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212518.jpg" target="_blank"></a></span><br/> <br/><br/><span>首先上桌的是兩道個人最常點的泰式料理：月亮蝦餅與椒麻雞，</span></p>
<p><span>前者沒給人深刻的印象，後者則反之份量很大且口感上不只單純地外酥內軟，</span></p>
<p><span>同時充滿著肉汁與帶點油脂，加上細切的高麗菜絲與微酸辣的調味很下飯。</span></p>
<p><span>　</span><br/><span>(月亮蝦餅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212519.jpg" target="_blank"></a><br/><br/>(美麗的色澤)</span></p>
<p><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212520.jpg" target="_blank"></a></span></p>
<p><span><br/>(椒麻雞)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212521.jpg" target="_blank"></a><br/><br/>(真得很下飯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212522.jpg" target="_blank"></a><br/><br/></span><span>接下來是青醬魚與打拋牛肉，皆為相當能充分展現出泰式料理特質的菜色，</span></p>
<p><span>前者使用青醬以翡翠椒大綠椒香菜梗與檸檬葉等混在一起打出來，</span></p>
<p><span>與義式的做法截然不同，微酸又透著點檸檬香氣，</span></p>
<p><span>加上鮮美又肉厚的鱸魚吃起來格外清爽可口。</span></p>
<p><span>後者顧名思義使用一種名為打拋的泰國香草一起下去炒，</span></p>
<p><span>這打拋的泰語實地發音為嘎拋，是一種聞起來很像九層塔的羅勒，</span></p>
<p><span>不過卻沒台式的香味來得強烈。</span></p>
<p> </p>
<p><span>(青醬魚)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212523.jpg" target="_blank"></a><br/><br/>(打拋牛肉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212524.jpg" target="_blank"></a><br/> <br/> </span></p>
<p><span><span>最後是兩道我給予相當高評價的蝦醬空心菜與泰式豆腐煲，</span></span></p>
<p><span>前者偏鹹卻不惱人，鹹中帶層次與散出濃郁的蝦醬香，</span></p>
<p><span>後者充滿著奶香配上外層有著豆皮口感內層軟嫩到極致的豆腐，</span></p>
<p><span>兩者實在都是太好吃了！</span><br/> <br/><span>(蝦醬空心菜)</span></p>
