<br/>
  (星巴克的六本木七週年限量隨行杯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    <br/>
    <br/>
    <br/>
　這一天同時拿到兩個期待已久的星巴克隨行杯款，<br/>
    <br/>
　一個是日本的六本木七週年限量隨行杯，<br/>
    <br/>
　另一款是香港的中環冰室限定隨行杯。<br/>
    <br/>
　兩款都是好心的朋友們幫忙代買回來的。<br/>
    <br/>
　<br/>
<br/>
    <br/>
　<br/>
    <br/>
　首先是六本木杯款，與知名的東名高速隨行杯一樣，<br/>
    <br/>
　都是少見的透名城市隨行杯款，不過有一點不大一樣，<br/>
    <br/>
　前者竟然是杯底可拆卸的回憶杯。<br/>
    <br/>
　配合著晶瑩剔透的透明杯面與彩色繽紛的精美塗裝，<br/>
    <br/>
　看來這一款杯子大概很快又要賣光光了。<br/>

　香港的中環冰室限定隨行杯亦是杯底可拆卸的回憶杯，<br/>
    <br/>
　雅致的黑白速寫圖圍繞在杯圍四周，<br/>
    <br/>
　造就出一種知性與書卷氣質的美感。<br/>
    <br/>
　中環都爹利街星巴克冰室是一處很特別的門市，<br/>
    <br/>
　2009年以六七零年代的香港懷舊冰室為設計概念，<br/>
    <br/>
　裡頭分為兩個區塊，入口處與全世界的星巴克門市雷同，<br/>
    <br/>
　裡頭的區塊則較復古，融入許多懷舊元素，<br/>
    <br/>
　非常特別。<br/>
