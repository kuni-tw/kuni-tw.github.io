<br/>
　臨時硬著頭皮,還是將就的紀錄一下,<br/>
<br/>
  雖然跟老大不熟,不過悠揚的旋律充滿著流動性,<br/>
<br/>
  活生生的又重新帶領我進入了石板路上疾行的馬車裡.<br/>
<br/>
<br/>
  愜意的踏著步伐的馬,像風清楊祖師爺一樣地瀟灑,走過瀏陽河上的橋,<br/>
<br/>
　親切的低聲嘶鳴,和著手風琴悅耳的和絃,橋下細水長流,<br/>
<br/>
　不知不覺中輪子慢慢停止了轉動...<br/>
<br/>
　順著進入車窗的光束往外頭瞧,看見了老舊又泛黃的褐色底片,<br/>
<br/>
  隨著巨輪的轉動逐漸清晰,愈來愈豐富,充滿著層次.<br/>
<br/>
<br/>
　外頭的光芒漸漸轉橘紅,像顆半熟的荷包蛋,灑上些許的玫瑰鹽,<br/>
<br/>
  白色中帶一點粉紅,間接一陣一陣地透出微微松露的香氣,<br/>
<br/>
  全身上下倘佯在凝脂裡,嗅著嗅著,漸漸昏昏欲睡,一動也不想動,<br/>
<br/>
  開一瓶殷紅色的Latour,過了半餉,維也納獨特的法國號揚起了序曲,<br/>
<br/>
  就這樣我又見到了歌德...<br/>
<br/>
