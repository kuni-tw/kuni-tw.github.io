<p><br/> <br/> <br/> 　(NY-Bagel在美麗的陽光之下)<br/> <br/> <br/> <br/> <br/> <br/> <br/> 　幾乎算得上是NY-Bagel常客的我，卻從沒為他們寫過食記還真是說不太過去，<br/> <br/> 　今天起個大早，加上前兩個學生臨時改期上課，一時興致勃勃就衝了去仁愛分店．<br/> <br/> 　和著週末晨間閃耀的陽光，一場美式早餐盛宴真是太應景了．<br/> <br/> 　<br/> 　大概是老天爺的眷顧吧，門口幾個小團體在排隊，我卻馬上拿到我最愛的雙人位，<br/> <br/> 　看來外頭的都在等四人以上的桌位吧．<br/> <br/> <br/> <br/> 　看了看菜單，意外得知NY前陣子推出不少新餐點，從附圖來看感覺都很好吃，<br/> <br/> 　看得我口水直流，可惜最近發福還是少吃點好，遂只點了一份"火腿班尼迪克蛋"，<br/> <br/> 　附上新鮮水果，橙汁與無限供應的熱咖啡加上10%的服務費才兩百二，<br/> <br/> 　感覺真划算呢．<br/> <br/> <br/> <br/> <br/> 　白色的大圓盤裡有番茄與黃瓜切大丁，烤馬鈴薯塊以及主菜：<br/> <br/> 　煎火腿，兩顆半熟的水波蛋與上面淋著滿滿蛋黃醬(又稱荷蘭醬)的英式鬆糕，<br/> <br/> 　為什麼說"鬆糕"，看似滿福堡的外觀其口感卻類似鬆餅，<br/> <br/> 　將半熟的水波蛋弄破，以鬆糕沾著暖暖的蛋黃，口感還滿不錯的．<br/> <br/> <br/> 　<br/> 　美中不足的則是副餐水果就寥寥兩三片蘋果，<br/> <br/> 　與主餐實在不搭以外，外觀更是沒什麼美感，<br/> <br/> 　這部分真希望NY-Bagel再下點功夫呀！<br/> 　　<br/> 　<br/> 　　<br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> (火腿班尼迪克蛋)<br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> 　(局部特寫:誘人的水波蛋)<br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> (熱絡的早晨)<br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> <br/> BY GRD2<br/> <br/> <br/> <br/> <br/> .</p></div>
