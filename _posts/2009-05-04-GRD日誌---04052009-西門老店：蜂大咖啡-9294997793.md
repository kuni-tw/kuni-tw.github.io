<br/>
    (蜂大著名的手工餅乾：核桃酥，雞仔餅，鮑魚酥)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>



下午有事在西門町，忙完後與友巧遇，<br/>
<br/>
　便提議在附近找地方坐坐，挑上了幾十年的老店"蜂大咖啡"，<br/>
<br/>
　店門口擺著幾只大玻璃罐，相當好認．<br/>
<br/>
　裡頭放滿各式各樣的手工餅乾，由店內一位香港的師傅所製，<br/>
<br/>
　也是蜂大的招牌與特色之一．<br/>
<br/>
　點了三樣餅乾(核桃酥，雞仔餅，鮑魚酥)，<br/>
<br/>
　與一小盒蛋捲，分別看了一二樓的環境後，<br/>
<br/>
　決定在光線較良好的二樓就座．<br/>
<br/>
　聽了店員介紹後，決定試試他們的蜂大綜合咖啡，<br/>
<br/>
　另外點了杯我常喝的義式摩卡以及一杯冰淇淋冰滴．<br/>
　<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　餅乾表現尚可，核桃酥還不錯，微甜，<br/>
<br/>
而雞仔餅味道有些類似新竹的名產竹塹餅，<br/>
<br/>
　剛好是我不喜歡的味道，<br/>
<br/>
　另外港式做法的蛋捲口感滿還細緻的，<br/>
<br/>
　金黃餅皮顏色均勻，又薄又脆．<br/>
<br/>
  價位分別是核桃酥一片15，雞仔餅一片12，鮑魚酥一片15，<br/>
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　<br/>
　相當超值！<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(金黃色的港式蛋捲，餅皮薄脆，口感十分細緻)<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(核桃酥，雞仔餅，鮑魚酥價位相當低，很超值呢)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　 (雞仔餅的斷層特寫，真可口呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　　
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

　咖啡就不得不稱讚一番了，<br/>
<br/>
　義式摩卡與冰淇淋冰滴表現中上，可惜的是稍稍偏貴了點...<br/>
<br/>
　但一杯才85的蜂大綜合，以巴西，摩卡，<br/>
<br/>
　曼特寧和哥倫比亞四種豆子的混合調配，<br/>
<br/>
　味道有如魔術一般，神奇又均衡，<br/>
<br/>
　以這價位來說，真是難得一見的好咖啡呢！<br/>
