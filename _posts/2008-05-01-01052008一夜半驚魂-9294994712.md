<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　與友一時興起遊山玩水,途中敗興折回至仁愛腹地.<br/>
<br/>
  就座之後對著過往來客指指點點不休,不巧也看到一位愛哭的毛姑娘,<br/>
<br/>
  與不知名的.....<br/>
<br/>
  友說:"她我認識耶!"<br/>
<br/>
　"喔你認識,誰呀?"<br/>
<br/>
  "看過,但不知道名字."<br/>
<br/>
  "呃,這不是等於白講了."<br/>
<br/>
  兩人漸漸地開始不著邊際,反正想得到的先插隊再說.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　後來看到三男不甘寂寞地請同行女性友人搭訕別桌,<br/>
<br/>
  友又說了:"你不覺得這些人很奇怪嗎?"<br/>
<br/>
  "還好啦,這種見多了"<br/>
<br/>
　我就開始述說起以前看到某男性冤大頭的往事,<br/>
<br/>
　友:"亨,這種人真是白癡!"<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　就這樣子,有一搭沒一大搭的順著話風,<br/>
<br/>
  晚風逐漸轉涼,玻璃杯見底,<br/>
<br/>
  遂清了款項啟動路邊等待多時的兩匹悍馬,隨著圓環直達林蔭大道,<br/>
<br/>
  不多時兩人已過萬重山水,就在此刻一瞥忽然間避開兩次陷阱,<br/>
<br/>
  心想這回可要鬥智了,兩人馬上調頭向左疾行.<br/>
<br/>
  我不時地盯著前方與後方,沒想到還是被識破了,<br/>
<br/>
  便拉高轉速迅速脫離,良久兩人默契的在巷口會合,<br/>
<br/>
  然後徒步兩公里找尋另一隻悍馬.<br/>
<br/>
<br/>
<br/>
<br/>
<br/> <br/>
<br/>
  心驚至此仍未完全平復.<br/>
