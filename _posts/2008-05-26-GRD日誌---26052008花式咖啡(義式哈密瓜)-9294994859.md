　半夜與小風去中和路永貞路交叉口的客喜康,<br/>
<br/>
  那是一家我半夜友事沒事會去光顧的一家好店,店長年輕有為,<br/>
<br/>                                                    
  跟我一樣很熱愛咖啡,<br/>
<br/>
　這一次,店長他那今年暑假要參加大師賽的弟弟臨時幫我弄了一份特製花式:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  取材自04年的比賽冠軍創作過的作品,暫且稱之為"義式哈密瓜"好了,<br/>
<br/>
  香甜牛奶與鮮奶油混進打好的冰鎮哈密瓜汁,<br/>
<br/>
　上面慢慢淋上冰濃縮,那哈密瓜的香氣不知為什麼與濃縮咖啡那麼的合,<br/>
<br/>
  真是味覺上的享受呀...<br/>
