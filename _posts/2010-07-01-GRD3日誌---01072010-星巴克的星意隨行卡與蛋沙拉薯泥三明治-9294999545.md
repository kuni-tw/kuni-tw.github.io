<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　七月一號發行的星巴克星意隨行卡相當貴，<br/>
    <br/>
　一張要價NT.350，與之前所不同的是裡頭附有兩點紅利，<br/>
    <br/>
　可供飲料或是限定商品兌換，另外更重要的是，<br/>
    <br/>
　在有效期限內，以此卡在星巴克消費時可享九折優惠。<br/>
<br/>
<br/>
<br/>
<br/>
  (星意隨行卡配上七月一號開始通用的新版常態性卡套)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721537.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(星意隨行卡配上專屬卡套)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721536.jpg" target="_blank"></a></span><br/>
<br/>
 <br/>
<br/>
<br/>
<br/>
   <br/>
　為了找尋特別的卡號，小弟一大清早趕到幾家門市，<br/>
    <br/>
　順便喝杯優惠價NT.65的冰香草蜜斯朵，<br/>
    <br/>
　點了沒吃過的蛋沙拉薯泥三明治，要價NT.45。<br/>
<br/>
<br/>
<br/>
　特價的香草蜜斯朵，無法提供以往星巴克所引以自豪的客製化服務，<br/>
    <br/>
　加上冰塊過量，飲用一段時間後會因冰塊逐漸溶解，<br/>
    <br/>
　會有味道變得過淡的問題。<br/>
    <br/>
　唯一的方法就是，拿到做好的飲料後，<br/>
    <br/>
　客氣地請夥伴將裡頭的冰塊挖出來，<br/>
    <br/>
　當然裡頭的飲料量也會實質地變少，<br/>
    <br/>
　但至少能保有較好喝的口感。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　蛋沙拉薯泥三明治在星巴克糕點類裡價位算相當低，<br/>
    <br/>
　實地品嚐時，卻驚異地發現相當好吃，<br/>
    <br/>
　薯泥配上蛋沙拉那綿綿細緻的口感，<br/>
    <br/>
　配上新鮮甜脆的生菜，加上頗具質感的胚芽三明治，<br/>
    <br/>
　相當可口，而且份量不算小，搭上大杯優惠香草蜜斯朵，<br/>
    <br/>
　飽足感相當夠呢。<br/>
    <br/>
　扣了杯扣，再以星意隨行卡結帳打九折，<br/>
    <br/>
　只花了NT.89，稱得上是相當相當划算。<br/>
    <br/>
<br/>
<br/>
