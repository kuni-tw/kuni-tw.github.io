-------------------------------------------------<br/>
芭蕾咖啡<br/>
地址：台北市民生東路五段36巷8弄62號<br/>
電話：(02)2763-1981<br/>
-------------------------------------------------<br/>
    <br/>
    <br/>
  在網上看到過不少網友們推薦芭蕾咖啡館的分享文章，<br/>
    <br/>
　一直以來總想抽個空過去看看，這天下午，<br/>
    <br/>
　趁著教課時間空檔臨時溜出去，<br/>
    <br/>
　到了知名的民生社區，在巷弄中找著了芭蕾咖啡，<br/>
    <br/>
　門邊聳立著一棵大樹，旁零零落落地擺著不少大盆栽，<br/>
　<br/>
　足添了不少綠意盎然，好美的一番景象呀。<br/>
   <br/>
<br/>
<br/>
<br/>
　　(門口一片綠意盎然)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721433.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
　進了店門，寬敞而又明亮的大空間，<br/>
    <br/>
　中間一張大桌供客人們併坐，<br/>
    <br/>
　桌底下的小貓被客人用逗貓棒逗得橫衝直撞。<br/>
    <br/>
　被服務人員帶位就坐後，看了牆上的點餐黑板，<br/>
    <br/>
　點了一杯糖烤瑪琪朵，要價NT.180，<br/>
    <br/>
  另外因嘴饞，順便點了普獲網友們好評的鄉村牛肉鹹派，<br/>
    <br/>
　要價NT.130。<br/>
<br/>
   <br/>
<br/>
<br/>
　　(躺在門口的貓)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721434.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(牆上的黑板菜單)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721435.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  等餐期間順便環顧了一下四周，<br/>
    <br/>
　看到了兩台熟悉的Mazzer Super Jolly，<br/>
    <br/>
　會熟悉可能也是因為前陣子才幫朋友賣出一台Robur的關係吧，<br/>
    <br/>
　詢問之下，原來一台是負責義式，另一台則是負責單品。<br/>
<br/>
   <br/>
<br/>
<br/>
<br/>
　沒多久後，鄉村牛肉鹹派與糖烤瑪琪朵陸續上桌，<br/>
    <br/>
　鹹派部分，盤上擺著生菜沙拉與番茄切片，沾醬與牛肉鹹派，<br/>
    <br/>
　生菜部分新鮮度夠，鹹派部分先淺咬一口，<br/>
    <br/>
　派皮口感稍軟但不酥，肉餡部分調味似乎過淡，<br/>
    <br/>
　配上酸甜的沾醬，可能是口味不合吧，<br/>
    <br/>
　個人沒特別喜歡。<br/>
<br/>
<br/>
<br/>
<br/>
　　(鄉村牛肉鹹派)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721436.jpg" target="_blank"></a><br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721437.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
  咖啡部分，將咖啡上的黑糖微炙過再送上桌，<br/>
    <br/>
　微焦的糖層散出絲絲甜香，但不知道是什麼原因，<br/>
    <br/>
　似乎沒有與咖啡融合在一起的感覺，<br/>
    <br/>
　之前在許多咖啡廳裡喝到過不少類似的咖啡，<br/>
    <br/>
　其中個人最推薦的應該是最近在永和菲瑪所喝到的黑糖卡布，<br/>
    <br/>
　微焦糖層與咖啡搭在一起竟可如此地天衣無縫，<br/>
    <br/>
　而且也較便宜呢。<br/>
<br/>
<br/>
<br/>
<br/>
  　　(糖烤瑪琪朵)<br/>
