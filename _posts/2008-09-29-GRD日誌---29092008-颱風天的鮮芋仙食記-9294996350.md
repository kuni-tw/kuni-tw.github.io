<br/>
  (鮮芋仙的招牌仙草)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　大颱風天的禮拜一中午與友在永和的知名老店老黃牛肉麵大快朵頤後，<br/>
<br/>
　老黃部分請請參閱：<br/>
<br/>
　<div><a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11461159" target="_blank">老黃牛肉麵</a></div><br/>
<br/>
　一行人走著走著經過一家鮮芋仙，一家我早已耳聞許久的甜品連鎖店，<br/>
<br/>
　這一兩年來快速的擴張分店數目．<br/>
<br/>
　當然網路上褒貶不一，以這樣大規模的經營方式在所難免，<br/>
<br/>
　就如同星巴克或８５Ｃ一般，生意做得愈大，愈容易引起批評．<br/>
　<br/>
<br/>
　記得第一次聽到這家的店名是在幾個月之前，友要求我幫他外帶甜食，<br/>
<br/>
　卻指名一定要鮮芋仙時，這三個字當時便在我腦海留下深刻的印象了，<br/>
<br/>
　今日既然有幸經過，五臟雖飽，卻仍有一碗甜食的預備空間，<br/>
<br/>
　最多晚上颱風下再騎個五公里山路．<br/>
<br/>
<br/>
<br/>
　老實說，個人向來是不吃這種東西的，以前剛回台灣時曾因為好奇吃過幾家，<br/>
<br/>
　後來便因為沒什麼感覺再也沒吃過了．<br/>
<br/>
　所以老實說，今天感覺有點像是去踢館，講這樣還真讓我感到不好意思呢！<br/>
<br/>
<br/>
<br/>
　點了一碗鮮芋仙招牌，內有仙草凍，仙草沙，芋圓與鮮奶油，<br/>
<br/>
　相當豐富的配料，要價50，這個價格算平價甜食裡偏高的了，<br/>
<br/>
　哇，要是不好吃，可真的虧大了．<br/>
<br/>
　另外還點了一杯冬瓜仙草凍，要價30．<br/>
<br/>
　內用的客人會得到一個與伯朗咖啡館一樣的電子摳機，<br/>
<br/>
　餐點好的時候會在桌面上震動與逼逼叫，提醒客人來領取．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(電子摳機響時，會亮黃燈！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先是冬瓜仙草凍，很特殊的內用瓶裝設計，一大杯，<br/>
<br/>
　很漂亮的色澤，一看就覺得很涼爽很解暑的感覺，<br/>
<br/>
　冬瓜汁部份爽口，偏甜，但是我喜歡．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(冬瓜仙草凍漂亮的色澤)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(冬瓜仙草俯瞰圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後是鮮芋仙招牌，真的是一大碗，滿滿的仙草，<br/>
<br/>
　上面擺著芋圓以及地瓜圓．<br/>
<br/>
　個人一開始看的時候還有點錯愕，畢竟也吃過九份還是深坑的芋圓，<br/>
<br/>
　那時候只覺得口感不錯，香味適中，可能跟個人口味有關吧．<br/>
<br/>
　我便咬了一口他們的芋圓，馬上嚐到濃濃的芋頭味，<br/>
<br/>
　喔，好香的芋頭味呀，我心裡忍不住驚了一下！<br/>
<br/>
　每個都大大飽滿，口感很不錯，這真是連鎖店的等級嗎？<br/>
<br/>
　難怪會有那麼多人排隊了．<br/>
<br/>
　這樣還沒結束呢，湯匙舀下去才發現底下竟還有玄機，<br/>
<br/>
　原來是仙草冰沙，整體說來是我相當喜歡的冰品！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(鮮芋仙的招牌仙草，份量頗大！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(加了奶球後漂亮的色澤！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(看起來真好吃...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　雖然網路上對他們的負評如此之多，不過，<br/>
<br/>
　這畢竟是每個人口味不同罷了，這才叫青菜蘿蔔各有所好呀，<br/>
<br/>
　而他們的確對上了我的胃嚕！<br/>
<br/>
  以連鎖店來說，味覺品管向來一直都是一個很難突破的問題，<br/>
<br/>
　鮮芋仙今天能做到如此地步，真的滿不容易的呢．<br/>
