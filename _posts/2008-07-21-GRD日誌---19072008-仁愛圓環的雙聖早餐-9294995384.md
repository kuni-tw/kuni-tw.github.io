<br/>
<br/>
<br/>
<br/>
<br/>
  禮拜六趁著早上臨時一個小空檔走去仁愛的雙聖吃一頓早餐．<br/>
<br/>
　雖然之前就去過好幾次了，不過一直沒寫過食記，<br/>
<br/>
　這一次碰巧隨身有帶著心愛的"GRD2"小DC，剛好也可以拍照．<br/>
<br/>
　<br/>
<br/>
<br/>
　走進門口之後，被服務生帶位，服務態度非常好，<br/>
<br/>
　這也是我很喜歡這一家餐廳的原因之一，不知怎麼的，<br/>
<br/>
　他們的服務態度老是給我一種很舒服的感覺，<br/>
<br/>
　並不是說特別殷勤或是什麼熱情之類的，但就是很放鬆，<br/>
<br/>
　我很喜歡．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這一次我點了他們的早餐蔬果沙拉，裡面附有新鮮生菜，西芹，胡桃，葡萄乾，<br/>
<br/>
　水果，原味的優格沾醬以及全麥吐司，飲料則是Cola Zero，<br/>
<br/>
　一看就知道是低熱量取向，想想真是慚愧呀，個人其實還滿想減肥的．<br/>
<br/>
　若要說好吃與否，其實是還好，就是清爽而已，<br/>
<br/>
　蔬菜水果的新鮮度還夠，沾醬還不錯，配上芹菜還滿爽口的．<br/>
<br/>
　加上服務費約兩百左右，在這種地段其實還滿正常的，<br/>
<br/>
　手裡拿著一份剛出爐的蘋果日報，<br/>
<br/>
　至少可以讓我享受一個氣氛很棒的週末早晨！<br/>
<br/>
