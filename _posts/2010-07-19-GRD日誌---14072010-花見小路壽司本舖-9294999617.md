<br/><br/><br/>　　(花見小路的勝手丼)<br/><br/><br/><br/><br/><br/>------------------------------------------------------<br/>   花見小路壽司本舖<br/>   地址：臺北縣永和市安樂路160號<br/>   電話：(02)8941-9802<br/>------------------------------------------------------<br/>    <br/>    <br/>  這一篇可能會寫得較不公平些。<br/>    <br/>　<br/>　<br/>　前一天才剛去過野壽司的小弟，第二天傍晚臨時肚子餓，<br/>    <br/>　加上曾聽到朋友推薦過，<br/>    <br/>　便決定去永和四號公園旁的花見小路叨擾一番。<br/><br/><br/><br/><br/>　進了店門，在門口的吧台前就座後，<br/>    <br/>　細看了菜單，點一份山藥海膽，要價NT.380<br/>    <br/>  和一份勝手丼，要價NT.260<br/>    <br/>    <br/>    <br/>  沒多久後，山藥海膽先上桌。<br/>    <br/>    <br/>    <br/>    <br/>　一大碗裝細切的山藥上鋪著滿滿的三大條海膽，<br/>    <br/>　外觀看起來頗為誘人。<br/>    <br/>　海膽本身型狀還算完整，新鮮度夠，<br/>    <br/>　據說是從日本進口而來的，<br/>    <br/>　不過味道似乎不是很濃郁，說不上來為什麼。<br/><br/><br/><br/><br/><br/>  (山藥海膽)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/></span><br/>　然後是勝手丼，一份擺得相當美的散壽司，<br/>    <br/>　醋飯上鋪著鮪魚赤身，鮭魚以及旗魚，<br/>    <br/>　三種較為常見的生魚片，厚度偏厚，<br/>    <br/>　相當新鮮，溫度恰到好處，<br/>    <br/>　另外還有兩隻甜蝦與一些生花枝。<br/><br/><br/><br/><br/>　(勝手丼)<br/><br/><span><br/><br/><br/><br/><br/></span><br/><br/>　雖然說前一天才剛吃到令人感動至極的野壽司，<br/>    <br/>　對於第二天的花見小路似乎有點不大公平，<br/>    <br/>　不過說實在的，以散壽司來說，同樣在永和，<br/>    <br/>　個人反而較推薦在中正橋頭的橋壽司，<br/>    <br/>　要價更低的海鮮丼，雖然小輸在新鮮度上，<br/>    <br/>　但上頭的魚料多到令人難以置信，<br/>    <br/>　若沒用筆寫絕對會漏記：<br/>-----------------------------------------------------<br/>　胭脂蝦一隻，甜蝦三條，熟蝦一隻，<br/>　兩個生的大干貝，鮪魚赤身，海鱺，紅甘，鯖魚，<br/>　生花枝，玉子燒以及鰻魚等等。<br/>-----------------------------------------------------<br/><br/><br/><br/><br/><br/><br/>　此時味增湯上桌，味道普普，<br/>　<br/>　令人有些小失望。<br/><br/><br/><br/><br/><br/>　(味增湯)<br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/></span><br/><br/>　聽師傅說，他也曾在日本的本店工作了好一段時間，<br/>    <br/>　而花見小路是日本總公司特地在台灣開的海外分店。<br/>    <br/>　此時小弟不禁好奇了起來，便加點一份赤身握，要價NT.160<br/>    <br/>　師傅馬上為我捏製端上桌。<br/>    <br/>　外表看起來，不論是魚片顏色還是型狀，皆相當漂亮。<br/>　<br/>　可惜的是，第一個壽司使用筷子夾起來後，<br/>    <br/>　底下的醋飯便散了開來。<br/>    <br/>　第二個只好小心地使用筷子捧起來，入口後，<br/>    <br/>　醋飯口感仍舊偏緊偏硬(這可奇了)，<br/>    <br/>　而且調味過重，另外魚片部分雖然夠新鮮，<br/>　<br/>　但卻切得過厚。<br/>    <br/>　平心而論，扣除野壽司裕二師傅的話，<br/><br/>　光論握功，花見小路師傅其實不會輸給台北那幾家高價位的。<br/><br/><br/><br/><br/><br/>　(鮪魚赤身握)<br/><br/><span><br/><br/></span><br/><br/>    <br/>　最後店家送的飯後點心是枸杞白木耳，<br/>    <br/>　味道清淡，大概吃了一口，就懶得吃完，<br/>    <br/>　直接去付帳了。<br/>  <br/><br/><br/><br/>　(枸杞白木耳)<br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/>BY KUNI AND GRD3<br/><br/><br/><br/><br/><br/>.</div>
