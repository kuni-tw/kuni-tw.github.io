<p><span>(美到令人感到屏息的漬筋子握)<br/><br/> <br/> --------------------------<br/>匠樂　割烹壽司<br/>地址：臺北市大安區四維路52巷31號<br/>電話：(02)7709-1188<br/> --------------------------</span></p>
<p><span>(門口招牌)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212556.jpg" target="_blank"></a></span></p>
<p> </p>
<p><span>前知名料亭Micasa以阿鴻師傅為首的部分團隊如今在九月初重新開店，</span></p>
<p><span>店址選在四維路知名牛排餐廳Danny &amp; Company旁取名叫"匠樂 割烹壽司"。</span></p>
<p> </p>
<p><span>(講究的漆具)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212557.jpg" target="_blank"></a></span><br/> <br/> <br/><span>這晚臨時訂位可惜板前早已客滿只好先屈就在一般用餐區，</span></p>
<p><span>簡單俐落的菜單上只列了三四種沒寫內容的套餐與價格，</span></p>
<p><span>初次來訪先試水溫遂點了樂旬套餐(NT.2380)</span></p>
<p><span>服務生態度細心與專業給人良好的印象，沒多久後先端上熱茶，</span></p>
<p><span>等餐期間環顧一下四周氣派而又極具質感的木造裝潢。</span></p>
<p> </p>
<p><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212558.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>首先是鮑魚無花果佐京都腐皮茗荷與北海道秋刀魚甘露煮兩道前付，</span></p>
<p><span>前者薄切鮑魚的嫩配上微酸微甜的無花果與香滑的芝麻醬是一道不錯的暖身運動，</span></p>
<p><span>正值產季的秋刀魚肥嫩不說，洋蔥經細心處理過(冰鎮？)已完全不嗆，</span></p>
<p><span>兩道皆為細緻的味蕾體驗。</span><br/> <br/> <br/><span>(鮑魚無花果佐京都腐皮茗荷)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212559.jpg" target="_blank"></a><br/> <br/> <br/>(微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212560.jpg" target="_blank"></a><br/> <br/>(北海道秋刀魚甘露煮)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212561.jpg" target="_blank"></a><br/></span></p>
<p><span> <br/>(真美的微距)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212562.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>接下來是比目魚薄造佐酸桔醋，薄切比目魚的美麗令人心曠神怡，</span></p>
<p><span>絕妙的緊實彈牙口感配合著清爽的肉質繼續讓人保持期待的心情。</span></p>
<p> </p>
<p><span>(比目魚薄造佐酸桔醋)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212563.jpg" target="_blank"></a><br/> <br/> <br/>(從一般席往板前看過去)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212564.jpg" target="_blank"></a><br/> <br/> <br/> </span></p>
<p><span>至此已發現一般座位區的上菜速度似乎有點偏慢，</span></p>
<p><span>不過服務生很貼心還記得小弟訂位時曾希望坐板前的要求，</span></p>
<p><span>此時板前客人已離去並重新整理完畢，便將小弟轉到阿鴻師傅前面，</span></p>
<p><span>同時綜合生魚片上桌裡頭包括鮪魚大腹，赤身，旗魚，真鯛，甜蝦，</span></p>
<p><span>紅甘與少見的青芋莖切片，最令人印象深刻的莫過於那顏色極淺的赤身，</span></p>
<p><span>與一般常見的深紅完全不同，雖然是遠自大西洋來的冷凍鮪魚，</span></p>
<p><span>但從其絕佳的口感表現更可見師傅其高超的處理手藝。</span></p>
<p><span>(綜合生魚片)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212565.jpg" target="_blank"></a><br/> <br/> <br/>(阿鴻師傅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212566.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>接下來是匠樂的拿手好戲煮物醋物及烤物三部曲：</span></p>
<p><span>手工雞肉丸子佐野生明蝦，大和芋佐八方醋與赤鯥魚大葉燒，</span></p>
<p><span>前者靠著熱度先將逐漸冷卻的胃重新暖和，</span></p>
<p><span>然後靠著甘美的柴魚湯頭與雞肉丸，明蝦，揚豆腐及南瓜等不同煮料，</span></p>
<p><span>以不同的口感組合重新挑起我的好奇心，醋物靠著微酸則繼續帶來清爽，</span></p>
<p><span>烤物赤鯥魚外皮烤得微酥微焦卻仍保有內層的軟嫩，</span></p>
<p><span>可惜魚刺有點過多不是很喜歡。</span><br/> <br/><span>(手工雞肉丸子佐野生明蝦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212567.jpg" target="_blank"></a><br/> <br/>(大和芋佐八方醋)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212568.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>(赤鯥魚大葉燒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212569.jpg" target="_blank"></a><br/> <br/> <br/>(微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212570.jpg" target="_blank"></a><br/> <br/> <br/></span><span>阿鴻師傅知道我偏愛握壽司所以接下來破例為我連捏了八貫，</span></p>
<p><span>這當中包括比目魚中段，小鯛，北寄貝，鯥魚(紅喉)，午魚，海膽與漬筋子等，</span></p>
<p><span>單以握壽司來說醋飯與握功是非常重要的，可惜這非匠樂的專長，</span></p>
<p><span>但阿鴻師傅大概是我吃過所有台北高級日式料亭裡最具親和力的料理長了，</span></p>
<p><span>姿態低且拿捏合宜而不惱人的風趣從頭至尾完全彌補了握壽司部分的不足，</span></p>
<p><span>而使用了大量北海道高檔馬份的軍艦捲，雖海膽形狀已快化掉，</span></p>
<p><span>但那極具層次的甜與鮮卻令人印象深刻。</span><br/> <br/> <br/><span>(比目魚中段握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212571.jpg" target="_blank"></a><br/> <br/>(小鯛握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212572.jpg" target="_blank"></a><br/> <br/>(北寄貝握)</span></p>

<p><span><span>當晚最棒的一貫非漬筋子握莫屬，</span></span></p>
<p><span>漬筋子乃將未排卵即連同卵巢取出來的鮭魚卵浸在醬油裡一段時間，</span></p>
<p><span>醃製出來的濃郁與新鮮卵液的爆漿稍微藉著醋飯來緩和令人至今難以忘懷。</span></p>
