  (花月嵐的極黑大蒜拳骨拉麵)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
--------------------------------------------------------------<br/>
　花月嵐拉麵西門店<br/>   地址：台北市萬華區漢中街52號1樓<br/>　電話：(02)23615358<br/>
--------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
　聽到之前在加州西門分館一樓原處開新門市的花月嵐，<br/>
<br/>
　在今年初重新推出季節限定款"極黑大蒜拳骨拉麵"，<br/>
<br/>
　這是一款一些熱愛美食的朋友曾提過的美味拉麵，並對此讚譽有加，<br/>
<br/>
　小弟抽了空，趁著中午空檔飆去西門町，<br/>
<br/>
　停好了車，看著剩餘時間不多，快步走至花月嵐西門店。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　進門後表示只有小弟單獨一位，沒想到竟開始遭到某男服務人員的刁難：<br/>
<br/>
　服務生態度不是很客氣，連小弟為了避免與他人併桌，所以希望將桌子分開，<br/>
<br/>
　一樣也遭拒絕，小弟也只好讓步，反正這是店家的自由，<br/>
<br/>
　但這一下子，花月嵐的印象分數馬上就扣到不及格，<br/>
<br/>
　拒絕也有拒絕的方式吧，又不是仇家，態度沒必要這麼討厭呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　(極黑大蒜拳骨拉麵大復活！！！！！！)<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(菜單與MENU點法)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　(桌上的調味料組)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(店內一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　依照預想的，點了極黑大蒜拳骨拉麵，要價190，<br/>
<br/>　加55點花套餐，有一杯任選飲料(小弟選Cola Zero)，<br/>
<br/>　以及一盤鮮脆沙拉，感覺上還滿划算的。<br/>
<br/>
<br/>
　<br/>
<br/>
　首先上桌的是鮮脆沙拉與可樂ZERO，<br/>
<br/>　擺盤漂亮的沙拉上有排列整齊的水煮蛋切片，薄小黃瓜片，<br/>
<br/>
　細切高麗菜絲與玉米粒，淋上特製沙拉醬，<br/>
<br/>
　整體來說生菜夠新鮮，口感爽口。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(附餐：可口的鮮脆沙拉！！！！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(可樂ZERO)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>　(一旁的筷子擺設真美呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　緊接而來就是重頭戲了：花月嵐當家的"極黑大蒜拳骨拉麵"！<br/>
<br/>
　美麗黝黑深邃而又充滿神秘感的色澤中飄出濃濃煎焙過的麻油香氣，<br/>
<br/>　淺嚐一口，真的，不是我蓋的，充滿深度與層次，<br/>
<br/>
　完整地填滿我的味蕾，讓我從閉目到回神足足花了半分鐘，<br/>
<br/>
　呃，突然覺得這樣形容好像有點過火了，<br/>
<br/>
　差點以為自己在形容的是高檔壽司。<br/>
<br/>
<br/>
<br/>
<br/>　接下來可要講些較難聽的嚕：<br/>
<br/>　其他配料部分表現得較為普通，尤其是麵條，<br/>
<br/>　吃起來與較粗的油麵沒什麼兩樣，沒有Q度，<br/>
<br/>　太可惜啦...　
 <br/>
<br/>
<br/>
<br/>
<br/>　但這美味湯頭實在是太厲害了，將整體分數提升到最高點，<br/>
<br/>　這樣一大碗真的足夠讓我HIGH整個下午的課了，<br/>
<br/>　美味就是可以如此簡單！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(重頭戲來嚕：極黑大蒜拳骨拉麵！！！！！)<br/>
