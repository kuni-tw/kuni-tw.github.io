　自從在某美食雜上看到介紹之後，就一直想親自去拜訪一次這一家冰店，<br/>
<br/>
　O-Sole-Mio冰淇淋通化店今年二月才開幕，是一家賣義式手工冰淇淋的專門店，<br/>
<br/>
　產品中人氣最高的莫過於他們的優格霜淇淋了，<br/>
<br/>
　標榜著以台東初鹿牧場的鮮奶為原料製成．<br/>
<br/>
<br/>
<br/>
<br/>
　可惜筆者當天大概是運氣不好，晚間工作甫完畢，<br/>
<br/>
　拖著老邁年高的身子來到時，才被告知優格霜淇淋均已賣完，<br/>
<br/>
　不過俗話說的好，既來之則安之，只見他們每日鮮烤的花籃餅多麼的誘人，<br/>
<br/>
　遂點了兩籃，後來又點了一籃，配上六種口味的冰淇淋，<br/>
<br/>
　一樣是吃得我直呼過癮，真是太好吃了．<br/>

<br/>
　拖著老邁年高的身子來到時，才被告知優格霜淇淋均已賣完，<br/>
<br/>
　不過俗話說的好，既來之則安之，只見他們每日鮮烤的花籃餅多麼的誘人，<br/>
<br/>
　遂點了兩籃，後來又點了一籃，配上六種口味的冰淇淋，<br/>
<br/>
　一樣是吃得我直呼過癮，真是太好吃了．<br/>
<br/>
<br/>
<br/>
<br/>
　這六種口味分別是：<br/>
<br/>
　1.苦甜巧克力---本日最愛，巧克力帶著淡淡的榛果味，<br/>
<br/>
　　推測是用70/100的巧克力作成的．<br/>
<br/>
　2.焦糖瑪琪朵---咖啡味有點太淡，要是能加強一點會更好，不過很順口．<br/>
<br/>
  3.牛奶糖---有種太妃糖的感覺，味道稍甜，表現很搶眼．<br/>
<br/>
  4.抹茶---個人相當喜歡，抹茶與牛奶味配合得剛剛好，真是值得一吃再吃．<br/>
<br/>
  5.青蘋果---有泡泡糖的感覺，很清爽．<br/>
<br/>
  6.義式巧克力---雖然是店員大力推薦，可是不大喜歡，<br/>
<br/>
　　沒什麼特色，感覺像是牛奶冰淇淋中參入巧克力碎片，味道平淡．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　雖然東西真的滿好吃的，尤其是冰品很爽口，吃多不膩，<br/>
<br/>
　重點是每一種口味都可以試吃之後再點．<br/>
　<br/>
　不過店員服務態度真的有待加強，有時候感覺愛理不理的，<br/>
<br/>
　這大概是唯一美中不足的地方吧...<br/>
