<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
  (慵懶的店貓)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  經咖啡界的朋友介紹，<br/>
<br/>
　認識了北市大安區雲和街51號立斐米緹咖啡(lapres-midi cafe)的老闆，<br/>
<br/>
　朋友在咖啡比賽使用的豆子也是由他幫忙準備的．<br/>
<br/>
　另外，由台北市政府商業處在公館新世界所主辦的2008精品咖啡大賞，<br/>
<br/>
　有14位咖啡好手展開激烈的咖啡競賽，並由莊宏彰代表立斐米緹咖啡館拔得頭籌，<br/>
<br/>
　獲得首屆台北咖啡殊榮．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(店內滿溢的咖啡熱情！)　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　位在師大夜市的巷子裡，店內外裝潢皆相當樸素低調，<br/>
<br/>
　整家店充滿著濃濃的咖啡香，幾位正在上網的顧客，<br/>
<br/>
　感覺是可以耗上一整天的好地方，點了杯冰濃縮與摩卡，<br/>
<br/>
　與健談的老闆相談之下，才得知他那一台相當稀有的經典咖啡機型號：<br/>
<br/>
　LaSanMarco 85型的活塞式咖啡機，操作上相當不易，<br/>
<br/>
　但是這些堅持卻完整的呈現在咖啡裡頭．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(咖啡機中的經典：LaSanMarco 85型的活塞式咖啡機)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不論是冰濃縮或是摩卡，香味非常濃厚，味道較厚，<br/>
<br/>
　推測為深度烘培，相當有個性，<br/>
<br/>
　搭配著店內滿溢的咖啡熱情細細品嘗，<br/>
<br/>
　都市間的煩躁漸漸煙消雲散...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(都市間的煩躁漸漸煙消雲散...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最後來一些滿推薦的咖啡廳延伸閱讀吧^^<br/>
