<br/>
  平常就是L'amour常客的我,跟老闆一樣都是熱情的音樂愛好者,<br/>
<br/>
  沒事還會一起討論音樂方面的話題.<br/>
<br/>
　今天中午抽空又去叨擾了一番,可不同以往點的都是三明治類,<br/>
<br/>
  這一次我可是留著空腹特地去品嘗他們有名的蛋糕呢.<br/>
<br/>
　當然配上平常愛喝的熱摩卡(上圖),感覺真是幸福.<br/>
<br/>
  蛋糕切片其一,櫻桃起司,<br/>
<br/>
  賞心悅目的鮮豔紅色調,美味又細緻的起司層與櫻桃香混合在一起,<br/>
<br/>
  入口不膩:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>

<br/>
<br/>
　其二,黑瑪莉,<br/>
<br/>
　上層的巧克力醬裡滿滿的果仁碎片,下層則是濃郁的巧克力蛋糕,<br/>
<br/>
  非常懷念以前在Vienna吃的Sacher Torte的我每一次只要遇到巧克力蛋糕,<br/>
<br/>
  都會不禁會仔細的嘗嘗;不過可從沒找到可以與其相提並論的濃醇巧克力味道,<br/>
<br/>
  這一次當然也不例外,但是爽口度還不錯,推薦^^<br/>
