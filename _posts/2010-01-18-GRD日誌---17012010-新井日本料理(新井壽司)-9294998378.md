<p><span> </span><br/> <br/><span> (新井的海膽壽司相當有水準)</span><br/> <br/> <br/><br/><span> ------------------------------------------------------------</span><br/><span> 新井壽司</span><br/><span> 地址：中和市保健路9號</span><br/><span> 電話：(02)29410010</span><br/><span> ------------------------------------------------------------</span><br/> <br/> <br/><span>經某網友推薦，就在丸居食酒屋附近，</span></p>
<p><span>有一家風評似乎不錯的日式料理店，新井壽司，</span></p>
<p><span>選在十七號中午上完課後過去吃頓午飯。</span></p>
<p><span>到了保建路路口發現這是一條很小的單向道，</span></p>
<p><span>乍看之下，倒比較像一般的小巷子，竟然會以"路"來命名，</span></p>
<p><span>還真是有趣極了。</span><br/><br/><br/><span> (位於中和市保健路的新井壽司)</span></p>
<p><span> </span></p>
<p><span>進了店門就座後，一位年紀較長的歐巴桑前來幫我們點餐，</span></p>
<p><span>詢問之下原來是老闆娘。</span></p>
<p><span>菜單上有不少琳瑯滿目的菜色，看樣子以後似乎可以多來幾次，大快朵頤一番，</span></p>
<p><span>不過今天就先只點咖哩豬排定食，要價一百九，</span></p>
<p><span>菜色內容包括：咖哩豬排，生菜沙拉，漬小魚乾，味增湯，白飯，</span></p>
<p><span>茶碗蒸，甜點。</span></p>
<p><span>點餐中老闆娘有推薦我多點一些其他菜色，不過我直說這樣就可以了，</span></p>
<p><span>老闆娘的口氣與表情馬上轉成不悅，跟之前的殷勤差很多，</span></p>
<p><span>然後她說不可以拍菜單，我說我只拍菜色就好，若不能拍那我就不吃了，</span></p>
<p><span>她才說那好吧。</span></p>
<p><span>與之後上來幫我服務的可愛小女生相比，老闆娘的臉色實在令人很反感，</span></p>
<p><span>那時候心想，若菜色沒什麼特別的表現，那這一次的可真是踩到大地雷了。</span></p>
<p> </p>
<p><span>個人向來覺得，品嚐美食，服務態度真的非常重要，管它好不好吃，</span></p>
<p><span>只要服務態度不對，個人就倒胃口，</span></p>
<p><span>而這老闆娘紮紮實實地踩到我的地雷區！</span></p>
<p><span>這部分可能深受以前熱愛吃的日式料理有關吧，</span></p>
<p><span>很久以前曾在歐洲一家便宜的日式料理店遇過一位日籍老師傅，</span></p>
<p><span>當地雖然沒什麼新鮮魚貨，但他那暖烘烘又貼心的服務，</span></p>
<p><span>配上一級的手藝，曾讓我愛上這深不可測的料理國度。</span><br/> <br/><span> 　</span><br/><span>沒多久後，可愛的女服務生端來了咖哩豬排定食，</span></p>
<p><span>外表上看起來就像一般的定食套餐，但是當第一口生菜沙拉進了胃道後，</span></p>
<p><span>一切開始變得不一樣了。</span></p>
<p><span>先從沙拉說起，沙拉最重要的莫過於生菜要新鮮，口感清脆，</span></p>
<p><span>然後是淋醬部分的配方，大部份的中低價位餐廳通常只能作到前者，</span></p>
<p><span>而新井的淋醬味道有層次感，將整盤沙拉的風味提升了許多，</span></p>
<p><span>裡頭的甜玉米塊更有畫龍點睛之妙，</span></p>
<p><span>類似在不少高檔日本料理店可以嚐到的北海道甜玉米作法，</span></p>
<p><span>竟然可以在190元的定食裡吃得到，傑克，這真是太神奇了呀。</span></p>
<p><span>其他的小菜表現一樣不俗，漬小魚乾口感濕潤，味道不單調，</span></p>
<p><span>味增湯味道濃郁鮮明，裡頭的小魚乾增加了不少咀嚼感，</span></p>
<p><span>茶碗蒸內物口感軟嫩。</span><br/> <br/> <br/><span>(生菜沙拉)</span></p>
<p><span> </span><br/><span> </span><br/> <br/> <br/> <br/><span>(漬小魚)</span></p>
<p><span> </span></p>
<p><br/> <br/><span>(味增湯與湯底的小魚)</span></p>
<p><span> </span><br/><span> </span></p>
<p><br/><span> (略乾的白飯上撒著芝麻)</span></p>
<p><span> </span><br/> <br/> <br/><span>(好味道的茶碗蒸)</span></p>
<p><span> </span><br/> <br/><br/><br/><span>主餐部分個人倒是有點小抱怨，豬排過乾，</span></p>
<p><span>裡頭幾乎完全沒有肉汁，不過肉體緊實，</span></p>
<p><span>看來是選用不差的部位去炸的。</span></p>
<p><span>配上了略為普通的咖哩醬，加上較乾的白飯，</span></p>
<p><span>個人不是很喜歡，還是上一次勝博殿好吃多了，</span></p>
<p><span>當然價位也有差，而且那裡的服務與環境氣氛也比較好呀。</span><br/> <br/> <br/> <br/><span>(炸豬排)</span></p>
<p><span> </span><br/> <br/> <br/><span>(淋上咖哩醬後)</span></p>
<p><span> </span><br/><span> </span><br/> <br/> <br/><span>(肉質頗乾硬的剖面圖)</span></p>
<p><span> </span><br/><span> </span><br/> <br/> <br/><span>吃到此時突然想看看他們的壽司表現，</span></p>
<p><span>趁著上一次在游壽司品嚐的海膽壽司至今記憶猶新，</span></p>
<p><span>便加點了一份海膽壽司：</span></p>
<p><span>真不敢相信，能在這裡嚐到具有相當水準的海膽壽司，</span></p>
<p><span>與游壽司比起來，外型上與技術性皆不輸，</span></p>
<p><span>但海膽色澤略暗沉一點，而且不知道是不是山葵的優劣關係，</span></p>
<p><span>躲在壽司裡的山葵相當嗆，蓋掉海膽不少風味，</span></p>
<p><span>有點可惜。</span><br/> <br/> <br/><span>(海膽壽司隨拍)</span></p>
<p><span> </span><br/><span> </span><br/><span> </span><br/> <br/> <br/><span>最後登場的甜點，紅豆湯，則畫下了完美的句點。</span></p>
<p><span>香甜的紅豆湯上灑著些許桂花，散出淡雅清香，</span></p>
<p><span>絕妙之處可不只如此，淺嚐一口後，</span></p>
<p><span>那甜味，我不禁嚇了一跳，絕對不是一般的砂糖，</span></p>
<p><span>特地走去大廚那裡(老闆娘的兒子)，詢問之下，</span></p>
<p><span>果然糖是選過的，是使用紅糖，糖放的量剛剛好，</span></p>
<p><span>配合著較稀的的紅豆粒與桂花香，</span></p>
<p><span>呈現出一首美妙的室內小品，真是漂亮。</span></p>
<p> </p>
<p><span>(散出淡雅桂花香的紅豆湯)</span></p>
