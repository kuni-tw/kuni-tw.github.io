----------------------------------------------------<br/>
   京華城地下美食街的京廚<br/>
   地址：台北市松山區八德路四段138號B2<br/>
----------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  八號下午至京華城美食街尋覓中餐地點，<br/>
<br/>
　許久未來這裡果腹了，繞了整個美食廣場，<br/>
<br/>
  最後看到京廚的一鍋一燒感覺相當誘人，<br/>
<br/>
　便點了豬排一鍋一燒套餐。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (京廚)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(京華城地下美食街)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　隔了一段不短的時間後終於領到套餐，<br/>
<br/>
　這倒是滿令人狐疑的，明明人就不是很多，<br/>
<br/>
　怎麼上菜速度竟會如此地慢，假日人潮時那還得了！<br/>
<br/>
<br/>
<br/>
<br/>
　為了保持鍋物溫度，鍋底下有盆火，應是使用固態燃料。<br/>
<br/>
　鐵盤上有胡椒豬排，油豆腐，海帶，筍絲與半熟荷包蛋等等，<br/>
<br/>
　趁著熱吃頗具滋味，另外豬排令人意外地相當軟嫩。<br/>
<br/>
　吃完燒物後喝點熱騰騰的湯，鍋料普通，口味屬清淡，<br/>
<br/>
　但海鮮湯頭還不錯。<br/>
<br/>
　總之，台幣一百多能在百貨的美食街吃到這樣子的搭配，<br/>
<br/>
　還滿划算的。<br/>
