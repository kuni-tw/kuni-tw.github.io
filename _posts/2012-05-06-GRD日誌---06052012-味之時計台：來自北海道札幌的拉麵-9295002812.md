味之時計台：北海道札幌拉麵(2012南西日本祭)<br/>
地址：台北市中山區南京西路12號9F<br/>
時間：Mon~Fri 11:00~21:30 Sat~Sun 11:00~22:00<br/>
------------------------------------------------------------</span></p>
<span>
<p><br/>
<br/>
<br/>
<br/>
　(南西日本祭)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255972.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
<br/>
　感謝上一次網友在小弟的一風堂拉麵文章裡拋磚引玉，</p>
<p>　總算還來得及趕在南西新光三越日本祭最後一天吃到味之時計台，</p>
<p>　一家來自北海道札幌的拉麵。</p>
<p><br/>
<br/>
<br/>
　(來自北海道札幌的味之時計台)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255973.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(師傅皆來自日本)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255974.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
<br/>
</span><br/>
　好不容易衝到九樓的日本祭只見人滿為患，</p>
<p>　在諸多攤位不停地穿梭中找了半天總算找到味之時計台，</p>
<p>　與前段日子堪稱瘋狂的日式拉麵排隊人潮相比今晚顯然冷清許多，</p>
<p>　點了一碗暫定價NT.250的味噌拉麵，當然沒收服務費，</p>
<p>　另外再從其他攤位買來蝦子一口天(NT.100)與烤干貝串(NT.150)。<br/>
<br/>
<br/>
<br/>
　(別家攤位的蝦子一口天與烤干貝串)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255975.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　前者味道不差但裡頭蝦子迷你小隻且數量稀少，</p>
<p>　後者干貝口感極軟尺寸巨大吃起來超級過癮。</p>
<p><br/>
<br/>
<br/>
　(烤干貝串口感超軟嫩尺寸巨大咬起來很過癮)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255976.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span><br/>
  　<br/>
　終於期待已久的味之時計台拉麵上桌了，</p>
<p>　淺嚐一小口微微感到詫異，湯頭濃郁度中等散出類似山頭火的海鹽鹹味，</p>
<p>　整體味道雖不會過鹹但卻少了層次與個性，叉燒口感不差使用的是台灣豬肉，</p>
<p>　麵條則以日本進口的麵粉全程在台製作，尺寸偏粗少了些口感，</p>
<p>　整體水準與前陣子才剛吃過的緣麵屋，山頭火，一風堂相比有一小段差距，</p>
<p>　推測是準備還不夠周詳吧。</p>
<p><br/>
<br/>
　(味之時計台的味噌拉麵)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255977.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(使用台灣本地豬製作的日式叉燒)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=34&amp;f=1089255978.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span></span></p>
<p>　與幕後老闆夫婦聊了一會兒已確定要引進台灣，</p>
<p>　但目前還在談合約與找尋地點當中。<br/>
