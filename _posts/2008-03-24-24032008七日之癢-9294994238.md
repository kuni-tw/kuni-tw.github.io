　最近閒著沒事就租些片子,看了看也沒留下多大印象,<br/>
<br/>
  直到昨晚的"七日之癢".<br/>
<br/>
<br/>
　半夜醒來臨時抽了一張就送進電腦放著看,<br/>
<br/>
  看到途中時鼻子突然莫名的就酸起來,想了想,<br/>
<br/>
  原來是不經意的想起一些令人難忘的回憶了.<br/>
<br/>
  那是某年不知名的暑假,正值盛開的年齡,<br/>
<br/>
  動不動與好友稱兄道弟,每晚豪氣干雲牛飲卻面不改色.<br/>
<br/>
  真不知道該說是當時太過放縱還是血氣方盛,<br/>
<br/>
  左擁右抱就不說了,當時真可算得上是打敗群雄得到了她.<br/>
<br/>
  只萬萬沒料到命運可以如此的捉弄人,<br/>
<br/>
　沒多久之後又遇到了她:這還真難形容,或說就是從那種一見鍾情的感覺吧,<br/>
<br/>
　甚至這也還不足以形容.<br/>
<br/>
  更貼切一點,這有如教父裡當麥可第一次見到阿波羅妮亞時那種被霹靂擊中的感覺吧.<br/>
<br/>
  兩人霎時就忘了時光,盡情的暢談直到曙光逐漸開始啃食街道上的黑暗.<br/>
<br/>
  細節就不多說了,徒增一些不必要的傷感罷了.<br/>
<br/>
  就如同男主角後來對女主角所說的:<br/>
<br/>
  "我不知道上天為什麼要捉弄我,讓我早一個星期結婚,<br/>
<br/>
  但是我不後悔,因為那樣我才去卡波,我才遇上妳,我才愛上妳,<br/>
<br/>
  對此我永遠不會道歉..."<br/>
<br/>
  看到此時我真的費了好大功夫才這十幾年瀕臨絕種的淚水止住,<br/>
<br/>
  可不是?<br/>
<br/>
  我能怪誰,怪命運嗎?<br/>
<br/>
<br/>
<br/>
  傍晚,帶著晚霞的祝福與學生的告別,<br/>
<br/>
  我盡興喫著晚餐...<br/>
<br/>
<br/>
<br/>
