　點了店裡的招牌力作：老兵霸王麵(滿漢全席)<br/>
　<br/>
　菜單上印有小與大之差別，差在麵量：小份NT.250，大份NT.270。<br/>
    <br/>
　但服務生告知說現在只供應大份的，加可續麵一次。<br/>
    <br/>
　不過別擔心，一樣只算小份的價錢：NT.250。<br/>
    <br/>
　但以個人號稱大胃王的食量來看，<br/>
    <br/>
　一般人要續的機會應該不多吧(笑)。<br/>

　沒多久後服務生端來一個大碗公(或稱三號大的花盆？)　　<br/>
    <br/>
　望裡頭一瞧不禁倒呼了一口氣，這根本是男人的夢想呀：<br/>
    <br/>
　有如一大片汪洋的牛肉湯海，當中滿滿的牛肉牛筋與牛雜們載浮載沉著。<br/>
    <br/>
　湯頭是個人較不愛的清燉口味，不過卻無損於這一碗麵的男人味。<br/>
    <br/>
　麵條不過軟且微彈牙帶勁，湯頭濃郁適中，偏鹹些但我喜歡。<br/>
    <br/>
　牛肉部分使用的是澳洲腱子肉，肉質佳，熟度拿捏的剛剛好，<br/>
    <br/>
　牛筋與其餘牛肚牛腸牛等煮得也夠透夠軟嫩，<br/>
    <br/>
　重點是主料量多到令人吃完後有種很撐的感覺，<br/>
    <br/>
　卻仍舊深感意猶未盡，實真乃男人之福也。<br/>
