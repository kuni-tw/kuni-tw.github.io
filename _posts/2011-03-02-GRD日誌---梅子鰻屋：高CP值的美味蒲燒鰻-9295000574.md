<p><span>(超級好吃的梅子鰻屋)</span><br/> <br/><br/> <br/><span> ----------------------------------------------------------</span><br/><span>地址：台北市林森北路107巷8號</span><br/><span>電話：(02) 2521-3200; 2521-2813</span><br/><span>時間：週一～週六：11:30~14:00 17:30~23:00</span><br/><span>週日：11:30~14:00 17:30~23:30</span><br/><span> ----------------------------------------------------------</span><br/><br/> <br/><span>自從上次在京都屋超級不愉快的經驗之後，</span></p>
<p><span>有好一陣子完全不想吃鰻魚類的料理，甚至歸類為拒絕往來戶。</span></p>
<p><span>沒辦法，那次的體驗實在是太震撼人心了。</span></p>
<div><span><a href="http://kuni.pixnet.net/blog/post/294999815-grd%E6%97%A5%E8%AA%8C---092010-%E4%BA%AC%E9%83%BD%E5%B1%8B%E7%9A%84%E9%B0%BB%E9%AD%9A%E6%96%99%E7%90%86(%E6%9C%89%E5%A4%A0%E5%B7%AE%E7%9A%84%E6%9C%8D" target="_blank">(詳情請用滑鼠點選這裡：服務態度不佳的京都屋)</a></span></div>
<p><span>不過人終究還是屬於好奇心強的類種</span></p>
<p><span>某一次聽到較懂吃的朋友強烈推薦梅子鰻屋，</span></p>
<p><span>潛意識裡就不禁開始有了"給個機會去試試看吧"的念頭。</span><br/><span>   </span><br/><span>找了天晚上臨時過去，位在六條通的梅子鰻屋相當好找，</span></p>
<p><span>停好了車便走進店門口，店內不大走的是溫馨的日式居酒風，</span></p>
<p><span>客人絡繹不絕。</span><br/><br/><span>(店內人口絡繹不絕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160829.jpg" target="_blank"></a><br/></span><br/> <br/><span>(桌上隨拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160830.jpg" target="_blank"></a></span><br/><br/><br/><span>(隨之送上來的茶水)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160831.jpg" target="_blank"></a></span><br/> <br/><br/><span>(店內一景)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160832.jpg" target="_blank"></a></span><br/><br/> <br/><span>因為不大餓，先點了一份大的鰻魚飯，NT.260</span></p>
<p><span>還有鰻肝，一串NT.30，沒多久後餐點便紛紛上足。</span></p>
<p><span>說真的那晚堪稱是美味無窮幸福滿溢呀，只可惜胃不夠大，</span></p>
<p><span>當下便暗下決心："一定要再過來吃吃看其他東西！"</span></p>
<p><span>幾天後不爭氣的肚子真的再度將疲累的小弟拖來，</span></p>
<p><span>這一次改點小份的鰻魚飯，NT.140，以便留胃口好加點其他東西。</span></p>
<p><span>超好吃的鰻肝這次竟然中午就賣光只好屈就些改點烤雞肉串，</span></p>
<p><span>店家還規定一次得點兩串，所以是NT.60</span></p>
<p><span>還有貴俗俗的炸明蝦，菜單上也規定得點兩隻，NT.320，</span></p>
<p><span>服務大哥當場告知剛漲價變NT.500，但還是忍痛點了下去，</span></p>
<p><span>心下當場淌血差點被送進醫院。</span></p>
<p><span>最後再加點一份蛤仔味噌湯，NT.90，因看每桌幾乎都有點，</span></p>
<p><span>不禁就嘴饞了起來。</span></p>
<p><span>首先上桌的是炸明蝦，比想像中來得好吃，</span></p>
<p><span>炸衣雖然偏油，但裡頭的蝦肉彈性足富肉汁，很好吃，</span></p>
<p><span>不過這價位真的有點太高了(血繼續流不停)</span><br/><span>     </span><br/><br/><span>(梅子屋的炸明蝦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160833.jpg" target="_blank"></a><br/> <br/>(盤裡的生菜)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160834.jpg" target="_blank"></a><br/> <br/><br/>(蝦頭)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160835.jpg" target="_blank"></a><br/><br/>(很棒的肉質)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160836.jpg" target="_blank"></a><br/><br/>(呀，看這濃郁的蝦膏呀)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160837.jpg" target="_blank"></a></span><br/><span> 　</span><br/><span>然後是小份的鰻魚飯，跟上次大份的比起來整整小了好幾號，</span></p>
<p><span>以價錢來看很顯然地大份較划算且吃起來過癮度也有差。</span></p>
<p><span>梅子的好吃鰻魚特點最主要有兩個地方，</span></p>
<p><span>其一，山椒粉成功地帶出鰻魚的香氣，有畫龍點睛的效果。</span></p>
<p><span>其二，鰻魚肉質夠肥厚，有著令人無法抗拒的好口感。</span></p>
<p><span>總而言之就是太好吃了，完全合小弟的胃口。</span><br/> <br/><span> </span><br/><span>(梅子鰻屋的小份鰻魚飯登場)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160838.jpg" target="_blank"></a><br/><br/>(令人流口水的一幕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160839.jpg" target="_blank"></a><br/><br/>(喔喔喔，實在是太好吃了)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160840.jpg" target="_blank"></a><br/><br/>(這個才叫做肥滋滋呀)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160841.jpg" target="_blank"></a><br/><br/>(上一次點的大份鰻魚飯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160842.jpg" target="_blank"></a><br/><br/>(說實在的，點大份真的比較過癮)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160843.jpg" target="_blank"></a><br/> <br/></span><br/><span>接下來的蛤仔味噌湯也令人相當驚艷，</span></p>
<p><span>蛤蠣尺寸很大且有兩隻，肉質又厚又軟，</span></p>
<p><span>在味噌與蛤蠣兩種甜味的搭配下有著一加一大於二的效果，</span></p>
<p><span>難怪隔壁桌有那麼多人點。</span><br/> <br/><br/><span>(蛤仔味噌湯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363170464.jpg" target="_blank"></a><br/> <br/></span><br/><span>烤雞肉串雖不難吃且醬汁也夠入味，</span></p>
<p><span>不過在前面幾道美味絕佳的表現下相形失色許多，</span></p>
<p><span>與上次柔軟至極的鰻肝串更是沒得比。</span><br/><br/><span>(烤雞肉串)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363170465.jpg" target="_blank"></a><br/> <br/>(經典的鰻肝)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363170466.jpg" target="_blank"></a></span><br/> <br/><br/><span>此時意猶未盡便想試試他們的生食，</span></p>
<p><span>點了一份鮭魚握壽司，一份兩貫NT.100</span></p>
<p><span>沒想到竟比想像中來得好，魚料肥美新鮮度夠，舍利鬆度與握工皆到位，</span></p>
<p><span>真是令人驚訝。</span><br/><br/><span>(漂亮的鮭魚握)</span></p>
