<br/>
<br/>
　使用GRD2好一段時間後，看到其他各大廠馬不停蹄地推陳出新，<br/>
<br/>
　Canon當家的G系列至今也出到G11，<br/>
<br/>
　LX3的出現更是造成3C界一股不小的風潮，<br/>
<br/>
　就在我納悶了好一段時間之後，Ricoh終於發聲了：<br/>
<br/>
　GRD3終於登場迎戰了。<br/>
<br/>
　<br/>
　果不其然，驚異的規格升級：<br/>
<br/>
　光圈從2.4進化到1.9<br/>
<br/>
　SNAP有更多選擇，更細緻的九宮格，更大的螢幕，<br/>
<br/>
　還有更漂亮的高ISO表現，等等，說都說不完。<br/>
<br/>
<br/>
　不過在網路上一堆又一堆的GRD3作品分享紛紛出籠後，<br/>
<br/>
  個人心中的那一團大火，瞬地就被澆熄了。<br/>
<br/>
<br/>
<br/>
<br/>
　一直以來，很喜歡一代那高反差相當漂亮的冷色調，<br/>
<br/>
　不過當二代出現後，二代的暖色調與更優異的操控性，<br/>
<br/>
　反而更貼近我的需求。<br/>
<br/>
<br/>
<br/>
<br/>
　因為個人最重視的就是DC的操控感，<br/>
<br/>
　其次才是畫質與成色表現。<br/>
<br/>
<br/>
<br/>
　當三代出現後，所有的毒照紛紛對我指向一個方向，<br/>
<br/>
　那就是非常精準的自動白平衡。<br/>
<br/>
　突然間，那獨特的GRD色調便消失了，<br/>
<br/>
　從三代的照片分享當中，似乎很難找到GRD那獨特的風格所在。<br/>
<br/>
　原本GRD系列的數位感相對於其他DC比起來就弱許多了，<br/>
<br/>
　但三代不可諱言的又更進一步，<br/>
<br/>
　多了更多底片感(這還真不知道該怎麼解釋)<br/>
<br/>
　<br/>
<br/>
<br/>
　上一次在BEST測試過GRD3的實機後，<br/>
<br/>
　不禁被他的優異操控性所迷住，<br/>
<br/>
　但因至今與GRD2的合作相當愉快，<br/>
<br/>
　加上個人沒那麼喜歡網路上GRD3作品分享的成色，<br/>
<br/>
　對於兩萬多的售價便實在是下不了手，<br/>
<br/>
　這不是價格多寡，而是C/P值的問題。<br/>
<br/>
<br/>
<br/>
<br/>
　有幸朋友出國前借了我一台GRD3，<br/>
<br/>
  便在二十六號下午帶著兩台兄弟機子，<br/>
<br/>
　找了間星巴克門市來測試個人最常作的：拍攝美食。<br/>
<br/>
<br/>
<br/>

　出門前先在機械車位前測試一下高ISO表現，<br/>
<br/>
　不過，兩機子相差實在是太大了，<br/>
<br/>
　刻意使用GRD2的ISO800對抗GRD3，<br/>
<br/>
　我事前就知道三代的高ISO表現已進步許多了，<br/>
<br/>
　但卻沒想到可以差到那麼多，GRD3的高ISO顆粒感表現好太多了，<br/>
<br/>
　加上1.9大光圈的加持，GRD2相較之下，表現實在是慘不忍睹，<br/>
<br/>
　也罷，待會就用GRD2最擅長的項目來回擊吧。<br/>

　結果...<br/>
<br/>
  在個人最常使用的自動白平衡上，GRD2完全沒有佔到多大便宜，<br/>
<br/>
　原本就是一把鋒利小刀的GRD2在GRD3面前似乎完全招架不住，<br/>
<br/>
  另外，三代的自動對焦又準又敏銳，比如在二代較不好對焦的鮮奶油上，<br/>
<br/>
　三代幾乎是游刃有餘。<br/>
<br/>
　三代更大的大光圈給了創作者更多的快門空間與題材自由度，<br/>
<br/>
　GRD3就像是一把更為鋒利的小刀，只是端看人怎麼使用而已。<br/>
