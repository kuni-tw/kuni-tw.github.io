<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
  在眾友圍繞在幸福裡的我,趁著酒意正濃揮灑白毫,將昭告天下的情懷寫下.<br/>
<br/>
  這些日子以來就像劃過天際的流星一般逝去,光芒耀眼不留白,<br/>
<br/>
  只見得精華濃縮在三百六十五只小瓶子裡頭,我數著數著,<br/>
<br/>
  不禁興致大發,再斟一盅黃河之水,管他天上地下來.<br/>
<br/>
<br/>
<br/>
  心裡滿滿,沉甸甸地找尋被遺忘的過去不被遺忘的未來.<br/>
<br/>
  現在可以成為未來,亦也可以是過去的種籽,只要不被蹧蹋就好.<br/>
<br/>
  手裡拿著忠實的目擊者紀錄著一剎那的靜止畫面,把能力所及所無法捕捉的,<br/>
<br/>
  一股腦兒掃進永恆的寶箱.<br/>
<br/>
<br/>
  就這樣,恭喜我又多了三百六十五只晶瑩剔透的小瓶子吧,<br/>
<br/>
  收藏品絡繹不絕呢!<br/>
<br/>
