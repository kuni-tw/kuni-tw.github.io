<p><br/><span>     </span><br/><span> ------------------------------------------------------------</span><br/><span>Bistro Le Pont 樂朋小館</span><br/><span>地址：台北市潮州街176號</span><br/><span>電話：(02)23965677</span><br/><span> ------------------------------------------------------------</span><br/> <br/><span>這是一家很特別又極為少見的餐廳，本質上以台式鵝肉料理為主，</span></p>
<p><span>但卻同時融法國料理的精神。</span></p>
<p><span>自之前在名傳極佳的鵝肉經驗開始，每逢看到鵝肉兩字，</span></p>
<p><span>精神就會抖擻起來。</span><br/> <br/> <br/> <br/><span>前陣子耳聞一家風評不錯的鵝肉專賣店，名叫樂朋小館(Bistro Le Pont)。</span></p>
<p><span>高雄知名的橋邊鵝肉店第二代曾留學過法國，</span></p>
<p><span>回國後嘗試在台北開一間融進法式料理精神的台式鵝肉店，</span></p>
<p><span>走少數的法國風。</span></p>
<p><span>小弟當然不肯放過機會，</span></p>
<p><span>第二天傍晚趁教課空檔直接飛奔至永康街附近。</span></p>
<p><span>停好車走到店門前時愣住：低調的深木色店外裝潢，</span></p>
<p><span>端的是再熟悉不過的歐式小館氣質，太棒了。</span><br/><br/><span>(樂朋小館)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178547.jpg" target="_blank"></a></span><br/> <br/> <br/><span>走進店內，純樸微暖的懷舊風是這裡的主題，</span></p>
<p><span>就座後先簡略讀菜單，然後一共點了：</span></p>
<p><span>橋邊鵝香飯豪華版，NT.50 or €1.2</span></p>
<p><span>橋邊鹹水鵝肉，NT.250 or €5.8</span></p>
<p> </p>
<p><span>橋邊鵝香米血，NT.60 or €1.4</span></p>
<p><span>川味紅油抄手，NT.80 or €1.8</span></p>
<p><span>西洋芹拌鵝腸，NT.120 or €2.8</span></p>
<p><span>傳統鵝香筍湯，NT.50 or €1.2</span><br/><br/><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178548.jpg" target="_blank"></a><br/> <br/>(點完菜後四處走走拍拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178549.jpg" target="_blank"></a><br/></span></p>
<p> </p>
<p><span>(店家自製自賣的黃金鵝油蔥酥)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178550.jpg" target="_blank"></a><br/> <br/></span><span>以傳統台式鵝肉來看，這裡的價位偏高。</span></p>
<p><span>橋邊鵝香飯豪華版與橋邊鹹水鵝肉首先上桌。</span></p>
<p><span>豪華與正常版最大的不同在於前者多了聖傑克醬，後者無。</span></p>
<p><span>這裡解釋一下：所謂聖傑克醬是指樂朋自製的黃金鵝油北海道干貝XO醬。</span></p>
<p><span>樂朋的鵝香飯非常好吃，鵝油拌在白飯內同時增添香氣與口感。</span></p>
<p> </p>
<p><span>(鵝香飯豪華版與鹹水鵝肉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178551.jpg" target="_blank"></a><br/> <br/> <br/>(鵝香飯光看起來很可口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178552.jpg" target="_blank"></a><br/> <br/> <br/></span><span>鵝肉部分肉質緊實微嫩有嚼勁同時<span>外皮</span>肥厚多脂，</span></p>
<p><span>份量足且相當大一盤，令人驚豔。</span></p>
<p><span>樂朋是極少數或者說是唯一加入養鵝協會的餐廳，</span></p>
<p><span>不知這是否與優質鵝肉有關。</span><br/> <br/> <br/><span>(肥嫩鹹水鵝肉切盤)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178553.jpg" target="_blank"></a><br/></span></p>
<p><span>鵝血糕則是店內當天自己拌米蒸好，</span></p>
<p><span>有著一般店家吃不到的嫩Ｑ嚼勁。</span><br/> <br/><br/><span>(美味的鵝血糕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178554.jpg" target="_blank"></a><br/> <br/> <br/></span><span>紅油抄手相當特別，入喉過幾秒鐘才慢慢感到微微麻辣，</span></p>
<p><span>另外辣油裡混的醋也比一般吃到的要多些，酸味明顯。</span><br/> </p>
<p><span>(川味紅油抄手)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178555.jpg" target="_blank"></a><br/> <br/> <br/></span><span>西洋芹拌鵝腸與前面菜色相比下較不突出，</span></p>
<p><span>鵝腸口感嫩配西洋芹的脆可惜表現尚可。</span><br/> <br/> <br/><span>(西洋芹拌鵝腸)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363178556.jpg" target="_blank"></a></span><br/> <br/><span><br/></span><span>鵝香筍湯部分雖然特地使用來自高雄六龜的筍，不過口感上並無特別之處，</span></p>
<p><span>是樂朋唯一兩道較令人失望的料理。</span><br/> <br/> <br/><span>(傳統鵝香筍湯)</span></p>
