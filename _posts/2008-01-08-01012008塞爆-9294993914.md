<br/>
<br/>
<br/>
<br/>
　看了場想看的,選在最不會塞的地域,本想外帶一杯幸福的聖誕入場,<br/>
<br/>
  後來在店裡赫然發現,排隊要花比台北新竹還要更長的時間,<br/>
<br/>
  馬上落跑:算了,減減肥喝一杯Zero就好.<br/>
<br/>
<br/>
  內容沒什麼好說的,事實上還滿失望的,<br/>
<br/>
　尤其是那種一定要犧牲某人來成就內定的造英雄情節;很明顯地,<br/>
<br/>
  作者本身並不太懂大場面氣氛的營造,擅自以自以為又殘破不堪的邏輯,<br/>
<br/>
  連結事先設定的要素,中間我不禁想籌錢徵一些新意,所幸沒人理我,<br/>
<br/>
  因為漏洞過多所以我預算不夠.<br/>
<br/>
<br/>
  回程時事與願違,某段路塞了我足足十五分鐘,雖然有著草地人行道石板地,<br/>
<br/>
　以及越野的優勢存在,加上十足的鑽功,不然大概會多花上一炷香吧.<br/>
<br/>
　趁著水溫飆高,轉速始終破不了三千之下,<br/>
<br/>
  就看著某遊覽車從後照鏡裡持續縮小,好像從沒離開過原來的位置一般.<br/>
<br/>
  此時的我千不該萬不該的竊笑,幸好AGV的Hi-tech款防霧功能一流:<br/>
<br/>
  鄭重感謝AGV以及我的荷包.<br/>
<br/>
<br/>
  與父母親大人道了安,幸福的揮一揮手,揮別了07年的眷顧,<br/>
<br/>
  電話裡稍微有一點擔心,不過失戀者應該會好好盡自己的本分吧?<br/>
<br/>
  這是個甜蜜的夜晚呢.<br/>
<br/>
<br/>
　接下來就是為著青春洋溢的春天要大傷腦筋了;<br/>
<br/>
　不安分又急著想長大的袋子還是能力所及,<br/>
<br/>
  但懶得動一公分的手腳,怎麼踹都沒用,這叫我如何是好?<br/>
<br/>
  下一次用針刺吧,不過肉厚肯定大失所望,還是學著提琴手說吧:<br/>
<br/>
  傳統,傳統,這就是傳統!<br/>
<br/>
