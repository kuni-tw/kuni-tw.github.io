<br/>
<br/>
　<br/>
  測試到今天稍微有些眉目了，漸漸地從眾人眼中吃力不討好的工作裡，<br/>
<br/>
　我得到了許多收穫．<br/>
<br/>
　確切的結論依舊懸著，如同我忐忑不安的心情一般，<br/>
　<br/>
　在深不知處的雲層裡找尋那一絲絲的線索，引向飄邈的未來．<br/>
<br/>
<br/>
　<br/>
　這樣得不到片刻安寧的緊張感，在我的生命當中似乎會永無止境的出現，<br/>
<br/>
　還記得考首席的那個週末嗎，譜上出現了成群未知的符號，<br/>
<br/>
　此起彼落的重疊，礙著面子硬生生的止住了顫抖，最後我大氣的下了台，<br/>
<br/>
　在漫長的等待時間裡靜候法官的敲槌．<br/>
<br/>
<br/>
<br/>
　後來也不知道從什麼時候開始，我開始無懼於這種時刻，<br/>
<br/>
　就算天塌了下來也無妨，反正除死無大事，我樂天了起來．<br/>
<br/>
　記得也曾在冰天雪地裡孤零零地躺在冷颼颼的地窖裡，心裡卻異常的平靜，<br/>
<br/>
　某日在車站底下為了愛情不支倒地，然後因為見證友情而變得勇敢，<br/>
<br/>
　然後我就免疫了．<br/>
<br/>
<br/>
<br/>
　什麼時候開始我以恐懼作為我的精神食糧呢？　<br/>
<br/>
<br/>
