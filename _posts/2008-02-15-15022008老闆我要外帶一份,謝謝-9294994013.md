<br/>
<br/>
  前陣子因為入手GRD,整個人像似喝醉酒一般,兩個禮拜以來瘋瘋癲癲...<br/>
<br/>
  說到這才想到,這一輩子我根本還沒有喝醉過,<br/>
<br/>
  所以根本無權使用喝醉這個詞呀.<br/>
<br/>
  不過話說回來,記得曾看過某學術報導,<br/>
<br/>
  指出人剛睡醒十分鐘之內跟喝醉酒其實沒兩樣,<br/>
<br/>
  咦,既然如此,其實我還是有間接資格的呢!<br/>
<br/>
　<br/>
　愈這樣徹底的實行漫步,愈感到生命中多了那一份音符的悸動,<br/>
<br/>
  愈接近大祕寶,愈了解深處的願望.<br/>
<br/>
  回到家衣服沒換衝到電腦前,打開檔案,細細品嘗整天下來的收穫,<br/>
<br/>
  一件件的仔細觀察,回想以及檢討圖資.<br/>
<br/>
  回想起那種半夜看到黑漆漆的雨水就撐著雨傘出門的孩子氣,<br/>
<br/>
　在細雨與冷風中換取無比的幸福,<br/>
<br/>
  這就是我要外帶的一份美滿呀:<br/>
<br/>
<br/>
  "老闆,請幫我打包,謝謝!"<br/>
<br/>
