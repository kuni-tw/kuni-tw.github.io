<div class="article-content-inner" id="article-content-inner">
　記得某天在高速公路上的休息站小憩了一下時,突然聽到有人喊我的名字,<br/>
<br/>
  身後冷不防地被敲了幾下肩膀,我回首一看不禁呆了.<br/>
<br/>
  一位壯年男子正瞪著我,這可不是我小學時代的班導師嗎!?<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  "阿...老師好!"<br/>
<br/>
  "不錯嘛,你竟然還記得我."<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這可奇了,老師竟然認得出我,將近十年沒見過面的師生就這樣,<br/>
<br/>
  在最沒有氣氛的高速公路上偶遇相認,整齣充滿著肥皂式劇情的張力.<br/>
<br/>
  老師說我一點兒都沒變,我卻完全感受不出來.<br/>
<br/>
  三十公分的身高差別姑且不論,光是人生當中變化最劇烈的青春期前後的易容術,<br/>
<br/>
  就可以讓我輕鬆的躲開通緝然後逍遙法外個幾十年,<br/>
<br/>
　更遑論那時候才十八九歲的我常常被冠上少年老成的評價了.<br/>
<br/>
<br/>
  一直到後來,每當我又想起這段往事時,我就會自問:"我到底有沒有變呢?"<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這或許是一種悲傷的想法,對於漸漸要拋棄大葛隔而去迎接薯叔的標籤,<br/>
<br/>
  似乎我還強裝著不知道,因為我無法接受我持續變老的事實.<br/>
<br/>
  我承認,若今日之我跟十年前之我是敵人的話,那後者只是個待捕的獵物罷了.<br/>
<br/>
　十年前曾夢想過的,追求過的,今天不過是例行公事般的無趣.<br/>
<br/>
  而我極度排斥這種變化,卻無力抗拒.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　每當此類思潮來臨時,我便需要奔馳在沙場上的快感,以換取過往雲煙中的折磨,<br/>
<br/>
  然後我又重新再次獲得新生.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
