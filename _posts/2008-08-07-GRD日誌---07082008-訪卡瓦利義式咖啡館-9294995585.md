<br/>
　甫看完電影，散場之後遂與朋友漫步在永康街，<br/>
<br/>
　經友人推薦到鄰近的卡瓦利義式咖啡館，據說他們自家烘的咖啡品質有一定水準，<br/>
<br/>
　便好奇的點了幾杯義式咖啡與一杯冰濃縮．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　店內裝潢典雅，座位寬敞，走古典歐風路線．<br/>
<br/>
　吧檯前坐了幾位聊天熱絡的熟客．<br/>
<br/>
　等了許久，咖啡終於送來，這也讓我吃了一驚，<br/>
<br/>
　喝過那麼多家咖啡館，還沒碰過要等那麼久的咖啡，<br/>
<br/>
　光是後來加點的單杯咖啡也一樣，這大概是唯一令人較不愉快的地方吧．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

  咖啡部分：他們的義式咖啡中規中矩，與一般咖啡廳沒差多少，<br/>
<br/>
　不過冰濃縮表現的可圈可點，該怎麼形容呢，<br/>
<br/>
　共行的友人與我一致覺得，苦與酸表現得非常平衡，<br/>
　<br/>
　我們都滿喜歡的呢．<br/>
