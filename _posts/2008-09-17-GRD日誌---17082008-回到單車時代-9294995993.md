<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  十六號，睽違了好幾年，我又重新買了一輛單車，<br/>
<br/>
　所不同的是，這一次完全地使用自己的辛苦錢．<br/>
<br/>
  當晚依照慣例滿懷熱情地騎著它到處溜搭，這樣一直持續到半夜，<br/>
<br/>
　先騎上烘爐地，再繞中永和，與友在咖啡廳小憩一會兒，<br/>
<br/>
　然後又繞了四號公園好幾圈．<br/>
<br/>
<br/>
<br/>
　我又回到當初，在烈陽下多腦河畔高喊著：Jetzt kommt die Bergwertung！<br/>
<br/>
　幾十台車就這樣，激烈著，先後互換領先位置，<br/>
<br/>
　輪胎在高溫與強力摩擦之下，變得愈來愈黏，<br/>
<br/>
　小石子與沙子噴灑在悶熱的藍色空氣中，<br/>
<br/>
　漸漸地，呼吸愈來愈急促．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　回到台灣後，單車成為起初最貼近我夢想的橋樑，<br/>
<br/>
　一台泛黃的老車陪我度過我最朝氣勃勃的歲月，<br/>
<br/>
　一大早五點我躍上了車，穿過市集與，兩顆荷包蛋，<br/>
<br/>
　白天，跳過忠孝橋與大直橋，載滿義氣與回憶．<br/>
<br/>
　如今，這個月我將滿懷感激的端詳我這幾年來的努力與變化，<br/>
<br/>
　僅此，我便已心滿意足．<br/>
<br/>
<br/>
<br/>
　今天，在另一處河岸上，心跳從五十，急遽加速至一百八十，<br/>
<br/>
　望著久違的心臟脈動與下臀部的疼痛，風切聲中帶著嚶語，<br/>
<br/>
　把手上傳來城市間的血液，面對身旁呼嘯而過的野獸們，<br/>
<br/>
　我，彷彿又活過來了．　<br/>
<br/>
　<br/>
