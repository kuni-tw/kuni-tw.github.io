<span><br/><br/><br/>　　(晶瑩剔透的星冰粽)<br/><br/><br/><br/><br/><br/><br/><br/>  最近星巴克星冰粽的預購活動也開始了一陣子，<br/>																			    <br/>　主打的口味有兩種：蜜芋頭與摩卡可可咖啡，<br/>																			    <br/>　這一晚路經京華門市時，點了一杯香草蜜斯朵，<br/>																			    <br/>　看到冰櫃裡還有冰粽，一盤有兩顆，要價NT.80，<br/>																			    <br/>　便順便點來嚐嚐看。<br/> <br/><br/>					    <br/><br/>　　(包裹得還滿精緻的)<br/><br/><br/><span><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/>　星巴克的冰粽尺寸相當地小，<br/>																			    <br/>　不過包裹得還滿精緻的，<br/>																			    <br/>　剝開層層粽葉後，晶瑩剔透的冰粽外皮探出了頭，<br/>																			    <br/>　外表看起來還滿討喜的，拍了幾張寫真後，<br/>																			    <br/>　兩種口味各先淺嚐幾口。<br/>																			    <br/>　外皮的口感相當彈Ｑ，軟中又帶有嚼勁，<br/>																			    <br/>　個人很喜歡。<br/> <br/>							   <br/><br/><br/>　　(晶瑩剔透的冰粽本體)<br/><br/><span><br/></span><span><br/>　<br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　內餡部分，蜜芋頭還滿不錯的，<br/>																			    <br/>　芋頭的香氣與味道有出來，也不會太甜，<br/>																			    <br/>　而摩卡可可咖啡味道則就較令人困惑了，<br/>																			    <br/>　不知道是什麼原因，香氣似乎有點混過頭了，<br/>																			    <br/>　感受不太到咖啡或是可可的香味。<br/>																			    <br/>　整體來說，這種中西合併的小點心能做到這樣子已相當不錯了，<br/>																			    <br/>　且外型有緻，口感又不差，<br/>																			    <br/>　但內餡要是可以再多花點心思，那就更棒啦。<br/>																			    <br/>　　<br/> 　<br/><br/>    (蜜芋頭)<br/><br/><br/><span><br/></span><br/></span><br/><br/><br/>　　(摩卡可可咖啡)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/>   (內餡寫真)<br/><br/><span><br/><br/><br/><br/><br/>　(內餡寫真)<br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/><br/>BY GRD3 AND KUNI<br/></span><br/><br/><br/><br/><br/><br/><br/><br/>.<br/><br/></div>
