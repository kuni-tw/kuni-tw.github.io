<br/>

<br/>
<br/>
<br/>
　這天陪爸媽慢跑完後被他們帶去附近的朱記餡餅粥，</span></p>
<p><span>　很久沒去的一家老字號餐廳，自1973年開幕至今主打北京餡餅粥與北方麵食。</span></p>
<p><span>　仁愛總店外觀飛簷碧瓦十足的古典中式建築，入門後在二樓就座。</span></p>
<p><span>　看完菜單點了牛肉餡餅(NT.40)，招牌豬腳(NT.80)，牛雜細麵(NT.120)</span></p>
<p><span>　炸排骨(NT.70)，鮮蝦餛飩抄手(NT.120)，蝦仁炒飯(NT.140)，</span></p>
<p><span>　當然一家人不可能只點了這些，這裡單列有吃到的。<br/>
<br/>
<br/>
<br/>
<br/>
　(筷子)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212529.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先上桌的是牛肉餡餅與媽說我非嚐不可的招牌豬腳，</span></p>
<p><span>　前者外皮煎得微焦微酥咬了一口後中等厚度的牛肉內餡湯汁四溢，</span></p>
<p><span>　後者充滿八角香偏外省口味與常見的台式軟爛不同，</span></p>
<p><span>　口感彈牙有嚼勁，不過個人還是比較喜歡軟膩又充滿著老滷香的富霸王。</span></p>
<span>
<p><br/>
<br/>
<br/>
<br/>
　(來這裡必點的牛肉餡餅)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212530.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(招牌豬腳)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212531.jpg" target="_blank"></a><br/>
<br/>
<br/>
</span></span></p>
<p>　接下來是牛雜細麵與炸排骨，前者牛雜煮得入味口感偏實，</p>
<p>　但卻是酸菜清燉湯頭配上細切薑絲有點喝不慣，不過爸連續湯好幾次，</p>
<p>　後者外皮炸得酥脆內層肉厚醃得入味。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(牛雜細麵)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212532.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(炸排骨)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212533.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最後兩道是鮮蝦餛飩抄手與蝦仁炒飯，抄手向來為小弟最愛吃的中式料理之一，</p>
<p>　每家紅油醬料配方皆不同，朱記的充滿著微辣的麻香非常特別。</p>
<p>　炒飯部分最重要的指標之一就是米飯的口感啦，</p>
<p>　朱記的炒得似乎不夠香飯粒偏軟有點可惜，但不難吃。</p>
