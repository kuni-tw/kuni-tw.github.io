<p><span>(欣匠的飛魚握)</span><br/> <br/> <br/> <br/> <br/> <br/> <br/><span> ---------------------</span><br/><span>割烹的欣匠</span><br/><span>時間：2011年/5月</span><br/><span>地址：台北市大安區泰順街6號</span><br/><span>電話：23695887</span><br/><span>時間：12:00~14:30 17:30~22:00</span><br/><span> ---------------------</span><br/><br/> <br/><span>日式握壽司向來一直是小弟最愛吃的料理之一。</span></p>
<p><span>可能跟個人喜好有關吧，整個大台北區裡真正令我滿意的握壽司並沒有幾家，</span></p>
<p><span>當然這部分還得考慮到等價因素，如前陣子剛吃過麗水街游壽司二店的中午套餐，</span></p>
<p><span>以那價位來看CP值就絕對是爆表。</span><br/><br/> <br/> <br/><span>這次去的店家"割烹的欣匠"剛開幕不久且與游壽司頗有淵源，</span></p>
<p><span>前者料理長欣匠相當年輕與游壽司老闆阿不拉有個共通點：</span></p>
<p><span>兩人均曾待過台北東區知名的日式料理亭東丸好一陣子，</span></p>
<p><span>只是從沒同事過。</span><br/> <br/><br/> <br/> <br/><span>(割烹的欣匠店門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197972.jpg" target="_blank"></a></span><br/> <br/><br/><span>(板前的保鮮玻璃櫃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197973.jpg" target="_blank"></a></span><br/> <br/><br/> <br/> <br/><span>試水溫先點了一份綜合定食，NT.480附生菜沙拉，兩種當日海釣生魚片，</span></p>
<p><span>四貫握壽司與細捲(改迷你散壽司，但價位相同)</span><br/> <br/><br/> <br/><span>(黑板上寫著當日食材)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197974.jpg" target="_blank"></a></span><br/> <br/> <br/><br/><span>首先端上的是生菜沙拉，新鮮度與口感還不錯。</span></p>
<p><span>然後是兩種當天海釣上來的生魚片：青龍針與黑鮪魚赤身，</span></p>
<p><span>沾點山葵與醬油入口後才發現魚片的溫度有點微妙，</span></p>
<p><span>詢問下欣匠桑嚇了一跳說沒想到會有客人發現，</span></p>
<p><span>因為他覺得一般店家生魚片端出來時溫度都偏過冰比較不好吃，</span></p>
<p><span>所以刻意的把玻璃保鮮櫃溫度調得比一般高一兩度左右。</span></p>
<p><span>從這裡可以看得到欣匠桑料理時的細心之處。</span><br/> <br/><br/><span>(生菜沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197975.jpg" target="_blank"></a></span><br/> <br/><br/> <br/><span>(青龍針)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197976.jpg" target="_blank"></a><br/> <br/><br/> <br/>(黑鮪魚赤身)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363197977.jpg" target="_blank"></a><br/> <br/> <br/></span><br/> <br/><span>接下來就是我最期待的握壽司節目了，　</span></p>
<p><span>依序是飛魚握，活花枝握，鰹魚茗荷握與尖梭佐柚子胡椒握，</span></p>
<p><span>魚料種類相當多元化我喜歡不過壽司尺寸有點偏小不大習慣，</span></p>
<p><span>舍利部分醋飯煮得不錯但比例有點偏過多拿捏似乎不夠穩定，</span></p>
