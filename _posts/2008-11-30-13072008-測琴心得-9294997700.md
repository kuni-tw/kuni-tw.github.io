  心得:<br/>
<br/>
<br/>
<br/>
  高檔的小提琴真是一樣很吸引人的樂器，有如頂級紅酒一般，<br/>
<br/>
　愈陳愈香，層次漸漸分明，口感愈來愈多變，<br/>
<br/>
　變化似乎永無止境地輪迴下去．<br/>
<br/>
<br/>
　<br/>
　若以紅酒拿來作比喻，還可以分成分新世界的Napa Valley與法國波爾多五大酒莊，<br/>
<br/>
　甚至是DRC，不同產區不同葡萄品種連試飲期都大相逕庭，<br/>
<br/>
　法區的因其單寧厚重多半要陳年較久，漫長的等待讓酒質愈來愈陳熟，<br/>
<br/>
　最後達到無法以言語形容的神秘境界．<br/>
<br/>
　美區的雖然有著可以馬上飲用的優勢，但依然有著陳年存放的單寧實力．<br/>
<br/>
　<br/>
<br/>
　小提琴似乎也是如此，只是保存方式與紅酒不同，需要常常積極的去催化，<br/>
<br/>
　一把琴如光擺著而不時時去拉奏一番，震動其木料，其生命也會漸漸縮短，<br/>
<br/>
　提早結束．<br/>
<br/>
<br/>
　<br/>
　最近試了不少好琴，有大多是幫學生挑選的，初級的，中級的或是演奏等級的都有，<br/>
<br/>
　不同產區的包括大陸製義大利德國法國甚至美國的，量產或是獨立手工，<br/>
<br/>
　也細心的長時間測試過，不得不驚訝現代製琴工藝水準竟然如此之高！<br/>
<br/>
　長久以來歐洲甚至是義大利名琴的優勢漸漸變得微乎其微，<br/>
<br/>
　美國首先慢慢追上，近幾年期間大陸更是突飛猛進，<br/>
<br/>
　真的說得上是令人瞠目結舌啞口無言．<br/>
<br/>
　<br/>
<br/>
　更有甚者，琴漆才剛乾沒多久，一拿來演奏音質如此讓我驚為天人，<br/>
<br/>
　音色豐富內斂厚重，餘韻甘美華麗，真不知道幾十年之後，<br/>
<br/>
　這樣一把琴音色會到達什麼樣的境界呀．<br/>
<br/>
<br/>
<br/>

<br/>
　等到弦上最後一絲餘震漸漸飄去之後才猛然想起忘了要測琴，<br/>
<br/>
　馬上重新拉奏一遍，這一次反而變成天馬行空，<br/>
<br/>
　悠悠漫步中卻又再次忘記本來的目的，也因此沒一次測琴輕鬆過，<br/>
<br/>
