----------------------------------------------------------<br/>
加州燒烤堡敦南店<br/>
地址：台北市敦化南路一段238巷1號<br/>
電話：(02)8771-9423<br/>
----------------------------------------------------------<br/>
    <br/>
    <br/>
    <br/>
　<br/>
　這一次朋友遠從台南來，帶他到處走動吃喝，<br/>
    <br/>
　其中一處便是加州燒烤堡的敦南店。<br/>
    <br/>
　許久之前曾去過他們位在永康商區的創始店，<br/>
    <br/>
　那時候印象相當好，後來他們留言邀請我去敦南的新拓點品嘗，<br/>
    <br/>
　可惜時間一直沒排出來，一兩年後的今天終於成行，<br/>
    <br/>
　也算是另類的順水推舟吧。<br/>
<span><br/>
<br/>
<br/>
<br/>
<br/>
　　(店內一景)<br/>
<br/>
<br/>
<a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376715576.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　在SOGO敦南館附近的加州燒烤堡，與永康區那一家相同，<br/>
    <br/>
　店外裝潢不甚明顯，在競爭如此激烈，<br/>
    <br/>
　房租如此高昂的地區，似乎有點過於低調了。<br/>
　<br/>
　晚餐時間進了店內，裡頭相當冷清，<br/>
    <br/>
　點了一份招牌牛肉漢堡，要價NT.195，<br/>
    <br/>
  加50附薯條或沙拉與冷飲。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(菜單)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376715577.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　沒多久後主餐便上桌，長方形的擺盤上將兩片漢堡麵包分開擺著，<br/>
    <br/>
　一片上有生菜，番茄與酸黃瓜片，看起來頗為新鮮，<br/>
    <br/>
　一片上則有漢堡肉，培根與起司，<br/>
    <br/>
　盤子上另外還擺著金黃色的薯條。<br/>
    <br/>
　首先品嘗到處都可以吃得到的薯條，<br/>
    <br/>
　真的得說，加州燒烤堡的薯條不用沾番茄醬就很好吃了，<br/>
    <br/>
　炸得適中，酥脆外皮中夾著柔軟的內裡。<br/>
<br/>
<br/>
<br/>
<br/>

　將兩片漢堡麵包與夾料和在一起，先咬了一大口後，<br/>
    <br/>
　眼神不禁閉了起來，配合著加州特製的漢堡醬料，<br/>
    <br/>
　又嫩又多汁的漢堡肉味十足，當天本來不餓的小弟，<br/>
    <br/>
　食慾馬上被激了起來，真是不簡單呢。<br/>
