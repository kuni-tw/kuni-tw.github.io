<br/>
<br/>
　　(敦信午間的沙發區：多麼閒暇愜意的一幕呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　午間坐臥在敦信二樓的沙發區，多麼閒暇愜意呀！<br/>
<br/>
　<br/>
<br/>
<br/>
　教課之餘，常常信步找一家鄰近的星巴克門市，<br/>
<br/>
　入了店門，聽到夥伴們熟悉的招呼聲，不禁心情也開朗起來．<br/>
<br/>
　點了杯常喝的摩卡，找了張閒適的位置窩了下來，<br/>
<br/>
　然後隨著咖啡色的時光流逝，內心逐漸平靜，<br/>
<br/>
　我便就像剛充飽的電池一般，<br/>
<br/>
　不知不覺中，嘴角揚起，又重新生龍活虎了起來！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　而這其中，最常讓我想投向懷抱的，<br/>
<br/>
　便是坐落在信義與敦化南路交叉口的敦信星巴克，<br/>
<br/>
　一家開不到兩三年的小門市，前身為奧地利連鎖咖啡廳Julius Meinl．<br/>
<br/>
　走進店裡點了杯飲料，然後上二樓，<br/>
<br/>
　馬上可以感受到那種令我著迷的違和感．<br/>
<br/>
　這裡沒有其他熱門門市的熱絡吵鬧，<br/>
<br/>
　低調又微暗的燈光與裝潢之下，<br/>
<br/>
　連牆壁似乎都能滲出濃濃的咖啡味來！<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　這裡的熟識夥伴們雖然有一半以上已調走，<br/>
<br/>
　不過新來的夥伴們也一樣貼心與熱忱，<br/>
<br/>
　使得我不管在什麼時段或是抱著什麼樣的心情來到這裡，<br/>
<br/>
　總是能滿載而歸的離去，走的時候手裡可不只持著一杯熱咖啡，<br/>
<br/>
　另外還有附帶一份窩心．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (窩心的敦信門市夥伴！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　<br/>
　週末閒暇時刻，客人們似乎也沉浸在充滿書卷氣的氣氛裡，<br/>
<br/>
　看著書，上個網，寫寫報表，桌上總有著一杯寧靜，<br/>
<br/>
　偶爾啜飲一口，令人忘卻世俗，漸漸地，<br/>
<br/>
　才驚覺窗外的陽光已變得昏暗，趕緊收拾行囊，<br/>
<br/>
　帶著微笑走下樓，與夥伴們道別然後回歸車水馬龍...<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
