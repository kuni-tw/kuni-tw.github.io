<br/>
<br/>
  (星巴克的草莓奶霜星冰樂)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　星巴克於今年六月二十四推出的共愛地球隨行杯(也就是俗稱的地圖杯)，<br/>
    <br/>
　小弟早已不知肖想了多久，<br/>
    <br/>
　從日本那時推出過Venti與Tall款後，這兩size在日拍上漲得相當驚人，<br/>
    <br/>
　尤其是前者漲得更是離譜，小弟當時一直覺得很肉痛，<br/>
    <br/>
　便遲遲一直沒下手。<br/>
<br/>
  <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  直到從友人那裡得知"台灣也要出地圖杯"的消息，<br/>
    <br/>
　當時還真是難以相信，上一次才剛出過日本零九年的櫻花杯，<br/>
    <br/>
　這一次又要出與日本一樣的地圖杯嗎？<br/>
    <br/>
　詳細詢問之下才得知有點微妙，原來這次要出的只有Venti是同size，<br/>
<br/>
　另一款則是Grande size，不過無所謂，<br/>
    <br/>
　霎時間小弟簡直開心到了極點，<br/>
    <br/>
　終於可以蒐集到肖想已久的Venti地圖杯嚕。<br/>
<br/>
<br/>
<br/>
<br/>
  二十四號當天同時間還推出了幾款飲料與糕點，<br/>
    <br/>
　第二天便帶了買好的杯子，找了家門市拍"純純"的地圖杯寫真集，<br/>
    <br/>
　邊拍邊嚐嚐星巴克的草莓奶霜星冰樂與青蔬沙拉米捲餅，<br/>
    <br/>
　前者Grande size要價NT.130，後者要價NT.80<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  首先嚐嚐草莓奶霜，上面有一層打得相當漂亮的厚實鮮奶油。<br/>
    <br/>
　說實在的，有別於之前星巴克出過的幾款新飲品，<br/>
    <br/>
　這次的草莓奶霜個人倒是沒什麼特別感覺。<br/>
    <br/>
　草莓香有出來，只是不甚明顯，不過甜味剛好，<br/>
    <br/>
　沒有過甜過膩之問題，可惜缺了點個性。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(草莓奶霜星冰樂)<br/>
<br/>

　(配合著門市的美妙光線，真美)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721520.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
　青蔬沙拉米捲餅倒有點意思，使用了"沙拉米"火腿，<br/>
    <br/>
  火腿部分翻錯啦，應該要說香腸或臘腸才是。<br/>
    <br/>
　沙拉米義大利原文為Salami，是一種義大利的風乾式香腸或稱臘腸，<br/>
    <br/>
　有經過特殊乳酸菌發酵過，在歐洲是相當常見的食物與配料。<br/>
    <br/>
　正統與高檔的Salami口感非常美妙紮實，油花分分布有如雪花般，<br/>
    <br/>
　粒粒分明清楚，肉色則呈深紅，通常會切得相當薄，<br/>
    <br/>
　濃郁的味道，光現在想到口水就止不住啦，<br/>
    <br/>
　思及至此，突然鼻酸，想念起以前求學的美好時光。<br/>
<br/>
<br/>
<br/>
<br/>
　(青蔬沙拉米捲餅)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721521.jpg" target="_blank"></a><br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721522.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
　可惜的是，星巴克的青蔬沙拉米捲餅使用的Salami相當普通，<br/>
    <br/>
　甚至吃不大出來是Salami，口感過軟，也沒什麼味道。<br/>
    <br/>
　話說回來，若真使用高檔的義大利Salami的話，除了成本過高外，<br/>
    <br/>
　也有台灣人口感接受度不高的問題風險會存在。<br/>
    <br/>
　整體說來，雖然Salami部分，小弟有個人偏好，所以較不喜歡，<br/>
    <br/>
　但嚴格來說，其實還滿好吃的。<br/>
    <br/>
　更是小弟思鄉情懷濃濃時不錯的選擇與替代品呢。<br/>
<br/>
<br/>
