<br/>
<br/>
<br/>
<br/>
    (我超愛的摩斯海洋珍珠堡！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  其實喜歡吃摩斯也不是一天兩天的事情了．<br/>
<br/>
　<br/>
<br/>
　幾年前SEVEN剛推出米漢堡時，第一次吃到以米飯代替麵包的漢堡，<br/>
<br/>
　那時候簡直不敢相信自己的舌頭，令人驚奇的美味印象至今記憶猶深．<br/>
<br/>
　記得當時不少朋友跟我說，SEVEN的根本無法跟摩斯的比，<br/>
<br/>
　我才知道原來摩斯有賣米漢堡．<br/>
<br/>
　不信邪的我馬上去摩斯買一個來試吃看看呢，<br/>
<br/>
　試吃的結果倒是令我意外：兩家的其實都差不多好吃呢，<br/>
<br/>
　可惜後來SEVEN將米漢堡下架，只剩冷凍的還有再繼續賣．<br/>
　<br/>
　<br/>
<br/>
<br/>
　慢慢地，摩斯漸漸地也成為我沒閒好好吃頓飯時的優先選擇，<br/>
<br/>
　在諸多菜色當中又以海洋珍珠堡與金黃薯條最能抓得住我的脾胃，<br/>
<br/>
　另外摩斯常常沒事會推出一些新產品，<br/>
<br/>
　喜歡嘗試新鮮的小弟當然也不會錯過，<br/>
<br/>
　比如最近剛推出的XO黃金蝦堡(NT.75)<br/>
<br/>
<br/>
<br/>
　這一次點了XO黃金蝦堡B套餐，要價135，<br/>
<br/>
　飲料升級為玉米濃湯，沙拉升級為寒天海藻鮮蔬沙拉，<br/>
<br/>
　另外單點我最愛的海洋珍珠堡與金黃薯條．<br/>
<br/>
　從玉米濃湯開始說起好了，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (摩斯的玉米濃湯奶油玉米味自然濃郁！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　摩斯的玉米濃湯向來在連鎖速食餐廳裡頭品質相當穩定，<br/>
<br/>
　奶油玉米味自然濃郁，還算滿可口的．<br/>
<br/>
　海藻鮮蔬沙拉這一次的水準稍稍下降了點，<br/>
<br/>
　稍稍有點不夠新鮮的感覺，倒是可惜之處，<br/>
<br/>
　但我還是很愛呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (海藻鮮蔬沙拉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (這一次稍稍不夠新鮮，真是可惜呢...)<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　黃金薯條有點炸過了頭，不過馬鈴薯部分味道仍然香味十足，<br/>
<br/>
　連續兩樣附餐表現失去以往水準，這對我來說倒是個警訊呢．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (黃金薯條馬鈴薯部分味道仍然香味十足唷！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這一次的大地雷大概就是XO黃金蝦堡了，<br/>
<br/>
　不知道是不是廚房工作人員疏失，總之麵包擠得乾癟癟的，<br/>
<br/>
　醬料只能以稀疏來形容，整體表現有失摩斯以往水準．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(這一次的大地雷：XO黃金蝦堡)<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
　但個人愛吃的海洋珍珠堡表現依舊亮眼，<br/>
<br/>
　濃郁的溫熱米飯堡相當中夾著海鮮餅，<br/>
<br/>
　海鮮料適當的依比例混在一起煎，咬下去軟中帶Q，<br/>
<br/>
　無比可口的美味也稍稍沖淡了剛剛的小失望呢！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(個人超愛吃的摩斯海洋珍珠堡^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (喔喔，看那海鮮餡料呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(真可稱之為摩斯的招牌菜色之一呢！)<br/>
