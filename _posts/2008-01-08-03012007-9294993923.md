<br/>
<br/>
<br/>
  斷了一半的制約,隔日又重生.<br/>
<br/>
<br/>
  渾渾噩噩的重新睡去,醒來時感覺清爽多了,馬上提了一杯石榴色的西班牙汁液,<br/>
<br/>
  重溫西南半島的清香:總是那麼的宜人.<br/>
<br/>
<br/>
　這要怎麼說呢,回想起八月的開端,就是從那時候起始的吧,直到03號的傍晚,<br/>
<br/>
  我意識到,長久以來克服不了的瓶頸,就在無意之間被解決了.<br/>
<br/>
  這看似很簡單的問題,困擾我足足有兩輪多了,或說打從我出生起就沒停過的樣子;<br/>
<br/>
  這與父母之間向來不為人所知的隔閡在外人來看簡直是不可思議,<br/>
<br/>
　但神奇的是,就在旋律以及歐洲文化的薰陶之下,小提琴與美食,<br/>
<br/>
  這兩者都具有藝術(virtuosity)的本質,漸漸地搭成一座橫跨時光的橋樑.<br/>
<br/>
  看著爸孩子氣的笑容不時地展開,也意會到我們所扮演的角色開始模糊化,<br/>
<br/>
　甚至終有一天有著完全對換的可能性;同時間,老媽洋溢著幸福的微笑,<br/>
<br/>
　似乎再也無法停下聊起往事的腳步,我在一旁就樂觀其成,<br/>
<br/>
  這也大概就是所謂下意識才是牽動著人心的主謀吧.<br/>
<br/>
  終於,德國木鐘也敲起了辛德蕾拉的緊張,<br/>
<br/>
　這一道故事也就此告一段落.<br/>
<br/>
