<p><span>二店店內空間較一店寬敞許多，就座後點了最熟悉的綜合定食，NT.420</span></p>
<p><span>內容包括沙拉，一大份散壽司，四貫握壽司，味噌湯及玉子燒。</span></p>
<p><span>觀察師傅們正細心地製做壽司順便與阿不拉聊了一會兒，</span></p>
<p><span>沒多久軟絲沙拉與散壽司依序上桌，前者料多新鮮比往日更為可口，</span></p>
<p><span>後者醋飯上撒著紫蘇細絲另舖上鮭魚卵玉子燒切丁與鮪魚鮭魚白旗魚等魚料，</span></p>
<p><span>份量似乎比以前多，一大碗很快地扒光馬上就有飽足感。</span><br/><span>     </span><br/><span>(軟絲沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191463.jpg" target="_blank"></a><br/> <br/>(阿不拉與柏丞)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191464.jpg" target="_blank"></a><br/> <br/>(柏丞正在製做我的散壽司)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191465.jpg" target="_blank"></a><br/> <br/>(散壽司登場！)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191466.jpg" target="_blank"></a><br/> <br/>(微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191467.jpg" target="_blank"></a><br/> <br/></span><span>這一次順便重新試試阿不拉的握功，</span></p>
<p><span>第一貫是甜蝦握，上頭一小撮蟹膏賦予甜蝦更為濃郁的甜味，</span></p>
<p><span>甜蝦口感軟黏鮮甜毫無腥味，舍利握得鬆中偏緊水準不差，</span></p>
<p><span>可惜醋飯比例稍微過多。</span></p>
<p><span>(阿不拉正在製作我的甜蝦蟹膏握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191468.jpg" target="_blank"></a><br/> <br/>(甜蝦蟹膏握登場)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191469.jpg" target="_blank"></a><br/><br/></span><span>第二貫是竹筴魚拌薑末軍艦捲，使用似乎相當優質的海苔入口極化，</span></p>
<p><span>是相當討喜的一貫。</span></p>
<p><span>第三貫是鰭邊肉軍艦捲，使用比目魚鰭邊肉微炙過上頭一小撮明太子，</span></p>
<p><span>味道不錯。</span></p>
<p><span>此時加點阿不拉推薦正值產季的螢烏賊軍艦捲與軟絲握，</span></p>
<p><span>前者堪稱當天的驚人之作，薄薄的外皮內盡是濃郁鮮美的精華，</span></p>
<p><span>一入口就欲罷不能，後者配上細膩的刀工肉質口感軟嫩又綿密。</span></p>
<p><span>在最後一貫上桌前先來一碗味噌湯吧，將胃重新回溫後最後一貫穴子魚握上桌，</span></p>
<p><span>穴子魚柔軟入味配上微甜的醬汁也難怪游壽司向來一位難求呀。</span><br/> <br/><span>(竹筴魚拌薑末軍艦捲)</span></p>
