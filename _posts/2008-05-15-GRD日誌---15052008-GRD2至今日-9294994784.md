<br/>
  入手GRD2也四個月了,仔細對照起一二月時的作品,<br/>
<br/>
  真難以相信我的進步有這麼明顯,甚至我可以感覺的到我的照片會說故事,<br/>
<br/>
  會散發出攝影者的心情,有時候帶點淘氣有時候又帶著些許的柔情.<br/>
<br/>
<br/>
  照得好不好我似乎還沒有資格評論,因為我真的覺得自己還有很大的進步空間,<br/>
<br/>
  但是這期間攝影真的帶給我無窮的快樂,一部分來自於這可貴的成就感,<br/>
<br/>
  另一部分就是捕捉最美麗的那一刻,那一份感動,那一份幸福,<br/>
<br/>
  以及那一份可以與人分享的喜悅,實在是太令人著迷了.<br/>
