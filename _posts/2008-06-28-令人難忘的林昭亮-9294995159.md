在求學時代，就先看過晨曦為林昭亮寫的小傳"天籟的傳釋者"，<br/>
<br/>
　為此試聽過幾片他的CD，後來買了他的海頓與魏歐當以及莫札特的協奏曲，<br/>
<br/>
　他的音色相當清晰明亮，非常動聽．<br/>
<br/>
<br/>
<br/>
<br/>
  首先是他與鄭京和的巴哈雙小提琴協奏曲第二與第三樂章，<br/>
<br/>
　兩個幾乎快算是系出同門的茱莉亞高材生，風格大相逕庭，<br/>
　(林的老師狄雷乃鄭京和老師葛拉米安以前的助教)<br/>
<br/>
　林昭亮的精緻優雅與鄭京和的熱情奔放呈漂亮的對比，<br/>
<br/>
　真是過癮極了!<br/>
