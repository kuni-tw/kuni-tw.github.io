傍晚吃飽飯運動完之後臨時起意想找一家有水準的咖啡館去品嘗一下單品,<br/>
<br/>
  挑了一家風評不錯的咖啡館"咖啡意識",位在永和的永利路109號.<br/>
<br/>
　店面不大,裝潢在第一印象時不是很起眼,但是細看之下非常具有巧思,<br/>
<br/>
  進門首先入眼簾的便是這一台相當有份量的烘豆機:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  老闆看起來相當年輕而且清秀:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  小聊了一下之後,點了以酸味著稱的肯亞AA-PLUS單品以及老闆特製的瑪琪朵,<br/>
<br/>
　後來還贈送一杯他的特調單品,不過名字倒是忘了:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  飲後感想:<br/>
<br/>
  肯亞AA-Plus真的是相當推薦的一款好單品,<br/>
<br/>
　口感微酸,後韻有點澀,但是是類似紅酒單寧那種細緻的澀.<br/>
<br/>
  瑪琪朵上面的焦糖醬則是其最吸引我的地方,是老闆親自熬出來的,<br/>
<br/>
  香味濃郁,那像似濃縮了其他許多無法形容的甜味真是迷人.<br/>
<br/>
  而奶泡更是打得很細緻,口感真是綿密呀! <br/>
<br/>
