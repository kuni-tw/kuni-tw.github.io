-------------------------------------------------------------------<br/>
   橋壽司<br/>
   地址：臺北縣永和市永和路二段 292 號(中正橋口，世界豆漿大王附近)<br/>
   電話：(02)8925-2223<br/>
   營業時間：11:00~14:30  17:00~23:00<br/>
-------------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    <br/>
    <br/>
  離橋壽司六月開幕時期的初訪已有三個多月了，<br/>
( <a href="http://www.wretch.cc/blog/kuni/11614426">http://www.wretch.cc/blog/kuni/11614426</a> )<br/>
    <br/>
　這期間橋壽司的知名度也大幅地提高許多。<br/>
    <br/>
　由於上次還有許多想吃的料理沒品嚐到，一直想再過去嚐鮮一次，<br/>
    <br/>
　這天晚上趁著傍晚教課空檔便又溜了去。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　客人依舊絡繹不絕，服務生都是新面孔。<br/>
    <br/>
　找了舊位置坐下後點了橋壽司丼，附鮮魚味增湯要價NT.580<br/>
    <br/>
  加NT.20升級為蟹味味增湯，另外照例一份蟹膏軍艦捲，NT.60。<br/>
    <br/>
　望著當日限定哈燒貨的白板許久，<br/>
    <br/>
　後來加點了一份牡蠣軍艦捲與大干貝握壽司。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先上桌的是尺寸驚人的干貝握，<br/>
    <br/>
　是少見的巨無霸SIZE，鮮度與握功尚可，<br/>
    <br/>
　醋飯部分已比初訪時進步許多，不會過於乾硬，<br/>
    <br/>
　而這麼大的生干貝一下子就填滿整個口腔，<br/>
    <br/>
　實在是過癮極啦。<br/>
    <br/>
　然後是牡蠣與蟹膏軍艦捲，前者沒什麼令人印象深刻的地方，<br/>
    <br/>
　後者以這價格來論仍舊是來此必點的一道美味呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (大干貝握)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483083861.jpg" target="_blank"></a></span><br/>

<br/>
　(牡蠣與蟹膏軍艦捲)<br/>
<br/>

<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483083862.jpg" target="_blank"><br/>

<br/>
<br/>
<br/>
<br/>
</span><br/>
　不過自此仍未見到熱湯上桌便提醒了一下服務生，<br/>
    <br/>
　服務生這才恍然大悟，隨後補來，這部分倒是可再加強些。<br/>
    <br/>
　蟹味味增湯裡頭一大份螃蟹，味道中規中矩，<br/>
    <br/>
　只是個人實在不愛吃帶殼的，連蝦殼都懶得剝。<br/>
    <br/>
　平常一碗單價是NT.60，相當公道。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(蟹味味增湯)<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483083863.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
</span><br/>
<br/>
<br/>
　終於，當晚的重頭戲"橋壽司丼"上桌，<br/>
    <br/>
　這價格與著名的金泰無敵蓋飯(NT.600)差不多，<br/>
    <br/>
　雖然後者多了兩顆大生蠔，不過前者的豐富魚料與多樣性亦不惶多讓。<br/>
    <br/>
　仔細觀察橋壽司丼裡頭的料後將之記錄下來：<br/>
    <br/>
　魚料部分有紅魽，鮭魚肚，真鯛(長尾鳥)，穴子魚，<br/>
    <br/>
　每一種說真的都相當大份，尤其是穴子魚將近有一整條了。<br/>
    <br/>
　另外還有兩粒北海道大干貝，兩粒廣島牡蠣，兩條牡丹蝦，<br/>
    <br/>
　一份安康魚肝(代替當天賣完的海膽)，<br/>
    <br/>
　不小份的鮭魚卵，數片酪梨與玉子燒。<br/>

<br/>
　後記：<br/>
    <br/>
　說真的，橋壽司丼裡的魚料部分之豐富真的只能說甘拜下風，<br/>
    <br/>
  重點是每種量還真不少，可以吃得很過癮。<br/>
    <br/>
　鮮度也比初訪時進步些，以量來說絕對超值，<br/>
    <br/>
　不過以質來論還是略輸金泰，但後者的量卻沒得比。<br/>
<br/>
<br/>
　另外關於有網友反應真鯛並非長尾鳥部分，這我自己也曾懷疑過，<br/>
<br/>
　不過基於尊重師傅回覆寫給我的紙條關係，所以還是照寫啦^^<br/>
