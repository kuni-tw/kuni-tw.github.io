  調好焦距之後照了下來,沒想到竟然成功了......................<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
　經同為咖啡愛好者的友人介紹之後,下午課餘之時抽空騎去湛盧咖啡館的大安本店坐坐,<br/>
<br/>
　地理位置相當好找,從信義路左彎進新生之後一下子就到了.<br/>
<br/>
  店家入口有搭個棚,就看到有客人坐著,桌上一杯咖啡一邊打著筆電.<br/>
<br/>
  店內裝潢還滿精緻的,選著內廳的牆邊小桌坐下之後,<br/>
<br/>
  點了一杯我最熟悉不過的肯亞AA-Plus,店員隨後將沖煮器具端上桌,<br/>
<br/>
  這也是他們的特色之一:在客人面前為客人沖煮咖啡.(圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這不知道算不算噱頭,但是我還滿喜歡的^^<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  下面是我點的肯亞AA-Plus:<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  飲後感:他們的肯亞AA-Plus酸味對我來說可能還不夠明亮吧,<br/>
<br/>
  不知道是不是淺烘的關係(這部分有跟店員求證過了),味道稍稍的薄了點,<br/>
<br/>
  但是整體而言還滿順口的.<br/>
<br/>
  店員更是貼心的來詢問我對他們咖啡的感受,聽完之後,<br/>
<br/>
  沒料到他們馬上說要重新沖一次給我嘗嘗,這真是讓人感到無比的窩心呢.<br/>
<br/>
  基於好奇的心理,雖然不太好意思,但我還是接受了他們的好意,<br/>
<br/>
  第二杯順口度以及酸味顯然的比第一杯更好,不過可能還是習慣問題吧,<br/>
<br/>
  或許我真的不太適合淺烘的豆子.<br/>
<br/>
<br/>
  後記:<br/>
<br/>
  這是一家咖啡質感與服務都兼具的好店家,不過若是沒有辦他們的會員卡,<br/>
<br/>
  價位會有一點稍稍偏高就是了.^^<br/>
<br/>
