<br/>
  這一次的08年TBC比賽，朋友也有去參加，<br/>
<br/>
　網頁地址：http://www.taiwancoffee.org/Barista_08_list.asp<br/>
<br/>
　在此之前也準備了許久，可惜因為求好心切重做了太多次，<br/>
<br/>
　超時扣分，不過為了比賽特地準備的豆子有剩，<br/>
<br/>
　特地跑去找他，請他用比賽的豆子做了一杯卡布．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　使用的豆子總共有六支，皆為莊園等級的，成本非常高，<br/>
<br/>
　其中三支為：<br/>
<br/>
  巴西的黃波旁<br/>
　<br/>
　坦尚尼亞的TRIPLE A<br/>
　                                                                                                                                                                          <br/>
  衣索比亞．哈拉．藍馬<br/>
<br/>
　請了厲害的師傅特地為比賽的機器量身烘培，大概是中深程度吧，<br/>
<br/>
　牛奶也是比賽使用的，脂肪含量較高，很香，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　另外使用Terra Keramik手工咖啡杯，看起來真精緻！                   <br/>
<br/>
<div><a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11399797" target="_blank">Terra Keramik手工咖啡杯</a></div><br/>
<br/>
<br/>
　<br/>
  這大概是我第一次喝到這種等級的咖啡吧，<br/>
<br/>
　那甜味不知如何的從哪裡引了出來，<br/>
<br/>
　可能也是第一次品嘗這種咖啡，香味很香卻無法形容，<br/>
<br/>
　簡單一句，就是好喝而已，真是令人難忘呀！<br/>
