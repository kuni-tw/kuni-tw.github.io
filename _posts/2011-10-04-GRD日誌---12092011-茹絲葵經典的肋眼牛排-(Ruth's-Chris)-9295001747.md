<p><span>(茹絲葵的肋眼：充滿美國西部風情的霸氣肉味)<br/><br/> <br/> ---------------------------------------------------------<br/> 茹絲葵 Ruth's Chris Steak House<br/> 地址：台北市松山區民生東路三段135號2樓<br/> 電話：(02)2545-8888<br/> ---------------------------------------------------------</span></p>
<p><span>(一樓的招牌)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212536.jpg" target="_blank"></a></span><br/> </p>
<p><span>好久沒回訪這家位在台北的經典高級牛排館了，</span></p>
<p><span>這一次因前陣子也吃了幾家同類型的餐廳剛好可以順便比較。</span></p>
<p> </p>
<p><span>(走沙龍路線的典雅裝潢)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212537.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/> <span>推開與Lawry's類似的典雅木門走進充滿沙龍氣氛的空間裡，</span></span></p>
<p><span>就座後看了菜單最後決定單點，共計有：</span><br/> <br/><span>紐奧良烹蝦(NT.420)，茹絲葵沙拉(NT.300)，馬鈴薯泥(NT.260)</span></p>
<p><span>焗青花椰菜(NT.260)，美國頂級肋眼十六盎司(NT.1690)三分熟加外層微焦，</span></p>
<p><span>依服務生建議搭配C.Sauvignon, Mondavi Private Selection 2009(單杯NT.360)</span></p>
<p><br/><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212538.jpg" target="_blank"></a></span></p>
<p><span> </span></p>
<p><span><span>這次的服務生從頭到尾都保持著微笑，雖有點做作不自然不過還算有禮貌，</span></span></p>
<p><span>但在點餐過程中有點強勢這點較讓我無法接受。</span></p>
<p><span>沒多久麵包與礦泉水先上桌，前者是免費供應的所以可無限續，</span></p>
<p><span>外層微烤過為保持溫度用白布包著，奶油事先用機器打過所以又香又鬆軟，</span></p>
<p><span>但麵包質地有點普通很可惜。</span><br/> <br/> <br/><span>(礦泉水與麵包)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212539.jpg" target="_blank"></a><br/> <br/> <br/></span><span>接下來是前餐紐奧良烹蝦與茹絲葵沙拉，</span></p>
<p><span>前者醬汁微酸微辣很開胃配上五隻鮮度夠富口感的蝦子是一道很棒的前餐，</span></p>
<p><span>後者上頭擺疊在一起的炸紅洋蔥圈，下方則是扮著醬汁與藍紋起司的生菜，</span></p>
<p><span>上桌前服務生先現磨一些胡椒，還滿好吃的。</span><br/> <br/> <br/><span>(紐奧良烹蝦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212540.jpg" target="_blank"></a><br/> </span></p>
<p><span><br/>(口感彈牙充滿嚼勁)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212541.jpg" target="_blank"></a><br/></span></p>
<p> </p>
<p><span>(茹絲葵沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212542.jpg" target="_blank"></a><br/> <br/> <br/>(服務生現磨胡椒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212543.jpg" target="_blank"></a><br/> <br/> <br/></span><span>終於主餐肋眼牛排上桌了，同時登場的還有馬鈴薯泥與焗青花椰菜兩位配角，</span></p>
<p><span>可惜配角們表現有點不夠理想，馬鈴薯泥散出濃濃大蒜香口感濃稠卻不夠細，</span></p>
<p><span>青花椰菜咬起來清脆非常好吃，可惜整體吃起來偏油容易膩口。</span></p>
<p><span>但主角果然夠霸氣，厚重又肉味十足的肋眼口感充滿嚼勁，</span></p>
<p><span>每一口都充滿著油花芬芳與濃濃美國西部風情，</span></p>
<p><span>與Lawry's的典雅肉味呈現出完全不同的畫面，兩家好吃程度實不分軒輊。</span></p>
<p><span>這一次搭配的Mondavi 2009不差但還不到令人滿意的程度。</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212545.jpg" target="_blank"></a><br/> <br/> <br/>(十六盎司的美國頂級肋眼)</span></p>

<p><span><span>後記小感想：</span></span></p>
<p><span>在裝潢上茹絲葵以精緻沙龍對抗著Lawry's的挑高氣派空間，</span></p>
<p><span>可惜在配餐與牛排搭配餐酒上卻無力抵抗，</span></p>
<p><span>但最終仍能以其難以模仿的霸氣肉質肋眼繼續迎接其他高檔牛排餐廳們的挑戰。</span><br/> <br/> <br/> <br/> <br/><span>相關延伸閱讀：</span></p>
