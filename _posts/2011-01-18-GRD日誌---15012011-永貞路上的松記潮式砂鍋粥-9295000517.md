<br/>　這幾天在網路上閒逛美食心得時，<br/>																			    <br/>　不經意看到永貞路上有一家剛開幕沒幾個月的宵夜好去處，<br/>																			    <br/>　賣的是少見的潮州砂鍋粥。<br/> <br/><br/><br/><br/><br/><br/><br/>　朋友聽說了，從網路上印來一張優惠券(現在已經沒了)，<br/>																			    <br/>　一張只要399可換任選半份砂鍋粥與一盤熱炒，相當划算。<br/>																			    <br/>　便挑了天黃道吉日跑去嚐鮮。<br/> <br/>   <br/><br/><br/><br/><br/><br/>　(松記潮式砂鍋粥店外)<br/><br/><br/><span></span>		  <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>																 <br/>																			    <br/>　到了永和熱炒店林立的永貞路上一下子便找著了，<br/>																			    <br/>　店內燈光明亮給人很舒服的感覺，不過客人不多。<br/>																			    <br/>　就座後點了半份鮑魚帝王粥(原價NT.550)一盤脆皮豬腳(原價NT.99)<br/>																			    <br/>　另外加點一盤櫻花蝦炒高麗菜(原價NT.60)<br/><br/><br/><br/><br/><br/><br/><br/>　(菜單)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　於所謂的潮州砂鍋粥是客人點餐後才直接下生米去煲粥，<br/>　<br/>　一般來說要費十五到二十分鐘左右，小弟為了節省時間便事先預約到達時刻，<br/>																			    <br/>　果然點完菜沒多久後主菜便開始陸續上桌了。<br/> <br/>		   <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(吃粥附的配料)<br/><br/><br/><span></span>	 <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>															  <br/>																			    <br/>　<br/>　首先是櫻花蝦炒高麗菜，份量中等，<br/>																			    <br/>　火候尚可，沒有附近知名熱炒店上介青來得好，<br/>																			    <br/>　不過菜葉還算頗嫩，配上櫻花蝦口感不錯。<br/> <br/> <br/>		 <br/><br/><br/><br/><br/>　(櫻花蝦炒高麗菜)<br/><br/><br/><span></span>		 <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/> <br/>																			    <br/>　然後是脆皮豬腳，外皮看起來偏乾澀不討喜，<br/>																			    <br/>　皮的部分口感還算軟嫩可口，可惜份量不夠多。<br/>											   <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(脆皮豬腳)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/></span><br/><br/><br/>　不過最後的重頭戲鮑魚帝王粥倒是滿正點的，<br/>																			    <br/>　雖然只有半份，不過實地測量的結果竟然有六碗半，<br/>																			    <br/>　而且使用的是來自罐頭的真鮑魚而不是一般的假鮑。<br/>																			    <br/>　一入口不禁愕然，竟然有花生香味，仔細一瞧還可以看到不少栗子切片呢。<br/>																																							    <br/>　一問之下原來有添加堅果類的醬料，可惜卻沒有吃到老闆娘提到的干貝。<br/> <br/>	  <br/><br/><br/><br/><br/><br/>　(鮑魚帝王粥登場嚕)<br/><br/><br/><span></span> <br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(仔細一瞧除了真鮑魚以外還可以看得到栗子切片)<br/><br/><br/><span></span><br/><br/>					 <br/><br/><br/><br/><br/><br/><br/>										 <br/>
