　　(傳說名杯：星巴克銀座十年紀念杯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這一次林老師，阿立與我，有幸跟到剛回國的BBQ團，<br/>
<br/>
　更有幸親眼看到傳說中的名杯：星巴克銀座十周年紀念杯　<br/>
<br/>
　超限量的殘酷事實，造就了網拍上的高結標價，<br/>
<br/>
　另外還得說有錢還不見得買得到，可遇不可求呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　林老師為了滿足我們幾個的心願，當場就如火如荼地拆了包裝，<br/>
<br/>
　喔喔喔，林老師一邊拆一邊大家的心臟幾乎都快跳出來了，<br/>
<br/>
　終於名杯登場，難以置信的質感，精美的圖裝，<br/>
<br/>
　打開杯蓋後才發現裡頭全部附品無一不俱全，<br/>
<br/>
　連蓋有銀座門市店章的咖啡卷都有...<br/>
<br/>
　"這賣家也實在是太有良心了吧！"林老師開心的說。<br/>
<br/>
   更令人驚奇的，杯裡竟然還有圖裝，<br/>
<br/>
   好精緻的設計感呀...這樣叫人怎麼捨得用呢XD<br/>
<br/>
<br/>
<br/>
<br/>
　阿立跟我當下也暗下定決心，<br/>
<br/>
　Ｘ的，真的要勒緊褲帶了...<br/>
<br/>
<br/>
<br/>
