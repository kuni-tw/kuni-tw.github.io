  (隨行杯裡好可愛的小兔兔)<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　感謝某板友釋放出一只最近夯到極點的小白兔隨行杯，<br/>
<br/>
　本來是要拿來當禮物的，現在確定可以拿到兩個，<br/>
<br/>
　總算可以留一個私下供在專屬的星巴克玻璃櫥櫃裡頭。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(我是一隻可愛的小白兔^^)<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(以下是寫真隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　本來有好一陣子沒買星巴克的隨行杯，<br/>
<br/>
　其一是因為怕磨到或摔壞，<br/>
　<br/>
　其二，隨行杯畢竟是消耗品，加上保溫效果其實滿弱的，<br/>
<br/>
　所以先買了幾個超好用的不銹鋼款，加上備用的蓋子，<br/>
<br/>
　交互輪流使用，這樣比較能延長杯子的壽命。<br/>
<br/>
　沒想到這一款創意十足的隨行杯款才剛上市，<br/>
<br/>
　瞬間就擄獲了我的目光，我終於又心動了，<br/>
<br/>
　好不容易拿到一只，便馬上來給它個紀念寫真隨拍，<br/>
<br/>
　說實在的，這種杯子真的很不好拍呀，<br/>
<br/>
　尤其是中間那一隻小兔子，藏得真好！<br/>
