<p><span>(令人屏息的魚道生秋刀握)<br/> <br/><br/> --------------------<br/>魚道生美學料理<br/>地址：台北市大安區信義路四段199巷13號<br/>電話：(02)2702-4323<br/> --------------------<br/> <br/>由於教課地點就在附近，<br/></span></p>
<p><span>老早就一直想去開幕沒幾個月的魚道生試試水溫了，<br/></span></p>
<p><span>除了劉料理長(又稱技安師傅)是從個人熱愛的野壽司出來以外，<br/></span></p>
<p><span>另外幾位學生家長也相當推薦(他們也算是從野壽司跳槽過來的吧XD)。<br/></span></p>
<p> </p>
<p><span>與野壽司的裕二相比之下，技安師傅較有客人緣，<br/></span></p>
<p><span>這部分從進入店內後技安師傅那極富親和力的笑容就可以看出來，<br/></span></p>
<p><span>而服務生的服務態度也是相當討喜，沒有任何一絲馬虎。<br/> <br/> <br/>(專心製做料理的技安師傅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098550.jpg" target="_blank"></a><br/> <br/>這一天中午某學生臨時調時間，<br/></span></p>
<p><span>趁著好不容易多出來的空檔信步走到了魚道生。<br/></span></p>
<p><span>低調的店外裝潢與門面，店內燈光偏暗但不惱人。<br/></span></p>
<p><span>在吧台前就座後點了最便宜的中午握壽司套餐，NT.680<br/> <br/></span></p>
<p><span>等餐期間先試吃黑色石板上的漬蘿蔔，<br/></span></p>
<p><span>味道與野壽司相同，不過做了點小改良，NICE。<br/></span></p>
<p><span>首先上前付，是明太子章魚，味道處理得很細膩，好吃。<br/></span></p>
<p><span>然後是因天氣冷而多附的花枝丸清湯，<br/></span></p>
<p><span>湯頭很清，散出甜甜柚香。<br/></span></p>
<p><span>生菜沙拉則相當細心，口感佳，份量不多不少剛剛好，<br/></span></p>
<p><span>總之至此小弟皆相當滿意。<br/> <br/>(漬蘿蔔)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098551.jpg" target="_blank"></a><br/> <br/>(明太子章魚)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098552.jpg" target="_blank"></a><br/> <br/>(花枝丸清湯)                </span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098553.jpg" target="_blank"></a><br/> <br/>(沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098554.jpg" target="_blank"></a><br/> <br/> <br/>(店門口的花藝擺設) </span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098555.jpg" target="_blank"></a><br/> <br/>打從小時候起小弟就向來超討厭吃香菇的，<br/></span></p>
<p><span>除了在老爸老媽面前較不敢放肆之外，<br/></span></p>
<p><span>其他時候絕對是每逢香菇必將其挑出來。<br/></span></p>
<p><span>直到某天在野壽司吃到那神奇的茶碗蒸香菇，<br/></span></p>
<p><span>那真是難以形容，簡單一句：就是好吃而已。<br/></span></p>
<p><span>魚道生的商業午餐裡一樣有茶碗蒸，<br/></span></p>
<p><span>使用的碗器則更為講究，蒸蛋部分也很美味，<br/></span></p>
<p><span>只可惜裡頭的香菇我還是吃不下去呀。<br/></span></p>
<p><span>熟食部分則是烤魚，可惜沒有特別好吃，<br/></span></p>
<p><span>所以也懶得問是什麼魚。<br/> <br/>(茶碗蒸)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098556.jpg" target="_blank"></a><br/></span></p>
<p><span>(烤魚)　</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098557.jpg" target="_blank"></a><br/> <br/>接下來就是這一次要試的水溫重頭戲了，<br/></span></p>
<p><span>技安師傅開始以熟練乾淨俐落的手法捏製握壽司。<br/></span></p>
<p><span>首先是花枝握，花枝料部分軟黏，但似乎切得不是很平均，<br/></span></p>
<p><span>中間部位厚很多，舍利(醋飯)部分握得比野壽司的裕二鬆些，<br/></span></p>
<p><span>用筷子夾起來後要小心力道，不然會散開。<br/></span></p>
<p><span>推測是醋飯部分的問題，野壽司的較為軟黏，<br/></span></p>
<p><span>似乎在握工與口感上較具優勢。<br/> <br/>(花枝握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098558.jpg" target="_blank"></a><br/> <br/> <br/>然後是鮭魚卵軍艦捲與甜蝦握，前者表現佳，<br/></span></p>
<p><span>後者醋飯比例似乎多了點，可惜。<br/></span></p>
<p><span>(甜蝦握因技安師傅要求馬上吃所以沒拍XD)。<br/></span></p>
<p> </p>
<p><span>(鮭魚卵軍艦捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098559.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>當然最想吃吃看的莫過於鮪魚赤身握了，<br/></span></p>
<p><span>說真的，與裕二相較之下，技安師傅真的給人很細心的感覺，<br/></span></p>
<p><span>加上爽朗又極具親和力的語調，讓每一道料理都加分了不少，<br/></span></p>
<p><span>雖然裕二的握壽司還是略勝一籌，但技安師傅的握壽司還是很好吃。<br/> <br/>(有如紅寶石般的鮪魚赤身握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483098560.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>此時技安師傅似乎是一時興奮了起來，<br/></span></p>
<p><span>用了當晚兩千等級以上的Omakase魚料做了一道秋刀握給我，<br/></span></p>
<p><span>外表無比絢麗魚料色澤與刀工，<br/></span></p>
<p><span>入口後我不禁閉上雙眼，享受那難以置信的幸福時刻，<br/></span></p>
<p><span>美味的渦流像龍捲風般地襲向我身，實在是太美妙了。<br/></span></p>
<p><span>為此我願意花時間找一晚空檔品嚐魚道生的Omakase呀。<br/> <br/></span></p>
