<br/>
<br/>
<br/>
    (Amedei經典的Chuao巧克力)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  熱愛巧克力的小弟，<br/>
<br/>
　好不容易同時買到了義大利天王級巧克力品牌Amedei的Chuao<br/>
<br/>
  與Porcelana，<br/>
<br/>
　便挑了天黃道吉日，帶了心愛的GRD2小相機，<br/>
<br/>
　幫這兩款可算得上是經典的黑巧克力拍了幾張寫真．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Chuao與Porcelana包裝外觀)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Chuao先單獨來一張^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Porcelana當然也不落人後呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(左邊是Porcelana，右邊是Chuao)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不過拍照過程還真是難熬呀，那濃醇誘人又高貴的巧克力香一邊在四周圍繞著，<br/>
<br/>
　而棕黑色的巧克力磚一邊在鏡頭前似乎在搔首弄姿，<br/>
<br/>
　像是在喊著：主人，怎麼不趕快來吃我呀！<br/>
<br/>
　待拍攝工作到一段落後，主人，喔，不，是我，<br/>
<br/>
　便開始慢慢細細品嘗．<br/>
<br/>
<br/>
<br/>
<br/>
　其實以前這兩款都吃過，尤其是Chuao更是不知道買過幾次了，<br/>
<br/>
　而Porcelana則只吃過一兩次，不過是薄片，較沒有那種過癮的感覺．<br/>
<br/>
　這一次竟然兩款都是買到較厚的巧克力磚，實在是太幸運了．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  先說較熟悉的Chuao好了，這一款的可可豆出產於委內瑞拉裡同名的Chuao村莊，<br/>
<br/>
　外觀來看，顏色較Porcelana深些，香味濃厚，口感細緻迷人，<br/>
<br/>
　但是真正最誘人的則是裡頭慢慢散開來的果酸味，<br/>
<br/>
　有點類似於葡萄乾的酸甜，在舌頭的味蕾上慢慢地化開，<br/>
<br/>
　留下的則是滿滿的幸福．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (Chuao的斷面圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  然後是Porcelana，一款以瓷器來命名的頂級黑巧克力，<br/>
<br/>
　號稱完全以產量最稀少的Criollo來製作，<br/>
<br/>
　咬下第一口時，小弟不禁暗自心驚，難以形容的細緻又"綿密"的口感，<br/>
<br/>
　沒錯，Chuao口感已經夠細緻了，但是Porcelana顯然地又技高一籌，<br/>
<br/>
　在巧克力化開的同時散出絲絲的牛奶味與微微的果酸，<br/>
<br/>
　尾韻時則帶較多點微苦．　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
　　(Porcelana的斷面圖，個人感覺Porcelana的質地較Chuao細些)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  兩款皆可帶給人難以形容的細緻高貴口感，Porcelana較典雅，<br/>
<br/>
　而Chuao則較熱情奔放些．<br/>
<br/>
　個人倒是比較喜歡Chuao多些...邊打字時，<br/>
<br/>
　一邊又剝開了一片...然後靜靜地倘佯在幸福的懷抱中...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(喔喔，看起來好又人的巧克力疊疊樂呀...口水！)<br/>
