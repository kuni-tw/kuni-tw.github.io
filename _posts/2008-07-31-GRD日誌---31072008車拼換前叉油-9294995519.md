<br/>
　今天臨時決定去仰慕已久的車拼幫心愛的小金(RV250)換前叉油．<br/>
<br/>
　<br/>
　其實這個構想已久，加上一直以來就聽說圓仔老闆作工非常細心，<br/>
<br/>
　自己的愛車平常雖然在板橋保養，但是前叉倒是沒動過，<br/>
<br/>
　加上陰錯陽差從DC板板主sunicer得知住址等等(他也打算去換)<br/>
<br/>
　為了怕時間不足，下午打個電話過去，問了價錢以及可以去光顧的時間，<br/>
<br/>
　傍晚就直奔和平西路二段了．<br/>
<br/>
<br/>
<br/>
　沒想到竟巧遇DC板板主sunicer與Biker板人稱電系高手的板大chungsen，<br/>
<br/>
  　前者則是帶著馬車去換前叉油，後者則是去觀摩(chungsen真是用功呀)<br/>
<br/>
　圓仔夫婦一聽到我們都是批踢踢的板主時(小弟是星巴克板的)<br/>
<br/>
　都滿開心的，原來圓仔也有上逼．<br/>
<br/>
　不過最近電腦壞了，可能要過一段時間之後他才會再上來看看．<br/>
<br/>
<br/>
<br/>
<br/>
　圓仔工作時，我則忙著在旁拍照，<br/>
<br/>
　果然名不虛傳，作工精細到不行，一件一件的拆出來擦拭，<br/>
<br/>
　弄完時順便幫我檢查胎壓，充充氣，<br/>
<br/>
　愛車完工之後，付了750之後，<br/>
<br/>
　我馬上騎去教課，一路上真的只有一個字可以形容，<br/>
<br/>
　那就是"爽"：<br/>
<br/>
　前叉整個Q到不行，入彎前小煞感覺超棒，<br/>
<br/>
　車子一整個穩，真是太爽了呀...<br/>
<br/>
<br/>
<br/>
　就這樣，放上今天在車拼的一些隨拍，大家看看吧^^<br/>
<br/>
