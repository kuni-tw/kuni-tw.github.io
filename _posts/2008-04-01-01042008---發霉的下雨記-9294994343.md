發霉的下雨記<br/>
<br/>
<br/>
  整天下來觀賞著各式各樣的雨滴直到膩為止,順著雨勢漸漸磅礡,<br/>
<br/>
  乘捷運沿著水流到達下一個轉折.<br/>
<br/>
  鮮少在大雨下跳祈雨舞的我,難得細數著每一吋溼地與水漥,<br/>
<br/>
  攜著形影不離的隨身機萃取淚水的傾訴,我決定要為妳主持公道了.<br/>
<br/>
  直到徬徨的傍晚來訪,詢問我是否永遠不畏邪惡我點了頭.<br/>
<br/>
<br/>
  晚間掌上托著一份黑黝黝的義式咖啡上浮著鮮奶油,趁著熱呼呼時吞下去取暖,<br/>
<br/>
  畢竟雨勢持續擴大,這濕冷也愈來愈難以忍受了.<br/>
<br/>
  嗅著這濃稠乳甜香,與雨果等人暢談人文思想,<br/>
<br/>
  不禁感嘆了起來...這無窮盡的的黑色底上混著白紋,<br/>
<br/>
  這與我內心裡的交戰竟如此吻合:<br/>
<br/>
  最近愈來愈對於某事的堅定,使得自信開始反噬,<br/>
<br/>
  我陷入了無底漩渦,但卻快樂的享受著這暫時性的無能為力.<br/>
<br/>
<br/>
  馬克杯終於見底,於是拎起貴重的家當望著清澈夜晚,<br/>
<br/>
  不知什麼時候開始淚水早已乾涸,因為來日將只有勇氣陪伴.</div>
