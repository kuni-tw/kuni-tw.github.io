<br/>
<br/>
  這兩三年以來最愛的咖啡廳之一，雨田咖啡館，<br/>
    <br/>
　終於走入了歷史。<br/>
    <br/>
　幾個月前，在華軒告知要讓他弟鄧豪前去法國進修後，<br/>
    <br/>
　加上合約即將到期，小弟基於長期的情誼，<br/>
    <br/>
　這段時間順便幫忙他們處理轉讓店面等相干事務。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>

    <br/>
  從一開始的調和Latte與花式，中期極具新意的藍色風暴，<br/>
    <br/>
　與後期以口感為本，視覺為輔的水果風Latte，<br/>
    <br/>
　簡餐逐漸地變得精緻，加上營業時間夠長，<br/>
    <br/>
　雨田早已成為個人整天工作完後，<br/>
    <br/>
　夜晚必赴的充電聖地之一了。<br/>
