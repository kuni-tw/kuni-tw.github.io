<br/>
<br/>
　　(25122008 nach dem Abschluss)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  一個月過去了，<br/>
<br/>
　這當中扁桃腺意外的發炎，我卻堅持選擇對抗自己的弱點，<br/>
<br/>
　這樣持續到最後一天，這完全燃燒的感覺真好．<br/>
<br/>
　是的，我克服了，也再一次的證明了自己的能力，<br/>
<br/>
　KUNI，這裡將頒給你一張小獎狀！<br/>
<br/>
　沒錯，三冠王，只是這樣的日子大概不會持續太久吧，<br/>
<br/>
　不久之後我慢慢地又得重回死老百姓的身分，<br/>
<br/>
　每天繼續與老化掙扎．<br/>
<br/>
<br/>
<br/>
　瘦到６４的感覺很奇妙，因為這是會讓我回想起不少時光的體重數字，<br/>
<br/>
　傷心欲絕的倒是沒有，因為那數字低於６０！<br/>
<br/>
　身體狀態愈來愈接近十年不見的巔峰階段，這樣的時刻，<br/>
<br/>
　使得我每天無時無刻都有機會重溫往日：<br/>
<br/>
　每天在球場上或是田徑場上的熱血時代，<br/>
<br/>
　無時無刻都在百尺竿頭上斤斤計較，<br/>
<br/>
　那時候眼睛兩旁有如賽馬一般地被蒙住，只看得到前方難以捉摸的目標，<br/>
<br/>
　卻從未注意到兩旁的花朵繽紛亮麗，風景如此開闊無比，<br/>
　<br/>
　是的，我好懷念那一段日子．<br/>
<br/>
　如今，成天與夢想打拼，與現實爭取榮譽與利益之間的平衡，<br/>
<br/>
　雖然過得很好，但是內心裡卻不想扔掉任何一絲與青春期有關的事物．<br/>
<br/>
　<br/>
<br/>
　再過不久又是另一番新挑戰，<br/>
<br/>
　十年前在音樂上的熱血隨著體能的極限也即將登場，<br/>
<br/>
　我已經好久沒有過那種真正從零開始奮鬥的感覺了，<br/>
<br/>
　那與之前小孩子般的上雅虎首頁，兩百五十分或財富是不大一樣的，<br/>
<br/>
　是一種男人很容易忘卻的心情，很容易讓人產生恐懼，<br/>
<br/>
　而我想去面對，這讓我害怕起來．<br/>
<br/>
　我擔心無法達到我給自己的標準與期待與無情的現實，<br/>
<br/>
　但這一次的失敗是不允許的，keine Chance！<br/>
<br/>
　Scheitern wird sicherlich nicht geduldet, so etwas hat in meinem ganzen<br/>
<br/>
  Leben auch nie geschehen, weil ich nie aufgebe!<br/>
<br/>
<br/>
<br/>
  我不想失敗，因為之後有更大，更大又更龐大的挑戰即將如火如荼的到來，<br/>
<br/>
　這些將充斥著我整個人生，永無止境，<br/>
<br/>
　每日都得小心踏實的走，心無旁鶩，<br/>
<br/>
　漸漸地，我得開始學習熟悉與接受轉換．<br/>
<br/>
<br/>
<br/>
　晨間與老爸老媽慢跑時我不禁想了好多，<br/>
<br/>
　人生當然不是為了安逸養老而存在著的，<br/>
<br/>
　而是為了這當中不停的衝刺來爭取無數的漣漪與浪花與光榮，<br/>
<br/>
　來為出生以來就帶在身旁的水彩盤調色．<br/>
<br/>
　我想，接下來冗長而又充滿未知數的對抗賽，<br/>
<br/>
　我一定要好好的品嘗與吸收它，使之成為養分，<br/>
<br/>
　一點一滴都不能浪費．　  　<br/>
<br/>
<br/>
<br/>
  　<br/>
