<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  最近時間很忙，好不容易終於有空來寫一篇食記，<br/>
<br/>
　便挑了今年甫開張沒多久的雨田咖啡．<br/>
<br/>
　對於小弟來說，中永和真可算得上是大台北地區的咖啡一級戰區，<br/>
<br/>
　許多小弟心中的前幾名的咖啡館幾乎也都開在這區域，<br/>
<br/>
　這些咖啡廳各有各的特色與強處，熱愛咖啡的小弟沒事就會去叨擾一番，<br/>
<br/>
　每杯咖啡後，瞬地褪去疲憊，精神也轉開朗，心中更是充滿著浪漫與幸福．<br/>
<br/>
　<br/>
　這一家雨田是小弟一位好朋友兼咖啡界的前輩為他剛入社會的老弟所開的，<br/>
<br/>
　個人相當欣賞他老弟對於咖啡以及美食的態度以及認知，<br/>
<br/>
　以及那種在許多老店家已愈來愈少見的咖啡熱情，<br/>
<br/>
　近來許多店家常常給人自視甚高的感覺，<br/>
<br/>
　對於客人的任何意見幾乎不放在眼裡，<br/>
<br/>
　多的是禮貌式的"馬式"回應："謝謝，我們會改進"，這就免了吧，<br/>
<br/>
　記得去年某松山區的咖啡廳更曾惹火了我，<br/>
<br/>
　那種冷漠以及近乎鴨子聽雷的服務態度讓我發誓再也不踏進那店家門口，<br/>
<br/>
　北市不少店家更是不惶多讓，高高在上姑且不論，<br/>
<br/>
　更常擺出一副客人若不認同他們的咖啡就是不懂咖啡的表情，<br/>
<br/>
　這當中當然也有例外，比如湛盧咖啡館，<br/>
<br/>
　單價雖然較高，但那無微不至的服務真是窩心到極點呢！<br/>
<br/>
　<br/>
　<br/>
　雨田的開幕給了我耳目一新的感受，這部分與裝潢無關，<br/>
<br/>
　而是那種對於咖啡的理想與認知，<br/>
<br/>
　完全可以從一位名叫鄧豪的年輕人身上感受出來，<br/>
<br/>
　也是現在許多咖啡廳所欠缺的！<br/>
<br/>
　這一次抽了空過去，點了杯鄧豪剛創的新花式口味：<br/>
<br/>
  藍色風暴(NT.160)，<br/>
<br/>
　另一杯則是小弟個人激賞的碳烤杏仁拿鐵，超值130，                                                 <br/>
<br/>
　還有非常用心製作的三明治塔塔照燒雞，<br/>
<br/>
　單點130，套餐附一杯100的飲料則168，<br/>
<br/>
　以及綜合水果鬆餅(單點110，套餐159)<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (雨田店內半夜一景:這裡很適合夜貓子來唷XDDDD)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  首先來的是藍色風暴，令人驚異的奇異顏色，<br/>
<br/>
　泛出透光的藍色與綠色本體，漂浮在最上層則是濃縮咖啡，<br/>
<br/>
　十足的視覺效果，真難以想像這是一杯咖啡呢！<br/>
<br/>
　靠近杯緣可以聞到清新的薄荷與柑橘香，<br/>
<br/>
　入口後口腔裡滿是大膽的味覺遊戲，<br/>
<br/>
　微酸微甜，配上奶香與咖啡香，表面上看來似乎有點不協和，<br/>
<br/>
　卻不令人反感，喝得出來不同味道的比例有細心搭配過，<br/>
<br/>
　這大膽的創意與細心的搭配令人感到激賞！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (藍色風暴特寫：看著那漂浮的碎冰，暑意頓消)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(藍色風暴：令人懾服的視覺遊戲呀！))<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　接下來登場的是個人最為推薦的碳烤杏仁拿鐵，<br/>
<br/>
　應該是前一兩個月才剛推出的，一樣是創意十足，<br/>
<br/>
　但是更吸引人的則是那極為融合的杏仁香與咖啡味，<br/>
<br/>
　不知道怎麼巧妙的搭配得天衣無縫，<br/>
<br/>
　兩個字：好喝！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(碳烤杏仁咖啡：絕妙的好味道呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (上面一層濃郁的鮮奶油與薄荷葉！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　　(好漂亮的近距離特寫呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後我超愛的塔塔照燒雞三明治終於登場嚕，<br/>
<br/>
　一整盤裡放著四份用竹籤固定住的可愛三角吐司，<br/>
<br/>
　一小份金黃薯條，口味不錯，有五香的感覺，<br/>
<br/>
　一小盤生菜沙拉，看得出來有細心切過；<br/>
<br/>
　微烤過的去皮吐司裡，夾著一層全麥吐司，<br/>
<br/>
　自製的塔塔醬，新鮮生蔬與照燒雞，口感非常好，<br/>
<br/>
　味道均衡，個人就喜歡吃這樣的三明治呢！<br/>
<br/>
　<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
<br/>

<br/>

<br/>
<br/>
<br/>
　　　(三角三明治特寫：喔喔喔...好想咬一口唷)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
  最後則是綜合水果鬆餅，相當討喜的外觀與擺盤，<br/>
<br/>
　雖然沒什麼很令人激賞的特色，不過，<br/>
<br/>
　可以從許多細節上看出雨田在簡餐上的用心，<br/>
<br/>
　比如水果的切工相當乾淨，<br/>
<br/>
　另外似乎是刻意使用橘黃色系來配色，整體感足夠，<br/>
<br/>
　口感不錯，但那一片薄荷似乎有點小多餘，<br/>
<br/>
　畢竟前面兩杯飲料上也都出現過了，<br/>
<br/>
　個人比較建議使用與冰淇淋與柑橘類水果更搭的東西．<br/>
<br/>
　整體來說，口感不錯，處理得依然保持一貫的細心，<br/>
<br/>
　仍舊是水準之上的簡餐呀！<br/>
