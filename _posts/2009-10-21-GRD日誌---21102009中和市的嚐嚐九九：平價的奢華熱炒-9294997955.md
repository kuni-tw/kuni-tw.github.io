<br/>
<br/>
    (嚐嚐九九的蒜泥扇貝)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  早在八月時，探聽到第四屆中華美食展世界廚藝邀請賽剛落幕，<br/>
<br/>
　冠軍就是中和市的平價熱炒店"嚐嚐九九"店長"溫國智"，<br/>
<br/>
　另一位儲備店長一樣不惶多讓，<br/>
<br/>
　乃去年第一屆台北廚王賽熱炒組冠軍"黃家洋"。<br/>
<br/>
　從那時起小弟便一直心癢癢地，<br/>
<br/>
　想找時間來拜訪這一家號稱平價的熱炒名店，<br/>
<br/>
　今晚總算如願以償。<br/>
<br/>
<br/>
　位在台北縣中和市連城路224-10號的嚐嚐九九，<br/>
<br/>
　電話是(02)82422199<br/>
<br/>
　門口有一大片的機車停車格，還滿方便的，<br/>
<br/>
  走進去，寬敞而又明亮的空間，滿滿都是饕客，<br/>
<br/>
　圍繞著寬廣又大片的落地窗。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(人滿為患的嚐嚐九九)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　　(不過用餐環境相當明亮寬敞呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　小弟點了當日的十元特價餐：蝦皮燴瓠瓜<br/>
<br/>
　另外還有和風蔬果鰻，蒜泥扇貝，橙汁排骨，泰式涼拌海鮮。<br/>
<br/>
　每道熱炒均一價：九十九元，白飯免費吃到飽。<br/>
<br/>
　點完餐後，走到盥洗室去洗個手差點以為走錯地方，<br/>
<br/>
　非常高檔的洗手台，旁邊還有先進的烘手機，<br/>
<br/>
　不過因為來往客人相當多，<br/>
<br/>
　沒有專人時時來打掃清潔倒是滿可惜的。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(本日特價與主廚推薦)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(高檔的盥洗室呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　沒多久蝦皮燴瓠瓜便先上桌了，<br/>
<br/>
　看似普通的家常菜，雖然才十元，<br/>
<br/>
　仔細一看可見切工毫不馬虎，<br/>
<br/>
　瓠瓜入口後，帶出輕淡的蝦皮香，<br/>
<br/>
  雖然顏色不大討好，但仍舊非常下飯。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(本日特價10元的蝦皮燴瓠瓜)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(蝦皮燴瓠瓜很下飯呢)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　接下來上桌的是泰式涼拌海鮮，<br/>
<br/>
　酸辣夠味，拌著又Ｑ又軟嫩的花枝，<br/>
<br/>
　蝦仁香甜多汁，實在是太好吃啦。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(泰式涼拌海鮮)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (看這蝦仁多麼美味呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　和風蔬果鰻的擺盤相當精緻，視覺上非常討喜，<br/>
<br/>
　多脂的鰻魚配上新鮮的聖女番茄，<br/>
<br/>
　鳳梨，小黃瓜等生菜，<br/>
<br/>
　口感味道皆精緻上乘。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (和風蔬果鰻)<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
     (再一張特寫)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　橙汁排骨大概是當中我最不喜歡的一道菜了，<br/>
<br/>
　肉質有點死硬，而骨頭也偏多，<br/>
<br/>
　吃不到多少肉...令人感到相當失望呢。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(本日的失望之作：橙汁排骨)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　蒜泥扇貝，這真是本日第一大驚奇，<br/>
<br/>
　說真的，這道菜賣九十九，真是不知道要賺什麼了，<br/>
<br/>
　蒜香辣味十足，配上鮮嫩的扇貝，<br/>
<br/>
　底下墊著嫩豆腐，實在是好吃到讓小弟差點將舌頭也吞了進去，<br/>
<br/>
　好好吃呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(歐伊西的蒜泥扇貝)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(喔，這我一個人可以吃兩盤沒問題！)<br/>
<br/>
<br/>
<br/>
<br/>
　飽餐後出來結帳，加上十趴的服務費共花了447，<br/>
<br/>
　真是令人無法置信，這樣的用餐環境，<br/>
<br/>
　深厚的熱炒功力，親切有禮貌的服務態度，<br/>
<br/>
　讓人吃得真的很爽快，推薦！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(這就是與美味不成比例的帳單呀！)<br/>
