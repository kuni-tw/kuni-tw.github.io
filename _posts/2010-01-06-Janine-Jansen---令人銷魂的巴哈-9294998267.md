<div class="article-content-inner" id="article-content-inner">
　小提琴裡的巴哈無伴奏組曲在音樂價值上是無與倫比的，<br/>
<br/>
　當中作著名的莫過於夏康舞曲，<br/>
<br/>
　厚重的夏康舞曲更是可以將小提琴的音色發揮得淋漓盡致，<br/>
<br/>
　完全將這"f孔霸主"的特色完整的凸顯出來。<br/>
<br/>
<br/>
　<br/>
　在這幾十年間的歲月當中，歷代的大師們幾乎已將此組曲詮釋得近乎完美了，<br/>
<br/>
　尤其是帕爾曼，近乎完美的抖音以及弓壓技巧，<br/>
<br/>
　竟可從頭至尾都以甜美又溫暖的音色呈現給聽眾，<br/>
<br/>
　或許有人覺得過於浪漫，過於煽情，<br/>
<br/>
　但是試問，這種等級的詮釋又有誰能做得出來呢？<br/>
<br/>
<br/>
<br/>
<br/>
　這十年來沒想到竟然能聽到更加純潔的巴哈，<br/>
<br/>
　那就是我最欣賞的年輕一代的小提琴美女：Hilary Hahn<br/>
<br/>
　十幾歲的Hahn完美的將巴哈錄至一個新的巔峰，<br/>
<br/>
　這幾年的成長與演奏成果，讓人不禁懷疑這小女生究竟有沒有談過戀愛呀：<br/>
<br/>
<br/>
<br/>
<br/>
　　Hilary Hahn的夏康舞曲：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　就這樣追尋與見證了無數Hahn的創作與突破好幾年，<br/>
<br/>
　以為再也不可能有這麼巔峰的年輕小提琴家後，<br/>
<br/>
　沒想到竟然又看到一個新的領域，那就是Janine Jansen，<br/>
<br/>
　與Hilary Hahn幾乎同年，不過成名較晚許多，<br/>
<br/>
　但卻絲毫無損她的巴哈無伴奏在我心目中的地位，<br/>
<br/>
　天呀，令人無法置信的細緻與柔情：<br/>
<br/>
<br/>
<br/>
Janine Jansen：Bach - Sonata for Violin solo No 1 g minor<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這真是音樂界的撼事之一，原來巴哈無伴奏竟然還有這麼多的空間供人繼續追求，<br/>
<br/>
　如此的令人振奮，令人欣喜...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最後再分享一個經典的大師名演奏吧，<br/>
<br/>
　評價雖然高，但名氣較沒那麼突出的Arthur Grumiaux，<br/>
<br/>
　相當完美與溫暖的演奏，簡直無法相信是1960年的錄音，<br/>
<br/>
　相當現代的演奏技巧與詮釋，令人為之動容：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　　Arthur Grumiaux，Bach Sonata No.1 in G minor<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　其實在音樂界裡，青菜蘿蔔，各有所愛，<br/>
<br/>
　這本來就是藝術界該有的想法與邏輯，也是學藝術者該有的認知，<br/>
<br/>
　學藝術是為了更接近幸福，然後傳達給其他人們，<br/>
<br/>
　可惜卻有不少人從一開始就誤入歧途呀...(嘆)<br/>
<br/>
