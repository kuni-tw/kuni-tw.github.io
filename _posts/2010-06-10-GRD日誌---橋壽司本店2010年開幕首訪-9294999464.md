<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721483.jpg" target="_blank"></a></span><br/> <br/><span>(橋壽司配料豐富的海鮮丼飯只要NT.250)</span><br/><br/> <br/><span> ------------------------</span><br/><span>橋壽司</span><br/><span>地址：臺北縣永和市永和路二段292號</span><br/><span>電話：(02)8925-2223</span><br/><span>營業時間：11:00~14:30  17:00~23:00</span><br/><span> ------------------------</span><br/><span>     </span><br/><span>友前幾天想約我去剛開幕的橋壽司，可惜有事未約成，</span></p>
<p><span>變成她自個兒去吃，事後一直跟我嚷著很棒又很便宜，</span></p>
<p><span>而且從開幕到六月十號，全面八五折，</span></p>
<p><span>聽得小弟心癢好一陣子。</span></p>
<p><span>十號當天晚上教完課後，外頭雖下著大雨，</span></p>
<p><span>仍擋不住那股致命的誘惑，飛也似地直奔中正橋口，</span></p>
<p><span>停好車信步順著永和路的騎樓走到橋壽司。</span></p>
<p><span>(橋壽司)</span></p>

<p><br/><span>(服務生引領顧客們就座)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721469.jpg" target="_blank"></a><br/></span></p>
<p> </p>
<p><span>(店內一景)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721470.jpg" target="_blank"></a><br/> <br/>(桌上擺設的竹筷盒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721471.jpg" target="_blank"></a><br/> <br/>(熱茶)</span></p>
<p><span> <br/> <br/>(櫃台一旁的酒櫃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721487.jpg" target="_blank"></a><br/> <br/></span><span>店內裝潢</span><span>明亮而清爽，</span><span>客人絡繹不絕。</span></p>
<p><span>一位穿著短褲的辣妹服務生引領小弟就座，</span></p>
<p><span>看牆上白板上寫著"當季限定哈燒貨"</span><span>興致就來。</span></p>
<p><span>馬上點份廣島牡蠣握壽司，兩貫NT.80，</span></p>
<p><span>然後依正常菜單點一份蘆筍手捲，NT.40，</span></p>
<p><span>蟹膏軍艦捲一份兩貫，NT.60</span></p>
<p><span>海鮮丼一份NT.250</span></p>
<p><span>本來要點加海膽的，只多40，可惜海膽早賣光了。</span></p>
<p><span>鮮魚味增湯，一碗NT.40</span></p>
<p><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721472.jpg" target="_blank"></a></span></p>
<p><span>這價格真的平價，而蟹膏軍艦捲倒是首次在平價料理店裡看到。</span></p>
<p><span>另外開幕期間全面八五折，真的有爽到，</span></p>
<p><span>而且是爽到腸子裡的那種。</span></p>
<p><span>可惜等餐真的花很多時間，不過也因小弟對於這裡太期待了，</span></p>
<p><span>幾乎忘了時間，等到真的很不耐煩時，加上服務生又頻頻送錯，</span></p>
<p><span>才不禁提醒他們一下，終於，第一道佳餚總算上桌啦。</span></p>
<p><span>一碗才四十的鮮魚味增湯究竟有什麼樣的驚喜呢：</span></p>
<p><span>第一眼望滿滿魚料之外，</span><span>還有一整個胭脂蝦頭。</span></p>
<p><span>湯裡魚肉與預期相同，是切剩帶骨魚塊，</span><span>份量很大，</span></p>
<p><span>另外詢問之下得知使用日本信州的味增。</span></p>
<p> </p>
<p><span>(鮮魚味增湯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721474.jpg" target="_blank"></a><br/> <br/>(誘人的蝦膏)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721475.jpg" target="_blank"></a><br/> </span><br/> <br/><span>然後又等很久，上桌的是蘆筍手捲。</span></p>
<p><span><span>橋壽司的蘆筍口感很棒，清爽中帶著無比自然的甜，</span></span></p>
<p><span>可惜離食堂屋還有一段距離：</span></p>
<p><span>食堂屋的蘆筍手捲只有一句話可以形容：</span></p>
<p><span>滿滿又巨大的蘆筍中充滿著濃郁的清爽香甜。</span></p>
<p><span>價錢一樣也是四十，恩，各位知道我在說什麼了吧。</span></p>
<p><span>(蘆筍手捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721476.jpg" target="_blank"></a></span><br/> <br/> <br/><span>然後又是一段不短的等待時間，</span></p>
<p><span>蟹膏軍艦捲上桌了。</span></p>
<p><span>外表看似毫不起眼，但入口後是螃蟹精華，</span></p>
<p><span>兩個才六十，只能說是超值的幸福。</span></p>
<p><span>(蟹膏軍艦捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721477.jpg" target="_blank"></a></span><br/> <br/> <br/><span>然後上桌的便是海鮮丼了：</span></p>
<p><span>無比大的KING-SIZE，料多豐富，</span></p>
<p><span>一邊吃一邊記裡頭有什麼魚料。</span></p>
<p><span>後來乾脆借一枝筆來寫，不過還是漏了兩三樣沒寫到，唉。</span></p>
<p><span>兩百五的海鮮丼上有：</span></p>
<p><span>胭脂蝦一隻，甜蝦三條，熟蝦一隻，</span></p>
<p><span>一大份鮭魚卵，兩個生的大干貝，</span></p>
<p><span>鮪魚赤身，海鱺，紅甘，鯖魚，</span></p>
<p><span>生花枝，玉子燒以及鰻魚。</span></p>
<p><span>早知道這麼大份，實在不該叫這麼多，</span></p>
<p><span>記得上次在號稱超平價的國屏吃過一份兩百的海膽蓋飯...</span></p>
<p><span>雖然平價，但除了四隻甜蝦</span><span>與海膽外，沒什麼魚料了，</span></p>
<p><span>更遑論不好吃的醋飯。</span></p>
<p><span>橋壽司的海鮮料種類多，量也多，</span></p>
<p><span>醋飯不難吃，使用的是越光米(應該是台灣種)</span></p>
<p><span>這樣台幣兩百五，就算不打折一樣超級划算。</span></p>
<p><span>(超值的海鮮丼)</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721480.jpg" target="_blank"></a><br/> <br/>(鮭魚卵)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721481.jpg" target="_blank"></a><br/> <br/>(大干貝)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721482.jpg" target="_blank"></a><br/> <br/>(多麼誘人！)<br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376721483.jpg" target="_blank"></a></span><br/> <br/><br/> <br/><span>最後左等右等就是等不到廣島牡蠣。</span></p>
<p><span>告知服務生後，又過好一段時間才得知賣完，</span></p>
<p><span>差點罵出聲，只好改點牡丹蝦握壽司，</span></p>
<p><span>一份兩貫NT.150，</span></p>
<p><span>等好一段時候上桌，微炙過的蝦頭靠在握工技術普通的壽司身上，</span></p>
<p><span>不過沒有想像中的好，有點失望。</span></p>
<p><span>只能說橋壽司</span><span>握工不算強項，來這裡吃丼飯就好。</span></p>
