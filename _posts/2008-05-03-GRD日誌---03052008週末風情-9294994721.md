　這美麗的週末就在一大早淋冷水浴的衝擊之下揭開了序幕,<br/>
<br/>
  我乘著風飛揚掠過整塊大安區,當時意識雖還未完全清醒,但仍舊神采奕奕.<br/>
<br/>
  午後享受一盤精緻沙拉拼盤,精神開始慢慢持續回升,<br/>
<br/>
  直到最後一位徒兒登場.<br/>
<br/>
<br/>
<br/>
  身為教育上的一環,我常思索著何謂可分離與不分離之責任之歸屬.<br/>
<br/>
  對於我來說,學生從來沒有什麼責任可言,<br/>
<br/>
  而這也造就我無比沉重的負擔,因為我無法接受任何一次的失敗.<br/>
<br/>
<br/>
<br/>
  這的確與好勝無關.<br/>
<br/>
<br/>
<br/>
　我在乎比賽輸贏,卻不會因此影響到我的情緒,<br/>
<br/>
  我深知勝敗乃兵家常事.<br/>
<br/>
  何況經驗也多了,大概也了解該怎麼臨門一腳,<br/>
<br/>
  贏常常不過是個手段罷了,有如泰山鴻毛.<br/>
<br/>
  不過,惟獨教育是我無法全盤控制的,為此我氣得急跺腳.<br/>
<br/>
  人性複雜,變因又多,更何況是成長當中的個體,<br/>
<br/>
  每一分每一秒都在蛻變羽化.<br/>
<br/>
<br/>
<br/>
　學生們,老師祝你們今晚都能有個好夢.<br/>
