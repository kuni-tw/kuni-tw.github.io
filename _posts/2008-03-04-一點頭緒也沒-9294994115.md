<br/>
<br/>
　換個題材好了,總是得找點什麼來紀錄一下此刻的心情.<br/>
<br/>
  其實說來說去,還不都是男人與女人?不過當同時間有三對男女併在一塊時,<br/>
<br/>
  這可就不好玩了,不過我說的可不是三P.<br/>
<br/>
<br/>
  下午看到熟悉的臉龐飄了過去,我不禁想喊叫,阿,這不就是妳嗎?<br/>
<br/>
  有時候緣分很奇怪,明知不可行卻依然反其道而行;<br/>
<br/>
  或是兩人之間只需要一把鑰匙,甚至只是一根沒形狀的筆直鐵絲,<br/>
<br/>
  趁著夜黑風高時適時的插進鎖孔,轉動齒輪個幾圈,十八段的銅甲鎖便應聲而開.<br/>
<br/>
　<br/>
　爬著爬著,蠕動著扭動著形體,沒事就看見一陣一陣美麗的胴體吹了過去,<br/>
<br/>
  數著數著,咕嚕轉動著的水晶體,無法持續將視線凝聚在同一個焦點身上.<br/>
<br/>
  走進了黑暗而吵雜的大廳,微笑環顧著五色斑藍,順便推算著本日的胃口多寡.<br/>
<br/>
  坐落在Bar檯旁此起彼落的鳳凰們隨時套換著位置以求得更好的風水,<br/>
<br/>
  就看著從遠古時代精進數萬年至今的心機彼此交流著.<br/>
<br/>
　然後互相喘息著,最後彼此又重新退回到最原始的陌生.<br/>
<br/>
<br/>
<br/>
　或許這些,一切的一切最一開始,不過是當初的猴子王趁著大家不注意時所立下的規矩,<br/>
<br/>
  這造成了今日某種程度上嚴重的妨礙自由,萬年以來,<br/>
<br/>
  難道沒人懷疑過這無謂的約定俗成嗎?<br/>
<br/>
