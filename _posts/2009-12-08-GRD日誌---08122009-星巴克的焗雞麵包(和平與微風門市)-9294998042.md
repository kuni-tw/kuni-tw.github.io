<div class="article-content-inner" id="article-content-inner">
<br/>
  (烤得如此漂亮的星巴克焗雞麵包)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　傍晚下完課後，幫學生們選購些配備，<br/>
<br/>
　不覺中發現肚子已經開始嘰嚕咕嚕，再不馬上補給，<br/>
<br/>
　鐵定命喪車水馬龍中，馬上小弟我找了家鄰近的星巴克(和平門市)<br/>
<br/>
　掏出愛用的不鏽鋼隨行杯(真是好用到爆呀！)<br/>
<br/>
　例行性的使用紅利點數換了杯冬季限量的中杯太妃核果，<br/>
<br/>
　看了櫥窗內的熱食後，決定點沒看過的焗雞麵包(NT.55)，<br/>
<br/>
　看起來應該是滿好吃的。<br/>
<br/>
<br/>
<br/>
　就坐後沒多久焗雞麵包便登場了，<br/>
<br/>
　媽呀，我簡直不敢相信自己的眼睛，<br/>
<br/>
　剛剛點的橙黃色麵包居然變成漂亮的金黃色，<br/>
<br/>
　馬上詢問夥伴，原來他們有使用微波烤箱，<br/>
<br/>
　真是令人驚喜連連，拍了照後，<br/>
<br/>
　馬上咬了一大口，外酥脆內軟，焗烤香味四溢，<br/>
<br/>
　真的得說星巴克愈來愈不簡單了！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (看起來就好好吃呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (烤過的金黃質感真令人食指大動呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　說到最近常喝的太妃核果咖啡，就不得不提一下最近幾次去的微風門市，<br/>
<br/>
　微風的夥伴速度向來相當快，而且服務貼心，<br/>
<br/>
　比如上一次前來時，因為帶錯隨行杯，帶成中杯SIZE的。<br/>
<br/>
　點了大杯後，還是扣了杯扣，較便宜了十元，<br/>
<br/>
　夥伴貼心地將多餘的部分倒到馬克杯裡，<br/>
<br/>
　另外最令人感到窩心的是，當夥伴弄好時將餐盤端到我桌上時，<br/>
<br/>
　你可以看到兩杯裡都打上了漂亮的鮮奶油。<br/>
<br/>
　老實說，這一陣子在其他家門市點太妃，<br/>
<br/>
　我是真的沒遇過打鮮奶油也如此細心的夥伴，<br/>
<br/>
　更何況這一次其中一杯還是多出來的。<br/>
<br/>
　鮮奶油的形狀其實對於口感是沒什麼影響的，<br/>
<br/>
　不過賞心悅目的畫面總是秀色可餐呀，<br/>
<br/>
　可惜那一次沒帶相機，沒能將如此漂亮的藝術品拍了下來，<br/>
<br/>
　真是太遺憾了呀！<br/>
<br/>
<br/>
　謝謝你們，親切的微風門市。<br/>
<br/>
