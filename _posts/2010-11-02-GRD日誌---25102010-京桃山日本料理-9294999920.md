(京桃山的海鮮散壽司)<br/><br/><br/><br/><br/><br/>-----------------------------------------------<br/>京桃山日本料理(美麗華門市)<br/>地址：台北市中山區敬業三路20號B1<br/>電話：(02)2175-3456<br/>-----------------------------------------------<br/> <br/> <br/> <br/>　晚上教課空檔閒來無事去美麗華買杯星巴克，<br/>																			    <br/>　肚子突然間餓了起來，信步走下樓到了B1的美食廣場，<br/>																			    <br/>　逛了幾圈，與其他各大百貨的美食街一樣，<br/>																			    <br/>　比如走南洋風的咖哩飯或是東洋風的炸豬排等，<br/>																			    <br/>　都是常見的簡餐類。<br/><br/><br/><br/><br/>　但今天心情好倒想吃點別的，看到京桃山的獨立店面與細讀過店門口的菜單後，<br/>																			    <br/>　便決定在此填飽當晚早已飢腸轆轆的五臟廟。<br/>																			    <br/>　點了海鮮散壽司套餐，<br/> <br/>　要價NT.300附茶碗蒸或沙拉(選後者)，漬物與味噌湯。<br/><br/><br/><br/><br/><br/>　(菜單)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/> <br/>　(店內裝潢)<br/><br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　不一會兒餐點便全部上桌，仔細一看才發現服務生送錯了，<br/>																			    <br/>　我選的是沙拉拿到的卻是茶碗蒸，經反映後服務生趕緊拿去換。<br/>																			    <br/>　京桃山的附餐沙拉相當普通，生菜部分有點冷藏過頭的感覺，<br/>																			    <br/>　新鮮度則有點不盡理想，菜葉切口已開始氧化變色。<br/>																			    <br/>　味噌湯則一般，沒什麼特別值得令人著墨的地方。<br/><br/><br/><br/><br/>  <br/>　(海鮮散壽司套餐)<br/>  <br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(沙拉)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/></span><br/><br/>　海鮮散壽司裡頭有玉子燒，花枝，鮪魚，鮭魚，干貝，鮭魚卵，<br/>																			    <br/>　熟蝦與烤鰻魚，底下的醋飯偏乾硬，<br/>																			    <br/>　魚料部分比預期要來得好些，腥味還不甚明顯，新鮮度還夠，<br/>																			    <br/>　烤鰻魚切片口感類似以前老媽常買回來的真空包熟食。<br/><br/>   <br/><br/><br/><br/><br/><br/><br/><br/>　(海鮮散壽司套餐)<br/><br/><br/><span></span><span><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　整體說來尚可，個人沒特別喜歡，<br/>																			    <br/>　畢竟屬於百貨地下樓的美食激戰區，租金與人事等成本高昂，<br/>																			    <br/>　要拿純C/P值來與外頭的餐廳比較當然也說不過去，<br/>																			    <br/>　不過私心上畢竟還是希望能以這價位吃到更優一點的品質啦。<br/><br/><br/><br/><br/><br/><br/>BY GRD3 AND
