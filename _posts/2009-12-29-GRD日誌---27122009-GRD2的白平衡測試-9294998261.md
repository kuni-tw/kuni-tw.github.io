<br/>
<br/>
  傍晚時在一家餐廳裡，使用GRD2作一下白平衡測試，<br/>
<br/>
  一是常使用的自動白平衡，二則是手動白平衡。<br/>
<br/>
　其他條件完全相同：<br/>
<br/>
　光圈：f/3.2<br/>
　曝光時間：1/23秒<br/>
　ISO：400<br/>
<br/>
　第一張是手動的，第二張則是自動白平衡，<br/>
<br/>
　結果卻相當有趣，GRD2自動白平衡出來的圖似乎比較討喜耶，<br/>
<br/>
　更怪的是，手動白平衡的色調似乎沒有自動來得準呢。<br/>
<br/>
　<br/>
　其實以前使用GRD2拍照時，就曾猶豫過到底該使用自動還是手動來作室內攝影，<br/>
<br/>
　可是若使用手動就會出現色調無法還原現場實景的問題，<br/>
<br/>
　加上GRD2的自動白平衡雖然不算百分之百準確，但還算相當夠用，<br/>
　<br/>
　所以後來拍攝時索性全部使用自動白平衡，圖也直出，完全不後製，<br/>
<br/>
　效果一直都還蠻滿意的，除了高ISO不夠用。<br/>
<br/>
　<br/>
　這一次測試更是加深了我對GRD2白平衡的信心呀。<br/>
