貳樓餐廳 Second Floor Cafe 貳樓餐廳<br/>
電話：(02)2364-1666<br/>
地址：台北市羅斯福路3段316巷9弄7號<br/>
時間：一~四 11:00~23:00 五~六 8:00~1:00 日 8:00~23:00<br/>
--------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　知名的貳樓餐廳(或稱貳樓咖啡)公館分店於去年十一月二十二號開幕，<br/>
    <br/>
　路過好幾次臨時想進去一探究竟皆被門口的人群擋了出來，<br/>
    <br/>
　實在是有點不甘心，一行人便在大年初三早上訂了位，<br/>
    <br/>
　趁著過年氣氛仍正值熱絡之時一同前去。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　公館分店其實嚴格來說已不能算是"貳樓"餐廳了，<br/>
    <br/>
　兩層樓獨棟式的建物包括了一樓，原有的貳樓意義已不翼而飛，<br/>
    <br/>
　門口立著一台老BMW重車，典型的對臥雙缸實在是魅力難檔，<br/>
    <br/>
　令小弟不禁駐足了好一會兒。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(貳樓餐廳門口)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363160813.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　經服務生帶領之下到了二樓就座後開始點餐，<br/>
    <br/>
　小弟點了美式炸牛排套餐，NT.480<br/>
    <br/>
　附沙拉，麵包，酥皮濃湯，飲料任選(加價選栗子拿鐵)及甜點。<br/>
    <br/>
　等餐期間拎著心愛的GRD3環顧一下四周：<br/>
　<br/>
　貳樓餐廳的店內裝潢完成度非常高，尤其是細節部分的拿捏，<br/>
    <br/>
　完全不會給人那種東拼西湊或者是礙於預算不得不妥協的感覺，<br/>
　<br/>
　非常具有令人感到舒服的一致性。<br/>
    <br/>
　好比有些新開幕的店家裝潢雖然新穎，甚至使用相當高檔的器材設備，<br/>
    <br/>
　但總常在細節上漏了些，有時為了預算不足繼續沿用前店家的舊品，<br/>
    <br/>
　有時餐具與裝潢不符，有時是廁所相較之下過於簡陋，<br/>
    <br/>
　而最常出現的問題則莫過於裝潢設計與店家風格不一或是定位不夠明確等。<br/>

　不一會兒服務生先送來麵包與沾著用的橄欖油酒醋，<br/>
    <br/>
　麵包一般，咀嚼起來似乎沒什麼味道與該有的麥香。<br/>
    <br/>
　然後是生菜沙拉與酥皮濃湯，前者口感與新鮮度皆夠，<br/>
    <br/>
　後者沒什麼令人記憶深刻的印象，整體表現普通。<br/>
    <br/>
　栗子拿鐵倒真是首次初嚐，頗具新鮮感的甜香，<br/>
    <br/>
　仔細回想起來倒有點近似於爆玉米花的甜奶油香，<br/>
    <br/>
　甜度相很高，有些人可能會不大喜歡。<br/>

　<br/>
　終於到了主餐美式炸牛排登場的時刻，<br/>
    <br/>
　真的要說是相當一大份：一整盤滿滿的主餐與配料疊得老高。<br/>
　<br/>
　薯條部分與配菜部分就略過不提了，沒什麼特別的，<br/>
    <br/>
　炸牛排倒是第一次吃到。<br/>
    <br/>
　躲在厚厚炸衣之下的沙朗牛排裡保留著大量的肉汁，<br/>
    <br/>
　口感一時之間還真是難以形容，有點類似於牛小排的嫩，<br/>
    <br/>
　肉香卻完全不同，沒有頂級牛排的軟嫩口感但也沒有豬排的緊實，<br/>
    <br/>
　但真的相當好吃，可惜整體而言有點偏油膩，<br/>
    <br/>
　加上份量很大，以至於後來端上的焦糖烤布雷沒吃到幾口就得放棄，<br/>
    <br/>
　真是太可惜啦。<br/>
