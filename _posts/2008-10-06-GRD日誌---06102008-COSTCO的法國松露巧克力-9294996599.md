<br/>
<br/>
<br/>
　　(誘人的法國松露巧克力！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  COSTCO的商品有幾個好處，以大賣場來說除了質優，就是價格較低了．<br/>
<br/>
　不過他們許多相當實惠性的商品份量通常都很大，<br/>
<br/>
　所以常常可以在網路上看到網友們徵求合購此類商品，<br/>
<br/>
　不然一次買太多，縱使再怎麼便宜，消化不完一樣是浪費．<br/>
<br/>
<br/>
　COSTCO熱賣的商品其中一樣就是法國松露巧克力：<br/>
<br/>
  TRUFFLES de FRANCE<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(法國松露巧克力的盒裝，質感還滿不錯的！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一公斤的盒裝松露巧克力裡頭分兩小包，<br/>
<br/>
　一次得購買兩盒，要價399，也就是說，<br/>
<br/>
　五百公克一包的法國松露巧克力只要一百元，<br/>
<br/>
　簡直是將巧克力秤斤來賣，難怪不少顧客們對此商品趨之若鶩．<br/>
<br/>
　一兩個禮拜前小弟接獲線報，COSTCO缺貨已久的松露巧克力重新上架，<br/>
<br/>
　當天果不其然，上午上架，下午一兩點便已銷售一空，<br/>
<br/>
　小弟有幸搶到手，不過在此保證全程沒使用過以前在校隊練過的卡位，<br/>
<br/>
　只是規規矩矩的看到商品就衝了過去而已．<br/>
<br/>
<br/>
<br/>
　今天晚飯後，打開了一包，甫從冷藏庫取出來的巧克力塊們香氣撲鼻，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(打開之後，巧克力塊們香氣撲鼻)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　黝黑中帶著巧克力特有的紅褐色，用刀子橫切之後，只見剖面頗有光澤，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(頗有光澤的巧克力剖面圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一咬下去，口腔中充滿著濃郁細緻的口感，非常迷人，<br/>
<br/>
　不過吃第三四顆時卻漸漸覺得有點過甜，這大概是美中不足的地方，<br/>
<br/>
　也好，免得一口接一口的停不下去，晚上又得多騎個幾十公里的腳踏車嚕．<br/>
<br/>
<br/>
<br/>
<br/>
　　(真是超好吃的呀！)<br/>
<br/>
