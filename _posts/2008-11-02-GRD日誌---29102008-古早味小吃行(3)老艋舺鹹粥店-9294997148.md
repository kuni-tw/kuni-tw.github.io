   (走清淡路線的老艋舺鹹粥，顏色很漂亮！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  十月二十九號古早味美食行前兩站結束後，  <br/>
  (第一站：<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11473764" target="_blank">台灣古早味小吃行(一)　阿華鯊魚煙</a>)<br/>
  (第二站：<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11474381" target="_blank">古早味小吃行(2)大橋頭筒仔米糕</a>)<br/>
<br/>
<br/>
　馬上又從大同區順著延平進入了萬華區，<br/>
<br/>
　首先拜訪的是位在西昌街117號的臺式鹹粥名店"老艋舺鹹粥店"，<br/>
<br/>
　道地的北部口味．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (臺式鹹粥名店"老艋舺鹹粥店")<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　鹹粥大體上是有南北之別的，北部通常為肉粥，以豬骨熬湯，<br/>
<br/>
　南部則為魚粥，粥料則較北部豐富了許多，<br/>
<br/>
　比如台南知名的廟口虱目魚粥．<br/>
<br/>
<br/>
　抵達時約下午兩三點左右，店內人潮剛散去，<br/>
<br/>
　店家的服務生開始交接換班，順便開始打掃，<br/>
<br/>
　進了店門，只點了一碗招牌肉粥，<br/>
<br/>
　沒辦法，總要留點胃口給接下去要拜訪的店家呀．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (抵達時約下午兩三點左右，店內人潮剛散)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　老艋舺鹹粥店的肉粥走的是清淡路線，<br/>
<br/>
　道地又細心的臺式做法：在客人點餐之後，<br/>
<br/>
　才將煮好的白飯放進事先熬煮好的高湯裡．<br/>
<br/>
　稍煮一會之後，再依序放入小白菜，豆皮，油蔥與紅燒肉，<br/>
<br/>
　顏色很漂亮，米粒保持得很完整，<br/>
<br/>
　吃起來真讓人感到很窩心呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (走清淡路線的老艋舺鹹粥)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(好誘人的紅燒肉與飽滿的米粒呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　後記，店家裡養著一隻很挑食的小貓阿祥，<br/>
<br/>
　據店家告知，平日與陌生的客人們"互動良好"，<br/>
<br/>
  不才小弟一直嘗試為牠拍照不成，<br/>
<br/>
　最後好不容易逮著牠進食之時才成功呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(西西，趁在店貓阿祥在進食之時偷拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
