<br/>
<br/>
<br/>
　不知何時開始，除了一些限定還是特別的杯款外，<br/>
    <br/>
　小弟開始較專注在不鏽鋼材質的杯款上了，<br/>
    <br/>
　不知不覺中竟成了友人間戲稱的"不鏽鋼杯控"。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　可能與使用習慣有關吧，平常出門並不會攜帶塑膠材質的隨行杯款，<br/>
    <br/>
  反而是較耐用的不鏽鋼杯。一方面擔心杯面有可能會被刮到，<br/>
    <br/>
  另一方面當心愛的杯子因粗心而墜地產生了裂痕，<br/>
    <br/>
　那種"資深杯控"所獨有的悲痛心情其實一般人真的很難體會或是了解。<br/>
    <br/>
　前一陣子趁著日本首度推出兩款不鏽鋼材質的櫻花杯熱賣後，<br/>
    <br/>
　沒幾個月再度推出兩個類似的相仿杯款，粉嫩配色的粉紅杯與黃杯。<br/>
    <br/>
　號稱鋼杯控的小弟當然不落人後，輾轉也託友人幫忙帶回。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(日本幾個月前推出的兩款不鏽鋼隨行杯)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076288.jpg" target="_blank"></a><br/>

<br/>
　這一次台灣也出了粉藍色的相仿杯款，<br/>
    <br/>
　所不同的則是杯蓋部分，日本那兩款則是類似果凍套的材質，<br/>
    <br/>
　台灣的則為一般的常見材質。<br/>
 <br/>
　另外同時推出的還有較少見的Venti與Grande不鏽鋼杯款，<br/>
    <br/>
　配色分別為綠色與紫色，與日本之前曾推出過的黑色VENTI杯屬同一款式。<br/>
<br/>
<br/>

　第四款則為經典款式的不鏽鋼杯，深邃的酒紅配色，<br/>
    <br/>
　趁著最近天氣溫度降低，外頭風大又細雨綿綿，<br/>
    <br/>
　晚上九點過後，拋開白晝的一切繁忙與疲累，<br/>
    <br/>
　一個人獨坐在咖啡廳趁著夜晚的寧靜時拿來使用還頗為應景的呀。<br/>
