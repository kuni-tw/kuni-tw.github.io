<br/>
<br/>
    (知名的公館芋頭大王)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  下午與網友面交肖想許久的一款星巴克隨行杯，<br/>
<br/>
　杯子好不容易到手後，幫學生選購一些消耗性物品，<br/>
　<br/>
　然後路過永康街時才想起一直沒有機會品嘗的芋頭大王。<br/>
<br/>
　個人向來相當熱愛芋頭類的甜品，<br/>
<br/>
　記得最高紀錄曾一口氣吃掉七八支芋頭冰棒，<br/>
<br/>
　尤其是芋頭那種綿綿密密的口感，在嘴裡慢慢化開，<br/>
<br/>
　實在是太銷魂了。<br/>
<br/>
<br/>
<br/>
　<br/>
　車停好後，順路走到之前常吃的冰館，<br/>
<br/>
  這部分詳情請參閱：<div><a href="http://www.wretch.cc/blog/kuni/11453387" target="_blank">永康街著名的冰館</a></div><br/>
<br/>
  看到熟悉的芋頭大王招牌，卻找不到店門口，<br/>
<br/>
　原來店家隱藏在裡頭的小吃店群裡頭，<br/>
<br/>
　看來若不做好口碑，這樣的地點還真是難做生意呀。<br/>
　<br/>
<br/>
<br/>
　<br/>
　因為中午吃得稍晚，肚子仍頗有飽足感，<br/>
<br/>
　便只點了一盤芋頭紅豆牛奶冰(NT$70)<br/>
<br/>
  沒多久便端了上來。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (芋頭紅豆牛奶冰)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　仔細觀察一番，打得相當細的剉冰上舖滿紅豆與蜜芋頭塊，<br/>
<br/>
　拿湯匙舀了一口芋頭塊，和著牛奶剉冰一起放入口中，<br/>
<br/>
　芋頭的甜味慢慢散出來，真的相當入味，<br/>
<br/>
　不過卻似乎有點死甜，或說是膩甜，<br/>
<br/>
　但是還是滿好吃的。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(芋頭紅豆牛奶冰局部特寫)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　　(芋頭紅豆牛奶冰局部特寫二)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (芋頭紅豆牛奶冰局部特寫三)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　與老板娘聊了一下，得知這裡的芋頭都是她公公花很多時間煮的，<br/>
<br/>
　是蜜芋頭的做法，總過程約三小時，<br/>
<br/>
　看著攤子裡還有不少其它配料，不禁心動了一下，<br/>
<br/>
　可惜肚子不爭氣，只好等下次有機會再來叨擾了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
