<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  一直以來就想找一把實惠又高檔的好弓,<br/>
<br/>
  沒事幫學生挑也幫自己看看,只要有機會就會試試.<br/>
<br/>
  這一次有兩把,第一把是在日本修琴許久的黃國瑞師父所製,<br/>
<br/>
  第二把則是開提琴工廠的學長好不容易找到乾燥幾十年的老巴西木(PERNAMBUCO),<br/>
<br/>
  請好師父製作出來.<br/>
<br/>
<br/>
<br/>
<br/>
  兩把各有不同的手感與音質,第一把很顯然的,使用顫弓時顯而易見地非常敏銳,<br/>
<br/>
  感覺跟法國老弓手感相當像,音質卻略悶,在彈性方面,在輕弓壓之下咬弦方面較不盡理想;<br/>
<br/>
  第二把在顫弓方面略輸,但Q度一流,尤其在慢速又輕盈中弓跳弓時,<br/>
<br/>
  還能保有其相當優異的避震性,此時使得弓毛咬弦非常溫和不會過於刺耳,<br/>
<br/>
  音質方面非常突出,不過顫弓能力輸第一把甚多.<br/>
<br/>
<br/>
<br/>
  自己學生時代使用的Charles Bazin老玳瑁金弓真的很捨不得拿來練習,<br/>
<br/>
  現在想想以前真是沒管那麼多,很不珍惜,一點也不知道手上握的是什麼好東西,<br/>
<br/>
  現在好法國弓又愈來愈少,唉...<br/>
