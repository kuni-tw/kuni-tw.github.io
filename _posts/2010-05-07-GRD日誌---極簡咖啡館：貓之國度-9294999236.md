<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696291.jpg" target="_blank"></a></span><br/><br/><span> (煙燻鮭魚捲)</span><br/><br/><span>     </span><br/><span> --------------------------------------------------------</span><br/><span>極簡咖啡</span><br/><span>地址：台北市泰順街2巷42號</span><br/><span>電話：(02)2362-9734</span><br/><span> --------------------------------------------------------</span><br/><span>     </span><br/><span>     </span><br/><span>終於找了天去了傳說中的貓咪咖啡餐廳：位於泰順街的極簡，</span></p>
<p><span>一家裡頭總共有三四十隻貓咪的國度。   </span></p>
<p><span>傍晚順著泰順街一路走下去，走到又暗又窄的二巷裡頭，</span></p>
<p><span>突然間一處燈火通明，前庭招牌燈上的極簡兩字綻放出耀眼光芒，</span></p>
<p><span>極為顯眼。</span><br/><br/> <br/><span> (極簡招牌與店門口隨拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696292.jpg" target="_blank"><br/> </a><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696293.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696294.jpg" target="_blank"></a></span><br/> <br/> <br/><span>順勢開了店門，馬上感受到幾隻小鬼頭的銳利眼神掃了過來，</span></p>
<p><span>沙發上，桌上椅上，甚至咖啡機上，</span></p>
<p><span>四處可見貓咪，有的趴著，有的漫步著，</span></p>
<p><span>有的嘻笑有的打鬧，不過絕大多數都在"閉目養神"，</span></p>
<p><span>只是睡姿相當誇張：</span></p>
<p><span>母親大人，有機會您應該來看看這兒，</span></p>
<p><span>這些毛茸茸的生物睡得比我還誇張。</span><br/> <br/> <br/><span>點了一杯摩卡(NT.180)與一份煙燻鮭魚捲(NT.200)</span></p>
<p><span>從價位上來看，咖啡稱得上是相當貴，</span></p>
<p><span>不過在還沒嚐到實品之前，暫先不下任和評語。</span><br/><span><br/></span></p>
<p><span> (菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696295.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696296.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696297.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696298.jpg" target="_blank"></a>　</span><br/><span>     </span><br/> <br/><span>等餐期間，熱愛貓科類動物的小弟便趁機到處去溜達一番，</span></p>
<p><span>帶著GRD3四處去捕捉這些小生物迷人的身姿與倩影，</span></p>
<p><span>在客人們四周，桌下腳下穿梭，各處沙發與家具上滿滿的抓痕。</span></p>
<p><span>看著如此不怕生的貓咪，面對陌生客人時張起水汪汪的大眼睛，</span></p>
<p><span>洋溢著滿臉的好奇心，不禁感到有趣。</span></p>
<p><br/><span> (貓咪寫真)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696299.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696300.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696301.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696302.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696303.jpg" target="_blank"><br/> </a><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696304.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696316.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696314.jpg" target="_blank"></a><br/></span></p>
<p><span>老闆娘剛好在店內的吧檯坐著與客人聊天，</span></p>
<p><span>沒多久後與她閒聊，得知這家店好幾年前剛開幕時，</span></p>
<p><span>並沒有刻意設定為寵物餐廳的，那時店內只有一兩隻貓，</span></p>
<p><span>其中一隻還是員工的家貓，那一隻後來隨著該員工的離職也返家了。</span></p>
<p><span>後來貓咪愈來愈多，便開始與動物醫院和協會等等合作，</span></p>
<p><span>有計畫的去飼養以及設計相關的裝潢，直到有今日的光景與規模。</span><br/><br/><span> (體貼貓咪的裝潢與設計)</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696306.jpg" target="_blank"></a><br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696307.jpg" target="_blank"></a>   </span><br/> <br/> <br/><span> (坐吧檯的客人)</span></p>

<p><span>沒多久後，摩卡與鮭魚捲上桌，</span></p>
<p><span>摩卡滿滿的鮮奶油上立著一顆咖啡豆，視覺上頗為誘人，</span></p>
<p><span>熱愛咖啡的小弟先小啜了一口，</span></p>
<p><span>熱牛奶加上濃郁的鮮奶油，與咖啡相當搭，</span></p>
<p><span>咖啡味相當明顯，但卻沒什麼苦味，</span></p>
<p><span>配合著巧克力的香甜，極簡的咖啡比想像中還要來得好。</span><br/> <br/><span>    </span><br/><span> (摩卡)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=51&amp;f=1376696309.jpg" target="_blank"></a> </span><br/><span>  </span><br/> <br/><span>以煙燻過的生鮭魚片將高麗菜絲與少量的洋蔥絲捲起來，</span></p>
<p><span>裡頭沾一點黃芥茉，擠一點附在盤中的檸檬片淋上去，</span></p>
<p><span>味道相當可口。</span><br/> <br/><span> (煙燻鮭魚捲)</span></p>

<p><span>但並無令人驚艷之處，不過氣氛宜人，</span></p>
<p><span>是愛貓者的天堂與久留之處。</span><br/><span>    </span><br/><span> (離去前跟貓咪與店員SAY GOOD-BYE)</span></p>
