  地址：台北市大安區忠孝東路4段205巷7弄11號(微風2館正後面)<br/>

<br/>
<br/>
　聽到消息說將主打日本相當風行的甜點"蜜糖吐司"的Dazzling　Cafe，<br/>
<br/>
  已在前幾天開幕了，這幾天實在是忍不住了，<br/>
<br/>
　便在禮拜三下午特地前去。<br/>
<br/>
<br/>
<br/>
<br/>
　到了忠孝敦化路口，從Haagen Dazs一旁的巷子穿進去，<br/>
　<br/>
　不其然地遇到了之前就很想去一次的乾杯BAR，<br/>
<br/>
  正值下午五點剛開店，見機不可失，<br/>
<br/>
  便先進去吃頓晚飯。 <br/>
<br/>
<a href="http://www.wretch.cc/blog/kuni/11595272" target="_blank">[ 詳情請見：乾杯BAR(請用滑鼠點選) ]</a><br/>

  吃個半飽後趕緊付了帳，信步走至鄰近不遠的Dazzling Cafe，<br/>
<br/>
  從外觀來看相當高雅，走十足的雅痞歐洲風，<br/>
<br/>
　裡頭人影晃動，燈火輝煌，走進店內，<br/>
<br/>
　就馬上發現那公主般的鑽石型門把設計，<br/>
<br/>
　真是夢幻，擺明就是要拉住所有少女們的心呀！<br/>

<br/>
　因客滿，在等候區待了好一陣子才有空位，<br/>
<br/>
　就座後馬上跟女僕妝扮的可愛服務生點了"就愛草莓蜜糖吐司"(NT.220)，<br/>
<br/>
  為什麼要稱"就愛"呢？<br/>
<br/>
　據斯文的店經理解釋是，因為除了外頭的草莓以外，<br/>
<br/>
　連裡頭的冰淇淋也是草莓口味的，所以才命名為"就愛草莓"，<br/>
<br/>
　相當有趣的說法。<br/>
<br/>
<br/>
　當下順便點了一杯經理推薦的白巧克力摩卡(NT.150)<br/>

<br/>

<br/>
　然後就是相當豪華的"就愛草莓蜜糖吐司"登場了，<br/>
<br/>
　無比夢幻般的蜜糖吐司有如少女漫畫裡常常出現的城堡，<br/>
<br/>
　上頭擺著直入天際的巧克力旋梯，與滿滿的草莓果實切片，<br/>
<br/>
　聽說是經理當天親自去濱江市場挑的大湖草莓呢。<br/>
<br/>
  風行日本的蜜糖吐司作法是將三分之一條土司挖空，<br/>
<br/>
　裡層按九宮格切加入楓糖與外層拿去烤，<br/>
<br/>
  烤完後放回挖空吐司裡，上面鋪滿冰淇淋與其他配料，<br/>
<br/>
　光形容起來就覺得是一道相當可口的甜點了，<br/>
<br/>
　更何況那夢幻般地城堡外觀，堪稱第一級的少女殺手呀。<br/>

<br/>
  這一道甜點吃法很特別，首先使用一旁附的鐵夾，<br/>
<br/>
　先將上頭的配料擺在盤子上，然後夾起裡頭的楓糖吐司條，<br/>
<br/>
　這時候可以將楓糖吐司條沾滿那冰淇淋再入口，<br/>
<br/>
　或是直接先吃上面的配料，或先吃草莓皆可，<br/>
<br/>
　純看各人喜好。<br/>

　看似夢幻的甜點似乎有一些可惜的地方，<br/>
<br/>
　首先是草莓似乎沒有選好，甜度不夠偏酸，<br/>
<br/>
　烤吐司部分其實口感也沒想像中的那麼好，<br/>
<br/>
　不過光是這高雅的氣氛，與宅男們酷愛的女僕裝，<br/>
<br/>
　華麗的蜜糖吐司配上童話般的白巧克力摩卡，<br/>
<br/>
　這實在不失為一處很適合情侶們來私奔的好地方呢。<br/>
