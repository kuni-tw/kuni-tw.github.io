-----------------------------------------------------------<br/>
厚切牛排<br/>
地址：台北縣板橋市南雅南路2段164號<br/>
電話：(02)8966-6801<br/>
-----------------------------------------------------------<br/>
    <br/>
    <br/>
　自某次在網路上不經意看到有關這家牛排店的食用心得後，<br/>
    <br/>
　小弟從此便得了不吃牛排會死的病，直到挑了黃道吉日，<br/>
    <br/>
　與友一同前去，說也奇怪，此病一下子竟不藥而癒了。<br/>
    <br/>
    <br/>
　小弟心目中的美食其實很簡單，基本上，食材高檔與否，<br/>
    <br/>
　常常不是重點所在，另外價位過高，當然也容易有距離感，<br/>
    <br/>
　但真正的重點在於，一，外在部分，比如餐廳裝潢與氣氛。<br/>
<br/>
　二，內在部分，比如食物美味與否。(道不道地與否則是另一層次問題)<br/>
    <br/>
　三，服務部分。　　<br/>
    <br/>
　當然先聲明一下，這些畢竟是個人主觀問題，<br/>
    <br/>
　比如有些人愛吃辣有些人愛吃酸，青菜蘿蔔各有其所好，<br/>
    <br/>
　食物美味與否本身並沒有對錯問題。<br/>
    <br/>
    <br/>
    <br/>
　噗，先進入正題吧，<br/>
    <br/>
　這一天傍晚抵達板橋的厚切牛排店門前時，<br/>
    <br/>
　望著這些圍繞在店門口的人群，不禁倒呼了一口氣，<br/>
    <br/>
　不過，人群表面上看似稀稀落落，其實不然。<br/>
    <br/>
　等到我走進店裡，跟服務生要了號碼，恩，記得是四十幾號，<br/>
    <br/>
　此時方知前頭竟然還有二十幾組號碼在排著，<br/>
<br/>
　看來這些土地蛇客人早有準備，紛紛趁等候時間四處去走走。<br/>
<br/>
　真是太有經驗啦，恩，那下一次我要帶一本小說來。 <br/>
<br/>
<br/>
<br/>
    <br/>
　個人向來是不愛排隊的，第一，浪費時間，<br/>
    <br/>
　第二，還是浪費時間。<br/>
    <br/>
　小弟的時間實在是太寶貴了，就算是非排隊不可的三味食堂，<br/>
    <br/>
　個人一樣嗤之以鼻，一看到人潮，馬上走人。<br/>
    <br/>
　吃到的唯一那兩次，小弟擔保絕對沒有排到隊，<br/>
    <br/>
　只是有挑對時間去而已。 　<br/>
<br/>
<br/>
<br/>
    <br/>
　但這一次小弟真的乖乖排隊了，沒別的原因，<br/>
    <br/>
　透過落地玻璃窗往裡頭一看，明亮乾淨的空間與宜人的氣氛，<br/>
    <br/>
　感覺相當新奇，加上門口的大字報上：美國進口牛肉<br/>
    <br/>
　從最近幾次吃牛肉的經驗來看，<br/>
    <br/>
　美國牛不知道什麼原因肉味似乎較濃， <br/>
<br/>
　反正年紀大了，為了肉味十足的牛排，<br/>
    <br/>
　排個隊似乎還說得過去。<br/>
    <br/>
<br/>
<br/>
  (厚切牛排店門口)<br/>

    <br/>
　就座後，馬上跟服務生點了上肩沙朗，20oz，NT.480，<br/>
    <br/>
  友點了特級肋眼沙朗，10oz，NT.400，都是五分熟。<br/>
    <br/>
　此時開始用碼表算時間，真的不誇張，<br/>
    <br/>
　點餐後，過了足足28分鐘，主餐才上桌，<br/>
    <br/>
　上桌後，才發現是12oz的沙朗，經我提醒後，<br/>
    <br/>
　服務生才不好意思的說，20oz的已賣完了。<br/>
    <br/>
　真是的，點餐時怎麼不說一聲呢？<br/>

<br/>
<br/>

(菜單)<br/>

<br/>
  等候餐點時，才發現現場沒有沙拉吧，<br/>
<br/>
　只有供客人自助烤的大蒜麵包，玉米濃湯及飲料。<br/>
    <br/>
　大蒜麵包真的相當推薦，蒜香四溢，有五香的感覺，<br/>
    <br/>
　要不是擔心吃太多待會牛排會吃不下，而且還得自己烤，<br/>
    <br/>
  不然我一定要一次吃十片(設計對白)<br/>

<br/>

　終於牛排上桌，望著友人點的特級肋眼沙朗，<br/>
    <br/>
　顏色偏淺，切下一口，肉質還不差，只是以四百的價錢來說，<br/>
    <br/>
　似乎有點偏乾硬，呃...<br/>
    <br/>
  我的12oz沙朗可就大不同了！<br/>
    <br/>
　難以置信的第一口：肉味肉汁滿溢，<br/>
    <br/>
　光寫這一段文字就足以讓小弟重新回想個半天，<br/>
    <br/>
　邊想嘴角邊流口水。<br/>
<br/>
　鮮嫩多汁，肉味十足，如此厚重的牛肉一塊塊的進入嘴裡，<br/>
    <br/>
　細細咀嚼，小弟突然間進入到人間天堂，<br/>
    <br/>
　這真是令人難以忘懷的美味呀。<br/>

　吃完後才發現隔壁有人一次點兩客，<br/>
    <br/>
　細問之下，原來這裡可以DOUBLE點餐，<br/>
    <br/>
　兩客加在一起可減70元，感覺真是划算，<br/>
    <br/>
　意猶未盡的小弟馬上又點了一客雞腿排，<br/>
    <br/>
　減掉70變成NT.100，沒多久雞腿排也上了。<br/>
    <br/>
　相當大的無骨雞腿趴在熱騰騰的鐵板上，<br/>
    <br/>
　肉質鮮嫩，但完全沒有剛剛那沙朗的驚人風味，<br/>
