<br/>
<br/>
<br/>
<br/>
<br/>
　下午抽了空沿著省道飆去了台中，晚上時順便去拜訪了朋友的朋友所開的咖啡廳，<br/>
<br/>
　那就是在網路上還頗有名氣的Mojo咖啡廳．<br/>
<br/>
　一進店門，馬上感受到那與眾不同的空間感，配合著光影，真是美極了，<br/>
<br/>
　加上事先早已得知老闆是個熱愛攝影的高手，本想與老闆聊幾句，<br/>
<br/>
　可惜他剛好在Retro：他另一家咖啡館．<br/>
<br/>
<br/>
　<br/>


　點了一杯冰單品，可惜忘記了是哪裡的豆子，<br/>
<br/>
　記得好像是蘇門達臘的，很好喝，感覺是支中烘的豆子，<br/>
<br/>
　後來也跟漂亮的女店員確認了．<br/>
<br/>
　他們的單品水準真的滿高的，順口加上香味適中．<br/>
<br/>
<br/>
<br/>
<br/>
　後來點了一杯烤布蕾其諾：他們的花式咖啡，<br/>
<br/>
　利用噴火槍將香草口味奶泡上的砂糖微烤，成為脆脆的焦糖表層，<br/>
<br/>
　配上淡淡的肉桂香，唔，真是可口的一杯饗宴．<br/>
<br/>
　可惜後來點的一杯摩卡，味道我覺得很普通，<br/>
<br/>
　鮮奶油放得有點過多，倒是上面那隻向我招手的小手攪拌棒吸引了我不少注意力．<br/>
<br/>
  另外還有點他們的手工餅乾：綜合堅果，櫻桃核桃與巧克力口味各一塊，<br/>
<br/>
　口感沒什麼特別的地方，不過配上他們的咖啡之後，真有畫龍點睛之妙呢．　<br/>
