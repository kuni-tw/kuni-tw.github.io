<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　從小在維也納長大的我,對於能找到這樣一家餐廳真令人感到莫名的興奮,<br/>
<br/>
  尤其沒事在台北就想來兩口咖啡,<br/>
<br/>
　卻被迫得入口"四不像的維也納咖啡"為此常常非常困擾,<br/>
<br/>
　自從不經意得知有這樣一家餐廳後,一直很想找機會來叨擾一番.<br/>
<br/>
　禮拜五下午一個空檔抓住機會,信步至SOGO敦南館的Kaffee Sound<br/>
<br/>
　因時間不夠加上還不餓,遂只單點一杯我所熟悉的Wiener Melange,<br/>
<br/>
　這是維也納人最愛喝的咖啡了,有時候簡稱就直接說 "ein Wiener bitte"(加一點口音)<br/>
<br/>
  做法非常類似台灣所謂的Latte,這以前也曾讓我感到困擾過呢.<br/>
<br/>
  Wiener Melange牛奶比例較Latte少,且奶泡打的非常綿密,澎澎的滿出杯緣.<br/>
<br/>
　碰巧以前在維也納有在餐廳裡打過工,當時也學過作各種義式咖啡,<br/>
<br/>
　因此深知Wiener Melange好喝與否與奶泡技術有絕對的關聯.<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　感想:這真是令人懷念的好咖啡呀,雖然個人偏好奶泡打得再高一點,<br/>
<br/>
　不過他們奶泡顯然的打得很用心,口感綿密,<br/>
<br/>
  Espresso部分中規中矩,苦味適中沒什麼令人印象深刻的地方,<br/>
<br/>
  事後曾試著詢問他們豆子是哪來的,不過服務的小姐並不清楚,<br/>
<br/>
  加上我時間也不夠就急著先走了,改天有機會再去問問好了.<br/>
<br/>
