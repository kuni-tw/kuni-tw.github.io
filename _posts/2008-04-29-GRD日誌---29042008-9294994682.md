<div class="article-content-inner" id="article-content-inner">
驚訝的發現時間又跳得老快,它一絲不苟,<br/>
<br/>
                                                                                
  緩緩地爬向床上掀起被子鑽了進去,然後躺在裡面作夢.<br/>
<br/>
                                                                                
  就這樣,現在竟然已四月三十號了,我的二十九號哪兒去了呢?<br/>
<br/>
                                                                                
　難不成是我整天太忙,所以冷落到了妳?<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  下午依舊抽空跑去想用喜愛的L'amour法式特製牛肉三明治,<br/>
<br/>
  咿呀,濃稠的麥香與微鹹帶點辛辣的味覺混著烤出來的酥脆外皮口感,<br/>
<br/>
  唔,真是過癮極了.<br/>
<br/>
  常常有種很深切的感覺是,好像不論何時何地,<br/>
<br/>
  不管我心裡藏有多少的憂愁,裹著深沉的低潮影子,<br/>
<br/>
  我似乎可以藉由簡單的美味將上述一掃而空.<br/>
<br/>
  這似乎是件好事,因為這代表我有辦法輕易的忘卻許多負面的回憶.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一路上我心想著這是多麼愜意的生活,然而天公卻不作美,<br/>
<br/>
  沒事戳了戳飽滿的鹽水袋,幸好數量不足,也沒些許妨礙到.<br/>
<br/>
  臨時決定半路停車繳清帳單,走在泥濘的工地上,<br/>
<br/>
  不禁擔心了一下蛇皮款,果不其然,新鮮而又黃橙的污漬正水汪汪的望著我.<br/>
<br/>
  可不是嗎,這裡將實用與花俏的對比下了最合適的注解了.<br/>
<br/>
  但往後我將拋棄花俏投奔實用嗎?<br/>
<br/>
<br/>
 　那可不,誰說花俏不實用的,那人肯定不實際.<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　就這樣,晚間重新落入了陷阱吃了過飽,這不當的輪迴來回地的跳針,<br/>
<br/>
  搞得我又開始與持續升高的血糖抗戰.<br/>
<br/>
  我快走一步想擺脫擺脫它的糾纏,卻無功而返,因為它熟悉我.<br/>
<br/>
  人類進化史上最重要的一環,不畏挫折,永遠堅持信念,<br/>
<br/>
  這成為今晚十一大課題之一,另十大好像從來也沒變過.<br/>
<br/>
<br/>
<br/>
  話說回來,庸庸碌碌若也可以順理成章,<br/>
<br/>
  那我也沒有白活了.<br/>
<br/>
<br/>
<br/>
