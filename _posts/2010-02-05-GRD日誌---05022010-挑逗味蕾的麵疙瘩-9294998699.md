<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這是一家已存在於食記清單裡許久的餐廳，<br/>
<br/>
　麵疙瘩是小時候最愛母親作的料理之一，更沒想到，<br/>
<br/>
　多年之後的今天，竟然可以在公館吃到麵疙瘩的創意版。<br/>
<br/>
<br/>
<br/>
<br/>
　座落在巷弄間的挑逗味蕾，單順著住址找去，<br/>
<br/>
　困難度其實相當高，走進巷子時碰巧又遇到幾台攤車出來，<br/>
<br/>
　一時之間因看不到招牌，便又折返了一次，<br/>
<br/>
　好不容易過了些時候，重新問路找回目的地。<br/>

　沒多久後，本日湯上桌，顏色還滿漂亮的羅宋湯，<br/>
<br/>
　番茄味相當濃郁，但嚴格說來，除了番茄味並沒有任何其他味道，<br/>
<br/>
　找了服務生一問之下，原來這是一碗顧及素食者需求的羅宋湯，<br/>
<br/>
　這倒是滿貼心的，不過個人不是很喜歡。<br/>

<br/>
　裡頭還有番茄，蘋果，柳橙及烤麵包丁，<br/>
<br/>
　老實說，醬料沒什麼特別的，與番茄湯，呃，羅宋湯一樣，<br/>
<br/>
　味道層次不夠。<br/>

　主餐麵疙瘩上桌之時，小弟才發現一個較怪異的現象，<br/>
<br/>
　那就是桌上從頭至尾竟然沒有擺餐巾紙給客人，<br/>
<br/>
　環顧四周一樣也看不到，只得跟服務生討，<br/>
<br/>
　得到的是一坨皺在一起的面紙，一時之間還頗納悶的，<br/>
<br/>
　直到算帳時才跟櫃檯反映，沒想到櫃檯竟回應，<br/>
<br/>
　他們都是放在桌上還是某處任客人自行取用的。<br/>
<br/>
　說實在的，到最後小弟還是看不到餐巾紙或是面紙的隱藏地點。<br/>
<br/>
　不過這是後話，暫先略過不提。<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　挑逗味蕾的麵疙瘩口感很棒，很好吃，<br/>
<br/>
　調和式辣味雞肉片辣得恰到好處，又香味宜人，<br/>
<br/>
　肉片雖然稍硬了些，不過還在可接受範圍，<br/>
<br/>
  可惜的是份量其實不大，加上黑糖凍吃完後個人竟然只有五分飽，<br/>
<br/>
　走出了門外，肚子竟然又餓了起來。<br/>
