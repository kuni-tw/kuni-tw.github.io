<p><span>(主廚綜合開胃菜)<br/><br/> -------------------------------------------------<br/> Primo Pizzeria<br/> 電話：(02)2528-2396<br/> 時間：11：30~21：30<br/> 地址：台北市光復北路11巷99號之一<br/> -------------------------------------------------</span></p>
<p><br/><span>去年八月就聽說過主打拿坡里式披薩的Trattoria di Primo準備要開二店的消息，</span></p>
<p><span>不過由於吃過總店不是很喜歡所以一直沒有拜訪二店的打算</span><br/> <br/><span>－－－＞<a href="http://kuni.pixnet.net/blog/post/294997442-grd%E6%97%A5%E8%AA%8C---08112008-trattoria-di-primo%E8%A9%A6%E8%B3%A3" target="_blank">(總店初訪請滑鼠點選：Trattoria di Primo)</a></span></p>
<p><span>這一日中午在京華城附近臨時有個小空檔，</span></p>
<p><span>查了一下網路發現二店就在附近腳程不到幾分鐘，</span></p>
<p><span>便直接走了過去。</span></p>
<p><br/><span>(店門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198004.jpg" target="_blank"></a></span><br/> <br/> <br/><span>店內裝潢還滿漂亮的，點了商業午餐托斯卡尼鄉村披薩，NT.300</span></p>
<p><span>附季節沙拉，Primo主廚湯，飲料選冰咖啡，</span></p>
<p><span>另加點一份今日主廚綜合開胃菜，NT.420</span><br/> <br/> <br/><span>(店內一景)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198006.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(桌邊隨拍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198005.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(桌上刀叉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198007.jpg" target="_blank"></a></span><br/> <br/><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198008.jpg" target="_blank"></a></span><br/> <br/><span>首先上桌的是凱薩沙拉，新鮮的蘿蔓心上灑滿現刨的義大利起司Grana Padano，</span></p>
<p><span>清脆的口感搭配著濃濃乳酪香讓口腔裡的味覺開始暖身做準備。</span></p>
<p><span>緊接而來的主廚綜合開胃菜是一道相當漂亮的擺盤，</span></p>
<p><span>裡頭包括帕馬火腿(Prosciutto)，義式臘腸Salami，番茄乳酪(Caprese)，</span></p>
<p><span>小牛菲力佐皮埃蒙特鮪魚醬蘆筍(Vitello Tonnato)，起司小點以及蘆筍。</span></p>
<p><span>是各種單點冷盤前菜的迷你綜合版，不同的口感與味道創造出驚人的衝擊，</span></p>
<p><span>馬上讓人感受到幸福，不過Salami部分希望Primo以後還能找到味道更濃郁點的。</span><br/><br/><span>(凱薩沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198009.jpg" target="_blank"></a><br/> <br/>(凱薩沙拉微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198010.jpg" target="_blank"></a><br/> <br/>(廚綜合開胃菜)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198011.jpg" target="_blank"></a><br/> <br/>(廚綜合開胃菜特寫)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198012.jpg" target="_blank"></a><br/> <br/>(小牛菲力佐皮埃蒙特鮪魚醬蘆筍)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198013.jpg" target="_blank"></a><br/> <br/>(帕馬火腿)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363198014.jpg" target="_blank"></a></span><br/> <br/> <br/><span>今天的主廚湯是小弟相當喜歡的牛肝菌濃湯，味道還不錯，</span></p>
<p><span>然後就是重頭戲托斯卡尼鄉村披薩了，上頭有著小弟很愛的Zucchini，</span></p>
<p><span>以前在Wien時常沒事自己買來煎，很香口感又棒。</span></p>
<p><span>餅皮部分感覺起來似乎有做過調整，</span></p>
<p><span>以前一店較類似蔥油餅的口感與麵香如今似乎已不復見，</span></p>
<p><span>取而代之的則是更Q軟的麵皮，造就更豐富的口感層次與更貼近小弟的味蕾。</span><br/> <br/><span>(附餐咖啡)</span></p>
