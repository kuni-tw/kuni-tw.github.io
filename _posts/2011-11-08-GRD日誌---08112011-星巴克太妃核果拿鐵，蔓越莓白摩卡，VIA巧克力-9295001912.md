<br/>
<br/>
<br/>
　(太妃核果拿鐵，蔓越莓白摩，VIA巧克力)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　期待了一整年的星巴克太妃季重新到來，在十一月八號。</span></p>
<span>
<p><br/>
　記得首次喝到她是在2002年冬天的台北，</p>
<p>　配合外頭冷澀澀的寒風走進玻璃門內一下子就被那甜美的太妃糖香氣所迷住，</p>
<p>　像戀愛般融出幸福滿溢在整家星巴克門市裡，</p>
<p>　從此成為每年冬天我必然期待的制約。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　今年的太妃來得似乎正是時候，這幾天台北在不停綿雨下感覺好像驟降了幾度，</p>
<p>　帶著感恩的心情與兩只平常收藏不用的馬克杯重新走回2002年的時空背景，</p>
<p>　點了熱太妃與首度出現的蔓越莓白摩卡(中杯減杯扣後各NT.115)</p>
<p>　另外加點一盒VIA咖啡巧克力(NT.250)</p>
<p><br/>
<br/>
　星巴克的太妃核果拿鐵很特別，喝了這麼多年一直到今天仍如此經典，</p>
<p>　微溫的鮮奶油適切擋住下層燙嘴的義式咖啡，</p>
<p>　藉著熱氣將太妃糖粒的香帶進我們的鼻腔，然後綿密滑順的香甜奶油咖啡登場，</p>
<p>　天氣越冷這太妃所提供的幸福就愈明顯愈濃厚。</p>
<p><br/>
　星巴克這一次推出的新款義式咖啡蔓越莓白摩卡較太妃甜許多，<br/>
　<br/>
　單喝容易因過甜而膩口但若配上微焦苦的VIA巧克力那甜將變得有層次，</p>
<p>　喝到一半時記得要使用木製攪拌棒攪和一下才能喝到底部沉澱的蔓越莓果粒唷。<br/>
