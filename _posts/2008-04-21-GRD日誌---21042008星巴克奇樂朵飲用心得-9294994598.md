<br/>
<br/>
  這一次星巴克甫推出的奇樂朵拿鐵算得上是對上我的舌頭了(大感謝),<br/>
<br/>
  剛推出的第一天夥伴們就開始熱情的邀請我們試喝,<br/>
<br/>
  那時候的印象其實還滿失望的,感覺跟焦糖馬奇朵沒差太多.<br/>
<br/>
  現在回想起來,猜測是當時試喝無法喝到它的優勢所在:<br/>
<br/>
　"那迷人的牛奶糖香!!!"<br/>
<br/>
　那一次試喝用的熱奇樂朵溫度早已降到微溫了,香味大概也揮發得差不多了,<br/>
<br/>
　這大概就是其失敗之處吧.<br/>
<br/>
  不管如何,禮拜六一大早又跑去敦信門市朝聖的我不信邪的又點了一杯,<br/>
<br/>
  而且這一次點大杯的,因為前天晚上只睡了三四小時,一大早又要教課,<br/>
<br/>
  整個人可說是面目可憎勒,沒有兩個Shot大概會撐不到兩小時吧,<br/>
<br/>
　另外再要求夥伴弄熱一點,對啦,個人偏好熱一點的口感.<br/>
<br/>
<br/>
<br/>
　那一次總算對到胃了:<br/>
<br/>
　"牛奶糖般的芳香,熱呼呼又偏甜的口感配上濃郁的奶香!"<br/>
<br/>
　<br/>
<br/>
<br/>
  雖然個人喜好偏甜的口感,但下一次我還是想試其他種不同的甜度組合,<br/>
<br/>
　而且冷的冰的我也還沒嘗過呢.<br/>
<br/>
<br/>
  最後總結:<br/>
<br/>
  雖然價位稍稍偏貴(Grande大小就要140嚕)<br/>
<br/>
　但是還是得說,我終於又找到一款可以陪伴我直到冬天太妃季來臨的飲品呢!<br/>
<br/>
