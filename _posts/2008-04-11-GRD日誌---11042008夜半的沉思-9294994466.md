　半夜常駐的NY BAGEL:<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  夜幕低垂,順著寂靜的臥房看著滿地的呻吟此起彼落,<br/>
<br/>
　望著活潑離我遠去,心情漸漸淹沒在成天下來堆積如山的疲累裡.<br/>
<br/>
  或許這時候我需要一盞搖曳的燭光,與黑暗與寧靜成明顯對比,<br/>
<br/>
  換來一絲類藝術的氣息,趕走心中的俗氣,然後昇華為早晨飛揚的雲彩,<br/>
<br/>
  這樣我就可以安然睡去.<br/>
<br/>
  或許我願意以十年壽命來交換我這一夜好夢,連續六小時沒有紛紛擾擾,<br/>
<br/>
  只有規律的鼻息聲與滴答聲伴著睡魔.<br/>
<br/>
<br/>
  乍聽之下真是不等價呀,但卻相當值得.</div>
