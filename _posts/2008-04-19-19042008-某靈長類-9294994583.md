<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
  說到這人呀有時候連我都恨得牙癢癢的,就算踹個幾十下,<br/>
<br/>
  還是無法彌補這廝的無藥可救之處,君不見號稱進化版長臂猿整天駕長車跨橋,<br/>
<br/>
  日上三竿而入眠,傍晚即起,刷牙漱口必迅速確實力求精簡.<br/>
<br/>
<br/>
  最近看了一些時事新聞,部分扯到現代藝術的爭議性,<br/>
<br/>
  尤其是當與世俗道德關相牴觸之時所引發的話題性.<br/>
<br/>
  夜深了與友對此議題深入探討了一番,不著邊際的漫談許久,<br/>
<br/>
  當中插入了為數不少的刻薄諷刺,算是另類的聊齋誌異.<br/>
<br/>
  漸漸進入自我表現的本質有時候並不是一種解脫,反而是負擔,<br/>
<br/>
  一種無窮無盡追求究極耶穌光的浪漫.<br/>
<br/>
<br/>
　這特殊的靈長類此生大概得不幸的蒙著賽馬眼旁的黑布,往前直行,<br/>
<br/>
  當障礙慢慢浮現時腳幅跨大,沒料到追上之後的世界仍舊深不可測.<br/>
