<br/>
<br/>
  當溫度指向漲停板時,手下開始驚慌失措六神無主,只因為老闆沒錢發餉,<br/>
<br/>
  主因至今未明.<br/>
<br/>
  直到晚間事態漸漸惡化,汗流浹背,頭兒顫慄,打開錦囊取出一帖妙計,<br/>
<br/>
  和著苦澀的心情吞入苦惱的胃袋.<br/>
<br/>
  三十分鐘後感受到第一波淨化,然後昏眩,失神,然後便是夜襲來臨:<br/>
<br/>
  走在伸手不見五指的地道裡,街鼠四竄,間或得堤防可怕的黑暗獵人,<br/>
<br/>
  直屬邪惡的惡魔.<br/>
<br/>
<br/>
  走著走著,不知已到了黃泉河畔...<br/>
<br/>
