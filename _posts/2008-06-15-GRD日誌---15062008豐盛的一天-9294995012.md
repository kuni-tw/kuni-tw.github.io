<br/>
<br/>
<br/>
　豐盛的晚餐:新鮮沒加明礬的澎湖海膽,北海道明太子,現磨的山藥泥,紅甘肚...<br/>
<br/>
  還有比這更豐盛的嗎?<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最近拍照時常常想嘗試亦或說是練習一些構圖技巧，<br/>
<br/>
　雖然幾乎每一次都失敗，我依舊樂此不疲，原因無它，<br/>
<br/>
　因為這就是學習．<br/>
<br/>
　<br/>
　請恕我直言，但我不認為這世上有任何所謂的天才，<br/>
<br/>
　這些人據說不論學習什麼都能一蹴可幾，無師自通．<br/>
<br/>
　許多人常常將天才這詞輕易的套用在成功者的身上，<br/>
<br/>
　乍聽之下頗為有理，實際上卻荒謬無比，<br/>
<br/>
　對那些比別人努力不知幾百倍，視挫折為知己的鬥士們真是一種屈辱呀！<br/>
<br/>
　<br/>
　向來在網路上常常看到一些沒品的好事者，唯恐鎂光燈照不到，<br/>
　　<br/>
　深深懼怕被人群遺忘，為此不惜犧牲一切，<br/>
　<br/>
　只要能站在聚光處，哪怕只是一秒鐘也在所不惜．<br/>
<br/>
　這也是台灣社會走向近年來的小趨勢：<br/>
<br/>
　許多年輕人漸漸好逸惡勞，加上家長們只看得到奢華的表面形象，<br/>
<br/>
　教育理念上的失敗導致學生眼光日益淺顯，只求得表面上的短暫成功模式，<br/>
<br/>
　然後，隨之而來的是面對挫折時束手無策進而坐以待斃．<br/>
<br/>
<br/>
<br/>
　唉，話一離題過遠有如黃河崩堤一發不可收拾！<br/>
　<br/>
