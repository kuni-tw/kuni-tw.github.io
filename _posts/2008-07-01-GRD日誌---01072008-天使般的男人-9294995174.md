<br/>
<br/>
<br/>
<br/>
<br/>
  最近看了不少感情討論區的文章，看著不少孤男寡女為情所困，<br/>
<br/>
　不禁搖搖頭，但一回想起自己以前莽撞癡情的純真模樣，<br/>
<br/>
　也不爭氣的笑了起來，喔，不，我可沒笑自己傻呢！<br/>
<br/>
<br/>
<br/>
  當然以今天的角度重新審視當初少不更事的我，那真是荒誕不經，<br/>
<br/>
　讓人笑掉大牙，但是那時候的我卻再認真不過了．<br/>
<br/>
　曾以紙片人的身分搖搖晃晃走在瑪莉亞街上，一度不到60公斤，<br/>
<br/>
　遇見情敵與她牽手漫步，於是我被對方推了一下，力道雖小，<br/>
<br/>
　我卻眼直冒金星，那一次真讓我印象深刻，<br/>
<br/>
　至此我曾發誓絕不再讓這種事情發生了！<br/>
<br/>
<br/>
<br/>
　我個人向來不喜歡將情愛這種陰柔的事大量的擺置在我的生活裡，<br/>
<br/>
　因為感覺很不健康，很沒有男子氣概．<br/>
<br/>
　總覺得這樣的男人其實也沒有什麼擔當，更沒有女人會看得起，<br/>
<br/>
　男人色可以，愛看A片，偶爾打打嘴炮也無傷大雅，<br/>
<br/>
　但是濫情真的就很頹廢了，不要說女人看你不起，連男人也是！<br/>
<br/>
<br/>
　<br/>
　說來說去，完美男人這角色可還真難當呀，<br/>
　<br/>
　平時成熟男人風，沒事又要有如大男孩般靦腆的笑容，<br/>
<br/>
　又要有女人般的體貼與善解人意，另，突然有一天女人有如晴天霹靂的發火，<br/>
<br/>
　怎麼解釋都以無效論時，只好模擬在拳擊擂台上休息時刻的情境，<br/>
<br/>
　在旁一邊遞上毛巾一邊按摩，當然，若能像店小二在旁陪笑那就更完美了！<br/>
<br/>
　<br/>
<br/>
<br/>
  當然，我也承認女人也不會好當到哪裡去！<br/>
