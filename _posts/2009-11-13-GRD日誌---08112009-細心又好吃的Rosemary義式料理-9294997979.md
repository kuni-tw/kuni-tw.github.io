<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
　　(Rosemary的番茄海鮮清湯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這個月總算要退伍啦，這一年來食記更新速度超慢，<br/>
<br/>
　只能趁放假有空時多寫幾篇。<br/>
<br/>
　看到留言板裡幾家餐廳邀請的留言，<br/>
<br/>
　其中一家似乎有一點印象，就是在中山站附近的Rosemary，<br/>
<br/>
　聽說他們才剛開幕兩三個月，餐廳外觀看起來還頗具質感，<br/>
<br/>
　便撥了電話過去，約了收假當天中午過去。<br/>
<br/>
　<br/>
<br/>
　餐廳資料：<br/>
<br/>
　Rosemary螺絲瑪莉義麵坊<br/>
　地址：台北市中山區南京西路12巷13弄9號<br/>
　電話：(02)2521-9822<br/>
<br/>
<br/>
<br/>
　地址看起來還滿複雜的，不過卻意外的好找，<br/>
<br/>
　進了巷口，童話般的圍牆與門口便映入眼簾。<br/>
<br/>
　進了門，寬敞的空間明亮又乾淨，<br/>
<br/>
　不過在這諾大的容積裡，桌椅擺置的卻相當少。<br/>
<br/>
　找了位置就座後，<br/>
<br/>
　看了菜單，發現完全沒有套餐類的組合，<br/>
<br/>
　親切的店長告知因為老闆刻意將單點價錢壓低，<br/>
<br/>
　讓大家都可以自己組合套餐。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (童話般的圍牆與門口)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
　　(餐廳內空間寬敞明亮)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(只有單點的菜單是其一大特色)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　看完菜單，聽了服務生建議點了：<br/>
<br/>
  1.海鮮燉飯(NT250)<br/>
<br/>
　2.菠菜海鮮焗烤麵捲(NT235)<br/>
<br/>
  3.番茄海鮮清湯(NT55)<br/>
<br/>
  4.奶油蘑菇濃湯(NT35)<br/>
<br/>
  5.提拉米蘇(價錢未知)<br/>
<br/>
　6.阿薩姆鮮奶茶(NT35)<br/>
<br/>
  7.鮮榨奇異果汁(NT55)<br/>
<br/>
<br/>
<br/>
  其實個人向來不喜歡在義大利餐廳點海鮮類的東西，<br/>
<br/>
　比如前幾次去的小蝸牛還是忠孝復興的Primo就都沒點，<br/>
<br/>
　原因是按照以往在台北吃過的經驗，<br/>
<br/>
　一，個人不大喜歡邊吃邊剝蝦殼的感覺，沒事要擦手又麻煩也就罷了，<br/>
<br/>
　　　口感又特別差！<br/>
<br/>
　二，海鮮類的料向來難控制火候，與其他配料相較之下，<br/>
<br/>
　　　口感較難與Risotto或是Spaghetti融合在一起。<br/>
<br/>
　還未上菜前，餐廳端上一盤免費招待客人的麵包條，<br/>
<br/>
　看起來相當普通，但入口宜人的甜味散出來，<br/>
<br/>
　一嚐就知道有用過心。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(麵包條香甜)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(店內裝潢一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　感想：<br/>
<br/>
  奶油蘑菇濃湯令人驚艷，才35塊的單價，<br/>
<br/>
　濃濃的湯裡頭滿滿的蘑菇，口感與調味皆超值。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(奶油蘑菇濃湯令人驚艷)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　番茄海鮮清湯，本日第一道驚奇，<br/>
<br/>
　兩尾相當大隻的蝦子，新鮮度驚人，<br/>
　<br/>
　蝦肉輕輕一拉便與尾殼脫落，乾淨俐落，<br/>
<br/>
　咀嚼之中竟散出少見的香甜味。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (番茄海鮮清湯，本日第一道驚奇!)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (乾就俐落的蝦殼，蝦子新鮮度驚人)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　菠菜海鮮焗烤麵捲，與一般的義式千層麵做法不同，<br/>
<br/>
　是用單片千層麵的麵皮捲起餡料，淋上白醬，<br/>
<br/>
　放在烤箱內焗烤。<br/>
<br/>
　老實說，以前在歐洲生活時，這是我最最討厭的義大利餐之一了，<br/>
<br/>
　口味超鹹，一直不懂為何義大利人對這個樂此不疲，<br/>
<br/>
　而Rosemary的口味並不是很道地，但是卻相當好吃。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(菠菜海鮮焗烤麵捲)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　個人最愛吃的義式料理其實是台北較少見的Risotto，<br/>
<br/>
　處理費時費工，而且正統又好吃的Risotto米其實取得不易，<br/>
<br/>
　不過因為以前求學時期曾吃過一家高檔餐廳，<br/>
<br/>
　當時的Risotto簡直讓我不敢相信自己的眼睛，<br/>
<br/>
　粒粒皆分明的飽滿又發亮的飯粒，<br/>
<br/>
　好像一道震懾人心的藝術品，<br/>
<br/>
　像一座小丘似地堆在白皙的圓盤中，<br/>
<br/>
　口感難以形容的Ｑ，味道中具有層次感，<br/>
<br/>
　令人難以忘懷。<br/>
　<br/>
  Rosemary的海鮮燉飯可惜用的不是正統的Risotto米，<br/>
<br/>
　而且水分偏多，不過仗著新鮮的海鮮料口感以及非常細心的調味，<br/>
<br/>
  還是很好吃。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (海鮮燉飯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　阿薩姆鮮奶茶，添加的是林鳳營鮮乳，<br/>
<br/>
　有牛奶糖的香味，一大瓶的奶茶要價才35，不錯。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　　(阿薩姆鮮奶茶)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  鮮榨奇異果汁，香濃清甜，<br/>
<br/>
　入口後馬上嚐到熟悉的味道，<br/>
<br/>
　馬上找來服務生，問道是否使用黃金奇異果來榨汁，<br/>
<br/>
　服務生不知，跑回廚房拿出來的果然是黃金奇異果，<br/>
<br/>
　這部分有加分呢！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(鮮榨奇異果汁，香濃清甜)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　提拉米蘇滿特別的，放在小馬克杯裡頭冷藏，<br/>
<br/>
　據店長解釋是因為刻意不放吉利丁，較不好凝固，<br/>
<br/>
　不過個人倒覺得還好，味道與口感皆普普。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (Rosemary的提拉米蘇使用小馬克杯盛著)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(提拉米蘇內觀)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(沒放吉利丁，口感較軟)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　整體說來，Rosemary的口味就是好吃，<br/>
<br/>
　雖然有些部分可能沒那麼正統，比如Risotto部分，<br/>
<br/>
　不過至少也比我在台北吃過的大部分義大利餐廳都好，<br/>
<br/>
　以價錢上來看，真的是相當超值。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (本人中文字真的很醜...)<br/>
<br/>
