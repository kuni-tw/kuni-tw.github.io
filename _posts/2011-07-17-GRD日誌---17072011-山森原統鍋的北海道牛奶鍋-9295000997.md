<p> </p>
<p><span><span>牛奶鍋名店山森原前陣子多次在網路上被熱烈地討論引起了小弟的好奇心，</span></span></p>
<p><span>平常不大吃所謂的台式平價小火鍋，這天中午趁著某位學生臨時請假空檔衝去。</span></p>
<p><span>山森原店面裝潢簡單低調以前路過多時都沒想進去看過，</span></p>
<p><span>今天可以一睹其廬山真面目了。</span><br/> <br/> <br/><span>(山森原統鍋店面樸素低調)</span><br/><br/><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207112.jpg" target="_blank"></a></span><br/><br/><span>不知從何時開始藉著全台的臭臭鍋熱潮逐漸興起了各式各樣的平價小火鍋，</span></p>
<p><span>鍋的湯底愈來愈多樣化舉凡麻辣，泡菜，麻油，牛奶與咖哩，</span></p>
<p><span>較高檔的諸如法式濃湯，壽喜燒還是石狩鍋等，</span></p>
<p><span>這些靠著價格上的優勢很受一般上班與學生族群們的歡迎，</span></p>
<p><span>這一次想吃吃看的便是其中的北海道牛奶鍋。</span><br/><br/><span>(店門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207113.jpg" target="_blank"></a></span><br/> </p>
<p><span>山森原的黃老闆曾在國內與國外多處任職過廚師，</span></p>
<p><span>四年多前終於決定獨立出來開店，當初因想壓低租金預算所以找到永和成功路的店面，</span></p>
<p><span>另外直接找原物料的源頭採買鍋裡的配料以便節省成本。</span></p>
<p><span>其中最值得一提的乃當家招牌人氣紅不讓的北海道牛奶鍋，</span></p>
<p><span>據老闆說使用的是北海道母牛的初乳，小弟就座後當然二話不說馬上點了一份。</span><br/><br/> <br/><span>(黃老闆本人)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207114.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(店內一景)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207115.jpg" target="_blank"></a><br/></span></p>
<p> </p>
<p><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207116.jpg" target="_blank"></a><br/><br/>(為"百元火鍋賺大錢"書中三家被挑中來訪問的店家之一)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207117.jpg" target="_blank"></a><br/><br/></span><span>山森原店內燈光昏暗裝潢有點簡陋，環顧四周發現用餐的幾乎都是學生族群，</span></p>
<p><span>自個兒斟了一杯這裡的冬瓜茶，不會死甜比想像中來得好喝。</span></p>
<p><span>沒多久後老闆親自端來輕冒白煙的大黑鍋到桌上，</span></p>
<p><span>撈了一會細看裡頭的料包括：兩三片豬肉，蔬菜，粉絲，豬血糕，蛤蠣，</span></p>
<p><span>豆皮，甜不辣，貢丸，阿...竟然有黑蝸牛肉(!?)</span></p>
<p><span>有趣的是鍋料上還放著幾片老闆自己做的豆花，</span></p>
<p><span>他說這非自己做不可，不然會不耐煮。</span><br/> <br/><span>(北海道牛奶鍋)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207118.jpg" target="_blank"></a><br/> <br/>(俯瞰一張，上面那片是豆花唷)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207119.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>喝了一會湯頭濃濃奶香四溢，實在是相當新奇的體驗，</span></p>
<p><span>沒多久後就得請老闆過來續湯，這可是不限次數的唷。</span></p>
<p><span>其實吃到最後有點撐了，但基於好奇心又加點了豆皮與凍豆腐各一份(NT.20)。</span></p>
<p><span>端來的時候看到那份量不禁愣住，豆皮似乎有經過特殊處理過耐煮好吃，</span></p>
<p><span>不過吃到最後實在是太撐了都沒吃完呀。</span></p>
<p><span><span>(後來加點的凍豆腐)</span></span></p>
