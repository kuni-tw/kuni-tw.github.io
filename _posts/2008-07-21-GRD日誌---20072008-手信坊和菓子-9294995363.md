<br/>
<br/>
<br/>
<br/>
  昨天下午在課餘時間閒逛了一下，不經意經過手信坊，<br/>
<br/>
　還滿喜歡日式和果子的我之前對於這一家店早已有所耳聞，<br/>
<br/>
　所以就信步走了進去．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
　因為時間不夠，只點了幾樣小東西帶回去與學生一起品嘗，包括：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　  花柿菓子，柿子乾的外皮裡包有麻糬，紅豆餡，<br/>
<br/>
　　外觀很討喜，一副就很美味的樣子，<br/>
<br/>
　　試吃時，外皮偏韌一點，紅豆餡不管是在甜味或口感上皆不夠細緻．<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
　　另外還有三種燒菓子，包括：              <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　1.小樽乳酪，內餡金黃，不過乳酪味顯然不太夠．            <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　2.宇冶金時，這個真的不錯，很濃郁的抹茶香呢．    <br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　3.九州鹿兒島-薩摩番薯，內餡色澤很漂亮，<br/>
　　<br/>
　　第一口就可以感受到些微細緻卻不會過多的肉桂味，很棒的組合呢．<br/>
