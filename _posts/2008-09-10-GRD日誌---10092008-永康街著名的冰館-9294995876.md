　　(超推薦的冰館綜合水果冰，這一盤尚未淋上煉乳)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
回台灣後自從去了一次永康街著名的冰館，就深深的迷上他們的新鮮水果冰，<br/>
<br/>
　我知道，總是有不少人對這一家名店有諸多批評，諸如：<br/>
<br/>
　"這家店愈來愈難吃了"<br/>
<br/>
　"又貴又不好吃，還是ＸＸＸ比較好吃，又多又便宜"<br/>
<br/>
　不過，對我來說，好吃就是好吃，沒什麼好質疑的．<br/>
<br/>
　我承認他們名氣愈來愈大，甚至今天中午去吃時，<br/>
<br/>
　坐在我前座的，從一開始的韓國口音，<br/>
<br/>
　沒多久竟變成日本口音，還真是國際化呀！<br/>
<br/>
　而永康街的冰館仗著巨額生意量，每日大量進貨新鮮的好水果，<br/>
<br/>
　直接在直銷農產地與果農簽訂供貨契約以保新鮮又漂亮的貨源！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
今天的綜合水果冰賣相依舊迷人，我看著那垂涎欲滴，黃澄澄的芒果，<br/>
<br/>
　切成大丁的奇異果上與鮮紅草莓沾滿著半透明狀的煉乳，<br/>
<br/>
　剉冰尖端上置放一顆美麗誘人的自製芒果冰淇淋，喔買嘎，<br/>
<br/>
　廢話不多說，我馬上一口咬下去！<br/>
<br/>
<br/>
<br/>
<br/>
　　(黃澄澄的自製芒果冰淇淋)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(切成大丁的奇異果上與鮮紅草莓沾滿著半透明狀的煉乳)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　雖然以前曾覺得他們的水果品質會有不穩的時候，不過，<br/>
<br/>
　也吃過幾家別人推薦過的水果冰，不是水果品質不夠，<br/>
<br/>
　就是糖水味過重，總之，我就是不喜歡．<br/>
<br/>
　而今天冰館的綜合水果冰表現真的非常好，也真是辛苦了這些果農呀！<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
  (絡繹不絕的冰館)<br/>
