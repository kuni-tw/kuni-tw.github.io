<div class="article-content-inner" id="article-content-inner">
不巧,雨天路滑,剛剛友敲我房門問起,<br/>
<br/>
  我納悶了一下,隨即嚇了一跳,一時之間語塞.<br/>
<br/>
<br/>
<br/>
  午後捎來一封悲傷的訊息,臨晨則是孤獨的,<br/>
<br/>
  昨晚則隸屬於俏皮.<br/>
<br/>
<br/>
<br/>
  五彩繽紛之下充滿著年輕時代的豪邁,<br/>
<br/>
  想起幾年前坐在圓椅上那面不改色的獨行者,單挑五大長老,<br/>
<br/>
  瞬間青綠色閃著螢光的汁液入喉燃燒,隨即黑褐色,土黃色,<br/>
<br/>
  最終打敗群俠得以一覽樂不思蜀的綺麗風光.<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　今夜隨著甜蜜的夢境歸去,<br/>
<br/>
　撒手撤回金牌換回無盡的憐惜.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
