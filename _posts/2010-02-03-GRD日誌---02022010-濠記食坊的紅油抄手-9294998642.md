<p><em><span> </span></em></p>
<p><span>(濠記的紅油抄手好吃！)</span><br/> <br/><br/><span> -----------------------------------------------------</span><br/><span> 濠記食坊</span><br/><span> 中和市宜安路55號</span><br/><span> (02)8668-5353</span><br/><span> -----------------------------------------------------</span><br/> <br/> <br/> <br/><span>早在許久以前，就曾看過有關一些有關於濠記食坊的分享文章了，</span></p>
<p><span>內容大多是以推薦其紅油抄手為主，不過一來，</span></p>
<p><span>個人不大吃辣，鮮少有例外，除了最近唯一好吃的麻辣例外：</span></p>
<p><span><a href="http://www.wretch.cc/blog/kuni/11587642" target="_blank">老四川巴蜀麻辣燙令人回味無窮(請用滑鼠點選)</a></span></p>
<p><span>二來，紅油抄手油脂量驚人，此類罪惡性極高的美食小弟向來唯恐避之不及，</span></p>
<p><span>更何況三來，美食清單上還有諸多未嚐過又不辣又健康的後補名單，</span></p>
<p><span>實在沒必要特地去找罪受呀。</span></p>
<p><br/> <br/><span>這一天晚上走在宜安路上，看著滿街不少有意思的店家，</span></p>
<p><span>均是在網路上仍未耳聞過的，本想挑一家來試試，</span></p>
<p><span>突然間看到了濠記愣了一下，一時之間還想不起來是從哪兒聽過的，</span></p>
<p><span>看到了裡面的紅油兩字，豁然間便明白了，</span></p>
<p><span>見機不可失，便進了去，馬上點了兩碗中辣的紅油抄手：</span></p>
<p><span>菜肉抄手(小份NT.50)，蝦仁抄手(小份NT.60)</span></p>
<p><span> 一碗乾拌麵(細拉麵NT.40)</span><br/> <br/> <br/><br/><span> (濠記食坊)</span></p>
<p><span></span></p>
<p> </p>
<p><span>(店內一景)</span></p>
<p><span></span></p>
<p><br/> <br/> <br/><span>上菜速度相當快，不一會兒三樣便分別上桌，</span></p>
<p><span>小弟迫不及待的當下先咬了一口蝦仁抄手，</span></p>
<p><span>霎時滿腔的紅油辣味與濃郁花椒香撲鼻而來，</span></p>
<p><span>舌頭上味蕾被麻辣軍團完整地佔領，</span></p>
<p><span>但很奇怪地，卻仍沒超越我所能容忍的極限範圍。</span></p>
<p><span>小份一整碗裡有六粒，每粒皆相當飽滿，</span></p>
<p><span>抄手裡的蝦仁又大又脆，足見廚師其用心之處。</span><br/> <br/> <br/><br/><span>(蝦仁抄手)</span></p>
<p><span> </span><br/><span> </span><br/> </p>
<p><span>菜肉抄手，飽滿的內餡裡肉質緊實配上細切菜絲口感迷人，</span></p>
<p><span>一碗裡只有四個，但是每個都相當於兩粒蝦仁抄手的大小，</span></p>
<p><span>尺寸實在大的驚人，一大口咬起來肉汁四溢配上店家特有的紅椒麻辣味，</span></p>
<p><span>這真是人生一大享受呀。</span></p>
<p> </p>
<p><span>(菜肉抄手)</span><span></span></p>
<p><span> </span><br/> <br/> <br/> <br/><span>但驚喜還沒完呢，千萬不要以為這裡的乾拌麵很普通，</span></p>
<p><span>吃下去包你吃驚連連，好久沒有吃到口感這麼Q又這麼緊實的細拉麵了，</span></p>
<p><span>將麵與底下的醬汁攪和在一起，濃郁的豬油香與紅蔥頭香散了出來，</span></p>
<p><span>吃一個抄手配一口乾拌麵，乾拌麵有效地解除麻辣感，</span></p>
<p><span>另外又是一番滋味，為此小弟願意墮落在罪惡的深淵呀。</span><br/> <br/> <br/><span>(乾拌麵登場嚕)</span></p>
<p><span> </span><br/> <br/> <br/><span>(口感驚人的麵條)</span></p>
<p><span> </span><br/> <br/> <br/><span>(記得要將麵條與底下的醬料攪和一下唷)</span></p>
<p><span> </span><br/> <br/> <br/><span>(吃一個抄手配一口乾拌麵...夫復何求矣)</span></p>
