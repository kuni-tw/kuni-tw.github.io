<div class="article-content-inner" id="article-content-inner">
  彷彿還留戀著那一段往事,不堪回首的幾回浪漫,是的,那一年身處異地的我,<br/>
<br/>
  白日趁著藍天白雲,騎著車,沿著藍色的多瑙河畔一路到鄰近小鎮克雷姆思.<br/>
<br/>
　在熾熱的午後面對著成群襲來的飛蟲睜不開眼,賣力的蹬著腳踏,成為暫時性的色盲.<br/>
<br/>
　傍晚啜著當季甫釀好的紅色龍捲風,香香甜甜,喝得頭暈眼花.<br/>
<br/>
　帶著七分的詩意重新拾起歸路,途中似快轉一般,一點也沒了印象.<br/>
<br/>
  傍晚重新返抵河畔中的球場,褪去了溼透的上衣,一直打到夜半月茫茫才罷手.<br/>
<br/>
  依依不捨的離去,循著微弱的路燈,口裡啣著當日第二顆蘋果,<br/>
<br/>
　乘著阿爾卑斯山下來的微風回家.<br/>
<br/>
<br/>
<br/>
  那一年的夏天不若後來的灰色暑假,乃黃濛濛的.<br/>
<br/>
  午後,至今仍不曉得是如何發生的,只知道那時我變得膽小,<br/>
<br/>
　看著那斷裂的兩截木頭想躲藏起來,深怕逝去的再也不回頭.<br/>
<br/>
　正當晨間美滿的花香鳥語之際我選擇了迴避,沉沒在為自己而設的夢靨裡,<br/>
<br/>
　成天與幻境牽扯不休,化童年純真為糜爛的自我,與一幫人追求可笑的夢想.<br/>
<br/>
<br/>
　<br/>
　不知道是哪日,耳酣之際夢見母親大人迢迢千里趕來打醒了我,<br/>
<br/>
  將躲在牆角的我拉了起來,瞧著我唯唯諾諾的模樣,不禁心疼,<br/>
<br/>
  便領我走過第一區的石板路將那兩截木頭托付給師傅.<br/>
<br/>
  "就這麼簡單?",我心想著,"原來當大人是這麼一回事呀!"<br/>
<br/>
　就這樣在十八歲那一年我偶然初嚐成年禮,意外的容易,<br/>
<br/>
  卻也分外的酸澀.<br/>
　<br/>
<br/>
