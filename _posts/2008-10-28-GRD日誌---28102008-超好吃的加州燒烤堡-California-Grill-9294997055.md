<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
   (加州燒烤的招牌漢堡真的超好吃！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  在台北我最愛吃的漢堡莫過於AN BURGER了，<br/>
　(詳情請參閱<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11463443" target="_blank">十足美式的AN BURGER漢堡</a>)<br/>
<br/>
  十足的美式風格，加上魄力十足的漢堡肉，<br/>
<br/>
　吃起來非常有"男人味"，尤其是以前在國外住久了，<br/>
<br/>
　回台灣後，更是非常想念那種走tex-mex的美式風餐廳呢！<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　這一次臨時決議的加州燒烤堡，總算是不虛此行，<br/>
<br/>
　位在"台北市永康街53-1號"的店面非常低調，從信義街口進來，<br/>
<br/>
　先過了公園，得走相當長的一段路才能找到．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　進了店門，簡約的美式裝潢，剛好還有幾位歐美年輕人正在裡面進食，<br/>
<br/>
　就座後得知點餐只能看牆上的類黑板菜單海報，真有意思，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(美式風的黑板菜單)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　我點了招牌漢堡套餐，要價230(單點則195)<br/>
<br/>
  起司漢堡內附培根，BBQ醬與大蒜美奶滋，<br/>
　<br/>
  而套餐另外則附薯條或是沙拉，個人因為想要品嘗所以點了薯條．<br/>
<br/>
　此時才得知套餐是不包括飲料的，這倒是沒意料到，<br/>
<br/>
　我點了許久沒喝過的ROOT BEER，一種美式的麥根沙士，<br/>
<br/>
　嚴格說起來味道比較像小時候常吃到的沙士糖．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(好懷念的麥根沙士呀)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(一副就很好喝的樣子！)<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不一會兒，主餐與副餐一起端了上來，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(外觀看起來不是很起眼的招牌漢堡套餐！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
　　(可口的番茄呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　望著其實不大討好的外觀，我不禁納悶了起來，<br/>
<br/>
　拍了幾張寫真，放下了GRD2，<br/>
<br/>
　將兩層麵包闔在一起咬了一口，我無言了好一段時間，<br/>
<br/>
　表情倒比較像肯德基廣告裡那陶醉在美食世界中的主角，<br/>
<br/>
　"阿，好好吃！"<br/>
　<br/>
　我不禁喊了出來，多麼令人懷念與熟悉的口感與味道，<br/>
<br/>
　清脆又嗆的新鮮洋蔥，軟嫩多汁的漢堡肉，<br/>
<br/>
  濃濃的Heinz番茄醬，這裡就要先註明一下了，<br/>
<br/>
　若不是經典的Heinz，不然漢堡裡加這麼多番茄醬通常會很難吃的！<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　　(招牌漢堡特寫！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(滿滿的Heinz番茄醬...喔...)<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (俯瞰圖，口水都快滴下來了...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後我望著那外表看起來乾澀的薯條，<br/>
<br/>
　抽了一根咬下去，喔，好濃郁的馬鈴薯味呀，真好吃！<br/>
<br/>
　馬上問了異常親切的服務生，原來是美國原裝進口來的薯條，<br/>
<br/>
　與AN BURGER稍嫌薄弱的薯條，<br/>
<br/>
　甚至是表現不錯的Bravo Burger的薯條相較之下，皆佔上風，<br/>
<br/>
　個人推測是馬鈴薯產地的問題，<br/>
<br/>
　造成先天性上等級的差異性！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(風味濃厚的美味薯條！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　套餐整體搭配著美國麥根沙士實在是太享受了，<br/>
<br/>
　要不是顧及到熱量問題，我真的很想再吃個兩份，<br/>
<br/>
　看來再度光臨的時刻不會太久呢...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　 (感謝這一次的加州燒烤堡讓我再度重溫舊夢呀！)<br/>
<br/>
　<br/>
<br/>
