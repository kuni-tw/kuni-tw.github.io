<p><span>(裕二的巨無霸牡丹蝦)</span><br/><br/><span> ---------</span><br/><span> 野壽司</span><br/><span> 地址：台北市吉林路22巷27號1樓</span><br/><span> 電話：(02)2581-7577</span><br/><span> ---------</span><span>  </span></p>
<p><span>今年老媽的農曆生日與老爸的父親節撞在一塊。</span></p>
<p><span>可惜週末工作行程排得滿滿，不好推掉，</span><span>決定提前在禮拜五晚上一起慶祝。</span></p>
<p><span>本來先預訂好中午的茹絲葵，</span></p>
<p><span>後來因公司中午繁忙，</span><span>所以決定改到晚上，在我三禮拜前才吃過的野壽司裡慶祝。</span></p>
<p><br/><span>雖然說離上次相隔不到一個月，</span></p>
<p><span>但小弟仍舊是非常期待裕二的手藝。</span><br/> <br/><span>下午教完課先提早到達場地，事先訂位時有特別要求坐裕二前面。</span></p>
<p><span>服務生很貼心保留好吧台前的角邊坐位，</span><span>那裡較適合四人一起聊天。</span></p>
<p><span>在爸媽到達前我先跟裕二討論，最後決定我的預算是兩千五。</span></p>
<p><span>爸媽部分則貴許多，因為希望他們能吃到較多點好料。</span></p>
<p><br/> <br/><span>這裡只講小弟吃的部分：</span></p>
<p><span>首先是前付，有滷蛤蠣，花枝佐醋味增與酒盜。</span></p>
<p><span>那酒盜是生花枝拌花枝內臟，</span></p>
<p><span>其典故為日本人覺得料理很好吃很下酒，</span></p>
<p><span>所以想偷酒來喝，生動的名字呢。</span></p>
<p><span>上次來訪就有吃到這個，</span></p>
<p><span>細緻的味覺享受，</span><span>不過這次似乎有點過鹹了。</span></p>
<p><span>(滷蛤蠣，花枝佐醋味增與酒盜)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076290.jpg" target="_blank"></a></span><br/> <br/><span>(漬白蘿蔔)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076291.jpg" target="_blank"></a></span></p>
<p><span>然後是生魚片，有紅坩與鮪魚大腹。</span></p>
<p><span>相較於上一次吃到的中腹，</span></p>
<p><span>這次大腹部位的口感似乎沒前者好，有點筋。</span></p>
<p><span>(紅坩)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076292.jpg" target="_blank"></a><br/></span></p>
<p><span>(鮪魚大腹)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076293.jpg" target="_blank"></a></span><br/> <br/><span>此時裕二開始展現他那看似漫不經心實際上卻異常純熟的刀工。</span></p>
<p><span>拿著一條黃瓜慢慢削成一片很長的薄片。</span></p>
<p><span>深綠色部分細切成絲，</span></p>
<p><span>淺色部分則拿來將青花魚與前面切好的黃瓜皮絲捲在一起。</span></p>
<p><span>然後一個個依序放在我們面前，</span></p>
<p><span>看似貌不驚人的青花黃瓜捲一入口，</span></p>
<p><span>清脆的口感配上黃瓜特有的清香還有毫無腥味的青花魚。</span></p>
<p><span>感覺就像是歌劇裡的序曲，為即將到來的美味揭開序幕。</span></p>
<p><span>(青花黃瓜捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076294.jpg" target="_blank"></a></span><br/> <br/><span>然後裕二突然拿起兩隻我從沒看過的巨大牡丹蝦。</span></p>
<p><span>置放在我面前的盤子上，細瞧之下異常的巨大。</span></p>
<p><span>他說這是接下來要給我們做的料理，</span><span>處理前先給我拍一下。</span></p>
<p><span>說真的，這種SIZE的牡丹蝦真的從來沒看過，現場看非常驚人。</span></p>
<p><span>(巨無霸牡丹蝦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076295.jpg" target="_blank"></a><br/> <br/>(看這多驚人呀)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076296.jpg" target="_blank"></a></span><br/> <br/><span>不一會兒端放在我們面前的是美到難以形容的藝術品，</span></p>
<p><span>潔白無瑕的牡丹蝦肉上擺著綻放出寶石光澤的蝦卵。</span></p>
<p><span>入口後，口感與以前所吃過的高檔牡丹蝦相比，</span></p>
<p><span>不論是在鮮度還是甜度上都極為類似，</span></p>
<p><span>但巨無霸牡丹蝦的口感更為軟黏，又滑又膩到令人難以忘懷。</span><br/> <br/><span>(像藝術品一般)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076298.jpg" target="_blank"></a><br/></span></p>
<p><span>服務生端來一盤漬秋刀魚，</span></p>
<p><span>一樣也是完全無腥味，</span></p>
<p><span>醃漬與調味程度細膩又恰到好處。</span></p>
<p><span>(漬秋刀魚)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076297.jpg" target="_blank"></a></span></p>
<p><span>接下來開始上熟食。</span></p>
<p><span>首先是水章魚燒烤串，將章魚腳上的大吸盤切下來串烤，</span></p>
<p><span>口感尚可，沒什麼特別印象。</span><br/> <br/><span>(水章魚吸盤燒烤)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076299.jpg" target="_blank"></a></span></p>
<p><span>然後是銀鱈西京燒與茶碗蒸，前者調味較淡些，</span></p>
<p><span>不過有將鱈魚肉質特有的滑嫩口感烤出來，很棒。</span></p>
<p><span>後者則稍微地與上次吃到的有點不同，</span></p>
<p><span>除了可圈可點的香菇外，裡頭還多了一小條海膽。</span><br/> <br/><span>(銀鱈西京燒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076300.jpg" target="_blank"></a><br/></span></p>
<p><span>(茶碗蒸)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076301.jpg" target="_blank"></a></span></p>
<p><span>這時得到爸媽首肯，便幫他們點了一瓶四五千的清酒，</span></p>
<p><span>我沒喝，不過有請裕二師傅一起喝，</span></p>
<p><span>看他們三人喝得還滿開心的，猜測酒質應該還不錯吧。</span><br/> <br/><span>(給爸媽的清酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076302.jpg" target="_blank"></a><br/></span></p>
<p><span>接下來開始我心目中的重頭戲了，</span></p>
<p><span>那就是裕二的握壽司，完全以技術先決的美食。</span></p>
<p><span>依序上來的有生花枝握，現場刨的柴魚海苔捲，</span></p>
<p><span>黃雞魚握，小鯛握，北寄貝握，尖梭魚握與醃鮪魚赤身握。</span><br/> <br/><span>(生花枝握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076303.jpg" target="_blank"></a></span></p>
<p><span>(柴魚海苔捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076304.jpg" target="_blank"></a><br/></span></p>
<p><span>(黃雞魚握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076305.jpg" target="_blank"></a></span></p>
<p><span>(小鯛握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076306.jpg" target="_blank"></a><br/></span></p>
<p><span>(尖梭魚握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076308.jpg" target="_blank"></a><br/></span></p>
<p><span>(醃鮪魚赤身握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076309.jpg" target="_blank"></a><br/></span></p>
<p><span>除了一貫的裕二握壽司那入口即化的美味口感外(這次飯煮得似乎沒上次好)，</span></p>
<p><span>不得不提一下那令我驚為天人的北寄貝握，簡直是出神入化！</span></p>
<p><span>將切好與處理過的北寄貝與醋飯握好後，</span></p>
<p><span>裕二用重手在上頭拍了一下，聲音非常響。</span></p>
<p><span>首先是爸媽先吃，就看到兩人一付很吃驚的樣子，</span></p>
<p><span>然後很誇張的用日語大聲說真好吃呀。</span></p>
<p><span>我那時心裡還偷偷笑了一下，心想爸媽兩人年紀畢竟也不小了，</span></p>
<p><span>但還是很像小孩子，真可愛呀。</span></p>
<p><span>不過等到我吃了下去後，竟然也衝動了起來：</span></p>
<p><span>"阿，真好吃呀！"(Todemo Oishi~)</span></p>
<p><span>從來沒吃過如此令人拍案叫絕的驚人口感，</span></p>
<p><span>難以形容的北寄貝，薄薄的一片，難以形容的脆度，</span></p>
<p><span>竟然可以與醋飯如此地契合，這到底是什麼魔術呀？</span></p>
<p><span>(北寄貝握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076307.jpg" target="_blank"></a></span></p>
<p><span>在上了味增湯後那一整瓶清酒也光了，</span></p>
<p><span>不知道是不是微醺的關係，</span></p>
<p><span>裕二突然神秘的微笑，將一個盤子放在我面前讓我先拍一下，</span></p>
<p><span>盤子中是海膽，圍繞在旁的有九種魚料。</span></p>
<p><span>待拍攝完畢後，他拿去將所有的料堆在一起做成軍艦捲，</span></p>
<p><span>說是這是給我們的生日蛋糕(Sushi cake)</span></p>
<p><span>看起來相當夢幻，一開始還真狐疑：這樣會好吃嗎？</span></p>
<p><span>(味增湯)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076310.jpg" target="_blank"></a></span></p>
<p><span>(Sushi蛋糕的魚料)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076311.jpg" target="_blank"></a><br/></span></p>
<p><span>(裕二的Sushi蛋糕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076312.jpg" target="_blank"></a><br/></span></p>
<p><span>(再一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076313.jpg" target="_blank"></a></span></p>
<p><span>不過入口後所有疑問霎時間一掃而空：</span></p>
<p><span>所有當晚吃過的味道從頭至尾在口腔裡重新RUN了一次，</span></p>
<p><span>真是一個很棒的收尾。</span></p>
<p><span>從開頭如此令人期待的序曲到最後竟然還可以有這樣華麗的結尾，</span></p>
<p><span>裕二真的是不簡單，這一次刷卡下來加服務費共吃了快兩萬，</span></p>
<p><span>但看到爸媽如此開心的模樣，這實在是太值得了。</span><br/> <br/><span>(甜點是地瓜紅豆泥)</span></p>
