御杉根時尚鍋物餐飲<br/>
地址：台北市大安區光復南路240巷5號<br/>
電話：02-27780505<br/>
-------------------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   這一天與友來到推薦的一家餐廳"御杉根時尚鍋物"，<br/>
<br/>
   來之前也沒細查，只聽說有好吃的燉飯。<br/>
<br/>
　以前在歐洲最愛吃的義大利料理，平價的莫過於PIZZA，<br/>
<br/>
　高檔的莫過於RISOTTO了，<br/>
<br/>
　可惜至今為止在台灣似乎還沒找到過一家願意使用正統RISOTTO米的義式餐廳，<br/>
<br/>
　不過上一次吃過喬尼，總算找到一家個人相當喜愛的，<br/>
<br/>
　義大利麵味道好吃又道地，可惜RISOTTO部分米還是不對。　　<br/>
<br/>
　請參閱：<a href="http://www.wretch.cc/blog/kuni/11606217" target="_blank">喬尼的私人廚房(使用滑鼠點選)</a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   這一次的御杉根時尚鍋物聽說有義式燉飯配鍋物，套餐價：580加上一成服務費，<br/>
<br/>
　但配合最近的買一送一活動，兩人吃580加兩份服務費，<br/>
<br/>
　等於一人出348...感覺相當划算呢。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(菜單)<br/>
<br/>
<br/>
<br/>
<a href="/album/show.php?i=kuni&amp;b=37&amp;f=1698883789.jpg" target="_blank"></a><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　鍋物部分選好了肉品後，沒多久香噴噴的義式燉飯便上桌了，<br/>
<br/>
　白色瓷盤上的米飯看似無奇，舀了一口，才驚覺那味道似曾相識，<br/>
<br/>
　"好吃"兩字不禁冒出口來，連我自己都忍不住笑了出來，<br/>
<br/>
　但這似曾相識的味道究竟從何而來呢？<br/>
<br/>
　問了曾來吃過的友人，他說不知道我指的是什麼，<br/>
<br/>
　只知道裡頭有放奶油與松露油...當下我恍然大悟，果然是松露，<br/>
<br/>
　只是沒料到會在三百多的套餐裡吃到而已！<br/>
<br/>
　而且這香味，絕對不是一般的松露油，推測是白松露油。<br/>
<br/>
　以前我也買過一小瓶，超貴，但味道沒這麼漂亮，<br/>
<br/>
　最後付帳之時還特地找了主廚問了一下，他將他的松露油給我們看， <br/>
<br/>
　果然是白松露，不過牌子沒看過，據說是西班牙來的，<br/>
<br/>
　下一次也來買買看好了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
   這一盤松露燉飯雖然也只是用一般的台灣米，不過松露油使用得宜，<br/>
<br/>
　還是很好吃，一下子就清光，意猶未盡之下，<br/>
<br/>
　開始涮鍋物裡的料，這裡的鍋物提供兩種湯頭：清淡與麻辣，<br/>
<br/>
　鍋物盤裡除了肉類還有擺得相當漂亮的菜盤，<br/>
<br/>
　裡頭有香菇，金針菇，白菜，南瓜，番茄，洋蔥，腐皮捲，與海鮮等，<br/>
<br/>
　份量似乎還足，肉類與海鮮類新鮮度也夠，<br/>
<br/>
　不過湯頭似乎較弱了點，涮起來似乎沒那麼過癮，<br/>
<br/>
　與前頭的燉飯似乎有了反差。<br/>

   飯後點心是自製巧克力蛋糕，鬆軟又濕潤順滑的蛋糕本體，<br/>
<br/>
　巧克力味道其實沒有那麼濃，有點像美祿，雖然後來有特地問過大廚，<br/>
<br/>
　得知是使用高級的巧克力製作，但味道並沒有特別令人印象深刻。<br/>
<br/>
   咖啡部分與一般餐廳的飯後咖啡相比，表現則尚可，<br/>
<br/>
　沒有特別令人驚艷之處。<br/>
