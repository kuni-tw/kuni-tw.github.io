　<br/>
<br/>
  最近阿殺力的買下了夢寐以求的GRD2,連續幾天下來廢寢忘食地,<br/>
<br/>
　整天在這光影交加的世界裡流連忘返,幾乎忘了原來我才是屬於真實的那一方.<br/>
<br/>
  或許也一直沒有發現自己對於捕捉那永恆的一刻有多麼的依賴,<br/>
<br/>
　也多麼想創造出那靜止的動態,更想以此紀錄此生最珍貴的動產:逝去的時光.<br/>
<br/>
  每每回憶起所經歷過的每一分每一秒,如同捕風捉影,<br/>
<br/>
　那美麗的畫面竟然沒一個是彩色的,像是蒙上一層不透明的紗一般,<br/>
<br/>
  像被老鼠啃食過,面目全非.<br/>
<br/>
　<br/>
　螢幕裡的畫面若無法與腦海裡的相吻合,那對我來說是沒有意義的.<br/>
<br/>
  我的心不是冷艷無缺的,更不是盲目的追隨,而是充滿著無數的藝術感,<br/>
<br/>
  每一片都是色彩繽紛,充滿著如癡如醉的情緒,醞釀著香醇的美酒.<br/>
<br/>
  也因為如此,一直以單眼為目標的我,似乎從來無法接受單薄又犀利的DC.<br/>
<br/>
  而對於選購單眼我又因慵懶而卻步不前了好多日子,一直到陰錯陽差的某天下午,<br/>
<br/>
  我終於看到了那一份感動.<br/>
<br/>
<br/>
　當我初次看到那貌不驚人的實體,並且探入了實際之後,<br/>
<br/>
　腦海裡就從未停止過那堅定的決心.<br/>
<br/>
  終於在漢口特區晃了晃幾圈,加上事先詳細的調查報告,我圓滿的達成初階段的目標.<br/>
<br/>
<br/>
　幾天下來,開始貪婪的蠶食所有身旁的感受,拒絕放掉任何的遺憾.<br/>
<br/>
　適可而止嗎,這是現況所不容所不允許的行為呀!<br/>
<br/>
  就在近乎瘋狂的搜括之下,我滿心的貪婪漸漸有了決堤的危機,<br/>
<br/>
  再也無法制止時光魔人的野心,直到下一階段開始...<br/>
<br/>
