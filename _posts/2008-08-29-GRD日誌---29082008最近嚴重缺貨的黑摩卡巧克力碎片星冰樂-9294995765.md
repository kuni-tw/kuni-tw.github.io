<br/>
<br/>
<br/>
      (最近在各大門市競相缺貨的黑摩卡星冰樂)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這大概是STARBUCKS所有星冰樂當中我最喜歡的口味了，可惜也是最貴的，<br/>
<br/>
　一杯黑摩卡巧克力碎片GRANDE大小為145NT(16oz)，特大杯為165(20oz).<br/>
<br/>
　當時幸好有隨行卡紅利點數兌換活動，得以有較實惠的方式來換購這消暑聖品，<br/>
<br/>
　可惜前陣子這項活動也結束了！<br/>
<br/>
<br/>
　最近碰到統一集團以旗下另一家關係企業SEVEN便利超商積點方式，<br/>
<br/>
　讓顧客們得以換取星冰樂買一送一卷，<br/>
<br/>
　使得在炎熱天氣特別受到歡迎的星冰樂最近好幾天以來大賣特賣，<br/>
<br/>
  各家門市均呈現供不應求的狀態，導致嚴重缺貨．<br/>
<br/>
　<br/>
<br/>
　下午心血來潮，問了幾家門市，好不容易有一家尚未斷貨，<br/>
<br/>
　便趕忙進了去，馬上點了一杯，並要求冰塊打勻一點，巧克力碎片多放一點，<br/>
<br/>
　這也是星巴克最吸引人的地方，依照不同客人需求將任何一杯飲料客製化，<br/>
<br/>
　多麼有魅力呀！<br/>
<br/>
<br/>
<br/>
　不一會兒，黑摩卡巧克力碎片星冰樂就到手了，<br/>
<br/>
　馬上為它上頭那灑下可可粉，打得美麗非凡的鮮奶油拍張近距離寫真，<br/>
<br/>
　幾分鐘後，中午的炎熱就這樣神奇的從我喉頭裡消聲匿跡．<br/>
<br/>
　雖然以後不一定有機會喝得到，但真的滿推薦的，有機會一定要試試：<br/>
<br/>
　微苦中散發出淡淡的甜味與摩卡香，<br/>
<br/>
  細緻的巧克力碎片經細心的打碎過口感真是令人難忘，<br/>
<br/>
　喝完之後望著杯裡還殘留著均勻佈滿的巧克力碎片，真是美極了，<br/>
<br/>
　馬上又是一聲快門，就這樣，2008年夏日那美麗的片刻回憶，<br/>
<br/>
　再也帶不走...<br/>
