<br/>
<br/>
<br/>
<br/>
<br/>
  七號下午臨時興起，抽空跑去永和的菲瑪咖啡，算是慕名而去吧．<br/>
<br/>
　沒辦法，對於美好的咖啡向來毫無抵抗力的我，只要聽說哪裡有好咖啡，<br/>
<br/>
　從此便會開始寢食難安，念念不忘．<br/>
<br/>
　店家裝潢感覺相當精緻，門口右側小沙發上擺了一把小提琴，<br/>
<br/>
　一看之下原來是初學者用的廉價琴，我不禁好奇起來，<br/>
<br/>
　細問之下原來是某客人借放一段時間的．<br/>
　<br/>
<br/>
　看了菜單，點了我前陣子瘋狂迷上的冰濃縮，後來加點一份摩卡．<br/>
<br/>
　店內氣氛相當宜人，老闆與老闆娘也滿健談的，老闆更是會不少紙模型手工藝呢．<br/>
<br/>
　<br/>
　先說說他們的冰濃縮好了，只有DOUBLE的大小可以選擇，<br/>
<br/>
　冰塊放得相當多，入口偏苦一點，但是入喉一段時間之後苦味慢慢消失，<br/>
<br/>
　不會殘留於舌根，尾韻不長卻有出乎意料的微澀口感，<br/>
<br/>
　這部分我滿喜歡的．不過對於個人來說，整體而言較沒什麼特色，<br/>
<br/>
　不若四號公園E61咖啡有著異國風情的偏酸，或咖啡意識的細緻口感．<br/>
<br/>
　另外後來上的摩卡口感則偏甜，感覺起來好像是摩卡醬放得較多的關係，<br/>
<br/>
　咖啡味不太感覺得出來，這一點可能是比較遺憾的地方．<br/>
