<br/>
<br/>
<br/>
  (攝於INTEL)<br/>
<br/>
<br/>
<br/>
  這幾天以來,最熱門的話題莫過於油價事件了吧.<br/>
<br/>
<br/>
<br/>
  經了幾番思考我最後仍舊沒搭上這一波搶油熱潮,一時之間,<br/>
<br/>
　真不知道該說自己是太過狂妄還是過於精打細算.<br/>
<br/>
  不知怎麼,我一點也沒感受到漲價所帶來的不便,<br/>
<br/>
　亦或說是我似乎也無法理解這些類似憂鬱症突發的恐慌,<br/>
<br/>
  也厭惡兩邊政客藉此炒作新聞,有如久旱逢甘霖一般,唯恐天下不亂.<br/>
<br/>
  總而言之後來算了算,花半小時當隻給油氣燻過頭的沙丁魚能省多少錢?<br/>
<br/>
　縱然以一小時九十的最低工資來論,抱著好心情勤奮工作等待升遷賺頭仍舊大多了.　<br/>
　<br/>
　不過這一次加油列隊長達好幾個街口,整座台北城裡處處皆如此,<br/>
<br/>
　這樣的壯觀場面倒是值得餓一整頓好吃的,帶著相機駐足在人群前面,<br/>
<br/>
  我一邊拍照,觀察人群,一邊與他們聊天,交流.<br/>
<br/>
<br/>
<br/>
  漸漸地,我也成了魚群中的一份子了.<br/>
<br/>
<br/>
