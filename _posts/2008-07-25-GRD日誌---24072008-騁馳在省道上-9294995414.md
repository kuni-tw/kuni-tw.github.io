<br/>
<br/>
<br/>
　騁馳在一號省道上，景象隨著逐漸升高的轉速消逝，      <br/>
<br/>
　其時已近傍晚，行程依舊撲朔迷離．<br/>
<br/>
　只見蛋黃般的日頭變得寬扁，然後我下了車歇息，買了一杯冷飲．<br/>
<br/>
　邊解渴邊觀賞著造物主的創作，您瞧，我們的雙眼多麼的配合，<br/>
<br/>
　他給我們什麼我們就看得到什麼，但是相機呢？<br/>
<br/>
<br/>
<br/>
　貧窮無辜的小相機卻背負著人類無比貪婪的願望，<br/>
<br/>
　近乎苛求的嘗試模擬我們兩隻無可取代的接受器，<br/>
<br/>
　從最明亮一直到最黑暗之處，一點一點的拼湊在一起，<br/>
<br/>
　可惜無功而返．<br/>
<br/>
<br/>
<br/>
　終於過了萬重山水，順著台中港路進了市區，<br/>
<br/>
　點了杯精品咖啡，澎湃激昂的心情慢慢降到Andante的節拍．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  夜半時候，終於依依不捨的與MOJO咖啡廳離別，<br/>
<br/>
　踏上一路上風塵僕僕陪著我的小金，心中默念著剛背熟的省道順序，<br/>
<br/>
　摸黑，重新騁馳在一號省道裡．<br/>
　<br/>
