<br/>
<br/>
  時間過得好快，沒幾個月後中將回到當初，<br/>
<br/>
　原本那一顆抱有的赤子之心：<br/>
<br/>
　以為不過就是操而已，那本來正是我想要的！<br/>
<br/>
　<br/>
　沒想到有如浪費時間一般，一堆過度自我膨脹的傢伙，<br/>
<br/>
　沒見過世面的小朋友與無法適應現實社會的LOSER．<br/>
<br/>
　表面功夫與阿諛奉承簡直是司空見慣...<br/>
<br/>
  但真正的問題卻比表面上看來的更嚴重，<br/>
<br/>
　我終於了解為何問過的那幾個，沒有壓力之下的回答令人驚異的一致：<br/>
<br/>
　這真是個鬼地方...<br/>
<br/>
<br/>
　<br/>
　不同的立場當然有不同的解釋，但每天從水果日報上看來的的確有其真實性，<br/>
<br/>
　處在那種地方，要不就天生個性扭曲，要不就是刻意扭曲自己，<br/>
<br/>
　團結與信任？其實根本只是幌子...<br/>
<br/>
<br/>
<br/>
　唉...我打從心底的討厭井底之蛙的世界，<br/>
<br/>
　尤其是那裡沒有一加一等於二的邏輯，<br/>
<br/>
　只有假象...</div>
