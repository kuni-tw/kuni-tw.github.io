<p><span>-----------------</span><br/><span>食堂屋</span><br/><span>地址：台北縣永和市中和路389巷2號之1</span><br/><span>電話：(02)8921-1222</span><br/><span>時間：11:00~21:00</span><br/><span>-----------------</span></p>
<p><br/><span>     </span><br/><span>好幾個月前某天嘴饞，跑去食堂屋吃了份豬排定食後，</span></p>
<p><span>從此念念不忘，後來又去吃了幾次。</span></p>
<p><span>只是剛好沒帶相機，總算這一次拍到了</span><span>。</span></p>
<p><br/><br/><span>一人份的炸豬排定食才NT.140，裡頭有些什麼呢？</span></p>
<p><span>配菜中有漬小黃瓜，生菜沙拉，紅豆湯，</span><span>白飯與咖哩醬。</span></p>
<p><br/><span>(定食的配菜)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076167.jpg" target="_blank"></a><br/> <br/> <br/></span><span>記得初次看到食堂屋的炸豬排主餐上桌時，不禁愣了一下：</span></p>
<p><span>"小姐，妳搞錯了，我只叫了一份。"</span></p>
<p><span>"這的確是一份呀！"，小姐也笑了出來。</span></p>
<p><span>兩大塊的炸豬排，外層微脆的麵衣裹著厚實肉排，</span></p>
<p><span>最難得是，如此厚度卻仍然擁有口感與滿溢肉汁，</span></p>
<p><span>廚師功力如此可見一般。</span><br/><br/><span>(初次看到時，還真難相信這是一人份的炸豬排)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076168.jpg" target="_blank"></a><br/><br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076169.jpg" target="_blank"><br/> </a><br/>(厚實與美味程度皆爆表)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076170.jpg" target="_blank"></a><br/> <br/></span><span>老實說，這樣的美味豬排在鄰近的台北並不是沒有，</span></p>
<p><span>當然還有一堆更好吃的。</span></p>
<p><span>但相對而言份量小許多，</span><span>且價格完全沒得比。</span></p>
<p><span>這次順便點了個人推薦的蘆筍手捲，</span></p>
<p><span><strong><span>蘆筍份量夠多，口感清脆鮮甜多汁。</span></strong></span></p>
<p><span>美味爆表，一份只要NT.40，</span></p>
<p><span>至今也仍未吃到同樣價錢之下可比擬的。</span><br/><br/><span>(一份才NT.40的蘆筍手捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076165.jpg" target="_blank"></a><br/> <br/> <br/></span><span>鮭魚卵手捲一份NT.90，魚卵部分放得足，</span></p>
<p><span>好吃不過不算特別。</span><br/><br/><span>(鮭魚卵手捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483076166.jpg" target="_blank"></a><br/> <br/> <br/></span><span>後記：</span></p>
<p><span>食堂屋雖然便宜好吃，</span><span>但也曾讓小弟踩過"地雷"。</span></p>
<p><span>記得是五月間去的，那天的豬排非常乾，</span></p>
<p><span>部分甚至很難咬，</span><span>蘆筍手捲更是失去以往水準。</span></p>
<p><span>一問之下，方知老闆因操勞過度，正在修養，</span></p>
<p><span>暫由兒子掌廚，但落差也太大了，</span></p>
<p><span>以至小弟後來有兩個月都不曾再去。</span></p>
