<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
　(咖啡弄的草莓冰淇淋鬆餅可稱得是上色香味俱全呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  個人向來沒有特別愛吃鬆餅之類的點心，<br/>
<br/>
　前陣子曾想過要去南京某間名店嚐鮮，<br/>
<br/>
　卻不是很茍同他們的高姿態做法，低銷也就罷了，<br/>
<br/>
　點一道吃的也得同時點一杯飲料也就罷了，<br/>
<br/>
　吃不夠想要加點鬆餅，竟然還硬性規定客人同時要加點飲料才行．<br/>
<br/>
　小弟在耳聞這樣的規定以及網路不少相關的負面心得之後，<br/>
<br/>
　便將之列為拒絕往來戶了，畢竟花錢歸花錢，花也要花得開心呀．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這一次在冷冽的寒流中，因友人的建議，<br/>
<br/>
　決定去拜訪在東區頂好商圈的咖啡弄，<br/>
<br/>
　地址：台北市大安區忠孝東路四段101巷45號<br/>
<br/>
　電話：(02）2711-1912<br/>
<br/>
　一樣也以香甜好吃的鬆餅著名，卻少了一些個人極為厭惡的商業手法．<br/>
<br/>
　低消90元也沒限定要點飲料，隨便點個鬆餅就好啦．<br/>
<br/>
　得知此店常常大排長龍，想事先定位，卻被告知客人太多請現場候位，<br/>
<br/>
　只好硬著頭皮親自過去候排啦．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　進了店門，店內裝潢配合著營造氣氛的燈光，<br/>
<br/>
　感覺很舒服，運氣很好一下子就排到吧檯前面的座位，<br/>
<br/>
　恩，燈光不錯，很適合今天想要的拍攝風格．<br/>
<br/>
　馬上點了一份草莓冰淇淋鬆餅，要價150元，<br/>
<br/>
　一杯熱炭燒，要價150元．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (咖啡弄吧檯內一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　咖啡弄的吧檯剛好設立在整間餐廳的中央，<br/>
<br/>
　在吧檯邊站著就可以看到裡面服務人員的工作狀態，<br/>
<br/>
　看著看，不一會兒，我的咖啡與鬆餅紛紛登場，<br/>
<br/>
　首先來的是炭燒咖啡，一整套的咖啡杯具頗有質感，<br/>
<br/>
　咖啡有中等水準以上唷，這一點還滿令我吃驚的．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(咖啡弄的咖啡杯具頗有巧思)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後就是令我驚喜連連的鬆餅了．<br/>
<br/>
　原來這就是令許多東區少女們為之瘋狂的點心呀，<br/>
<br/>
　我心裡不自主的讚嘆了一下，草莓大顆且色澤漂亮，<br/>
<br/>
　鬆餅顏色均勻且光滑細緻，雖然小弟在國外長大，<br/>
<br/>
　不過也從沒看過這樣"漂亮"的鬆餅，<br/>
<br/>
　難怪以前不會特別想吃這種東西呀．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(在這樣的燈光之下草莓更顯得嬌豔呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(令人垂涎欲滴的草莓鬆餅^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (再一張好了，因為實在是太美了！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (鬆餅顏色均勻的真的很漂亮！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (鬆餅與草莓冰淇淋搭配在一起顯得相得益彰呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　草莓與冰淇淋與鬆餅整體搭配起來相當棒，<br/>
<br/>
　色香味俱全，而令人最印象深刻的莫過於鬆餅的口感了，<br/>
<br/>
　外皮甜脆偏硬，內部紮實中充滿著細緻，<br/>
<br/>
　這偏硬的外皮口感在吃到一半時得到了解答，<br/>
<br/>
　原來一口接一口配上冰淇淋或是醬料，照理來說，<br/>
<br/>
　鬆餅應該會慢慢塌掉，但咖啡弄的鬆餅卻不會，<br/>
<br/>
　從頭至尾保持著相當不錯的支撐性，<br/>
<br/>
　與配料相得益彰，使得客人可以吃得很盡興，<br/>
<br/>
　真是令人感到窩心的鬆餅呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (吃到最後一口仍然非常盡興呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (店內充滿氣氛的一面牆)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(店內一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(結帳時才發現到的，是棉花呢，真漂亮！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(用棉花來當擺飾很適合這裡的氣氛呢^^)<br/>
