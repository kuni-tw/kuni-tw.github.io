<br/><br/><br/><br/>　(麥當勞的無敵大麥克)<br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>--------------------------------------------<br/>　麥當勞(永和永安店)<br/>　地址：新北市永和區中和路503號<br/>　電話：(02)2922-8242<br/>　時間：24H<br/>--------------------------------------------<br/><br/><br/><br/><br/>　吃了二十多年的麥當勞這次趁著"炫彩酷罐杯"的發行，<br/>																			    <br/>　終於吃到四層漢堡肉的大麥克了。<br/>																			    <br/>　當然不可諱言的那模仿可樂鋁罐造型的玻璃杯是賣點之一：<br/>																			    <br/>　購買無敵大麥克套餐或是普通大麥克薯條飲料加大便送一只杯子，<br/>																			    <br/>　總共有六色供蒐集。<br/><br/><br/><br/><br/><br/>　與學生中午到了永安市場捷運站對面的麥當勞四層樓透天門市(無敵大門市？XD)<br/>																			    <br/>　只見人潮擁擠不堪，好不容易找著窗邊的坐位後點了無敵大麥克套餐，NT.119　　　<br/>																			    <br/>　好久沒有吃麥當勞了，自從開始到處走訪美食後對於大型連鎖速食反而愈來愈陌生，<br/>																			    <br/>　有時臨時肚子起義沒時間吃飯通常就隨便找一家小七果腹，<br/>																			    <br/>　這一次重溫學生時代的回憶感覺實在是溫馨無比。<br/><br/><br/><br/><br/><br/><br/><br/><br/>　(麥當勞的永安捷運站透天門市)<br/><br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(這款杯子個人還滿喜歡的，準備要好好蒐集嚕)<br/><br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　不可諱言的四層牛肉咬起來一定比雙層的要過癮許多，<br/>																			    <br/>　尤其是對於像小弟這型的純種肉食性動物(說歸說不過以前也曾吃過一年多的素唷)<br/>																			    <br/>　但畢竟是大型連鎖企業品質上要完全控管實在是相當不容易，<br/>																			    <br/>　當中當然也有例外比如個人頗為喜愛的摩斯米漢堡。<br/>																			    <br/>　麥當勞的漢堡肉與薯條個人覺得皆有點偏乾且味道較不足，<br/>																			    <br/>　加上這幾年來瘋狂竄升的便利商店速食大軍來勢洶洶，<br/>																			    <br/>　麥當勞叔叔您可要加把勁呀。<br/><br/><br/><br/><br/><br/><br/><br/>　(四層牛肉的無敵大麥克)<br/><br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>BY GRD3 AND KUNI<br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>.</div>
