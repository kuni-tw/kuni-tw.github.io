   魚丰魚丰就是海鮮蓋飯<br/>
   地址：台北市錦州街446號一樓<br/>
   電話：(02)25099236<br/>
   時間：11:00~21:30<br/>
------------------------------------------------------<br/>
<br/>
    <br/>
　這陣子連續吃了好幾家海鮮蓋飯，其中一家就是位於錦州街的魚丰魚丰，<br/>
    <br/>
　全名則為"魚丰魚丰就是海鮮蓋飯"，念起來既有趣又俏皮，<br/>
    <br/>
　是台北東區"就是好客"海鮮燒肉所創立的第二品牌。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(魚丰魚丰錦州街店門口)<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　　　<br/>
<br/>
　這晚點了豪華海鮮蓋飯，NT.398，附一碗味噌湯，<br/>
    <br/>
　另單點一份胭脂蝦，NT.40。 <br/>
    <br/>
　沒多久後味噌湯，單點胭脂蝦與蓋飯陸續上桌，味噌湯味道偏淡令人頗失望，<br/>
    <br/>
　單點胭脂蝦一盤四十元共有四隻，感覺不會太貴。<br/>

    <br/>
　豪華海鮮蓋飯比想像中的還要小份些，魚料部分有：<br/>
    <br/>
　鮭魚鮪魚旗魚生魚片，海膽，干貝，蟹肉棒，鮭魚卵，蝦卵，胭脂蝦，<br/>
    <br/>
　機蛋黃與曼波魚皮。<br/>
    <br/>
　魚料新鮮度有中上水準可惜以四百元這價位來說魚料種類與份量感覺皆有點過少，<br/>
    <br/>
　個人會期待更多內容些，不過嚴格來說整體表現算還不錯了。<br/>
