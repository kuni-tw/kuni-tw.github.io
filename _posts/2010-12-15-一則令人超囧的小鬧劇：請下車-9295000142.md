<br/>
<br/>
<br/>
<br/>
<br/>
　好久不曾搭公車了。<br/>
<br/>
<br/>
　小時候的公車是每個禮拜去老師家與樂團團練的好搭檔，<br/>
    <br/>
　記得小學三年級時還曾一個人搭公車到北投去呢。<br/>
    <br/>
　國一後開始流亡歐洲十數年，坐的是當地不用打卡驗票的市區大眾運輸系統。<br/>
    <br/>
　回國後這幾年來連搭捷運的次數手指頭都可以數得出來，更遑論是公車。<br/>
    <br/>
<br/>
<br/>
<br/>
　今天下午有了意外之插曲，臨時沒了車鑰匙，這可怎麼辦才好呢？<br/>
    <br/>
　馬上先打給有我備用鑰匙的好友，可惜：<br/>
    <br/>
　"兄弟拍謝啦，我待會是滿堂所以走不出去耶。"<br/>
    <br/>
　為了能順利趕課，只好開始搭乘計程車趕場。<br/>
    <br/>
　到晚上終於開始閒了，加上某學生臨時改時間，<br/>
    <br/>
　便改搭捷運到了好友的補習班拿了鑰匙，<br/>
    <br/>
　然後動一動好久沒動的雙腿漫步在下著小雨的敦化南上，<br/>
    <br/>
　好不容易找到一台能到永和的公車，著實興奮地等了半天，<br/>
    <br/>
　終於公車到了：<br/>
    <br/>
　"恩，我可還記得台灣的公車有分上下車投錢呢。"<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　踏進車廂後："司機先生，請問這裡是上車投錢還是下車投呀？"<br/>
    <br/>
　我還問的相當理所當然呢。<br/>
    <br/>
　司機："請問你是要到永和嗎？"<br/>
    <br/>
　我回得超興奮："是的，我要到永和。"<br/>
    <br/>
　司機："那要下車。"<br/>
    <br/>
　我聽了嚇了一跳趕緊跳下車，<br/>
    <br/>
　想說細鶴，差點誤上了賊船到時不知道會被載到哪裡去說。<br/>
    <br/>
　此時司機又重新開了門喊著：<br/>
    <br/>
　"先生，你是要到永和嗎？"<br/>
    <br/>
　我趕緊又跑回車上："對，我要到永和。"<br/>
    <br/>
　司機："下車。"<br/>
    <br/>
　我本能的反應動作又下了車，不過心想不對勁於是又跑回去敲門，<br/>
<br/>
　司機先生似乎有點不耐煩了，開門後我又衝上去問：<br/>
    <br/>
　"司機先生，請問您剛剛是到永和下車再付錢的意思嗎？"<br/>
    <br/>
　司機："是低。"<br/>
    <br/>
　剎那間我感到超"囧"的，<br/>
    <br/>
　坐在前端的乘客們非常地開心看著我上上下下的短劇演出，<br/>
    <br/>
　我：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  "那......請載我到永和吧。"(臉紅)<br/>
    <br/>
  太可憐了我，以後絕對不可以忘了鑰匙呀。<br/>
    <br/>
    <br/>
