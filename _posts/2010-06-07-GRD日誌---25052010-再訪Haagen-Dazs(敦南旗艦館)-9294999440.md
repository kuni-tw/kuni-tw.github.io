<br/>　便一時興起，信步走到了忠孝敦化路口。<br/><br/><br/><br/><br/><br/>　　(Haagen Dazs本館門口迎面而來一位動人的妙齡女郎)<br/><br/><span></span><br/><br/><br/><br/><br/><br/>　許久不見的Haagen Dazs旗艦本館，<br/>																			    <br/>　記得好多年前剛落成時，奢華與夢幻般的獨棟建築物，<br/>																			    <br/>　曾給我過無比的震撼，如今時過境遷，<br/>																			    <br/>　館身與裝潢上已增添了不少歲月的痕跡，<br/>																			    <br/>　卻仍舊不減其魅力。<br/><br/><br/><br/><br/>  (夢幻般的階梯直上二樓)<br/><br/><span><br/><br/><br/><br/>　　(內部裝潢)<br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　點了份花之戀，裡頭附六球自選口味的冰淇淋，<br/>																			    <br/>　置放在脆餅花籃裡頭，上頭點綴著幾片草莓與青蘋果，<br/>																			    <br/>　要價NT.280。<br/>																			    <br/>  冰淇淋部分則選了草莓起司蛋糕，焦糖牛奶，芒果百香果，抹茶，<br/>																			    <br/>  藍姆葡萄，焦糖巧克力布朗尼，不一會兒便被端上桌來。<br/>													   <br/><br/><br/><br/>　Haagen Dazs向來除了以高品質的冰淇淋著稱外，<br/>																			    <br/>　另外還有多變性的細膩口味創新，比如這一次的草莓起司，<br/>																			    <br/>　起司酸奶味與微鹹配合著草莓的酸甜與果香，<br/>																			    <br/>　或芒果百香果，兩種都是以濃厚的獨特香味著稱的水果，<br/>																			    <br/>　混搭在一起竟然可以如此的契合與互補。<br/>																			    <br/>　這些較獨特的口味變奏曲，當然並不是只有在Haagen Dazs中才見得到，<br/>																			    <br/>　不過對於個人來說，相較於他牌，<br/>																			    <br/>　Haagen Dazs似乎更能掌握到不同風味的細節以及特色。<br/><br/><br/><br/><br/><br/><br/><br/><br/>　　(花之戀)<br/><br/><br/><span><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　不過不知是否溫度沒控制好，冰淇淋剛端上桌，<br/>
