 標題  0801的早晨<br/>
 時間  Tue Jan  8 07:14:21 2008<br/>
───────────────────────────────────────<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  同樣是早晨,卻沒有燦爛的陽光直射,沒有該有的活力,<br/>
<br/>
  手中握著一只夜光杯,黑眼圈中帶著一絲疲憊,懷著滿滿的疑問:<br/>
<br/>
  難道不是?飛揚的塵埃哪兒去了?<br/>
<br/>
<br/>
<br/>
  美麗的娘子婀娜多姿地緩緩走來,隨著慢板的節奏踏著細碎的步伐,<br/>
<br/>
  溫柔的拿著一碗愛情,和著失傳已久的儂儂吳語,<br/>
<br/>
  有如一朵初綻放的茶花,美到窒息.<br/>
<br/>
<br/>
<br/>
  失魂落魄:男人隨著征戰不休不眠,滿臉的鬍渣與骯髒的鬢角,<br/>
<br/>
  油膩膩的滿臉上歷盡滄桑,充滿著雨水與泥水,<br/>
<br/>
  連夜風塵僕僕地趕了回來,望著溫柔,倒地不起.<br/>
<br/>
<br/>
<br/>
  沒有比失魂落魄的早晨更具有潦倒的美了.<br/>
<br/>
