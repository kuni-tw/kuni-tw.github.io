<br/>
<br/>
<br/>
　到今天二十五號GRDII也使用了一個月,本來沒有預計要寫些什麼的.<br/>
<br/>
  但是今天整天照了幾十張,上傳了幾次,刪了刪,篩選又篩選,<br/>
<br/>
  挑剔到了最後乾脆把整份資料夾丟給資源回收桶.<br/>
<br/>
<br/>
　構圖已經不夠標準了呢!<br/>
<br/>
<br/>
  心下也訝異了一番,想了想,最近練琴也發生了類似的事情,<br/>
<br/>
  搞到最後其實沒一樣東西是滿意的,這大概就是我最常遇到的每下愈況吧.<br/>
<br/>
  以前打球也是,一開始可能在籃下奮力投擲幾百球之後,漸漸要求連進二十次,<br/>
<br/>
　三十次,等到都成功後,改成左手依樣畫葫蘆,然後繼續加大距離.<br/>
<br/>
  過了不久,連出力方式都要管,練完之後再單腳跳操場幾圈...簡直是無限輪迴.<br/>
<br/>
<br/>
　出門逛了幾家看了些DC,本想買些備用的便宜機種,查好了價錢,<br/>
<br/>
　後來SONY的W系列跟Fuji的F四五十系列操弄到了最後,買的竟然是乾燥包,<br/>
<br/>
  心想,最近WINTER琴盒的溼度節節飆高,可能已經快到紅線區了,<br/>
<br/>
  相機所需的強大除濕應該是足夠琴盒使用,當場就下手送了張紅色飯票.<br/>
<br/>
　沒辦法,當使用過了GRDII之後,連G系列都覺得頗不足了,<br/>
<br/>
　更何況是以CP值為前提的.<br/>
<br/>
　不過這下問題可大了,因為當初是早就有了心理準備甚至是盡全力查詢了所有價格,<br/>
<br/>
  才放手去店家的,沒想到是這樣的結果,看樣子我也得了GRD病狀了.<br/>
<br/>
<br/>
<br/>
  這一個月以來,GRDII給了我非常滿足的時光!<br/>
<br/>
  最新狀況是昨天的資訊:原來GR本來的意思是Greatest Ricoh呀,<br/>
<br/>
  現在我終於了解了這高檔的規格是從何而來了.<br/>
<br/>
  練習了很久,因為操控上有無窮的可能性,以至於很難滿足,<br/>
<br/>
　但有趣的是,它終能達成目標.<br/>
<br/>
　到了今天我總算可以稍稍地確認我有能力將腦子裡的想法付諸於現實,<br/>
<br/>
　雖然離完美還有十萬八千里,但是至少在現階段我是滿意的.<br/>
<br/>
　只是,腦子也不得不承認還趕不上GRDII的能力範圍呢.<br/>
<br/>
<br/>
<br/>
  老兄,我看你從現在開始再重新研究一次新的可能性吧.<br/>
<br/>
