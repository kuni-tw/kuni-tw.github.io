<br/>
    (個人超推薦的芝麻抹茶拿鐵加一份濃縮咖啡)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
---------------------------------------------------<br/>
竹林門市<br/>
服務地址： 台北縣永和市竹林路222-1號<br/>
服務電話： 02-2921-2560<br/>
營業時間： (日~四)07:30~22:00 (五~六)07:30~22:30<br/>
---------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
  最近星巴克推出台灣區限定的芝麻抹茶拿鐵，<br/>
<br/>
　銷售狀況出乎預料地好，可說是趨之若鶩，<br/>
　<br/>
　一堆星友們連嚐都沒嚐到就已賣到缺貨。<br/>
<br/>
　<br/>
　昨天聽說有些門市已重新開始補貨，今天便帶著心愛的GRD2，<br/>
<br/>
　到竹林門市點了一杯Grande大小，另外加一份SHOT，<br/>
<br/>
　這樣原本要價是145，不過使用了最近正夯的好友卡，<br/>
<br/>
　可免費升級，所以只要付中杯的價錢即可，再加上杯扣折10，<br/>
<br/>
  所以實際上付120，<br/>
<br/>
　另外還有點一份辣味燒肉帕里尼(NT.65)以及焦糖核果塔(NT.40)。<br/>
<br/>
<br/>
<br/>
　竹林門市因為地理位置相當好，附近住著不少學生以及上班族，<br/>
<br/>
　平常門庭若市，可惜與鄰近的水源門市相比之下，<br/>
<br/>
　營業時間不夠長，殊為可惜。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(竹林門市內隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　黑芝麻抹茶拿鐵本身若不加SHOT的話，奶味較重。<br/>
<br/>
　類似胡椒餅的香味似乎是從芝麻來的，聞起來令人精神為之一振，<br/>
<br/>
　加一份SHOT後，星巴克獨特的濃縮咖啡味與芝麻香緊密結合在一起，<br/>
<br/>
　完成度相當高，實在是繼太妃核果拿鐵以外又一次成功的味覺饗宴。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(黑芝麻抹茶拿鐵加SHOT超好喝！！！！！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　焦糖核果塔很特別的口感，內餡鬆脆有點類似沙其瑪，<br/>
<br/>
　卻更緊實些，厚重的奶油榛果味，偏甜，<br/>
<br/>
　所以一塊剛好，過多就膩嚕。<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (焦糖核果塔)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (內餡鬆脆有點類似沙其瑪)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(邊享受美食邊看著竹林門市外的街景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　辣味燒肉帕里尼則令人感到有點小失望了，<br/>
<br/>
　味道毫無獨特性，似辣非辣，有點類似叉燒包的調味，<br/>
<br/>
　但卻又不夠到位。<br/>
<br/>
　所幸的的是麵包品質夠，口感與嚼勁還足。<br/>
<br/>
　<br/>
<br/>
<br/>
