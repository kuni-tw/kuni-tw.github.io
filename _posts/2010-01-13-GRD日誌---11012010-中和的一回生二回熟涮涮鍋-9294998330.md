一回生二回熟涮涮鍋<br/>
台北縣中和市中和路296號 (捷運永安市場出口左轉步行約3分鐘)<br/>
02-29222822<br/>
11:00 am － 12:00 pm<br/>
---------------------------------------------------------<br/>
<br/>
<br/>
傍晚將近約十點時，終於有時間可以大快朵頤，<br/>
<br/>
　就在永安市場站附近步行了好一會，<br/>
<br/>
　走著走著就看到一家看起來相當有質感的涮涮鍋店：一回生二回熟，<br/>
<br/>
　在這麼晚的時間點，這麼冷的天氣裡，<br/>
<br/>
　竟然能看到明亮又有氣氛的燈光與裝潢，<br/>
<br/>
　店內一股又一股的縷煙緩緩升起，激盪出溫暖的食慾，<br/>
<br/>
　不覺之中右手便伸了去，門就這樣地，神奇的自動開啟。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
　　(菜單與點餐選擇)<br/>

<br/>
<br/>
<br/>
　平常相當討厭鍋類的小弟，自上一次老四川的好經驗後，<br/>
<br/>
　便重新燃起了對於涮涮鍋的好奇心，<br/>
<br/>
　在吧檯前就座後，點了一盤溫體黑毛梅花豬肉，<br/>
<br/>
　配上漢方回味鍋底，套餐價250，<br/>
<br/>
　選了細冬粉，甜點則是手工法式焦糖布丁。<br/>
<br/>
　後來吃到一半又加點了一份芋頭(NT.40)與一盤花雕酒滑雞肉(NT.120)<br/>
<br/>
<br/>
<br/>
<br/>
　首先服務生端來了漢方回味鍋底，一股特殊的香氣散了出來，<br/>
<br/>
　中藥類的湯底並不是沒聞過，而這鍋底的香味令我驚奇的是，<br/>
<br/>
　竟然沒有過重而又令人討厭的中藥味。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(漢方回味鍋底)<br/>

<br/>
<br/>
　然後依序上桌的是菜盤，粉絲以及溫體黑毛梅花豬肉，<br/>
<br/>
　菜盤與肉盤擺得整齊好看，料洗得很乾淨，<br/>
<br/>
　包括後來另外加點的芋頭與花雕酒滑雞肉。<br/>
<br/>
　供客人自行調配的醬料桌也是，相當吸引人的光線與擺設，<br/>
　<br/>
　醬料選擇頗多而不落俗套，個人吃到後來甚至興致一起，<br/>
<br/>
　總共調了三次不同的組合，味道都還滿喜歡的，<br/>
<br/>
　相當討人開心。<br/>

<br/>
　小弟相當重視這些小細節，記得好久以前吃過的幾家涮涮鍋，<br/>
<br/>
　同樣的價位，桌子也沒擦得很乾淨，擺盤雜亂無章，<br/>
<br/>
　調味桌上永遠是那幾款醬料，一成不變，<br/>
<br/>
　那時候就覺得，既然如此那幹嘛還要吃這種吃不飽的日式火鍋呢？<br/>
<br/>
<br/>
<br/>
<br/>
　現在說說一些小感想吧，首先是豬肉部分，<br/>
<br/>
　其實還滿好吃的，不過不知怎麼的，<br/>
<br/>
　每一次涮肉時，似乎總會涮過熟，撈起來時肉質已偏老，<br/>
<br/>
　花雕雞部分則反，選的是去骨大雞腿部位，切得較厚，<br/>
<br/>
　一開始很難拿捏時間，很容易吃到生的，<br/>
<br/>
　不過煮久一點，其實還滿好吃的。<br/>

<br/>
　鍋底部分，香氣與味道皆不錯，<br/>
<br/>
　溫和宜人而又自然，味道有層次，<br/>
<br/>
　建議加點芋頭，吃到一半時倒進去一起煮，<br/>
<br/>
　湯頭會愈來愈濃郁，很棒呢。<br/>
<br/>
<br/>
<br/>
<br/>
　點心則令人稱奇，呃，我真的得這麼說，<br/>
<br/>
　單點一份才18的手工法式焦糖布丁，<br/>
<br/>
竟然是我這幾年來吃過最好吃的焦糖布丁之一，<br/>
<br/>
　外表看起來平淡無奇，但上面那一層焦糖才是重點所在，<br/>
<br/>
　非常薄的一片焦糖，富有著又鬆又脆的細緻口感，<br/>
<br/>
　有著畫龍點睛的效果，呃...<br/>
<br/>
有哪一家咖啡廳還是蛋糕店願意花功夫作這一道甜點嗎，<br/>
<br/>
　記得要賣18元，小弟一定飛奔過去叨擾呀！<br/>
<br/>
<br/>
<br/>
<br/>
　吧檯後的擺架放著不少沒看過的梅酒，<br/>
<br/>
　後來大姊服務生拿了其中一瓶(藏元梅酒)，請我喝了一兩小杯，<br/>
<br/>
　淺酌一兩小口後，層次感浮出來，<br/>
<br/>
　鼻腔裡多了一道舒服而又難以形容的香氣，<br/>
<br/>
　問了一下價錢，一瓶要價400，<br/>
<br/>
對於這相當少見的梅酒風味來說，<br/>
<br/>
　還滿值得另外購買回家嚐鮮。<br/>
<br/>
