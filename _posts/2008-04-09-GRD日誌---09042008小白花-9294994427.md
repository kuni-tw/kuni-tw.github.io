<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
　看著這一張潔白無暇卻有些淡然無味的小白花照片,心裡霎時間多了份恬淡之意.<br/>
<br/>
  不禁訝異了起來:"我究竟是什麼時候開始對大自然有了認同呢?"<br/>
<br/>
  小時後每當週日爸媽定拎著我跟弟弟出遊,有時候開車到東北海岸吹風,<br/>
<br/>
  嚐鮮或是看海,有時與外公外婆爬山,有時候回北投洗溫泉.<br/>
<br/>
  在記憶當中,似乎從來沒有一起去看過什麼電影,逛街逛遊樂場.<br/>
<br/>
  小孩子不喜歡看山看海的,更無法了解爸媽動不動停住步伐觀賞路邊的野花野草,<br/>
<br/>
  但他們對此似乎樂此不疲.<br/>
<br/>
  後來媽陪我出國唸書,一樣地,只要是假日她一定一大早就把我叫起,<br/>
<br/>
  吃完早餐之後細細研究地圖上的路線,然後帶著我坐電車去市郊探險.<br/>
<br/>
  記得有一次找到了聞名的Beethovengang,順著羊腸小徑走到了田野,<br/>
<br/>
　聽到那河水淙淙流過的聲音,媽媽整個人神采飛揚,至今令人難忘呀.<br/>
<br/>
<br/>
  或許我也只是小恐慌了一下,深怕自己也漸漸蒙上中年人的陰影,<br/>
<br/>
　不,我不會讓這事發生的,<br/>
<br/>
  因為現在不正是小白花盛開的時候嘛!<br/>
<br/>
