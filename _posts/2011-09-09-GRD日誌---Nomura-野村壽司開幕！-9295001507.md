<p><span>---------<br/>NOMURA 野村壽司<br/>地址：台北市安和路一段78巷34號<br/>電話：02-2755-6587<br/> ---------</span><span> </span></p>
<p><span>自裕二師傅離開台灣後，小弟再也沒回去過野壽司。</span></p>
<p><span>這次得知師傅本人要重新在台北開業，並以其為店名叫野村壽司。</span></p>
<p><span>當下興奮無比馬上撥電話訂位，可惜晚間時段全滿只剩中午。</span><br/> <br/><span>(野村壽司門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212421.jpg" target="_blank"></a><br/></span></p>
<p><span>(桌子是相思木材)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212422.jpg" target="_blank"></a><br/></span></p>
<p><span>(備前燒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212423.jpg" target="_blank"></a></span></p>
<p><span>在台北美食界以傳統江戶前壽司來說，裕二師傅絕對是我的首選。</span></p>
<p><span>在精緻度上或許不夠細心與嚴謹，但在味覺饗宴上從沒讓小弟失望過。</span></p>
<p><span>野村壽司最終地點選在前助手濟安開的魚道生附近倒是始料未及。</span></p>
<p><span>安和與信義路巷弄中的店面低調，充滿著高雅與氣質。</span></p>
<p><span>店裡以原色木造裝潢為主，十足的傳統日式風情完整度高。</span></p>
<p><span>客人們陸續抵達，並在裕二師傅熱情招呼下就座。</span></p>
<p><span>這時發現一位女子手裡抱著未足歲的小孩，</span><span>原來是裕二師傅的妻兒。</span></p>
<p><span>她在師大語言中心學過一段時間的中文，</span><span>彌補了裕二師傅與客人在溝通上的問題。</span></p>
<p> </p>
<p><span>(裕二師傅)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212424.jpg" target="_blank"></a><br/></span></p>
<p><span>(裕二師傅正在捏製壽司)</span></p>
<p><span> <br/></span></p>
<p><span>在開幕階段中午只有一種套餐，NT.1200，共十貫握壽司與細捲等。</span></p>
<p><span>首先是軟絲沙拉與鮪魚花椒醋，</span></p>
<p><span>前者調味清淡，</span><span>後者味道較濃。</span></p>
<p><span>鮪魚口感偏乾有點不太喜歡，</span><span>兩者藉著微酸與麻達到開胃的效果。</span></p>
<p> </p>
<p><span>(軟絲沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212426.jpg" target="_blank"></a><br/></span></p>
<p><span>(鮪魚花椒醋)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212427.jpg" target="_blank"></a><br/></span></p>
<p><span>第一貫是軟絲握，睽違已久的正統江戶前握壽司終於回來了。</span></p>
<p><span>切得稍厚的生軟絲口感綿密緊實，配合微溫濕黏又入口即化的醋飯，</span><span>令人懷念呀。</span></p>
<p><span>(軟絲握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212428.jpg" target="_blank"></a><br/></span></p>
<p><span>接下來是小鯛握與紅甘握，兩貫肉質細嫩，與美味的醋飯為一體。</span></p>
<p><span>第四貫是白蝦握，不知是不是錯覺，醋飯比例似乎過多了。</span></p>
<p><span>這現象似乎在接下來的幾貫也曾出現過，甚至連一旁的客人也特地跟師傅反映。</span></p>
<p><span>果不其然師傅經老婆翻譯，告訴大家這次的米有特別選過。</span></p>
<p><span>是從日本運來的一整年份糙米(好幾百公斤)，</span></p>
<p><span>然後另外買台精米機每天現碾，這樣才能確保醋飯的品質</span><span>。</span></p>
<p> </p>
<p><span>(小鯛握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212429.jpg" target="_blank"></a><br/></span></p>
<p><span>(紅甘握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212430.jpg" target="_blank"></a><br/></span></p>
<p><span>(白蝦握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212431.jpg" target="_blank"></a><br/></span></p>
<p><span>第五貫黑鮪大腹握是裕二的拿手好戲，</span></p>
<p><span>每次吃他這貫壽司都會激動一番，是台北最棒的。</span></p>
<p><span>品嚐起來，就像將所有美味凝聚在短短一兩秒鐘的感覺。</span></p>
<p><span>然後閉起眼睛，將剛流逝的幸福努力地去記住。</span></p>
<p><span>(裕二的大腹握堪稱極品)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212432.jpg" target="_blank"></a><br/></span></p>
<p><span>第六貫是日本馬份海膽軍艦捲，海膽品相輪廓完整，一看便知是高檔貨。</span></p>
<p><span>至此1200早就值回票價了。</span></p>
<p><span>(海膽軍艦)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212433.jpg" target="_blank"></a><br/></span></p>
<p><span>(冰鎮清酒用的木造桶)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212434.jpg" target="_blank"></a><br/></span></p>
<p><span>(另外單點的清酒)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212435.jpg" target="_blank"></a><br/></span></p>
<p><span>第七貫是青花魚昆布握，覆蓋在魚料上的昆布顏色極淺透明。</span></p>
<p><span>來自日本的青花正值產季，油脂豐厚肥美無比好吃極了。</span></p>
<p><span>(青花魚昆布握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212436.jpg" target="_blank"></a><br/></span></p>
<p><span>此時中間穿插的鐵火捲上桌，</span></p>
<p><span>可惜切工不是很平整，</span><span>不過醋飯煮得好似乎怎麼做都好吃。</span></p>
<p> </p>
<p><span>(鐵火捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212437.jpg" target="_blank"></a><br/></span></p>
<p><span>接下來是竹莢握與赤身握，前者魚料上佐些薑末。</span></p>
<p><span>後者沒有醃漬過，出乎我意料，</span></p>
<p><span>但上面仍刷些醬料比醃漬過的多了點清爽。</span></p>
<p><span>(竹筴握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212438.jpg" target="_blank"></a><br/></span></p>
<p><span>(赤身握)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212439.jpg" target="_blank"></a><br/></span></p>
<p><span>突然一份迷你缽上桌，從沒看過這麼小份的鮭魚子丼飯。</span></p>
<p><span>使用的鮭魚子來自於北海道沿岸以定置網捕獲的鮭魚。</span></p>
<p><span>尺寸較小且卵膜極柔軟，卵液相當甘甜。　</span></p>
<p><span>(鮭魚子丼)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212440.jpg" target="_blank"></a><br/> </span></p>
<p><span><span>此時再度穿插一份小鯽細捲，與前項細捲一樣有切工不平整的缺憾。</span></span></p>
<p><span>(小鯽細捲)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212441.jpg" target="_blank"></a><br/></span></p>
<p><span>最後一貫是煮穴子魚握，細心的將刺完全挑掉肉質煮得極軟。</span></p>
<p><span>此時味噌清湯上桌，融進濃郁昆布精華的湯頭鮮美爽口有層次。</span></p>
<p><span>最後則是野村壽司使用明膠自製的豆花甜點，別出心裁。</span></p>
<p><span>以1200來說，令人驚艷。</span></p>
