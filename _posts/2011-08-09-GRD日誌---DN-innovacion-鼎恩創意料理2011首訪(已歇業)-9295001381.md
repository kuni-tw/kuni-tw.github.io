<p><span><br/> --------------------------------<br/>DN innovacion 鼎恩創意料理<br/>官網：<a href="http://dntaipei.com">http://dntaipei.com</a><br/>店址：台北市信義區松仁路93號<br/>電話：(02)8780-1155<br/> --------------------------------<br/> <br/>(DN Innovacion)<br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207215.jpg" target="_blank"></a><br/> <br/> <br/>終於又到了一年一度的父親節，本來預定要吃的是茹絲葵，</span></p>
<p><span>後來臨時想起前EL TORO主廚Daniel Negreira去年十月重開一家高檔餐廳，</span></p>
<p><span>就是鼎鼎大名的DN innovacion，當下馬上打去訂位。</span></p>
<p><span>Daniel擁有在台灣料理界相當顯赫的地位尤其是相當完整的資歷，</span></p>
<p><span>曾在西班牙許多一流名廚手下做事，待過的高檔餐廳名單多的嚇人，</span></p>
<p><span>包括：</span></p>
<p><span>El Bulli(知名分子料理餐廳，米其林三星，2002，2006~2009排名世界第一 )</span></p>
<p><span>Akelarre(米其林三星)</span></p>
<p><span>Mugaritz(米其林三星，2007排名世界第七，2008~2009排名第四)</span></p>
<p><span>Arzak(米其林三星，2006~2008世界排名第八，2009世界排名第七)</span></p>
<p><span>Berasategui Lasarte(米其林三星)<br/> <br/>(大門口)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207216.jpg" target="_blank"></a><br/> <br/> <br/>這麼棒的人後來怎麼來到台灣呢？</span></p>
<p><span>很幸運地他後來成為台灣女婿所以來到寶島定居，由此可見愛情有多麼偉大呀。<br/> <br/>(餐廳內裝潢)<br/> <br/> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207217.jpg" target="_blank"></a><br/> <br/> <br/>(服務生幫斟礦泉水)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207218.jpg" target="_blank"><br/> </a><br/> <br/>(氣泡式礦泉水加檸檬片)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207219.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>傍晚與家人一同來到位在松仁路上的DN innovacion，</span></p>
<p><span>進入餐廳內極為現代化的裝潢映入眼簾，四處瀰漫著</span><span>優雅與奢華，</span></p>
<p><span>冷色調與偏暗的燈光成就質感與寧靜。</span></p>
<p><span>就座後才發現隔壁包廂裡坐著不少媒體常客比如陳文茜等，</span></p>
<p><span>四人點了鼎恩父親節特別套餐2011(NT.2688+10%)</span></p>
<p><span>裡頭包括：<br/></span></p>
<p><span>1.精選手工麵包佐三式麵包佐料<br/></span></p>
<p><span>2.西班牙伊比利BELLOTA火腿佐蕃茄沙沙<br/></span></p>
<p><span>3.主廚自製鵝肝醬佐Ganny Smith青蘋果襯胡蘿蔔&amp;薑吐司<br/></span></p>
<p><span>4.雪莉酒香氣放山雞澄清湯佐碳烤義大利野生牛肝菌<br/></span></p>
<p><span>5.碳烤二分之一波士頓龍蝦襯主廚Daniel三式風味醬汁<br/></span></p>
<p><span>(媽選爐烤腰果，粉紅胡椒大干貝襯椰奶綿密泡沫)<br/></span></p>
<p><span>6.義大利夏季新鮮黑松露燉飯襯一年熟成特級大米“Riso del Pavesse”<br/></span></p>
<p><span>7.低溫式爐烤西班牙伊比利乳豬<br/></span></p>
<p><span>(爸選碳烤澳洲和牛MB5"肋眼"襯主廚精製配菜)<br/></span></p>
<p><span>8.D.N 特製父親節甜點「金色巧克力世界」<br/></span></p>
<p><span>9.咖啡或茶<br/></span></p>
<p><span>另加NT.1888的Wine Pairing，請侍酒師Tony共配六支酒。<br/></span></p>
<p><span>上菜前服務生先端來長條狀的脆麵包搭配三種不同的高級橄欖油，</span></p>
<p><span>透過那微微散出的清香與芬芳我竟然可以嚐得出每一種橄欖油的細微差異，</span></p>
<p><span>這實在是太讓我驚訝了。</span></p>
<p><span>不過麵包條有分芝麻與抹茶兩種口味，尤其是前者味道似乎會蓋住橄欖油的，</span></p>
<p><span>倒有點可惜，不過想了想若改使用無味的麵包條效果似乎也沒有比較好。<br/> <br/></span></p>
<p><span>(脆麵包條與三種不同的高級橄欖油)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207220.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>然後侍酒師Tony端來第一支配酒：Dom Perignon 2000</span></p>
<p><span>味道均衡，泡沫與香氣細緻，相當適合當餐前酒。</span></p>
<p><span>與台北Robuchon如出一轍DN Innovation隨後也端來麵包籃，</span></p>
<p><span>麵包本身部分口感偏軟沒什麼嚼勁這部分個人較偏好Robuchon，</span></p>
<p><span>但白松露奶油實在令人驚艷，香味隨著奶油化開悠長而有層次。<br/> <br/>(Tony拿的是Dom Perignon 2000)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207221.jpg" target="_blank"></a><br/> <br/>(橄欖油，羅勒與白松露奶油)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207222.jpg" target="_blank"></a><br/> <br/>(麵包籃)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207223.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>接下來是西班牙伊比利BELLOTA火腿佐蕃茄沙沙，</span></p>
<p><span>使用的是西班牙國寶級熟成五年以上的伊比利BELLOTA火腿，</span></p>
<p><span>肉片呈深褐紅色泛出滿滿油光如紅寶石般艷麗無比，</span></p>
<p><span>嚼勁綿密軟嫩，配合充滿核桃芬芳的油脂與微鹹令人陶醉在其中，</span></p>
<p><span>貼切詮釋何謂至高無上的幸福。<br/> <br/>(西班牙伊比利BELLOTA火腿佐蕃茄沙沙)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207224.jpg" target="_blank"></a><br/> <br/>(紅寶石般的色澤)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207225.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>終於第一道"類"分子料理上桌：</span></p>
<p><span>主廚自製鵝肝醬佐Ganny Smith青蘋果襯胡蘿蔔&amp;薑吐司，</span></p>
<p><span>其中部分鵝肝醬做成細沙狀，另外那黃色的"類"魚子醬其實是用橙汁製成。</span></p>
<p><span>光聽就覺得誇張，而當我將那細沙舀進口腔裡後腦海裡竟然真出現鵝肝醬，</span></p>
<p><span>但整體而言美味程度完全不及前面伊比利BELLOTA火腿所給我的震撼。<br/> <br/>(主廚自製鵝肝醬佐Ganny Smith青蘋果襯胡蘿蔔&amp;薑吐司)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207226.jpg" target="_blank"></a><br/> <br/> <br/>(好美的逆光照)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207227.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>此時Tony帶來西班牙的白酒Finca Allende Blanco 2006</span></p>
<p><span>初嚐一口香氣濃郁非常特別，是從來沒聞過的，</span></p>
<p><span>這支酒是為了搭配接下來的前餐使用的。</span></p>
<p><span>服務生先端來一只加溫過熱白瓷盤，裡頭端坐牛肝菌與雞肉捲。</span></p>
<p><span>這兩位主角在此之前被Daniel以低溫69度燉煮了八小時(可憐XD)，</span></p>
<p><span>然後服務生在我面前將熱湯斟入盤內，大概是擔心那兩位主角過於脆弱，</span></p>
<p><span>淨泡熱湯過久會逐漸喪失口感吧。(笑)<br/></span></p>
<p><span>不過似乎沒什麼用，牛肝菌香氣與味道雖然不錯但口感似乎有點過軟散。</span></p>
<p><span>在此時小啜了一口酒不禁驚訝萬分，酒香氣突然像似升級一般極為濃郁，</span></p>
<p><span>甜味層次被拉長與擴大許多，充滿熱情與奔放，令人回味無窮，</span></p>
<p><span>這就是侍酒師的功力！<br/><br/>(Finca Allende Blanco 2006)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207228.jpg" target="_blank"><br/> <br/></a></span></p>
<p><span>(服務生在我面前將熱湯斟入盤內)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207229.jpg" target="_blank"></a><br/> <br/>(雪莉酒香氣放山雞澄清湯佐碳烤義大利野生牛肝菌)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207230.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>這時Tony帶來Pravis Polin 2009。</span></p>
<p><span>一支義大力產區的白酒，香氣中庸酸與甜味較前一支少，</span></p>
<p><span>搭配接下來由Daniel親自挑選回來的波士頓龍蝦。</span></p>
<p><span>不愧是</span><span>頂尖廚師看上的食材，波士頓龍蝦本身號稱龍蝦界的鐵甲武士，</span></p>
<p><span>但這是我第一次吃到如此豐富嚼勁與口感的波士頓龍蝦，實在太棒了。</span></p>
<p><span>美中不足的是竟然是這一次所搭的配酒，</span></p>
<p><span>前一支西班牙Finca Allende Blanco 2006可能是太濃郁了，</span></p>
<p><span>以至於義大力這支顯得力道不足，就這部分與Tony聊一會，</span></p>
<p><span>這本身似乎就是一種矛盾，又不可能將龍蝦往前挪，</span></p>
<p><span>雖仍舊無解但還是聊得很開心。<br/> <br/>(Pravis Polin 2009)</span></p>

<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207236.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>然後Tony帶來Lopez de Heredia Vina Tondonia Blanco Reserva 1992，</span></p>
<p><span>連續第三支白酒，</span><span>搭配是至今這幾年來在台灣吃過最棒的義式燉飯。</span></p>
<p><span>以前在歐洲生活時曾吃過一次印象非常深刻，</span></p>
<p><span>那一餐花了當時仍身為在職窮學生的我足足兩三周的生活費，</span></p>
<p><span>難以形容的Ｑ彈嚼勁與剛好煮透的米心口感至今仍令人難以忘懷，</span></p>
<p><span>輾轉吃過台北那麼多家的燉飯一直無功而返今天卻讓我在DN Innovation重新吃到了。</span></p>
<p><span>但嚴格來說似乎還是有那麼一點點微妙的差異，</span></p>
<p><span>應該說跟記憶中那一頓比起來在彈牙口感上還是弱了點，但這無傷大雅，</span></p>
<p><span>使用置放一年以上的Risotto上藉著熱氣散出黑松露薄片的微妙香氣，</span></p>
<p><span>這讓我很佩服Daniel那靈敏的嗅覺與巧思，超級厲害。</span></p>
<p><span>這一次搭配的白酒給我感覺似乎介於前兩支的中間，</span></p>
<p><span>明顯的蘋果酸給人清爽舒服的感覺，藉著濃郁的Risotto再度提昇了一個層次，很棒。<br/> <br/>(Lopez de Heredia Vina Tondonia Blanco Reserva 1992)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363207237.jpg" target="_blank"></a><br/> <br/>(義大利夏季新鮮黑松露燉飯襯一年熟成特級大米“Riso del Pavesse”)</span></p>

<p><span>最後搭配伊比利乳豬的是Alvaro Palacios Camins del Priorat 2008，</span></p>
<p><span>與牛肝菌同樣使用低溫做烘烤(攝氏150度)然後外皮微炙。</span></p>
<p><span>裡頭肉汁似乎有點偏乾，外皮過韌，但由於豬種實在是太優了，</span></p>
<p><span>肉愈嚼愈有味道與散出微妙又難以形容的香氣，反而到後頭變得還滿好吃的，</span></p>
<p><span>這一支紅酒表現似乎沒留下令人印象深刻的感覺。</span></p>

<p><span>最後是Lustau Almacenista Sherry Angel Zamorano Oloroso 1/16</span></p>
<p><span>用來搭配DN特製的父親節甜點"金色巧克力世界"，</span></p>
<p><span>這是一支極少見的雪莉酒酒體，充滿著奶油與堅果香。</span></p>
<p><span>當服務生將溫熱的不知名液體澆在金色巧克力球上面時，</span></p>
<p><span>球體突然崩塌散開露出裡頭的果餡。</span></p>
<p><span>加上溫度升高的關係，巧克力口感變微軟增添不少趣味。</span></p>
<p><span>此時拿起雪莉酒小酌一口，蜂蜜甜突然變得明顯，</span></p>
