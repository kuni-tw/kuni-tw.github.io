(02)25525719<br/>

<br/>
<br/>
  國內具指標性的複合式廣場京站已開幕幾天了，趁著幾位朋友北上來玩，<br/>
<br/>
　就作個東道主帶他們去京站逛逛，順便瞧瞧新鮮，<br/>
<br/>
　乘著捷運到了台北車站，下車後直接有引道通到京站入口，<br/>
<br/>
　非常方便。<br/>
<br/>
　<br/>
　入口處一排諾大的京站時尚廣場立體字，配合著適量的光源，<br/>
<br/>
　成就出一款相當精緻又富有質感的光影畫面，<br/>
<br/>
　這樣的氣氛持續蔓延在整棟京站大樓裡，充滿著一貫的主題性，<br/>
<br/>
  真是非常棒的設計。<br/>

　<br/>

<br/>
　白色系另兼具大理石質感的古典設計真是令人驚艷，<br/>
<br/>

<br/>
　這一次在知名的Cloudy Cupcake選了四款小蛋糕：　　<br/>
　<br/>
　北歐巧克力乳酪(NT.65)，爵士巧克力(NT.45)，提拉米蘇咖啡乳酪(NT.65)<br/>
<br/>
  以及草莓巧克力海(NT.65)。<br/>
<br/>
　以外觀上來說，Cloudy的小蛋糕作工相當精緻，<br/>
<br/>
　不過實地品嘗時，部分口味卻與視覺上有了些許的小落差。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (Cloudy Cupcake蛋糕展示櫃內)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(好漂亮的蛋糕)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(叮叮咚　叮叮咚　蛋糕多響亮...XD)<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　味覺感想：<br/>
<br/>
　北歐巧克力上面那一小團巧克力實有畫龍點睛之妙，<br/>
<br/>
　微苦卻異常濃郁的高水準巧克力膏狀物，相當美的旋在杯子蛋糕本體上，<br/>
<br/>
　口感與味覺皆上乘。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (北歐巧克力)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　爵士巧克力不知道是不是用料等級的關係，<br/>
<br/>
　巧克力部分一整個弱掉，這部分提拉米蘇咖啡乳酪亦有相同的困擾，<br/>
<br/>
　尤其是上面那一粒巧克力豆，口感味道皆不夠搭，有點可惜。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (爵士巧克力)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　　(提拉米蘇咖啡乳酪)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　草莓巧克力海表現較亮眼些，一樣得歸功於草莓下那一團濃郁的巧克力，<br/>
<br/>
　與北歐巧克力上的相比，味道較苦澀，有不一樣的香氣風貌，我喜歡，<br/>
<br/>
　理論上來說，這濃郁又澀苦的巧克力配合著草莓，<br/>
<br/>
　一定會有一加一大於二甚至三的亮眼演出，<br/>
　<br/>
　可惜卻不然，相當酸質的草莓亦是小敗筆之處呀。<br/>
<br/>
