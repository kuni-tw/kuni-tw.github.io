<br/>
<br/>
<br/>
<br/>
  (星巴克經典的不鏽鋼杯，左為16oz右為12oz)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  平常熱愛蒐集星巴克各式各樣隨行杯的小弟，<br/>
<br/>
　杯子買來都只擺在玻璃櫥櫃裡供著，<br/>
<br/>
　能力許可的話一種杯子至少要買個兩個以上。<br/>
<br/>
　<br/>
　但日常真正在使用的卻是耐用又兼具質感的星巴克不鏽鋼隨行杯，<br/>
<br/>
　簡潔的曲線中蘊含著銀色的打磨金屬光澤與黑色橡膠手握腰身，<br/>
<br/>
　樣式相當純樸低調。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (五款鋼杯大合拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　少了點現代奢華的繁縟，多了點簡潔的歐式雅痞美感。<br/>
<br/>
　堅固耐用的品質，堪稱中上的密閉性與極佳的保溫效果，<br/>
<br/>
　使用上卻又較保溫瓶方便許多。<br/>
<br/>
　相當簡易的構造使得洗滌容易，加上杯蓋可另外購買，<br/>
<br/>
　間接地增加其使用壽命，雖然一只16oz大小的就要價NT.900，<br/>
<br/>
  價格上佔了相當大的劣勢，但是有鑑於以上之優點，<br/>
<br/>
　這一款兼具實用性與型的保溫隨行杯，<br/>
<br/>
　成為個人每天出門必隨身攜帶的家當之一。<br/>
<br/>
　<br/>
<br/>
　除了16oz的大小之外，另還有12oz的可供選擇，<br/>
<br/>
　而且最近星巴克又出了一款新的16oz常態性不鏽鋼隨行杯，<br/>
<br/>
　少了點經典曲線美，多了點陽剛氣息，蓋子與前代合用，<br/>
<br/>
　也是不錯的選擇。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>

<br/>
<br/>
　有基於對於素色不鏽鋼杯的熱愛，每看到新款的被展示出來，<br/>
<br/>
　小弟不禁就會多注意一番，最近又買下兩款，<br/>
<br/>
　一款是日本的巨大樣式：20oz的Venti鋼杯(Venti本意就是20)<br/>
<br/>
　另一款則是託朋友從美國帶回來的。<br/>
<br/>
　不過基於這兩款在國內能見度不高，<br/>
<br/>
　他們只好靜靜地躺在櫥櫃裡，<br/>
<br/>
  無聲無息望著玻璃櫥窗外的世界...<br/>
