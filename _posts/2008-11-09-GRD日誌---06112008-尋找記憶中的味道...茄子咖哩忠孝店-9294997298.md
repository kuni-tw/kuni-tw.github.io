<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
　　(這張藍帶雞排咖哩寫真看起來還真是很誘人呀...XD)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　咖哩對我來說，一直以來都是一道很神奇的料理，<br/>
<br/>
　記憶中我對它如此著迷，就是從小時候開始，<br/>
<br/>
　每逢回祖母家時，她就會弄最拿手的咖哩雞肉燴飯給我吃，<br/>
<br/>
　熱騰騰的白飯上淋上濃稠稠的深紫色咖哩醬汁，<br/>
<br/>
　配上鮮嫩的雞肉，喔，只記得一口又一口滿滿扒飯的我，<br/>
<br/>
　可以三兩下就再添一碗白飯，<br/>
<br/>
　全身上下香汗淋漓之下連吃五大碗呢！<br/>
<br/>
<br/>
<br/>
<br/>
　咖哩的記憶就隨著後來在歐洲的成長之下，<br/>
<br/>
　慢慢地從我深刻的回憶中慢慢褪去了，<br/>
<br/>
　直到十多年後終於又回到了台灣，某次初訪茄子咖哩，<br/>
<br/>
　那份早已蒙塵許久的悸動又奇蹟般的復活了，<br/>
<br/>
　日式咖哩的魅力也從此後深植在我心中．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(已改名為茄子洋膳的茄子咖哩忠孝店店內空間)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　禮拜四下午臨時空檔，靈機一動，<br/>
<br/>
　想到也好久沒去光顧茄子咖哩了，<br/>
<br/>
　便隻身衝去東區的忠孝店，上了樓，<br/>
<br/>
　一如慣例挑了可以眺望忠孝東路的窗邊座位，<br/>
<br/>
　點一份藍帶雞排咖哩套餐，中辣，米飯可以免費續，<br/>
<br/>
　附洋蔥湯與小份凱薩沙拉加服務費則是341．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(等餐期間眺望午後的忠孝街景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　<br/>
　首先上的是洋蔥湯，深褐色的湯底，<br/>
<br/>
　香味口感尚可，然後是凱薩沙拉，相當小一份，<br/>
<br/>
　份量小味道也普普，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(副餐之一：洋蔥湯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(副餐之二：凱薩沙拉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　我不禁將希望放在許久不見的主菜老友身上，<br/>
<br/>
　然後又是一次的小失望：<br/>
<br/>
　咖哩醬顏色變了，不像以前那樣的深紫褐色，<br/>
<br/>
　色調較淺，雞排外皮不是酥脆，而是稍有點老韌，<br/>
<br/>
　米飯則偏乾！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(今天下午的重頭戲上場嚕：藍帶雞排咖哩！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(唉，我還是敵不過日式咖哩的誘惑呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　不過咖哩畢竟是我最愛的料理之一呀，<br/>
<br/>
　一不小心又將那整盤飯扒光，<br/>
<br/>
　不爭氣的我只好跟服務生要求加飯，<br/>
<br/>
　還真不愧對我這貪吃王的稱號呀...<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
　　(一口接一口，不小心又扒完一整盤飯了...)                                                                                <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (只好不爭氣的服務生要求"續"飯啦XD)<br/>
