<br/>
<br/>
　　(在Costco買到的蝦子與干貝，超鮮美！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  小風一句：就算沒約成也要烤！<br/>
<br/>
　在颱風天聽到這樣的決心，我便知道今晚例行的頂樓烤肉將會相當難忘，<br/>
<br/>
　首先去大潤發，買足了部分食材後，看到許久沒喝的15年的麥卡倫Fine Oak，<br/>
<br/>
　配上兩只美麗煥發的新款威士忌酒杯，價格又漂亮，<br/>
<br/>
　身為純種的麥卡倫FAN，我還是禁不住的拿進推車裡．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　後來又在風雨之中去了Costco，買了賣相還滿不錯的草蝦，干貝等等．<br/>
<br/>
　因為前幾天跟老爸老媽去外面共進午餐時，店家送的明治雪糕令我記憶猶新，<br/>
<br/>
　順便也挑了一大盒，記得是裡頭有包餡的：牛奶糖跟抹茶<br/>
<br/>
　本來還想挑兩枝NAPA Valley的紅酒，可惜在電話中得知朋友已陸續到來，<br/>
<br/>
　便趕緊回去．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (準備中的食材之一：去骨雞腿肉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(準備中的食材之二：與烤肉一起夾著吃的生菜與海苔)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(準備中的食材之三：醃鮭魚排)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (準備中的食材之四：草蝦)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　 (準備中的食材之五：Mozarella起司配番茄淋上橄欖油)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
      (端到頂樓的食材一景)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
      (小風烤肉英姿)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
      (歷經幾次中秋節烤肉的露營燈，超好用！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　　 (大夥後來竟然開始烤棉花糖了！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　在屋裡將食材準備完畢之後，端到頂樓，雖然風雨交加，<br/>
<br/>
　但是在四十坪的巨大遮雨蓬之下，完全不受到任何影響．<br/>
<br/>
　在吵雜的風聲與附近大廈頂樓急遽的排水聲之中烤肉，還真別有一番滋味呢！<br/>
<br/>
　不知道是否與颱風有關，炭火的溫度似乎一直不太令人滿意，<br/>
<br/>
　除非烤得夠久，不然食材熟得似乎都很慢，跟以往差很多．<br/>
<br/>
　後來將烤肉架調低，拉近與炭火的距離，這狀況有好一點，<br/>
<br/>
　不過仍舊差強人意，看來明年可以考慮找找有沒有較好的高溫炭，<br/>
<br/>
　哈哈哈，我已經開始期待下一次了！<br/>
<br/>
<br/>
　烤完後，回到屋內大夥一起看電視，颱風天似乎節目也比較好看些，<br/>
<br/>
　配著Godiva巧克力，Micky Mouse造型月餅，<br/>
<br/>
　以及過了不久之後此起彼落的麻將聲與風切聲，<br/>
<br/>
　08年的中秋之夜就此圓滿地落幕了！<br/>
<br/>
<br/>
