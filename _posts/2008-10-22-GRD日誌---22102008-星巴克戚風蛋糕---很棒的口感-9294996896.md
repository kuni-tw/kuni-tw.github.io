<br/>
<br/>
<br/>
　(推出沒多久的星巴克草莓戚風蛋糕，口感真的很棒！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  在連鎖咖啡界裡面對來勢洶洶的85Ｃ以向來較為拿手的平價糕點與點心挾擊之下，<br/>
<br/>
　星巴克真的有一場不好打的仗，尤其在價錢上又稍處弱勢，<br/>
<br/>
　若不推出較有特色的產品出來，恐怕就真的只能靠咖啡了．<br/>
<br/>
<br/>
　今年九月推出的草莓戚風蛋糕外觀上相當"低調"，一直沒引起我的注意，<br/>
<br/>
　反而是同時間上戰場的南瓜塔較引人注目，小弟也因此，<br/>
<br/>
　在十月初先試吃了南瓜塔，印象不錯呢：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　今天教課之餘又來到了星巴克，看到櫥窗裡一塊簡約樸實的白色蛋糕，<br/>
<br/>
　卻被打上"草莓戚風蛋糕"的名稱，我仔細瞧瞧蛋糕本體，<br/>
<br/>
　一直努力地去尋找任何一點點的草莓塊，碎片甚至足跡也行，<br/>
<br/>
　完全徒勞無功，這也激起了我異樣的好奇心：<br/>
<br/>
　"星巴克到底是基於什麼樣的理由要推出外觀上這樣低調的糕點呢？"<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (樸實又內斂的草莓戚風蛋糕！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(口感清爽的鮮奶油層)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  馬上點了一份，要價55，用GRD2拍完照切了一小塊試吃，<br/>
<br/>
  外層的鮮奶油口感與一般的不大一樣，很清爽，　　<br/>
<br/>
　內部的海綿蛋糕本體第一眼看起來乾乾的，一點也沒有蛋糕該有的濕潤感，<br/>
<br/>
　這似乎是個敗筆，不過卻在入口的那一剎那給了我很棒的驚喜：<br/>
<br/>
　與不討喜的外觀有著極大反差的內層海綿蛋糕口感特別細緻與軟嫩<br/>
<br/>
　口感驚人中散著草莓香氣，真的很好吃！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(愜意的品嘗蛋糕)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　後來好奇心重的我，問了夥伴，得知草莓戚風蛋糕的鮮奶油是用低脂牛奶打的，<br/>
<br/>
　整份蛋糕才一百多卡，相當驚人的低卡與清爽取向，<br/>
<br/>
　整體來說加了不少分，真是非常漂亮的糕點戰爭呀！<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
　　(草莓戚風蛋糕，真是非常內斂又有氣質的小蛋糕呢！)<br/>
<br/>
