<br/>
<br/>
<br/>
<br/>
<br/>
  睽違已久的ABQ終於又將踏入台灣本島了，當初的記憶卻幾已毫無存底，<br/>
<br/>
　記得某日在大廳裡看著他背著咖啡色的方盒快步的衝了出來．<br/>
<br/>
　許多年輕學子們見狀，紛紛趕忙搶了過去，畢竟教授的時間寶貴，<br/>
<br/>
　深怕一不留神便再也無緣一見了．<br/>
<br/>
　當初那與優雅的華爾滋節拍強烈成為對比的急步狀聲詞，至今依舊鮮明，<br/>
<br/>
　或許，這也解釋了為何維也納華爾滋較傳統的急促許多，<br/>
<br/>
　這也曾令當時初學的我如此暈頭轉向，為此感到極度困擾，<br/>
<br/>
　不禁想強烈質疑約翰史特勞斯的意圖．<br/>
<br/>
<br/>
<br/>
　當演奏家開始敬業的運起弓來，此時巨大的音樂素材開始充斥著木箱的四周，<br/>
<br/>
　聚在一塊到處流竄，鞭打左手的指節，令其無法升起，鉗住右手手腕，<br/>
<br/>
　令其無法擺動，猛烈的程度有如炮烙．<br/>
<br/>
　焦黑的手指卻背負著不可失敗的責任，從零時零分開始，<br/>
<br/>
　持續到有如轟雷般的鼓譟聲為止，繼續與惡魔搏鬥．<br/>
<br/>
　背，開始灼熱，<br/>
<br/>
　眼，分不清淚水與汗水，<br/>
<br/>
　足，和著六十四分音符一起舞動著，<br/>
<br/>
　木嘶吼，毛和鳴，耳卻不聞．<br/>
<br/>
　當最後三和絃揚起勝利旗幟之時，苦難有了終結，<br/>
<br/>
　他卸下了眼罩與世界重新接壤，這世界因而對他有了重新的認識．<br/>
<br/>
