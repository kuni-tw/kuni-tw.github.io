<br/>
  四月甫拿到期望已久的Pierre Marcolini,<br/>
<br/>
  今天試吃這一盒"Les Saveur du Monde",<br/>
<br/>
  也就是世界各大產區的巧克力薄片(75% Cacao)<br/>
<br/>
　包括:Equateur 75% Cacao, Madagascar 75% Cacao,<br/>
<br/>
  Java 75% Cacao, Ghana 75% Cacao, Venezuela Cacao 75%, Trinite Cacao 75%<br/>
<br/>
  還有Marcolini特調的混合款.<br/>
<br/>
  為此特地傍晚沒吃晚飯,只喝了幾杯拿鐵,<br/>
<br/>
  晚上教完課之後拿了一杯冷開水空腹開始品嘗.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  因為看過一些名人的Marcolini的心得,<br/>
<br/>
  所以決定將Marcolini特調擺在最後.<br/>
<br/>
<br/>
<br/>
  賞味心得: <br/>
<br/>
  1. Equateur 75% Cacao<br/>
<br/>
  現在回想起來,這一款真不應該最先品嘗,<br/>
<br/>
  這味道真是極為強烈呀,當巧克力在舌上開始溶化時,<br/>
<br/>
  那香味非常特別,但卻似曾相識,<br/>
<br/>
　想了好久才想起這味道曾在以前吃的紅燒鰻魚罐頭裡出現過,<br/>
<br/>
　嚴格說起來是些許焦炭味,微酸,有點類似咖啡裡的酸味,但不是很明顯.<br/>
<br/>
  跟後面的幾款相較之下,味道算是最有個性的.<br/>
<br/>
<br/>
  2. Madagascar 75% Cacao<br/>
<br/>
  入口之後苦味較重些,有牛奶味,<br/>
<br/>
　整體來說味道清新.<br/>
<br/>
<br/>
  3. Java 75% Cacao<br/>
<br/>
  這一款很不錯,尾韻悠長,<br/>
<br/>
　酸味一開始較為明顯,之後漸弱轉為些許苦味,<br/>
<br/>
　入口過了一段時間我仍神遊在這虛幻的味覺世界裡.<br/>
<br/>
<br/>
  4. Ghana 75% Cacao<br/>
<br/>
  甜味較為明顯,不過層次感較沒那麼重,<br/>
<br/>
　吃完之後閉上眼睛好一陣子,微苦的尾韻才慢慢浮現.<br/>
<br/>
<br/>
  5. Venezuela 75% Cacao<br/>
<br/>
  入口之後果香味四溢,酸味與甜味很均衡,<br/>
<br/>
　很討喜又很讓人享受的味道.<br/>
<br/>
<br/>
  6. Trinite 75% Cacao<br/>
<br/>
  入口時感到詫異,味道是這幾款中最淡的,<br/>
<br/>
  細細品嘗之後有些許核果味,層次感較不易感受出來,<br/>
<br/>
　令人開心的是尾韻微甘呢.<br/>
<br/>
<br/>
<br/>
<br/>
  最後重頭戲:<br/>
<br/>
  Pierre Marcolini 特調 75% Cacao<br/>
<br/>
  有趣的組合---&gt;幾乎上述幾款出現過的味道都出現了.<br/>
<br/>
  不過很奇妙的是,期待的尾韻並沒有出來,有點虎頭蛇尾的感覺(笑)<br/>
<br/>
  但是真的得說,在吃完上述所有口味之後再試吃Marcolini的特調,<br/>
<br/>
  突然間,這些仍然存留在記憶中的味道又重新浮現真是很棒很棒的感覺,<br/>
<br/>
　很過癮.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　結論:<br/>
<br/>
　這真是一場十分鐘的味覺饗宴呀!<br/>
