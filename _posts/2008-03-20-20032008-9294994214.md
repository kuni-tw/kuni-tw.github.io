<br/>
  Irgendwie doch nix gebracht,<br/>
<br/>
  zugegeben auch unbeschreiblich enttaeuscht.<br/>
<br/>
<br/>
  回程時更精細的瞄著左右後照鏡,來回轉頭,<br/>
<br/>
  趁著西風,此時發狂的我打開所有的探照功能以及最先進的自動導航系統,<br/>
<br/>
  然後將模式調到最高的紅線區域,藉此打擊無辜者的信心.<br/>
<br/>
<br/>
  然後只有脆弱.<br/>
<br/>
<br/>
  晚間看了爸媽的笑容,我突然回想起以前他們充滿著愛心的照顧,<br/>
<br/>
  不禁感到了一絲絲溫暖,卻仍舊還有著些許的無力感,<br/>
<br/>
　後來回到了臥房,我沉默了好一陣子,腦子裡一片空白...<br/>
<br/>
<br/>
  今晚需要朋友與美酒.<br/>
