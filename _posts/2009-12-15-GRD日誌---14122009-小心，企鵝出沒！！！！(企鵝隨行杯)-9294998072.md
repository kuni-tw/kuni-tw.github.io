<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
  (小心，企鵝出沒！！！！！！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　在小兔兔小企鵝熱潮之下，不只我，連身邊的朋友也被感染了，<br/>
<br/>
　好不容易拿到一只板友願意割愛的兔兔杯，詳情請參閱：<a href="http://www.wretch.cc/blog/kuni/11587374" target="_blank">星巴克正夯的小白兔隨行杯寫真隨拍</a><br/>
<br/>
　結果某看到兔兔的友人，第一句是：<br/>
<br/>
　"阿，好可愛，KUNI，我們是好朋友對不對！？"<br/>
<br/>
　其他朋友也不惶多讓，紛紛詢問這個杯子在哪裡可以買得到，<br/>
<br/>
　看來到聖誕節前，這兔兔企鵝之火還會燒一陣子。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　  (企鵝鵝登場了)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　最近看到兔兔孤單落寞的樣子，<br/>
<br/>
　不禁想為牠找一隻企鵝當個伴，不過除非跟網拍低頭，<br/>
<br/>
　不然還真是難找。<br/>
<br/>
　昨天有幸得知某網友願意釋出多餘的一個後(他有兩組)，<br/>
<br/>
　價錢只比原價多一點，晚上時便大老遠騎車到他那，<br/>
<br/>
　一路上歷盡冷風摧殘，剛拿到手一時之間超興奮的，<br/>
<br/>
　馬上趕到水源門市為牠拍個寫真。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(企鵝鵝正面微距照)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　這兩只杯子對我來說，算是星巴克又一次的新突破。<br/>
<br/>
　常常觀察他們的杯款，即可發現，<br/>
<br/>
　最近一兩年以來，新花樣與設計突然增多了。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(杯上的雪花圖案散出著聖誕節的氣氛)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
　　(這叫做兔兔與企鵝鵝相見歡)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　幾年前先出了透明浮雕款，那時記得超經典的，這種杯款後來年年熱賣，<br/>
<br/>
　比如去年甫發行就銷售一空的星星麋鹿隨行杯。<br/>
<br/>
　前幾年也有了第一款透明的城市隨行杯(東名高速杯)，<br/>
<br/>
　後來陸陸續續星冰樂杯，龍鳳隨行杯款登場了，<br/>
<br/>
　常態性不鏽鋼也有了TALL SIZE，<br/>
<br/>
　連日本的黑色VENTI也到達台灣成為常態性商品，<br/>
<br/>
　沒想到今年聖誕節竟然還設計得出這麼可愛的兔兔跟企鵝，<br/>
<br/>
　星巴克真是太厲害了...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(孤單的企鵝鵝)<br/>
