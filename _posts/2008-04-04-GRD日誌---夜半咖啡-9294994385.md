　常常夜半之時與好友一同露宿Cafe,<br/>
<br/>
<br/>
<br/>
  回程趁著正值涼爽的微風當道,漫步在富有層次感的四號公園裡頭.<br/>
<br/>
  走著走著,隨著時間漸漸流逝,凝佇在步道中央,伴著昏黃夜空...<br/>
<br/>
<br/>
<br/>
<br/>
  我逐漸地走進那只神奇的時光抽屜裡頭,拉起冰冷的搖桿,<br/>
<br/>
  將指針轉到91年,那暑假,那惶恐與驚慌失措的某日.<br/>
<br/>
  那一天我失去了所有,因為變賣了充滿著自信的家當.<br/>
<br/>
  進入Lauda座艙就座之後,冷氣變得無情起來,透過氣窗往外頭瞧,<br/>
<br/>
  不論是移動式的扶梯或是載貨車,路邊的草地甚至是耀眼的太陽,<br/>
<br/>
  看得見的全都是黑白.<br/>
<br/>
  記得當時我強行抑制著緊張,正襟危坐,<br/>
<br/>
  但不時開啟身上的安全扣環,沒事就跟空姐討罐可果美這些舉動,<br/>
<br/>
  反而使我看起來更顯得不安.<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這些記憶,這些看似殘缺不堪的,實際上卻從沒錯漏過些什麼的,<br/>
<br/>
  如今終於成為我孩子氣未減的根源...<br/>
<br/>
<br/>
<br/>
