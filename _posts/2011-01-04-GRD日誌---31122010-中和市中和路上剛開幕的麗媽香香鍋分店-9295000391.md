麗媽香香鍋<br/>  新北市中和區中和路209-1號<br/>  電話：(暫無)<br/>--------------------------------------------------------<br/>										   <br/><br/><br/><br/><br/><br/>									 <br/>																			    <br/>　說真的，回台灣後這麼多年來從來沒有吃過什麼臭臭鍋的，<br/>																			    <br/>　小弟一直不知道這到底是什麼東西。<br/>																			    <br/>　上一次去車行換輪胎時因等候時間過長，<br/>																			    <br/>　跟師傅講了一聲便去了附近的某臭臭鍋店吃頓午飯，<br/>																			    <br/>　一吃驚為天人，沒想到一百塊的小火鍋內放了這麼多料。<br/>																			    <br/>　熱騰騰地直冒著蒸氣，配著一小碗白飯吃得我汗流浹背大呼過癮，<br/>																			    <br/>　在最近寒流不斷來襲時吃這種東西實在是太享受啦。<br/><br/><br/><br/><br/><br/><br/><br/>　一連好幾天便往不同的臭臭鍋店報到，<br/>																			    <br/>　其中一家就是最近剛在中和市中和路上開幕的麗媽香香鍋了。<br/>																			    <br/>　走進店面，畢竟剛開幕沒幾天，裝潢還相當新，<br/>																			    <br/>　與友點了一份大腸臭臭鍋與另外要求麻辣的鴨血鍋，<br/>																			    <br/>　每一鍋各要價NT.100。<br/><br/><br/><br/><br/><br/><br/><br/><br/>　(上桌前先上酒精膏的爐座)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/></span><br/><br/><br/>　經過連續幾天下來的臭臭鍋料理洗禮，<br/>																			    <br/>　也似乎愈來愈能分辨出味道上的細微差異了。<br/>																			    <br/>　麗媽香香鍋(不知怎麼地老記成麗香媽媽鍋)的白飯是免費續的，<br/>																			    <br/>　飲料部分也免費自取，鍋料份量部分則比三媽系統多。<br/>																			    <br/>　不過這部分還沒考慮到不同分店的差異性，<br/>																			    <br/>　比如三媽部分我拿來比較的是大直區域附近的分店。<br/>																			    <br/>　與這些老闆們聊過後，會發現同系統下不同分店的做法還是會有些小小差異。<br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(免費自取的飲料區)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　這一次麗媽香香鍋的大腸臭臭鍋出現了小失誤，<br/>																			    <br/>　竟然忘了放大腸，而裡頭臭豆腐部分也沒有三媽來得入味，<br/>																			    <br/>　經反映後老闆馬上致歉，另外送來一盤大腸與滷臭豆腐，<br/>																			    <br/>　可惜大腸部分似乎沒處理好，咬不動，<br/>																			    <br/>　沒有三媽的口感來得好。<br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(大腸臭臭鍋)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(店家後來補上的大腸與滷臭豆腐)<br/><br/><br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　但鴨血鍋部分表現就很不錯，裡頭的主料份量比三媽多，<br/>																			    <br/>　加上白飯可以免費續，價格又比三媽便宜了足足二十元，<br/>																			    <br/>　C/P值頗高，缺點是麻辣部分沒有三媽來得夠味有層次，<br/>																			    <br/>　比較像是單獨加了幾匙辣醬下去的感覺。<br/><br/><br/><br/><br/><br/><br/><br/>　(鴨血鍋)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　(寒流之下吃這個實在是太過癮啦)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>BY
