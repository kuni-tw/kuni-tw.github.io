　與友小風夜半出門漫步至中和路與永貞路交叉口的客喜康,<br/>
<br/>
　年輕有為的店長年紀比我還小呢,熱愛咖啡,也熱愛攝影.<br/>
<br/>
  這一次跟他臨時借了他的Terra Keramik手工咖啡杯拍幾張寫真,<br/>
<br/>
  Terra的咖啡杯都是純手工,且使用德國的黏土製造,<br/>
<br/>
  所上的透明釉料,更是標榜著無鉛無鎘的安全釉,經過三段燒製工法.<br/>
<br/>
  特別之處在於像是咖啡杯握把處,鍍上的是一層白金!<br/>
<br/>
  每個杯子上還有他本人用白金親手簽上"TERRA"(資料取自店長朋友的網誌"http://tw.myblog.yahoo.com/dunway999/article?mid=8388")<br/>
<br/>
  真是美極了!<br/>
