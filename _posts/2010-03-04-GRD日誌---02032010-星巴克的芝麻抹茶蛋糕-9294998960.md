    (星巴克的芝麻抹茶蛋糕與黑芝麻抹茶拿鐵)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
---------------------------------------------------<br/>
星巴克忠新門市<br/>
地址：台北市中正區忠孝東路二段134巷2號<br/>
電話：(02)2395-7870<br/>
<br/>
星巴克龍門門市<br/>
地址：台北市大安區忠孝東路四段134號<br/>
電話：(02)2740-6782<br/>
---------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這一次趁星巴克推出新產品之時，<br/>
<br/>
　與幾位熟網友們約在星巴克忠新門市相見，<br/>
<br/>
　除了敘敘舊，聊聊最近的新收藏之外，<br/>
<br/>
　順便也點了剛推出的芝麻抹茶蛋糕。<br/>
<br/>
<br/>
<br/>
　外表看起來相當精緻的芝麻抹茶蛋糕，<br/>
<br/>
　由芝麻，抹茶與巧克力層組成，<br/>
<br/>
　小切一口後，芝麻味道相當濃厚，<br/>
<br/>
　順滑濕潤的口感，配合著微甜，<br/>
<br/>
　雖然幾乎感受不到抹茶香，但是還是很好吃。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(芝麻抹茶蛋糕寫真隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　沒幾天後，在龍門門市再度點了一次，<br/>
<br/>
　配合著黑芝麻抹茶拿鐵加一份濃縮咖啡，<br/>
<br/>
  芝麻香濃郁地散在四周，香甜順口，<br/>
<br/>
　兩者實在是竟然超乎想像的契合。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
<br/>
<br/>
　　(在龍門門市搭配黑芝麻抹茶拿鐵，兩者非常契合)<br/>
