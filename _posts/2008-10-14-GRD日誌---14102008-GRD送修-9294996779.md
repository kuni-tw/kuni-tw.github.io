<br/>
<br/>
　　(這是九月四號照的，還滿好玩的^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　連續好幾個月黏在我身邊的GRD2終於得暫離一段時間了，<br/>
<br/>
　雖然說恆隆公司貨有三年保固，還有免費專人到府收件送修服務，<br/>
<br/>
　不過我個人天性甚懶也不是一天兩天的時日了，<br/>
<br/>
　其實鏡頭從剛買沒多久就開始有低機率性的伸縮故障問題，<br/>
<br/>
　有時候正常有時候鬧脾氣，頑皮得很，<br/>
<br/>
　而我一直懶得撥個電話過去，這樣也好一段時間了，<br/>
<br/>
　直到最近GRD2脾氣愈來愈大，鏡頭時縮時不縮，<br/>
<br/>
　為了能順利拍照食記，每一次都得在菜端上來之前一直嘗試開機，<br/>
<br/>
　使得每一次的GRD日誌都相當難產．<br/>
<br/>
　這一次狠下心來，冒著大雨騎著RV250直接殺到林口去，<br/>
　<br/>
　與親切的工程師哭訴愛機的問題之後，<br/>
<br/>
　忍痛的，依依不捨的將GRD2交給了他，過程中本人一直流著淚，<br/>
<br/>
　混著臉上滿滿的雨水，也有泥水，眼神中充滿著滿溢的愛與關懷，<br/>
<br/>
　左手拭淚，輕揮著右手，風切聲中和著沙啞的嗓音：<br/>
<br/>
　"親愛的GRD2，我們很快就會就再見面的！"<br/>
<br/>
　在場者紛紛為之動容，真是感人的一幕呀...<br/>
<br/>
  okay，掰完了，請原諒小弟有時候會有脫序行為的傾向呀！<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
  既然沒食記，講講最近都在幹麻好了XD<br/>
<br/>
<br/>
<br/>
<br/>
  最近每天半夜都會夜騎烘爐地，一處很陡峭的小山，<br/>
<br/>
　山上有一處很靈驗的土地公廟，加上個人排名前三的北市夜景，<br/>
<br/>
　每逢三更半夜總是可以看到一堆信眾上山到此求香．<br/>
<br/>
　而我呢，每一次騎到山頂，總是會紀錄著時間與換檔成績，<br/>
<br/>
　回家後再重訓，這樣不間斷地也持續了一兩個禮拜．<br/>
<br/>
<br/>
<br/>
　前幾天，換了肉球手把與輕量踏板之後，成績又進步到了十分十四秒，<br/>
<br/>
　下一次想換馬吉斯的防刺胎，猜想破十分應該是輕而易舉的吧．<br/>
<br/>
　今天則是測試高迴轉速上坡，因為之前習慣性的踩重檔，<br/>
<br/>
　這也是以前比賽時的習慣，上坡轉速維持在60左右，<br/>
<br/>
　而今晚全程至少以 70rpm up踩上去，沒想到只花了十分十六秒，<br/>
<br/>
　肌肉竟然感覺一點都不酸，下山後到平地測試次大檔衝刺試到85rpm，<br/>
<br/>
　對於慢慢步入中年的我而言，這大概是本日最令人快樂的消息吧！　<br/>
