<br/>
<br/>
<br/>
  (好可愛的和果子呀！！！！)　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　終於，已在網路上銷售日式和果子一段時間的唐禾本舖，<br/>
<br/>
　其實體店面在十月七號也正式開幕營業了，<br/>
<br/>
　地址是北市中正區羅斯福路二段48巷3號．<br/>
<br/>
　為了嚐鮮，小弟也肉痛的買了"水鳥"，"柿子"與"雪玉兔"，<br/>
<br/>
　加上開幕期間去消費，即享有九折優惠以外另附贈一隻"大衛鹿"，<br/>
<br/>
　回程時不小心壓壞了雪玉兔，真是可惜，<br/>
<br/>
　拍完美美的寫真照之後，<br/>
<br/>
　泡了一小杯熱茶，便開始品嘗這些美麗的"工藝品"了．<br/>
<br/>
　為什麼平日對茶唯恐避之不及的小弟要泡茶呢？<br/>
<br/>
　這是因為以前在日本，和果子是在茶會中用之搭配的茶點，可中和抹茶的苦味，<br/>
<br/>
　所以享用和果子的時候，配上一壺好茶更能帶出其中的韻味．<br/>
<br/>
　不過小弟家中沒啥好茶，所以只好將就一下便宜茶包了．<br/>
<br/>
<br/>
　心得<br/>
<br/>
　外表無比可愛的大衛鹿，黑糖風味外皮內裹著奶黃餡，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(有著黑糖外皮的大衛鹿寫真，首先是橫向圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(然後是側邊圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(微距表皮寫真)<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　柿子，要價150，新開發的口味，類饅頭口感的外皮配上白豆沙內餡，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(紅澄澄的柿子寫真)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(真細緻呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (微距表皮寫真)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　水鳥，要價200，糯米製的外皮內裹著抹茶餡，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (啾，可愛的水鳥)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(看起來就很歐依稀的樣子！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(微距寫真看起來更為誘人呢^^)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  不過外觀如此討喜與精緻的唐禾和果子，<br/>
<br/>
　可惜了小弟實在是吃不大出來其中的美味之處，<br/>
<br/>
　餡料的香味沒特別明顯，口感細緻度是還不差，<br/>
<br/>
　甜度還可以，可惜稍微單調了點，三個吃下來味道似乎沒啥差異性與特殊性．<br/>
<br/>
　配上了茶，比較像是茶中和了和果子的甜味！<br/>
　<br/>
　據老闆說，與日本傳統的和果子相較之下他已特地降低了甜度，<br/>
<br/>
　這一點到是真的，可是小弟還是比較喜歡日本那裡的，<br/>
<br/>
　日本當地的和果子的確是更甜，卻較有個性．<br/>
<br/>
　就算是甜，也充滿著不同風味，比如甜中帶苦或是帶酸，<br/>
<br/>
　不同的調和之下，有著不同的風景映入眼簾...<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　<br/>
<br/>
<br/>
  (剛開幕的唐禾本舖有許多優惠唷！)<br/>
