<br/>
    終於等到了這一天，台灣星巴克的桐花季再度來到，<br/>
    <br/>
　15號特地請朋友買桐花紀念磁性夾，幾個朋友一起合買，買了約十個，<br/>
    <br/>
　然後找機會去門市挑了張阿奈卡，今天趁機會找了家門市拍幾張寫真。<br/>
　　<br/>
    <br/>
    <br/>
　這一次的桐花磁性夾，色調相當粉嫩，<br/>
    <br/>
　裡頭怡然自得的桐花與咖啡桌還有星巴克沙發一道載浮載沉，<br/>
    <br/>
　十個隨機排列起來，看起來非常美妙，<br/>
    <br/>
　不禁多按了幾下快門，畢竟十姊妹難得齊聚一堂，<br/>
    <br/>
　終有一日得要各自遠走他鄉呀。<br/>
