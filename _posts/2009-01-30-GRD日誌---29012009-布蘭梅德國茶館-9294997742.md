<br/><br/>
<br/>
　　　(布蘭梅德國茶館的茶凍作得好精緻呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  下午在大安區附近教課之餘，拜訪了位在附近的布蘭梅德國茶館，<br/>
<br/>
　詳細地址為：台北市大安區光復南路278號<br/>
<br/>
　電話：(02) 27736155<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
　<br/>
<br/>
   (店家大門與開放式空間外觀)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　店家大門外觀相當漂亮，裝潢雅致，<br/>
<br/>
　給人一種優雅享受下午茶的氣氛，非常棒．<br/>
<br/>
　室內裝潢充滿著宮廷式的味道，高檔與質感不錯的擺設，<br/>
<br/>
　不過門口的開放式座位也一樣迷人呢．<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　看了菜單，裡頭茶品種類繁多，五花八門，<br/>
<br/>
　價位都相當高．<br/>
<br/>
　點了一杯調味紅茶，名字叫奶油，要價280，<br/>
<br/>
　加點全套茶點，要加200，<br/>
<br/>
　一人份裡包括手工餅乾或糕點，特製茶凍，新鮮水果盤，<br/>
<br/>
　份量相當大，老實說，<br/>
<br/>
  號稱大胃王的小弟也建議一人點兩人分享比較好，<br/>
<br/>
　因為每樣茶點的SIZE都大得令人驚訝呢！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　服務生馬上端上一杯漂著萊姆片的白開水，<br/>
<br/>
　使用滿漂亮的威士忌酒杯，很好看．<br/>
<br/>
　沒多久特製茶凍就上桌了，<br/>
<br/>
　看到櫻桃等水果與玻璃盛碗的比例就可知有多麼大份了，<br/>
<br/>
　茶凍部份口感Ｑ又彈牙，味道清爽！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
(使用威士忌酒杯裝的白開水裡放上一片萊姆頗具巧思)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　　(一旁調味用的糖罐)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　　(庭園開放式空間裡裝潢得很漂亮呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
(特製茶凍上場嚕，看起來作得很精緻呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　　(真的是滿滿一大份！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(來一張茶凍俯瞰圖吧！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(馬上舀一湯匙來品嘗看看...恩,真是可口呀！)<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　手工糕點部分我選的則是栗子蛋糕，<br/>
<br/>
　份量約一個拳頭大，表面充滿著濕潤感，<br/>
<br/>
　裡頭除了巧克力海綿蛋糕之外，還有栗子餡，<br/>
<br/>
　甜而不膩，非常好吃！　<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(栗子蛋糕SIZE超大！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(栗子泥外觀充滿著濕潤感)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(栗子蛋糕俯瞰圖)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(馬上也來切一小塊品嘗看看！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　水果拼盤裡頭水果的擺置頗具巧思，<br/>
<br/>
　也是一大份，裡頭有聖女番茄，<br/>
<br/>
　橘子，芭樂，蘋果，香蕉，西瓜與鳳梨，<br/>
<br/>
　從水果新鮮度來看，布蘭梅德國茶館真的滿用心的．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(水果拼盤登場！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(裡頭有聖女番茄，橘子，芭樂，蘋果，香蕉，西瓜與鳳梨)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(也是相當一大份呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　我點的調味紅茶名約奶油，菜單上的茶點都有附上英文與德文，<br/>
<br/>
　英文是Cream，德文則是Sahne．<br/>
<br/>
　這裡補充一下，德語區有些地方則稱之為Schlagobers．<br/>
<br/>
　奶油果然名副其實，滿滿一大杯中傳來濃郁的奶油香，<br/>
<br/>
　杯組似乎與東區的咖啡弄使用的相同，<br/>
<br/>
　不過卻整整大上一到兩個SIZE，不過好不好喝就見仁見智了，<br/>
<br/>
　個人則覺得香味質感不錯，整體配上茶點感覺還滿不賴的．<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(調味紅茶名約奶油，附上一大片手工餅乾)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(奶油香氣非常濃郁！))<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　到最後其實根本沒吃完，因為份量真的真的很大，<br/>
<br/>
　根本就是變相的下午茶吃到飽．　<br/>
<br/>
　下午若有充足的閒時間，<br/>
<br/>
　來這裡舒服的享受茶點美食，看看書報，<br/>
<br/>
　感覺真的很棒呢！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/><br/>
<br/>
　　(庭園裡一景)<br/>
