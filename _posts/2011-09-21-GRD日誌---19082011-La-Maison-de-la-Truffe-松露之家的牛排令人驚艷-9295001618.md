  松露之家 Maison de la Truffe<br/>
  地址：台北巿敦化南路一段246號B1 (太平洋sogo敦化館)<br/>
  電話：(02)2778-7955<br/>
  時間：11:00~22:00<br/>
-------------------------------------------------------------------</span></p>
<span>
<p>　　　<br/>
<br/>
<br/>
　這家爭議性頗大標榜著以高級食材松露為主力的法國高檔餐廳，</p>
<p>　是台北知名麵包店Paul老闆葉兩傳在今年初花重金引進台灣的。</p>
<p>　小弟觀望許久終於在這一天晚上心血來潮走去吃頓晚飯。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(松露之家門口)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212475.jpg" target="_blank"></a></span><br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　SOGO敦南館地下一樓這位置已換過不少餐廳，包括來自奧地利的Kaffee Sound，</p>
<p>　業績據說都不大好，葉老闆似乎是一個滿有勇氣的人。</p>
<p>　走進餐廳內才發現裡頭裝潢質感比預期優，真令人欣喜，</p>
<p>　服務生在專業度上有些落差但在良好而不做作的服務態度上贏過不少高檔餐廳，</p>
<p>　點了晚間套餐，NT.2880，附冷前菜松露鮪魚片酪梨沙拉，</p>
<p>　熱前菜森林風開心果鵪鶉無花果，湯品本來應該是胭脂蝦白花椰濃湯，</p>
<p>　但不是很喜歡經反映後服務生幫我改為蘆筍卡布奇諾綴拜耶里魚子醬，</p>
<p>　主餐則選巴黎風烤牛排佐黑鑽松露，甜點是蛋糕與咖啡，</p>
<p>　另加點一份經典松露炒蛋(NT.780)。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先上桌的是餐前麵包配雉雞葡萄醬與餐廳贈送的松露香檳，</p>
<p>　麵包尚佳口感較紮實醬料可口卻嚐不出任何葡萄味。</p>
<p>　香檳聞得到松露香口感微澀但在酸度上有著良好的平衡。</p>
<p><br/>
<br/>
<br/>
　(前麵包配雉雞葡萄醬)<br/>
<br/>
<br/>
<span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212476.jpg" target="_blank"></a><br/>

　(松露香檳)<br/>
<br/>
<br/>
</span><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363212477.jpg" target="_blank"></a><br/>

<br/>
</p>
<p>　接下來是松露煎蛋與松露鮪魚片酪梨沙拉，前者蛋皮煎得很嫩聞得到松露香，</p>
<p>　會想點這道是因想重溫以前在歐洲學生時期的回憶，</p>
<p>　松露香有沒有散出來這點真的很重要，之前在台北吃過太多家沒香味的松露菜色了，</p>
<p>　連台北的L'atelier de Joel Robuchon都有發生過。</p>
<p>　後者兩片鮪魚外層微煎過上頭各擺些魚子醬與酸豆一旁躺著兩片松露，</p>
<p>　配合著酪梨的綿密口感可惜少了點個性。<br/>
<br/>
<br/>
<br/>

　　　<br/>
　然後是森林風開心果鵪鶉無花果與蘆筍卡布奇諾綴拜耶里魚子醬，</p>
<p>　前者吃不太出師傅的本意，肉質嫩但上頭的開心果碎片感覺沒特別搭，</p>
<p>　無花果的微酸似乎與前一道鮪魚有著同樣的開胃意圖。</p>
<p>　後者蘆筍卡布奇諾綴拜耶里魚子醬則令人感到開心，</p>
<p>　濃濃打至發泡的卡布奇諾裡有著蘆筍特有的清甜香，</p>
<p>　自此胃漸漸開始暖和起來。<br/>

　終於重頭戲巴黎風烤牛排佐黑鑽松露登場，服務生先將黑松露塊與牛排端上桌，</p>
<p>　牛排是美國Prime等級的濕式熟成菲力，上頭那一大片黝黑裝飾物大有文章，</p>
<p>　使用墨魚汁和著麵粉模仿松露那網狀內裡烘烤而成，在視覺上加分不少。</p>
<p>　然後大方而不吝嗇地將新鮮的黑松露一片片刨下來，</p>
<p>　不過這卻不是濃濃松露香主要的來源，</p>
<p>　吃到後來可以確定那黑松露馬鈴薯泥才是提升香味的最關鍵之處。</p>
<p>　而最令人驚艷的莫過於那好吃到令人無法抗拒的美味菲力，<br/>
　<br/>
　肉質軟嫩充滿肉汁同時肉味十足絕對是這幾年來吃過的前幾強。</p>
