丸居食酒屋<br/>
地址：台北縣中和市景安路10號<br/>
電話：(02)2246-8328<br/>
-----------------------------------------------------------------<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這是一家才剛開幕一兩個多月的居酒屋，<br/>
<br/>
　與最近慢慢開展分店的嚐嚐九九屬同一連鎖系統，<br/>
<br/>
　嚐嚐九九打著每日特價主菜一道只要十元，<br/>
<br/>
　丸居食酒屋也一樣，打著北海道松葉蟹內用一盤只要39，<br/>
<br/>
　當然只限一道，想加點的話原價可是240呢。<br/>

<br/>
<br/>
<br/>
　上一次去吃一回生二回熟時，順道剛好有看到這一家，<br/>
<br/>
　門外的廣告內容分外引人注目，沒隔幾天中午便過去登門拜訪。<br/>
<br/>
　裡面的裝潢質感不是很好，有點陳舊，<br/>
<br/>
　選個中間的涼亭，脫了鞋，進入就座。<br/>
<br/>
<br/>
　點了一份中午套餐：生魚片定食(要價NT.180)<br/>
<br/>
  附白飯免費續碗，漬物，小菜，味增湯與茶碗蒸各一份<br/>
<br/>
　飲料任選(包括可爾必思，蔓越莓汁與柳橙汁)<br/>
<br/>
　另外當然要加點松葉蟹了，這可不客氣呀。<br/>
<br/>
<br/>
<br/>
　沒多久定食與我的可爾必思上桌，<br/>
<br/>
　飲料相當大杯，不錯。<br/>
<br/>
　小菜部分只是一小份漬黃瓜與白蘿蔔，與漬物及味增湯表現一樣，<br/>
<br/>
　味道平淡無奇，作得不夠用心。<br/>
<br/>
　茶碗蒸裡有一整隻大蝦仁，算比較有料，不過口感偏硬。<br/>
<br/>
　生魚片切功平整，不過過厚，而且尚未完全退冰，<br/>
<br/>
　肉裡筋多，容易塞牙縫，當下有給與我聊天的副店長一些建議。<br/>
<br/>
　若考量到成本問題，生魚片其實可切薄一點，<br/>
<br/>
　視覺與口感都會升級許多。<br/>
