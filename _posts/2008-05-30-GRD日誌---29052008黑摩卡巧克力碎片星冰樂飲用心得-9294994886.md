<br/>
<br/>
<br/>
<br/>
  繼之前有如牛奶糖一般的奇樂朵拿鐵成功推出之後,<br/>
<br/>
  最近星巴克這一次趁勝追擊,除了討喜的夏季商品之外,<br/>
<br/>
  另外就是可以以隨行卡點數兌換的黑摩卡巧克力碎片星冰樂了.<br/>
<br/>
  昨晚教課中的休息時間至星巴克的敦信門市,順便點了幾杯,<br/>
<br/>
  除了自己想試之外順便請學生喝.<br/>
<br/>
<br/>
<br/>
<br/>
  從商品名稱裡黑摩卡這三個字中,我似乎嗅出一點這一次星巴克想玩的花樣.<br/>
<br/>
  上一次的奇樂朵對許多人來說算得上是偏甜,不過卻也對上了不少螞蟻們的胃.<br/>
<br/>
  這一次的黑摩卡碎片明顯的想利用黑巧克力香來襯托出不一樣氣質,<br/>
<br/>
　擺脫最近新推出飲品漸漸偏甜的趨勢.<br/>
<br/>
　從一開始意外的清新黑巧克力香馬上令我為之一亮,<br/>
<br/>
  使用附上的長湯匙攪拌攪拌之後,吸入第一口,<br/>
<br/>
  馬上在舌尖上嘗到巧克力碎片顆粒般的口感.<br/>
<br/>
  整體來說非常清爽有氣質,飲用完之後,令人感到非常愉悅,<br/>
<br/>
  當然唯一的缺點:偏貴.<br/>
<br/>
  想享受GRANDE大小就得掏出145台幣了(約16oz),特大杯則160(20oz).<br/>
<br/>
　這也是最近星巴克新飲品的趨勢:價格愈來愈高!<br/>
