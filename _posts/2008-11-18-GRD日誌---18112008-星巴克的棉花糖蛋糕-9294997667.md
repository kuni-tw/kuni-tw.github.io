<br/>
<br/>
<br/>
    (棉花糖配上巧克力幕斯，蛋糕像似在呼喊著：快來吃我呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  這一次星巴克趁著本人最愛的太妃季到來，<br/>
<br/>
　順勢又推出一些不錯的糕點類與商品．<br/>
<br/>
　其中最容易引人注目的莫過於棉花糖巧克力蛋糕了．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　怎麼說呢，其實很簡單，<br/>
<br/>
　綿綿"蜜蜜"的棉花糖口感配上濃稠香純的巧克力，<br/>
<br/>
　阿，這樣的蛋糕似乎一直在對我說話：KUNI，你怎麼不趕快來吃我？<br/>
　<br/>
　因此，小弟便也不好意思推辭．<br/>
<br/>
　首先點了杯當季甫出的黑櫻桃摩卡，熱呼呼的，<br/>
<br/>
　上面一層厚厚的鮮奶油灑上巧克力粉，<br/>
<br/>
　散出淡淡的櫻桃香，偏甜，而且不是我喜歡的那種，<br/>
<br/>
　看來十二月時我還是繼續對我的太妃死忠好了．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (星巴克08年冬季新產品：黑櫻桃摩卡)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(厚厚的鮮奶油灑上巧克力粉，好香呀...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　然後就是棉花糖巧克力蛋糕了，<br/>
<br/>
　滿滿的棉花糖佈滿在巧克力幕斯與布朗尼蛋糕上，<br/>
<br/>
　真是驚人的視覺效果，在昏黃的燈光之下，<br/>
<br/>
　彷彿可以將人的視線吸住，然後開始分泌唾液．<br/>
<br/>
　切下一小塊，從切口中看到那微微的濕潤感，真棒呀！<br/>
<br/>
　開始品嘗，恩，<br/>
<br/>
　細緻的棉花口感與香甜的巧克力幕斯，<br/>
<br/>
　相當協和的搭配呢！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (看倌們，有請棉花糖巧克力蛋糕登場！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(看呀，不管什麼角度都很美呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(滿滿的棉花糖佈滿在巧克力幕斯上...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (實在是太美啦！！！！！！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(切下一小口：喔，您看那微微的濕潤感，太棒了！)<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　另外我也點了經典巧克力布朗尼，<br/>
<br/>
　上頭灑下些許的蔓越莓果乾，擠上一些白巧克力醬，<br/>
<br/>
　微苦的巧克力配上核果的香味，<br/>
<br/>
　不過對我來說口感稍稍有點乾，<br/>
<br/>
　味道也較為單調些．<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　　(看官們，有請經典巧克力布朗尼出場！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(好美的剖面圖呀！！！)<br/>
