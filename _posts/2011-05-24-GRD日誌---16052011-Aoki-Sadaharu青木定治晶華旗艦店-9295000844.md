地址：台北市中山北路二段41號B2(晶華酒店)<br/>  電話：(02)2511-5000<br/>--------------------------------------------------------<br/> <br/> <br/>																			    <br/>　前陣子比較常來晶華偶爾會經過Aoki的臨時櫃，<br/>																			    <br/>　有一次聽到他們將在這裡繼Bellavita後再開一家旗艦店，<br/>																			    <br/>　日期是四月二十二號，加上最近因為興趣到處找尋好吃的可頌，<br/>																			    <br/>　便在五月某天表演結束後迫不及待趕忙跑過去。<br/><br/><br/><br/><br/><br/>　(亮白而華麗的門口)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　位在晶華底層B2的Aoki相當大間裡頭空間夠寬敞，<br/>																			    <br/>　淺色系裝潢與明亮的打光與Bellavita旗艦店相同，<br/>																			    <br/>　就座後點了原味可頌，NT.60，巧克力麵包，NT.60<br/>																			    <br/>　Valencia瓦倫西亞，NT.200<br/>																			    <br/>　及Sadaharu AOKI為慶祝晶華店開幕特別設計的"Trio 迷情三重奏"，NT.290<br/>																			    <br/>　(內用另加10%服務費)<br/><br/><br/><br/><br/><br/>　(玻璃櫃內的糕點)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　最近剛吃過他們家的杏仁可頌，口感與味道都很不錯，<br/>																			    <br/>　但相較之下原味款便顯得普通許多，<br/>																			    <br/>　口感更不及這幾天所吃到的Kayser Maison與Paul。<br/>																			    <br/>　巧克力麵包剝開後可以看到底層有兩個深咖啡色的小圓點，是融化後的細巧克力棒。<br/>																			    <br/>　嚴格說起來Aoki的麵包其實還滿好吃的，不過卻無法與後來端上的蛋糕相比，<br/>																			    <br/>　畢竟糕點才是他們家的強項呀。<br/><br/><br/><br/><br/><br/><br/>　<br/>　(可頌)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/>　(巧克力麵包)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/><br/>　(巧克力麵包內層)<br/><br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　首先是華倫西亞，一款橙香巧克力榛果蛋糕，<br/>																			    <br/>　靠著上頭華麗的糖網吸引著所有人的目光是青木定治2007年得獎的作品。<br/>																			    <br/>　巧克力創造出甜與香，榛果粒則賦予微脆的口感配合著柑橘酒類的酸，<br/>																			    <br/>　堪稱視覺與味覺相交織出一首浪漫的舞曲。<br/><br/><br/><br/><br/><br/>　(華麗的舞曲：Valencia)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/>　(美麗的倒影)<br/><br/><br/></span><span></span><br/><br/><br/>																			    <br/>　Trio迷情三重奏則更討人喜歡，味覺上較瓦倫西亞單純但卻造就味蕾上更多的衝擊，<br/>																			    <br/>　利用Domori黑巧力白巧克力與牛奶巧克力等搭配法國的百香果泥，<br/>																			    <br/>　入口後明亮而又濃郁的酸甜被巧克力的微苦一整個帶了出來，<br/>																			    <br/>　突然間咀嚼出榛果片的脆帶給我無比的驚喜，實在是太棒了。<br/><br/><br/><br/><br/>　(Trio迷情三重奏)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/>　(微距一張)<br/><br/><br/></span><span><br/><br/><br/><br/><br/><br/><br/>　(帶給我味覺上無比美好的衝擊)　<br/><br/><br/></span><span><br/><br/><br/><br/><br/>　(友點的鹹派套餐)<br/><br/><br/></span><span><br/><br/><br/></span><br/><br/><br/><br/><br/><br/><br/>　BY
