<br/>
<br/>
<br/>
    (10/12/2009 星巴克公館水源門市終於開幕嚕！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  一聽到喜愛的星巴克，即將在我經常出入的公館永福橋口，<br/>
<br/>
　設立一家新門市，小弟特別興奮，<br/>
<br/>
　由於位在台大水源校區旁，加上宿舍就在鄰近，<br/>
<br/>
　裡頭的校風素養與氣氛可想而知。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (十點半到達時，排隊人群已清晰可見！)　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
(開幕前的大門口)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　個人常去的星巴克門市有幾個特徵，<br/>
<br/>
　其一就是不要太吵鬧，<br/>
<br/>
　二是不想與一堆沒品的顧客鄰坐，<br/>
<br/>
　三是交通方便，最好位在交通中樞地帶，但不要太熱鬧。<br/>
<br/>
  看來水源門市極有可能成為我最愛叨擾的門市之一呀！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(好悠閒呀，看來是懶得排隊了，這才是真正的咖啡人呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (門口的紅磚步道空間感十足，很棒！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一大早，約莫十點半，呃，對我來說的確是滿早的，<br/>
<br/>
　下了橋停好車，已看到不少人開始排隊了。<br/>
<br/>
　心想著糟糕，畢竟還是來晚了，早知道昨晚就少喝一杯。<br/>
<br/>
  沒多久後一位親切的夥伴就開始出來發號碼牌了，<br/>
<br/>
　這我倒是第一次拿到，原來開幕當天有免費的可以喝呀，<br/>
<br/>
　那聽說明天京站門市也要開幕耶...<br/>
<br/>
　唉，貪杯誤事，教課要緊，明天還是乖乖上課吧。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
　　(快十一點了，許多人似乎已開始磨拳擦掌了！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (隊伍最尾端)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(熱情的夥伴在隊伍尾端作解說)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　好不容易等到十一點，大門一開，人群開始有規矩的按順序排隊進去，<br/>
<br/>
　我則等不及先到附近甫開幕的禮客逛一下，等人潮開始退了之後才進去，<br/>
<br/>
　反正，企鵝跟兔兔杯買不到啦(原來這才是真正的目的呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (十一點一到，大門敞開，大夥按照順序列隊進門！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  　(開始塞車嚕)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (難得看到一位GRD愛用者：先生您真有品味呀！) <br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　用了號碼牌點了一杯大杯摩卡，加上驕傲的不鏽鋼隨行杯，<br/>
<br/>
　只付了十五元，喔，這實在是太划算了...<br/>
<br/>
  喔，看那碩大又肥嫩的鮮奶油，這實在是太不划算了。<br/>
　<br/>
　唉，今天似乎客人太多，夥伴的鮮奶油打得有點馬虎，不是很漂亮，<br/>
<br/>
　而且在過量的狀態之下，咖啡感覺仍稍稍過苦，有點可惜呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (人潮繼續塞車，看來該先轉移陣地嚕！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(水源門市門口人群絡繹不絕，是個好兆頭！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　水源門市裡坐位夠多，不過感覺起來並沒有布置得很漂亮，<br/>
<br/>
　甚至與常去的其他幾家門市相較之下，有一點陽春，<br/>
<br/>
　不過，這絕妙的地理位置實在是太吸引我啦，<br/>
<br/>
　以後一定會常去光顧的。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(終於進來了)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(水源門市內一景)<br/>
