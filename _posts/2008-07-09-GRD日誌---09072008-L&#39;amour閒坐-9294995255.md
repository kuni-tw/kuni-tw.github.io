  下午肚子餓著了，趕緊跑到坐落在公館並久未叨擾的L'amour，<br/>
<br/>
　與同樣古典熱愛音樂的老闆相見歡，馬上點了我最愛的煙燻牛肉法式三明治，<br/>
<br/>
　與一盤特製的大沙拉：肚皮呀肚皮，你今天可過得很開心哪．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

　　這是一家我平常在下午很喜歡去閒坐的蛋糕咖啡館，<br/>
<br/>

                                                                                
　他們自製的蛋糕真的非常可口，價格上也相當實惠，<br/>
<br/>


                                                                                
　只可惜今天時間不夠，沒來得及嚐嚐他們的新口味，<br/>
<br/>

                                                                                
　只能吃簡單的下午套餐，還附上一杯冰摩卡，真是爽到腸子裡去了！<br/>
<br/>

                                                                                
                                                                                
　這一次的沙拉看起來色調清新，非常新鮮的口感，<br/>
<br/>

                                                                                
　煙燻牛肉法式三明治依然美味可口，尤其是微微烘烤過的法式麵包剖開後，<br/>
<br/>

                                                                                
　毫不吝嗇的夾上豐富的三色沙拉與煙燻牛肉片，鹹度適宜，外皮又香脆，<br/>
<br/>

                                                                                
　真是大快朵頤呢！<br/>
