<p><span>(平山家的豪華散壽司定食)</span><br/> <br/><br/><span> --------------------------------------------------------------</span><br/><span>平山家日本料理</span><br/><span>地址：台北市新生南路三段106號3樓之1</span><br/><span>電話：(02)2369-9017</span><br/><span>時間：11:30-14:00 17:30-21:00</span><br/><span> --------------------------------------------------------------</span></p>
<p><span>最近特地連續挑了好多家散壽司品嚐，</span></p>
<p><span>原因無他，因為這是小弟少數幾乎吃不膩的料理之一，</span></p>
<p><span>個人實在是太愛生魚片了。</span><br/> <br/><span>走到台大大門對面的大樓前抬頭一望就可以看到平山的壓克力招牌，</span></p>
<p><span>搭電梯到了三樓，相當樸素與帶點歷史風韻的風格與裝潢，</span></p>
<p><span>就座後先點了招牌散壽司定食，</span></p>
<p><span>整套定食附茶碗蒸沙拉與味噌湯要價NT.580，單點則NT.480，價位相當高。</span><br/> <br/><span>(仰頭一望便可看見平山家的招牌)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191490.jpg" target="_blank"></a></span><br/> <br/><span>(走進大樓後要坐電梯上三樓)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191491.jpg" target="_blank"></a></span><br/> </p>
<p><span>(菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191492.jpg" target="_blank"></a></span><br/> <br/><span>沒多久後茶碗蒸與沙拉先上桌，</span></p>
<p><span>前者蒸蛋部口感細緻滑嫩令人懷念起以前小時候常吃的梅村，</span></p>
<p><span>後者菜葉部分似乎沒什麼挑過且和風醬汁偏清淡過了頭，個人不是很喜歡。</span><br/> <br/><span>(店內一景)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191493.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(茶碗蒸)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191494.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(和風沙拉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=64&amp;f=1363191495.jpg" target="_blank"></a></span><br/> <br/> <br/><span>然後是味噌湯與主角招牌散壽司，前者味道令人印象不甚深刻</span></p>
<p><span>後者新鮮度水準中上但魚料部分種類與量偏過少</span></p>
<p><span>內容包括紅干，鮭魚，胭脂蝦，金目鯛，鰹魚與鮭魚卵等，</span></p>
<p><span>據阿姨說平常還有海膽與星鰻不過當天沒給就是了。</span></p>
<p><span>若改單點散壽司也要NT.480另加服務費個人覺得稱得上是偏貴， </span></p>
<p><span>且說到服務費不禁有點小小不滿：</span></p>
<p><span>用餐時才發現醬油不夠，看到鄰旁服務小姐正在隔壁桌收拾，</span></p>
<p><span>禮貌地請她拿些醬油給我不料竟回："不好意思請等我先弄完！"</span></p>
<p><span>就這樣邊等邊欣賞她做事足足有五六多分鐘， </span></p>
<p><span>先將桌子收拾擦乾淨與細心擺桌後才終於走回去拿了醬油過來，</span></p>
<p><span>呃，又不是客滿狀態為何不能先幫我服務呢？</span></p>
<p><span>最後等小弟終於拿到醬油時胃口也差不多了。(笑)</span><br/> </p>
