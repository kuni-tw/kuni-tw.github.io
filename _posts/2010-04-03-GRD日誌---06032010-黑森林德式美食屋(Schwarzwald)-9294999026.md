黑森林德式美食屋(Schwarzwald)<br/>
地址：台北市安和路一段112巷10號<br/>
電話：(02)23253506<br/>
營業時間：11:30~23:00<br/>
-------------------------------------------------<br/>
<br/>
<br/>
  禮拜六向來要教滿堂整天，甚至連中午也沒吃飯的時間，<br/>
<br/>
　這一天中午突然出現一小時的空檔，<br/>
<br/>
　馬上拎著剛下課的學生一起去附近垂涎許久的德國餐廳：<br/>
<br/>
　黑森林德式美食屋。<br/>
<br/>
<br/>
　"小朋友，今天老師心情好，老師請客吧。"<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (黑森林德式美食屋)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　從小在歐洲長大的我，對於德式料理可說是再熟悉不過了，<br/>
<br/>
　當然對於這一家餐廳也會抱有極大的期望，<br/>
<br/>
　進門後，由服務生帶位就座，師生倆看了看菜單，<br/>
<br/>
　點了德式香腸總匯，要價399，<br/>
<br/>
  紅酒燉牛肉，要價300，<br/>
<br/>
  兩者皆加150變為套餐(平日則加100)，<br/>
<br/>
　附時湯，飲料，全麥麵包，生菜沙拉與餐後甜點。<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (菜單)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(餐前隨拍)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　首先熱湯上桌，熱騰騰的熱湯裡頭有馬鈴薯丁與德國酸菜，<br/>
<br/>
　熟悉的酸味似乎再度重現，雖然可口開胃，<br/>
<br/>
　不過味道較清淡，似乎是為了適應台灣人的口味而更改過，<br/>
<br/>
　令我感到滿失望的。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(黑森林的本日湯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　德國的全麥麵包，被當地的華人統一稱之為黑麵包的Schwarzbrot，<br/>
<br/>
　總算有點像樣了，而且似乎融合了台灣的麵包技術，<br/>
<br/>
　麵包好吃，有淡淡麥香與嚼勁，少了較為傳統的微酸與紮實，<br/>
<br/>
　但回國也好幾年了，老實說，這麵包我還滿喜歡的，<br/>
<br/>
　現在叫我重新去吃以前的傳統黑麵包，搞不好還會不適應呢。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(德國黑麵包)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　飲料是酸梅汁，有點愕然，<br/>
<br/>
　自從上一次在北平都一處喝到令人拍案叫絕的酸梅汁後，<br/>
<br/>
　個人對於酸梅汁也有了更不同的見解，<br/>
<br/>
　舉凡酸味，香味與甜味等等，都有了新的標準，<br/>
<br/>
　這裡的酸梅汁喝起來較似一般酸梅粉所泡出來的，<br/>
<br/>
　酸味香氣皆不足。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
   (酸梅汁)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　生菜沙拉夠新鮮，口感中上，<br/>
<br/>
　還滿好吃的。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　主菜部分首先上來的是學生的紅酒燉牛肉(Rindsgulasch)，<br/>
<br/>
　濃郁的醬汁裡透出紅酒的香氣，<br/>
<br/>
　配上兩坨德國常見的馬鈴薯泥，好吃，不過，<br/>
<br/>
　醬汁味道不夠重，馬鈴薯泥也不夠香，<br/>
<br/>
　跟以前在歐洲時所熟悉的味道還是有一段不小的差距。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (紅酒燉牛肉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　再來的是個人較推薦的德式香腸總匯，<br/>
<br/>
　五條不同色調的德國香腸(Wurst)，<br/>
<br/>
  光是用眼睛看還滿過癮的，當中小弟最愛吃的也是以前的最愛之一：<br/>
<br/>
　Kaesekrainer，中文應該算是起司香腸吧，<br/>
<br/>
　起司同時與絞肉一起灌入腸衣裡頭，<br/>
<br/>
　熱熱的吃，咬一口，裡頭的起司四溢的感覺真是無比過癮，<br/>
<br/>
　每逢冷天時，街口的Wurststand(香腸攤站)，<br/>
<br/>
　看起來就像是沙漠中的綠洲一般，如此美妙與充滿著溫暖。<br/>
<br/>
　這時候突然懷念一樣是百年未見的Leberkaese，<br/>
<br/>
  與Semmel一起夾著吃，與Kebab堪稱學生中的平價美食呀。<br/>
<br/>
  可惜的是，這些香腸雖然說應該是來自德國進口，<br/>
<br/>
　不過應該都是屬低價位的，味道在我眼中很普通，<br/>
<br/>
　連當地的香腸攤站賣的都比不上呀。<br/>
<br/>
　但好幾年沒吃過道地家鄉味的我，邊咀嚼邊懷念起往日歲月，<br/>
<br/>
　心想，早日應該重新拜訪一次溫德呀。<br/>
<br/>
　<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (德國香腸總匯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　最後上桌的是兩道皆普通的甜點，<br/>
<br/>
　尤其是Sachertorte，一種號稱維也納最經典的巧克力蛋糕，<br/>
<br/>
　也是個人的夢中情人之一，<br/>
<br/>
  可惜這裡的仿製品與維也納的正統Sachertorte實在是天差地遠呀。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(兩款飯後甜點，個人各只吃了一口)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　濃郁的巧克力味道不稀奇，在台灣多的是，<br/>
<br/>
　依Sachertorte傳統一定要加一坨鮮奶油在旁一樣也不稀奇！<br/>
<br/>
　真正道地的Sachertorte，<br/>
<br/>
  濃郁的巧克力味中還同時有著一股無法形容的獨特果香與酸味，<br/>
<br/>
　這就是Sachertorte自1832年至今仍無人成功複製的秘密之一。<br/>
<br/>
　據小弟那時候聽到的一則較不可靠的消息是，<br/>
<br/>
　此蛋糕據說是產生自某蛋糕師傅一次不經意的失誤中，<br/>
<br/>
　將做好的巧克力蛋糕在冰冷的地窖中存放過久，<br/>
<br/>
　因而產生出一種特別的酸味而聲名大噪，<br/>
<br/>
　當然真實情況似乎也不可考了，<br/>
<br/>
　這故事就當作是茶餘飯後的一則趣事來看好了。<br/>
