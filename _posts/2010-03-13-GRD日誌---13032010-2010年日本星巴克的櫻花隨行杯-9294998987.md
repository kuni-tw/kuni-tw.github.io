<br/>
<br/>
<br/>
<br/>
    (日本星巴克2010的限量櫻花不鏽鋼杯組)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不知道從哪一年開始，日本星巴克在櫻花季推出的限量商品，<br/>
<br/>
　總會造成一股旋風，記憶中，最令人瘋狂的莫過於08年的透明隨行杯款，<br/>
<br/>
　粉紅系的色調中飄著無數櫻花瓣，令人愛不釋手，<br/>
<br/>
  恰好那年世界第一款透明的星巴克城市杯也剛好上市，<br/>
<br/>
　那就是現在正炒得火熱的東名高速隨行杯。<a href="http://www.wretch.cc/blog/kuni/11392305" target="_blank"> ( 詳情請參閱：東名高速杯與鳳凰杯與外拍 )</a><br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
　去年打著08年櫻花杯狂賣的氣勢，09年的櫻花杯賣得也相當好，<br/>
<br/>
　霧狀杯面讓整體成色變得較為夢幻，不過可能因與前年相比之下較無特色，<br/>
<br/>
　後來似乎有點氣勢不足的現象，而08年杯款價格如今在網拍上節節攀高，<br/>
<br/>
　就在此時，10年款的櫻花杯終於有了眉目：<br/>
<br/>
　那就是日本即將推出櫻花不鏽鋼隨行杯款，一聽到如此令人振奮的消息，<br/>
<br/>
　小弟馬上託人從日本代買，不過因為不鏽鋼款單價相當高，<br/>
<br/>
　兩款櫻花不鏽鋼杯就要價6200日圓，這還不包括運費與包裝等等。<br/>
<br/>
　最後忍痛各買下兩只，加上12oz的櫻花隨行杯"10年正常版"。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(這是12oz的正常版櫻花杯)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(2010年的不鏽鋼櫻花杯組)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　終於，經過了重重波濤洶湧的海浪，小弟的杯子在台灣上岸了，<br/>
<br/>
　早上教課前先在星巴克龍門門市拆封挖寶一番，<br/>
<br/>
　對著新鮮的杯款仔細檢視，愈看愈是開心，<br/>
<br/>
　精美的塗裝與大和民族的櫻花魅力實在是太迷人了。<br/>
<br/>
  最令人驚訝的是，這一次正常版的12oz櫻花隨行杯比想像中的漂亮許多，<br/>
<br/>
　杯身的塗裝裡紙有作點微皺處理，增加其細緻度，<br/>
<br/>
　淺色系的配色使得10年款同時擁有了09年的夢幻元素，<br/>
<br/>
　非霧面處理較易有刮傷之虞，所以使用時可要小心保護杯子的安全呀，<br/>
<br/>
　不然杯子摔了可就要心疼個好幾天呢。<br/>
<br/>
<br/>
<br/>
　不鏽鋼款靠著簡潔有力的櫻花圖騰，告知了天下它的意圖，<br/>
<br/>
　小巧的杯身看起來相當秀氣。<br/>
<br/>
　堅固的不鏽鋼材質配上氣質柔弱的的色調，一點也不具衝突感。<br/>
<br/>
　簡潔的珠光白與粉紅，在橡膠手握，不鏽鋼杯身與杯蓋上分別作兩種配色，<br/>
<br/>
　兩只杯子擺在一起簡直有如孿生地密不可分，真是美極了。<br/>
<br/>
　<br/>
<br/>
