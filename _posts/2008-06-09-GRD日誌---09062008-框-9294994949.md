<br/>
<br/>　我試著找尋一些突破，給予自己繼續在比賽裡存活的免死牌，<br/>
<br/>　於是整天提心吊膽的望著那兩針滴答滴答的繞呀繞的．<br/>
<br/>　十二圈之後我開始活躍起來；我說思緒從此階段開始熱絡，<br/>
<br/>　但卻與詩意毫無牽連，而是屈服於現實的反省．<br/>
<br/>
　這樣腦中的磁碟開始趨向高轉，因為我開始同時使用播放與儲存這兩功能鍵,<br/>
<br/>　同時間開了十幾個視窗，以致於音樂檔時而間斷，到後來，<br/>
<br/>
　有七八個視窗毫無反應，我便先將程式關閉，然後斷絕網路，<br/>
<br/>
　等到重新開啟之時，路就變得又寬敞又舒適了．　<br/>
