<p><span>(令人無法忘懷的散壽司)</span><br/> <br/> <br/><br/> <br/><span>----------------------------------</span><br/><span> 三大日味屋</span><br/><span><span> 地址：台北市中正區汀州路二段217號</span></span><br/><span><span> 電話：(02)2366-0103</span></span><br/><span><span> 營業時間：11:30～14:00/17:00～21:30</span></span><br/><span><span>(每週一公休，週二僅下午時段)</span></span><br/><span> ----------------------------------</span><br/> <br/><span>這一下午被朋友臨時邀去汀州路上一家平價日本料理，</span></p>
<p><span>聽說是夫妻兩人上一次不經意經過吃了一頓，</span></p>
<p><span>覺得非常好吃，所以這一次想帶我一起來。</span><br/> <br/><br/><span>走進店裡，明亮而暖調的燈光配合著簡潔中上質感的裝潢，</span></p>
<p><span>造就出相當討喜和舒服的用餐氣氛，</span></p>
<p><span>明亮的玻璃質牆壁與廚房外的黑板上寫滿了一堆推薦菜色，</span></p>
<p><span>便點了招牌散壽司(NT.280)，味增湯(NT.20)，紫蘇蘋果手捲(NT.50)，</span></p>
<p><span>軟殼蟹(NT.160)，炸豆腐(NT.80)，明太子烤馬鈴薯(大份要價NT.120)</span></p>
<p><span>(店內裝潢與菜單隨拍)</span></p>
<p><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690953.jpg" target="_blank"></a></span><br/><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690954.jpg" target="_blank"></a></span><br/><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690955.jpg" target="_blank"></a></span><br/><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690956.jpg" target="_blank"></a></span><br/> <br/> <br/> <br/><span>首先上來的是味增湯，裡頭置放一塊嫩豆腐，調味適中，</span></p>
<p><span>湯頭味道很棒，有著自然的微微甘甜。</span></p>
<p><span>然後是個人相當喜愛的紫蘇蘋果手捲，</span></p>
<p><span>第一次吃到將紫蘇包進海苔裡的手捲，</span></p>
<p><span>酸甜的滋味很棒，不過海苔似乎有點受潮了，可惜。</span></p>
<p><span>炸豆腐則普普，竟然要價80，個人覺得是稍微貴了。</span></p>
<p><span>不過軟殼蟹可不一樣了，諾大的一份軟殼蟹，</span></p>
<p><span>炸得金黃，鬆軟的口感締造了絕佳的享受，</span></p>
<p><span>實在是太好吃了，</span></p>
<p><span>光這一道料理就足以讓小弟開始考慮下一次來訪的日期。</span></p>
<p><span>明太子烤馬鈴薯，香濃又充滿海洋風味的明太子醬裹在熟馬鈴薯上，</span></p>
<p><span>醬料部分很棒，香氣濃郁，不過與馬鈴薯似乎沒有很搭。</span><br/> <br/><br/><span>(紫蘇手捲)</span></p>
<p><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690957.jpg" target="_blank"></a></span><br/> <br/> <br/><span>(味增湯)</span></p>

<p><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690960.jpg" target="_blank"></a></span><br/><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690961.jpg" target="_blank"></a></span><br/> <br/><span>(烤明太子馬鈴薯)</span></p>
<p><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690962.jpg" target="_blank"></a></span><br/><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690963.jpg" target="_blank"></a></span><br/> <br/> <br/><span>重頭戲來了，要價280的散壽司，基本上個人不覺得這稱得上完全的平價，</span></p>
<p><span>上頭鋪滿鮭魚，鮪魚，蝦卵，透抽，鮭魚卵、明太子，花枝等等，</span></p>
<p><span>乍看之下並無特殊之處，不過淺嚐一口後，才發現竟然是如此的美味，</span></p>
<p><span>真的是很好吃，魚料部分切得較薄，不知道是淋上什麼醬料，</span></p>
<p><span>軟嫩中充滿著濕潤感，真是令人驚奇，</span></p>
<p><span>可惜跟朋友熱絡的聊天之中並未特意的去檢視其中的細節，</span></p>
<p><span>也忘了問店家了，改天再去請較一番呀。</span><br/> <br/><span>(充滿濕潤感的神奇散壽司)</span></p>
<p><span> <a href="/album/show.php?i=kuni&amp;b=51&amp;f=1376690952.jpg" target="_blank"></a></span></p>
<p><span>以平價度來說，這一家完全沒得跟新井，食堂屋甚至是永康的游壽司比，</span></p>
<p><span>比如手捲，食堂屋的四十元蘆筍手捲那豐富的配料C/P值絕對更高，</span></p>
<p><span>比如散壽司，游壽司的散壽司三百多附四貫握壽司，而且還是定食。</span></p>
<p><span>不過，這裡有一堆令我無法忘懷的好菜，</span></p>
<p><span>光那軟殼蟹還是神奇又充滿濕潤感的散壽司，</span></p>
<p><span>想著口水又要流出來了。</span></p>
