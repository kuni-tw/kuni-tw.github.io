<div class="article-content-inner" id="article-content-inner">
<br/>
<br/>
<br/>
  (視覺上很誘人的85度c草莓乳酪)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　下午空檔之餘，臨時找了間85度C來打發時間，<br/>
<br/>
　點了杯法式經典摩卡，透著玻璃櫥窗思考了頗久，<br/>
<br/>
　依稀記得在網路上好像有看過網友們介紹過草莓乳酪，<br/>
<br/>
　加上那一大顆圓滾滾又紅透透的誘人大草莓，<br/>
<br/>
　不禁就點了一份．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (草莓乳酪俯瞰圖，圓滾滾的草莓真可愛)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　不一會兒，櫃檯叫了號，便去領取我的摩卡與慕斯，<br/>
<br/>
　而咖啡部分就不必說了，真是經典，乍喝之下還以為是冰可可，<br/>
<br/>
　一點咖啡味也沒有！<br/>
<br/>
　望著牆上大幅的85度Ｃ廣告上寫著斗大的"五星級的咖啡"時，<br/>
<br/>
　我還是不爭氣的笑了出來...<br/>
<br/>
<br/>
<br/>
  草莓乳酪可就不同了，85度C雖說是平價連鎖店，也開了不知道幾百家了，<br/>
<br/>
　但糕點類相較之下真的頗具水準．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(平價連鎖店85度C的糕點類頗具水準呢！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　慕斯上那顆大草莓看起來真是令人流口水．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(口水止不住嚕！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　用湯匙對著慕斯舀起來，才發現裡頭另有玄機：<br/>
<br/>
　原來不是整塊慕斯而已，還有蛋糕夾層呢，<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(用湯匙對著慕斯舀起來)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　口感其實還好，與前陣子吃得那些較高檔的糕點類仍然有一段差距，<br/>
<br/>
　但在草莓酸甜的自然滋味配合之下，還滿不錯的唷！<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　(咬了一口的草莓剖面：真令人垂涎欲滴呀！)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>
<br/>
　最後還真是得說，那一杯摩卡也真是太經典了，<br/>
<br/>
　85度C，你們在咖啡上真的要改善一點啦...<br/>
<br/>
<br/>
<br/>
