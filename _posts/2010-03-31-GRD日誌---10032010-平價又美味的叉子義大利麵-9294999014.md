<div class="article-content-inner" id="article-content-inner"><p><span></span></p>
<p><span> </span></p>
<p><span> 　　(叉子義大利麵的凱薩沙拉)</span><br/> <br/> <br/> <br/><br/> <br/> <br/> <br/> <br/><span> ---------------------------------------------------</span><br/><span> 叉子義大利麵</span><br/><span> 地址： 台北縣永和市文化路110號</span><br/><span> 電話：02-2925-4506</span><br/><span> ---------------------------------------------------</span><br/> <br/> <br/> <br/> <br/> <br/><span>　在永和的叉子義大利麵，向來以價美物廉著稱，</span></p>
<p><span> 　小弟在此之前不知已耳聞過多少遍有關於這一家的讚譽，</span></p>
<p><span> 　尤其是低價位之下仍舊保有一定的美味與分量。</span></p>
<p><span> 　這一天小弟趁著月黑風高之時，從永和路路口切進文化路，</span></p>
<p><span> 　沒多久後便找著了：相當不起眼而又低調的店門口，</span></p>
<p><span> 　旁邊竟然是一家機車行，感覺還滿突兀的，</span></p>
<p><span> 　從門口便可從開放式廚房看到老闆與其他廚師們，</span></p>
<p><span> 　正如火如荼地翻炒著新鮮的醬汁，看著看著，</span></p>
<p><span> 　真是難以想像，這樣一家有點類似熱炒店風格的裝潢之下，</span></p>
<p><span> 　竟然主打的是義大利麵呢。</span><br/><br/> <br/> <br/><span>　(在店門口候位的情侶正專心地看著廚房)</span></p>
<p><span> </span><br/><br/> <br/><span> 　　(廚房內的老闆與其他廚師們)</span></p>
<p><span> </span><br/><span> </span><br/> <br/><br/> <br/> <br/> <br/><span> 　走進店裡後，一切相當自助，</span></p>
<p><span> 　餐具與餐巾紙一切自行取用，勾好了菜單，</span></p>
<p><span> 　點了青醬蘆筍蝦麵，要價130，</span></p>
<p><span>　一盤凱薩沙拉，要價40，</span></p>
<p><span>　酥皮本日濃湯，要價50(今天為南瓜濃湯)</span><br/> <br/><br/> <br/><span> 　　(供客人自行勾選的菜單)</span></p>
<p><span> </span><br/><br/><br/> <br/><span> 　</span><br/><span> 　沒多久後，沙拉先上桌，</span></p>
<p><span> 　說實話，擺盤雖然較普通，</span></p>
<p><span> 　但生菜部分夠新鮮，切得大小適宜，</span></p>
<p><span> 　配上好吃的醬料，口感清爽。</span><br/> <br/><br/><span> 　　(凱薩沙拉)</span></p>
<p><span> </span><br/><br/> <br/> <br/><span> 　酥皮濃湯，湯頭真是不簡單，</span></p>
<p><span> 　夠濃郁，分量夠大，建議以後點的時候不要加酥皮，</span></p>
<p><span> 　這樣只要30，好喝美味不減，但可以保留較多點胃，</span></p>
<p><span> 　不過這是後話。</span><br/> <br/><br/> <br/> <br/><span> 　　(酥皮本日濃湯)</span></p>
<p><span> </span><br/><span> </span><br/> <br/> <br/><br/><span> 　最後主餐登場，真的是相當大的分量，</span></p>
<p><span> 　用叉子與湯匙捲好了麵條，先淺嚐了一口，</span></p>
<p><span> 　頓時終於明白為何有這麼多人如此推崇這家店了，</span></p>
<p><span> 　濃稠，香噴噴又量多的醬汁，調味適中，</span></p>
<p><span> 　同時又有著熱炒店的熱度，淋在量質皆優的麵條上，</span></p>
<p><span> 　真是令人大呼過癮，</span></p>
<p><span> 　這絕對是少見的超級台式而又美味的平價義大利麵，</span></p>
<p><span> 　同時嚐了一口同行友人點的培根蛋麵(NT.100)，</span></p>
<p><span>　一樣是驚人，甚至好吃到讓我想要馬上再點第二份。</span></p>
<p><span> 　可惜此時胃早已毫無任何空間可以再放進任何美食了，</span></p>
<p><span> 　真是令人難以忘懷的平價美食呀。</span><br/> <br/><br/> <br/><span> 　　(青醬蘆筍蝦麵)</span></p>
