(店門口)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/>　六星烤雞永和店地理位置相當好找，離福和竹林路口約幾十公尺處。<br/>																			    <br/>　店門口寫著從99年12/24至100年01/07有優惠，<br/>																			    <br/>　比如烤全雞由168變成139，氣水類飲料一杯只要十元。<br/>																			    <br/>　店內空間寬敞明亮，門口處有一個大烤爐，<br/>																			    <br/>　可惜沒什麼客人非常冷清。<br/> <br/>　餐點一下子便被端過來。<br/><br/><br/><br/><br/><br/><br/>　(點餐櫃台)<br/><br/><br/><span></span><br/><br/><br/><br/><br/><br/><br/><br/><br/><br/>　脆薯部分是市面上常吃得到的五香口味薯條，<br/>																			    <br/>　基本上應該是跟中央廚房進貨的。<br/>																			    <br/>　玉米濃湯尚可，沒什麼特別。<br/>																			    <br/>　由於六星烤雞是卜蜂集團的關係企業，<br/>																			    <br/>　自然進的也是他們自有的肉品加工廠雞肉，<br/>																			    <br/>　另外使用泰式烤爐烘烤。<br/>																			    <br/>　客人使用店家附贈的半透明薄手套以手扒方式享用烤雞，<br/>																			    <br/>　不過不知是什麼緣故，個人沒有特別喜歡六星的烤雞，<br/>																			    <br/>　最主要是部分肉質偏乾些且不夠入味，<br/>																			    <br/>　跟之前吃過的二十一世紀風味館烤雞相比之下，<br/>																			    <br/>　後者似乎比較好吃些。<br/><br/><br/><br/><br/>　(六星的烤全雞，脆薯以及玉米濃湯)<br/><br/><br/><span><br/><br/><br/><br/><br/><br/><br/><br/>　(好久沒有用手扒方式吃烤雞嚕)<br/><br/><br/></span><span></span><br/><br/><br/><br/><br/><br/><br/><br/>　不過一月一號之後開始會販售烤雞便當，<br/>
