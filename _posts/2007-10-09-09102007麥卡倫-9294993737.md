09102007麥卡倫<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　好不容易搞到紫標,<br/>
<br/>
  今夜與友暢懷,<br/>
<br/>
  雖因整日臥病在床而與華佗先生有短短一面之緣,<br/>
<br/>
  卻依然興致不減,<br/>
<br/>
　這就是無差別的魅力.<br/>
<br/>
<br/>
  從美麗的國度裡取回遠古記憶猶存的書卷,<br/>
<br/>
  看著一頁一頁的精華,<br/>
<br/>
  協力番找出其中,<br/>
<br/>
  的遺忘國度.<br/>
<br/>
<br/>
  深藍的海洋鬼魅,<br/>
<br/>
  椰林羊腸的巨人,<br/>
<br/>
  愛好和平的幾何伉儷,<br/>
<br/>
  深山中的臥龍先生,<br/>
<br/>
  乘著颶風與我共度一堂.<br/>
<br/>
<br/>
　啜飲著智慧的結晶,<br/>
<br/>
  登時忘了被上天派遣而來的目的,<br/>
<br/>
  亦或說這就是?<br/>
<br/>
<br/>
<br/>
  卻無妨,<br/>
<br/>
　帶著淡淡的情懷入眠,<br/>
<br/>
  明日的台北多麼的期待,<br/>
<br/>
　期待的明日多麼的台北.<br/>
