17062008---隨想<br/>
<br/>
<br/>
  這幾天忙著一些瑣碎雜事，常常也沒什麼重點，<br/>
<br/>
　有點漫無目的的感覺，卻無法置身事外，原因無他，<br/>
<br/>
　只因為我想做的事情太多，稱得上是貪得無饜！<br/>
<br/>
　<br/>
  觀察著平常在管理的網路看板最近見怪不怪的亂象，<br/>
<br/>
　不少人將情緒合理化，甚至倒果為因，利用假象的多數給自己些藉慰，<br/>
<br/>
　我不禁想搖搖頭！<br/>
<br/>
　老實說，我向來支持自掃門前雪，這樣說起來似乎相當自私，<br/>
<br/>
　且不符合我們從小所被灌輸的社會倫理，但實質上呢？<br/>
<br/>
　這些規矩似乎不過是聰明人愚弄多數人的手段罷了！<br/>
<br/>
　別的不說，舉交通規則來當例子吧，試想，<br/>
<br/>
　市井小民違規就得乖乖受罰，這聽起來還滿合情合理的．<br/>
<br/>
　但路上隨處可見那些警察們違規，卻從未被自己人開過罰單，<br/>
<br/>
　不但便宜行事，甚至振振有詞，更別說那些為大眾們制定法律的立法委員了！<br/>
<br/>
　<br/>
　倫理該不該"愚守"，這裡我且強烈提出質疑！<br/>
<br/>
<br/>
  為什麼剛剛提到自掃門前雪呢？<br/>
<br/>
　只因為那些重度網路使用者或者是"重度自我藉慰者"，<br/>
<br/>
　與我其實毫無直接利益關係．<br/>
<br/>
　且，當此社會愈來愈亂時，也正是能力者大放異彩之時，<br/>
<br/>
　這也是為什麼我懶得對這社會有任何建設性的批判，<br/>
<br/>
　當然除了懶得管以外還有就是反正這些人也絕對聽不進去的．<br/>
<br/>
　但，話說回來，<br/>
<br/>
　這亂象真的不是台灣之福，<br/>
<br/>
　我們卻無法做任何改變，只能祈求其他國家此現象更為嚴重而已．</div>
