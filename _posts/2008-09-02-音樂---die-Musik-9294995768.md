die Musik<br/>
<br/>
<br/>
<br/>
  我如是說，<br/>
<br/>
　夜半原本只想舒發情緒的，看租來的一部片，<br/>
<br/>
　看到中途隱約的感到不對勁，然後我就哭了．<br/>
<br/>
<br/>
<br/>
<br/>
　我看到我的生命中最美好的那一段文字與風采，就在那當下的悸動，<br/>
<br/>
　與數不清的音符相互交流著．<br/>
<br/>
<br/>
<br/>
　我是否還有能力回到充滿無數感動的那一刻呢，我這樣問自己，<br/>
<br/>
　生命是否還未結束，或許，亦可這樣：<br/>
<br/>
　我血液中仍存在著青春時期的音樂與不可數的繁星，鞏罩在天際，<br/>
<br/>
　隨著小橋流水開始，緩緩地隨著母親與少年的腳步，踏在Beethovengang的河邊，<br/>
<br/>
　和諧的田園旋律從遠處毫無重音的起始，我閉目和著它呢喃著．<br/>
<br/>
<br/>
<br/>
　某天晚上，在南奧某處舞台邊，兩把熱戀中的小提琴各自奏出旋律，<br/>
<br/>
　慢板的和絃交互變化，不若調情，只有小宇宙的火花與熱焰狂暴著所有星系，<br/>
<br/>
　十六分音符在主旋律下竄了出來，不停的向前奔，拋棄世俗的羈絆，<br/>
<br/>
　剎那間，萬物失去了呼吸，為完美而靜默恆久，彼此心靈繼續交流，<br/>
<br/>
　突然間我回到了某下午空教室裡，用力踩著踏板，使勁地踩，<br/>
<br/>
　午後的餘光從滿是塵埃的窗透了過來，包圍整台鋼琴與生命，<br/>
<br/>
　奏出我的心靈與潛意識，是的，音樂是不需要技術的，<br/>
<br/>
  最完美的音樂只是一種熱情而已！<br/>
<br/>
<br/>
<br/>
　記得在我第一次聽到Daniel的Pathetique時，看著他左手有如蝴蝶般的伴奏，<br/>
<br/>
　右手重搓著琴鍵，只有澎湃與激昂才能與此匹敵的音響才能重創我的孩子氣，<br/>
<br/>
　我被霹靂擊到了，是的，有如教父裡的麥可初見阿波羅妮雅的時候，<br/>
<br/>
　我稱它為大霹靂，一種與熱戀過於相似的形容詞，我不禁長嘆一口氣，<br/>
<br/>
　與Daniel借了樂譜，一跟手指頭一跟手指頭地踏著白鍵與黑鍵，<br/>
<br/>
　走過一頁之後，我加快了速度與節奏，慢慢地我接近了它，一處藏寶地點，<br/>
<br/>
　然後毫不喘息的又走了一遍，又加快了層級，然後我感覺我擁有了它！<br/>
<br/>
<br/>
<br/>
　然後某一天我遇到了張小嫻，傷害了我純潔的心靈，<br/>
<br/>
　被迫地接受了無情的摧殘，那一夜，我睡不著，一直傷心到次晨，<br/>
<br/>
  Ja, das ist die Welt, die eigentlich nicht Dir gehoeren soll.<br/>
<br/>
  沒錯，這本來就不是屬於你的世界，不過死心吧，你從此再也無法離開它了．<br/>
<br/>
　我似乎漸漸地忘了所有的感動，我似乎只會利用著它所留下來的遺產，<br/>
<br/>
　而忘了內心裡的那隻精靈，一隻快要絕種的生物，全身不時地，<br/>
<br/>
　隨著節拍或是心跳聲而發光發熱，不需要觀眾就可以整夜在空曠的山谷裡唱歌，<br/>
<br/>
　我忘了它："我對不起你！"<br/>
<br/>
　所以我哭了，這好幾年來唯一的一次，我抽噎著一直到結束．<br/>
<br/>
　<br/>
<br/>
　突然間，我忘了該如何使用帶髒的字眼，只記得聖潔與無國際，<br/>
<br/>
　望著白色的耀光與雨中的澎湃，山中的巨靈依著歌德的囑咐搖搖晃晃地走到山角，<br/>
<br/>
　大地顫抖著，塵沙暴飛揚起來，撩向無邊的天際，<br/>
<br/>
　然後又是一陣默哀(靜)<br/>
<br/>
<br/>
<br/>
　心跳聲持續著，狂朗著，我重新拾起回憶，<br/>
<br/>
　然後奔向高原，向大地怒吼，我想要回應，<br/>
<br/>
　那一晚我哭了，面向行人繁忙的Kaerntnerstrasse，<br/>
<br/>
　我記得我曾在那間隔整齊的石板路上發過誓，我要學習如何感動，<br/>
<br/>
　學習如何與大自然對話，與天際心靈交流，與山川散步，<br/>
<br/>
　與白雲遊蕩，與百年的石塊相擁，與Schwedenplatz的冰淇淋融合，<br/>
<br/>
　與多瑙河合唱，與當下甫釀出來的Sturm搖搖欲墜，<br/>
<br/>
　與手風琴搖起阿爾卑斯山角下的民族舞蹈，與熱戀共譜愛之曲，<br/>
<br/>
　與晚年坐在燭臺邊取暖...<br/>
<br/>
<br/>
<br/>
  我像似失去了所有，忘了曾經的那一段生命，<br/>
<br/>
　我真沒料到，那一段生命竟然地這麼短促，而我卻渾然不覺．<br/>
<br/>
　漸漸地，曙光慢慢浮起，我眼前的世界開始跟我招手，我伸向前去，<br/>
<br/>
　卻觸碰到堅硬無比的一塊萬年冰，透明無瑕，我懷著淚目送它，<br/>
<br/>
　嘗試藉著酒精將控制我十幾年之久的理智推入火坑，縱然一秒鐘也好，<br/>
<br/>
　我只要留住那片刻，知道如何歸去的那一段路線，<br/>
<br/>
　因為那是我僅存的自然與單純．<br/>
<br/>
　<br/>
<br/>
  維也納的巧克力隨著濃濃的4.5%牛奶褪去苦澀，<br/>
<br/>
　奶香四溢，甜味散開，心裡卻充滿著酸酸的甜味，<br/>
<br/>
  Strauss的華爾茲配上Bernstein的Opus One與Radetzky Marsch，<br/>
<br/>
　從容地遞上了所有一與二小提琴，奮力的奏出新年音樂會的鐘聲，<br/>
<br/>
　早晨我們漫步在冷颼颼的馬路上，吃著熱呼呼的烤起司香腸淋上黃色的芥茉醬，<br/>
<br/>
　邊咬邊滴下，吃得滿手都是，然後似乎空氣中仍舊充滿著音樂與節拍，<br/>
<br/>
　配合著自動剪票機的咖嚓聲與開門前的預告，此時最適合一杯Soda，<br/>
<br/>
　扔進維他命補給片，聽著它在四處流竄的氣泡中漸漸散發，<br/>
<br/>
　多麼美的音樂呀！<br/>
<br/>
<br/>
<br/>
  然後淚也乾了<br/>
