  自從吃過葵元和食與游壽司他們超值的午間定食後，<br/>
  這部分請參閱：<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11470373" target="_blank">游壽司好吃的中午定食</a><br/>

              還有<a href="http://www.wretch.cc/blog/blog.php?id=kuni&amp;article_id=11414033" target="_blank">葵元和食</a><br/>
<br/>
<br/>
  小弟便開始對於相同性質的餐廳感到極大的興趣，<br/>
<br/>
　最近鎖定的其中一家便是太卷壽司了！<br/>
  (地址：北市仁愛路四段345巷15弄2-1號)<br/>
<br/>
　太卷壽司是東區知名的MASA日本料理前任熟食師傅Travis，<br/>
<br/>
　在前幾個月前開的，主打的是價位較平易近人的日本料理．<br/>
<br/>
　前晚預先定了位，順便告知預算約為六七百左右，<br/>
<br/>
　與爸媽約好中午來這裡吃午飯，依著住址好不容易才找到店家，<br/>
<br/>
　外觀真的是低調到不行，外側也看不到醒目的招牌，<br/>
<br/>
　位置相當難找，不過另外卻意外的發現JB漢堡就在一旁，<br/>
<br/>
　馬上加進我的食記待吃名單當中(不過依照順位還有得等低...)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  (板前桌上餐具擺設)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(Travis師傅細心的研磨山葵)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  店內空間雖然不大，不過因為也才兩張桌子，<br/>
<br/>
　其餘則由板前十人座來吸收，所以空間還滿足夠的，<br/>
<br/>
　就座後沒多久先上了沙拉：<br/>
<br/>
　裡頭有番茄，柿子，洋蔥與較不常見的石榴，<br/>
<br/>
　灑上一點芝麻，不錯，很開胃．<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(好吃的前菜沙拉)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　　(裡頭有番茄，柿子，洋蔥與較不常見的石榴)<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　<br/>

<br/>

<br/>
　烤得剛好，外熟內軟，<br/>
<br/>

<br/>
