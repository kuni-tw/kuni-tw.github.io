<p> </p>
<p><span><br/> -----------------------------------------------<br/>名傳鵝肉<br/>地址：台北市北投區石牌路二段183號<br/>電話：(02)2826-1401<br/> -----------------------------------------------<br/><br/>這是一家朋友強力推薦的鵝肉專賣店，<br/></span></p>
<p><span>向來不吃鵝肉的小弟這一次總算被說動了，<br/></span></p>
<p><span>教完課後當晚與友直接衝去許久不見的北投，也順便泡個溫泉。<br/>     <br/></span></p>
<p><span>從承德路在石牌路口右轉後一路直行沒幾分鐘後便找著了。<br/></span></p>
<p><span>與一般台灣小吃店無異的店內裝潢，<br/></span></p>
<p><span>可惜就座後發現桌子表面有點髒且過於油膩，<br/></span></p>
<p><span>只好請阿姨拿抹布過來重新擦一次。<br/> <br/></span><span>(名傳鵝肉店外)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119253.jpg" target="_blank"></a><br/> <br/>(櫃台)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119254.jpg" target="_blank"></a></span><br/> <br/><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119265.jpg" target="_blank"><br/></a></span><span>首先點了鵝肉，鵝胗，鵝肝與鵝腸各切一份，各依時價計算。</span></p>
<p><span>除鵝肉那盤算NT.130外，其餘每一份皆算NT.70。</span></p>
<p><span>另外一份鵝血糕，NT.35，兩碗魯肉飯，一碗NT.25，</span></p>
<p><span>燙青菜，NT.30。</span></p>
<p> </p>
<p><span>(牆上的菜單)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119255.jpg" target="_blank"></a>    </span></p>
<p><span>菜色一下子便上齊了，首先燙青菜使用的是菠菜，</span></p>
<p><span>表現與一般小吃店差不多，只不過菠菜剛好是個人最愛的葉菜類之一。</span></p>
<p><span>然後是魯肉飯，白飯部分煮得還不錯，肉燥部分放得偏多些。</span></p>
<p><span>(燙菠菜)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119256.jpg" target="_blank"></a><br/> <br/>(魯肉飯)</span></p>
<p><span><a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119257.jpg" target="_blank"></a></span><br/> <br/> <br/><span>重頭戲來了，名傳的鵝肉當真名不虛傳，</span></p>
<p><span>偏淺粉紅的肉色充滿著濕潤感，光看肚子就開始叫了。</span></p>
<p><span>入口咀嚼起來有著鵝肉特殊的紮實口感卻不會過韌難咬，</span></p>
<p><span>另外同時兼有肥嫩與肉汁，實在是令人難以忘懷，</span></p>
<p><span>光這一道料理就讓小弟覺得今晚沒有白來了。</span></p>
<p><span>(名不虛傳的重頭戲：名傳鵝肉)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119258.jpg" target="_blank"></a><br/> <br/>(看多麼可口呀)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119259.jpg" target="_blank"></a><br/><br/>(誘人的一幕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119260.jpg" target="_blank"></a><br/> <br/></span></p>
<p><span>鵝腸部分口感又脆又嫩，淋上沾醬頗為下飯。</span></p>
<p><span>鵝血糕還不錯，比常吃的豬血糕還要來得可口些，</span></p>
<p><span>同行的友人非常喜歡。　</span></p>
<p><span>鵝肝則與鵝心算在一起切，再與鵝胗擺在同一盤，</span></p>
<p><span>前者吃起來不怎麼令人印象深刻，但鵝胗卻比想像中來的要好吃許多，</span></p>
<p><span>比雞胗口感更軟嫩，卻又不失微脆。</span><br/> <br/><span>(鵝腸)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119261.jpg" target="_blank"></a><br/> <br/>(鵝血糕)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119262.jpg" target="_blank"></a><br/> <br/>(鵝肝與鵝胗)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119263.jpg" target="_blank"></a><br/> <br/>(鵝肝與鵝胗微距一張)</span></p>
<p><span> <a href="http://www.wretch.cc/album/show.php?i=Kuni&amp;b=33&amp;f=1483119264.jpg" target="_blank"></a><br/><br/></span><span>最後感謝名傳重新喚起我對於鵝肉的興趣，</span></p>
<p><span>真是沒想到竟然可以到這麼好吃的地步呢。　</span><br/> <br/><span>(的確名不虛傳)</span></p>
