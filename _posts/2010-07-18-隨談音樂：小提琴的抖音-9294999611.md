<p><span> </span><br/> <br/> <br/><span>　(奧伊斯特拉克的Tschaikowsky Violin Concerto in D)</span><br/> <br/> <br/> <br/> <br/><span>　在器樂裡，尤其是管樂或是弦樂，</span><br/><span>     </span><br/><span> 　抖音的使用率非常高。</span><br/><span>     </span><br/><span> 　它本身被視為模仿人聲的方式之一，</span><br/><span>     </span><br/><span> 　藉由這技巧，可將音色等等做巧妙裝飾以及提升，</span><br/><span>     </span><br/><span> 　達到接近人聲的境界，甚至做出一些戲劇性的效果。</span><br/> <br/> <br/> <br/> <br/> <br/> <br/><span> 　在廣大的世界演奏台上，技巧性好的小提琴家比比皆是，</span><br/><span>     </span><br/><span> 　但抖音可就不一定了，很多大師們往往都有很棒的抖音與音色。</span><br/><span>     </span><br/><span> 　比如奧伊斯特拉克的手腕抖音，幅度相當大，音色溫暖。</span><br/> <br/> <br/><span> 　曼紐因則抖音頻率較快，音色較燦爛與絢麗。</span><br/> <br/> <br/><span>   (曼紐因的Beethoven Violinkonzert in D)</span><br/> <br/><span> </span><br/> <br/> <br/> <br/> <br/> <br/><span> 　另外再舉兩個也是以抖音聞名的當代高手，</span><br/><span>     </span><br/><span> 　一個是帕爾曼，他的抖音技術驚人，</span><br/><span>     </span><br/><span> 　而且從前代大師們的演奏中學到不少，將最有效果的保留下來，</span><br/><span>     </span><br/><span> 　帕爾曼厲害的地方在於，他可在抖音的幅度與速度上，</span><br/><span>     </span><br/><span> 　做出相當細膩的層次感。</span><br/><span>     </span><br/><span> 　當然這可能也部分要歸功在他異常粗大的手指，</span><br/><span>     </span><br/><span> 　單獨使用指尖抖音便可達到別人指腹抖音的溫暖音色效果，</span><br/><span>     </span><br/><span> 　抖音的速度卻又更自由許多。</span><br/> <br/> <br/> <br/><span>　帕爾曼的辛德勒名單抖音：</span><br/> <br/><span> </span><br/> <br/> <br/> <br/><span> 　另一個便是知名的歐洲女小提琴家穆特(Anne-Sophie Mutter)</span><br/><span>     </span><br/><span>　堪稱是手腕抖音中的教學範本。</span><br/><span>     </span><br/><span> 　有趣的是，在這幾年來，她開始嘗試少用抖音，</span><br/><span>     </span><br/><span> 　尤其在巴哈無伴奏的演奏上可以看見，</span><br/><span>     </span><br/><span> 　有些長音部分甚至完全沒有加抖音，只用右手來做音色上的層次變化。</span><br/><span>     </span><br/><span> 　她在某次訪問說到，她希望慢慢改善現今世代的抖音現象，</span><br/><span>     </span><br/><span> 　抖音不應該像小七一樣，到處都是。</span><br/><span>     </span><br/><span> 　這想法與她以前的演奏風格實在是有著極大的反差呀。</span><br/> <br/> <br/><span>　穆特的泰伊絲：</span><br/> <br/><span> </span><br/> <br/> <br/> <br/> <br/><span> 　最後，這裡再推薦一個新世代的女小提琴家，</span><br/><span>     </span><br/><span> 　音樂處理手法如此驚人的細膩，Janine Jansen，</span><br/><span>     </span><br/><span> 　極富層次的抖音與弓壓，在在展現出她無比細緻至極的音樂性，</span><br/><span>     </span><br/><span> 　Tchaikovsky Violin concerto 1. and 3. movement</span><br/> <br/> <br/><span> </span><br/> <br/><span>
