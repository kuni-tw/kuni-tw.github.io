  八號中午吃飽後，臨時與好友一同去拜訪直走咖啡，<br/>
<br/>
　這是一家剛開幕沒幾個月的小私人咖啡廳，<br/>
<br/>
　從汀洲路一路走過來，人煙愈來愈稀少，<br/>
<br/>
　走進巷子裡只見得整排的民房，<br/>
<br/>
　一度懷疑是否抄錯了住址。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　好不容易找到了店家，霎那間簡直不敢相信眼前所見，<br/>
<br/>
　一樓的外牆直接打空充作大型透氣落地窗，<br/>
<br/>
　走進一看，窗下有一條長型厚木板架在水泥磚上的長椅，<br/>
<br/>
　相當克難的設計卻頗具巧思。<br/>
<br/>
　店內更是令人嘆為觀止，從天花板吊著的長條形木板桌，<br/>
<br/>
　配合著四處滑動的辦公桌椅，<br/>
<br/>
　刻意讓部分油漆剝落使水泥牆裸露出來的作法相當類似於潮褲，<br/>
<br/>
　還有正在放映片子的投影機，昏暗的燈光讓人感到放心與舒服。<br/>
<br/>

　跟老闆點了一杯熱摩卡(竟然才NT.80，屬隱藏菜單)，順便聊了一下，<br/>
<br/>
　得知老闆的理念是提供給客人一處很放鬆很自在的環境，<br/>
<br/>
　這裡所使用的咖啡豆屬少見的淺烘，咖啡味較沒那麼濃，<br/>
<br/>
　以拿鐵來說，嚐起來會稍甜點。<br/>
<br/>
　後來上桌的摩卡味道令人驚異的自然與調和，<br/>
<br/>
　配合著巧克力的微苦與香氣，展現出老闆細心的作風，<br/>
<br/>
　這的確是一杯讓我願意暫時停下腳步休息一會兒的好咖啡呀。<br/>
