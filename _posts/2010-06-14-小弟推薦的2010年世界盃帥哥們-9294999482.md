 <br/>
<br/>
　 世界盃終於又開打了，從1990義大利那屆開始算起，<br/>
    <br/>
 　至今也看了六屆，不知道從什麼開始，<br/>
    <br/>
　足壇上的帥哥愈來愈容易吸引人的目光。<br/>
    <br/>
　可不是嗎，在廣大的草原上飛也似不停地奔跑，<br/>
    <br/>
　男人與男人之間近乎肉搏式的奮戰，<br/>
    <br/>
　遍地呼喚著，混著滿天飛揚的熱血與汗水，<br/>
    <br/>
　為著自己民族與國家的榮耀背水一戰，<br/>
    <br/>
　套句海賊王裡的湯姆名句："是男人，就去放手一搏"<br/>
    <br/>
　還有比這更有男人味的嗎？<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
　一直到之前個人較欣賞的幾位足球帥哥，<br/>
    <br/>
　多半是屬於成熟型的，且技巧高超，<br/>
    <br/>
　通常也是球隊裡的中心人物，比如葡萄牙的Figo，<br/>
    <br/>
　與Zidane，<br/>
    <br/>
　還有一位大家較不熟的奧地利國家代表隊知名守門員，<br/>
    <br/>
　Michael Konsel，曾效力於AS Rom。<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
    (退休已久的老Konsel仍舊很帥呀)<br/>
<br/>

<br/>
　俗話說的好，長江後浪推前浪，<br/>
    <br/>
　頂尖的職業足球員的運動生涯並不久長，<br/>
    <br/>
　像巴西球王比利那種拿過三屆世界盃冠軍的怪物，<br/>
    <br/>
　真只能說是絕無僅有。<br/>
    <br/>
　如今2010年的世界盃也已開打了，<br/>
    <br/>
　足球帥哥牌理所當然地也要重新洗過。<br/>
    <br/>
　部分帥哥前輩們除以上提過的幾位外，<br/>
    <br/>
　另外還有金童貝克漢等，本屆將不再批上國家隊戰袍，<br/>
    <br/>
　而幾年前的小毛頭們，如今終於"初長成"，<br/>
    <br/>
　這一次小弟順手推舟推薦個幾位，<br/>
    <br/>
　當中有中有老有少，大家將就著看吧：<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
  首先是葡萄牙的美男子，Cristiano Ronaldo，<br/>
    <br/>
　中文普遍稱之為"西羅"，1985二月五號出生，<br/>
    <br/>
　身高185，位置是中場或前鋒。<br/>
    <br/>
　腳下技巧驚人，曾為葡萄牙摘下04年歐洲盃亞軍。<br/>

　世界盃裡個人向來最愛看德國隊的比賽了，<br/>
    <br/>
　冷靜無比的紀律以及組織戰，配合著高超體能，<br/>
    <br/>
　與那無比強大的九十分鐘日耳曼精神力，太精采了。<br/>
    <br/>
　這一屆的老將克羅斯，Miroslav Klose，<br/>
    <br/>
　1978年6月9號出生於波蘭，身高182cm，<br/>
    <br/>
　位置是前鋒，創下連續兩屆世界盃皆進五球之紀錄，<br/>
    <br/>
　目前距離羅納度的世界紀錄還差五球(至2010年六月十五號為止)。<br/>

    <br/>

　於1984年3月20號出生的他，身高185，<br/>
    <br/>

    <br/>
