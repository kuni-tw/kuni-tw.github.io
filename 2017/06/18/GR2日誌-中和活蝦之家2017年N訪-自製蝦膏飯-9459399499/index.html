GR2日誌---中和活蝦之家2017年N訪：自製蝦膏飯！

Jun 18, 2017

(來這裡通常點蒜頭蝦居多)

 

--------------
活蝦之家
地址：新北市中和區景安路60號
電話：0933377937
時間：17:00~賣完為止
--------------

 

偶爾就是想單純過過活蝦癮，愈肥，愈活跳跳，愈好。

景安路上的中和活蝦之家是我心目中首選，價格划算。

現在查得到的母泰國活蝦網購一斤約三百多(10~12隻)，

還有更貴的。

中和活蝦之家這尺寸一斤做成料理四百三起跳，

有桌位有冷氣還有白飯吃到飽，太適合男人下班後前去。

一個人獨自嗑一兩斤，不用跟朋友搶，太爽了。

 

(景安路簡單的攤位與店面)

 

(活蝦池)

 

(撈蝦)

 

(一斤約10~12隻大小)

 

用餐內容：
--------------------
青菜(NT.100)
菜脯蛋(NT.100)
蒜頭蝦(NT.480)
--------------------

 

(店內有電風扇與冷氣可吹)

 

(菜單簡易)

 

來這裡就別奢求精緻度了，首盤高麗菜火候意外足，

簡單的辣椒與蒜調味已能配飯。

沒多久蒜蝦上桌，正處火燙熱度所以無法直接徒手剝殼，

但一聞到那股油煎蒜末所引出的甲殼香實在食指大動。

索性鐵了心，以嘴代左手，右手持筷夾起肥美蝦身，

咬下蝦頭後繼續以雙齒一片一片啃掉蝦殼。

蝦頭內深橘紅蝦膏豐富一如既往，心念一動以筷掏出盛碗，

等蒐集完畢要另外做一份蝦膏飯。

今天蝦肉依舊鮮美結實，咀嚼中充滿有力度的勁道。

11隻一斤的大小一口氣完食，要不是顧及膽固醇，

不然差點跟阿姨再續一斤，胡椒蝦也好吃啊。

 

(炒高麗菜)

 

(蒜頭蝦)

 

(蒜香四溢)

 

(蝦肉厚實充滿嚼勁)

 

(蝦膏)

 

(菜脯蛋)

 

最後的自製蝦膏飯堪稱得來不易，完整取自一斤活蝦。

一開始蝦頭熱燙，只得擱一旁碗中等放涼。

降溫後一隻一隻以筷取出，紅澄澄鋪在白飯上，

飯雖煮不好，但還是美極了。

一口蝦膏白飯，一口甲殼鮮美蒜香；

如此閒情逸致，如此褪去日間煩悶。

 

(一斤活蝦的蝦膏全部取出完畢)

 

(自製的蝦膏飯，太爽了！)
