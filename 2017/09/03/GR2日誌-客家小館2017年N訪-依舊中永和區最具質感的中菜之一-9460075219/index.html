GR2日誌---客家小館2017年N訪：依舊中永和區最具質感的中菜之一

Sep 3, 2017

(斑魚茄煲下飯好吃)

 

-------------------
客家小館
地址：新北市永和區智光街22號
電話：(02)2948-3358
時間：11:00~14:00 17:00~21:00
-------------------


感謝長輩請客，終於又可以吃客家小館了。

這是一家位在中永和區的頂尖中式餐廳，

名曰客家，但實際卻是江浙菜系。


一段時日沒來，外觀變很多，看來有重新裝潢過。

本日點的菜色：
-------------------
燻魚(NT.170~200)
蜜汁芋頭(NT.100)
醉雞(NT.350)
斑魚茄煲(NT.380)
竹笙豆腐煲(NT.350)
時鮮青菜(NT.150)
招牌魯白菜(NT.350)
-------------------

 

(客家小館)

 

(餐廳內)

 

(二樓包廂)

 

(菜單)

 

(小菜一定要提早挑，不然很快就空了)

 

起先桌上招待的菜脯意外不錯。

切得厚實，但由於醃製夠水嫩，實在好吃。

蜜汁芋頭綿密與甜美，燻魚相較下遜色些。

 

(菜脯)

 

(燻魚)

 

(蜜汁芋頭)

 

(白飯)

 

醉雞酒香適度，但肉質不知為何偏乾。

石斑茄子煲雖名曰石斑，可惜魚肉毫無存在感，

不過九層塔香氣迷人，茄子熟度優質，軟爛剛好。

另外一鍋竹笙豆腐煲是美味的一品，

蟹管肉，蟹黃與蛋豆腐藉著熱度讓人胃口大開，

但又是名曰竹笙，卻吃不到什麼竹笙的一道。

 

(醉雞)

 

(斑魚茄煲)

 

(竹笙豆腐煲)

 

(蟹管肉量不少，但幾乎撈不到竹笙)

 

(時鮮青菜)

 

吃到尾端等水果時才想起魯白菜一直沒端來。

外場得知後趕忙進廚房，隔一段時間後倉促上桌。

可能是燉煮過於急切吧，菜葉竟不夠軟與入味，

吃沒幾口便停筷。

結帳時店家相當客氣地沒將這一道算進去，

危機處理還不錯，難怪連平日中午都會滿位。

 

(魯白菜)

 

(招待水果)

 

客家小館總的來說或許沒幾年前驚艷了，

但確確實實還是中永和區最具質感的中菜之一。
