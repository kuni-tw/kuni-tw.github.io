GR2日誌---錵鑶蒸火鍋板橋店：2017年初體驗

Sep 14, 2017

(剛蒸好的蛤蠣實在鮮美！)

 

----------
錵鑶蒸火鍋
地址：新北市板橋區觀光街4號
電話：(02)2958-0987
時間：17:30~00:30
----------

 

沒想到錵鑶也跨足蒸火鍋，聽說連學生都想去吃，

好奇心之下不禁訂了位。

錵鑶體系餐廳愈開愈多間，展店飛快始料未及。

我從聖凱最初四號公園的小Hana壽司開始吃，

到如今他成為網路名人，錵鑶教室的影片不斷被分享。

現在壽司料亭更逐步往中部擴散，先是新竹，

最近聽說台中分店也將落成，太驚人了。

 

(門口)

 

晚間與友相約抵達，帶一支玉乃光。

價格看似不差，尤其沙公划算，一整隻只要700。

難怪規定每層不能點一樣的，

若都只點沙公的話大概會倒店吧。

 

首次吃到蒸火鍋這種東西。

底層趁湯頭滾時先舀一口試試，順口但稍嫌平淡。

服務生算準時間過來打開蒸籠。

石斑溫度偏低，但其他層包括螃蟹熟度都okay，

尤其蛤蠣，剛蒸好的鮮味最迷人，

等涼了肉質緊縮就不好吃了。

今天的玉乃光真棒，非華麗，但細緻均衡。

可惜剛帶來還不夠冰，只得浸冰桶，

隨冰鎮溫度降低，有倒吃甘蔗的感覺。

首度的蒸火鍋體驗還算不錯。

美中不足是底層湯頭，不知為何覺得缺了點深度。

另外部分食材包括沙公與石斑有預先蒸過，

這讓客人不須花太多時間在現場熱蒸。

且確保每層食材能同時蒸熟，算是有效率。

缺點是由於現場熱蒸時間變短，

流進底層的湯汁精華也相對來說減少了，

連帶影響最後的雜炊。

不過整體而言，以這價位來說食材新鮮度夠水準。

喜歡這種現場熱蒸海鮮的調性，

尤其剛蒸好的蛤蠣，是難以言喻的鮮美。
