GR2日誌---給她ㄌㄚˋ：有點意思的麻辣牛肉麵

Oct 16, 2017

(給她ㄌㄚˋ牛肉麵)

 

---------------------
給她ㄌㄚˋ牛肉麵(給她辣)
地址：新北市三重區安和路23號
電話：0989-325-309
時間：12:00~14:00 17:00~20:30 (不定時公休)
---------------------


台灣餐飲界流行取怪名字早已見怪不怪。

近年常見使用地址的號碼，然後數字大寫，

好比北市的參食壱或桃園的貳捌海鮮丼。

名字的特殊性總會留給客人印象，

在行銷上幫助不小。

三重這家麵店直接用注音來拼，

"給她辣"寫成"給她ㄌㄚˋ"也算另一種噱頭。

不過招牌上打的是輕聲的"ㄌㄚ˙"，

而非四聲的"ㄌㄚˋ"，聽說是廠商製作失誤，

後來一直沒更正。

 

由於路途遙遠，出發前特地做好功課。

關於這家麵店的傳聞不少，包括公休日不定，

這跟永和常休店的中田咖哩有一較高下的本錢。

另外由於生意太好，排隊人潮多，

具謠傳還曾發生過五點營業前麵就賣光的紀錄，

不知是真是假！

 

我趕在五點前抵達，店門口與店內意外沒什麼人。

看來要不就傳言有誤，要不就是人氣已減。

本日用餐內容：
---------------------
黃金泡菜(NT.25)
豬血糕(NT.25)
給她ㄌㄚˋ牛肉麵(NT.145)
---------------------

 

(店門口)

 

(店內桌椅)

 

(調味料)

 

(小菜櫃)

 

(菜單)

  

小菜每碟都是台幣25元統一價，

只拿兩種，表現不俗。

當中尤其豬血糕醬汁微鹹微辣，風格討喜。

黃金泡菜酸甜，辣度較高，也好吃。

 

(黃金泡菜與豬血糕) 

 

今天的牛肉麵挺有意思，

我花了好一段時間思考該如何形容。

入口第一念頭竟是麻辣，而非紅燒。

重看菜單不禁笑了出來，底下註明麻辣兩字，

原來點餐時沒仔細閱讀。

湯頭椒辣明顯，層次稍嫌不足。

腱心切得大且厚實卻難得一點也不韌，

俐落咬斷的輕鬆口感相當喜歡。

麵條不弱，類似以前某家乾拌麵，

不屬強而有力的筋道，但微Q微彈。

最後湯頭沒喝完，留下一大半，

主要是這辣度若沒相配合的深度輔佐，

吃到中途便顯膩口與單調，是可惜之處。

 

(屬麻辣風格的牛肉麵)

 

(腱心大塊且厚實) 

 

臨走付帳時才發現老闆娘正使用製麵機，

原來每一碗麵條都是下麵之前才新鮮製作的。

這樣的堅持在台式麵店中算是極少見，

至於有其無必要性與效果則見仁見智了。

 

給她ㄌㄚˋ，

若不考量距離與陰晴不定的營業時間，

的確是有意思的一碗牛肉麵。
