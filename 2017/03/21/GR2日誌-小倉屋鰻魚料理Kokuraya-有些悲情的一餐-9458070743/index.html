GR2日誌---小倉屋鰻魚料理Kokuraya：有些悲情的一餐

Mar 21, 2017


(台北少見走關西風純燒烤的鰻重)

 

 

--------------------
小倉屋鰻魚料理 Kokuraya(田舍庵)
電話：(02)2721-8555
地址：台北市大安區光復南路280巷27號
時間：11:00~14:30 17:30~21:00
--------------------

 

小倉屋的媒體行銷夠厲害。

我同時被不同朋友揪去吃開幕，甚至還同一天巧遇。

總之這家號稱日本百年鰻魚飯來台展店，

幾乎是這幾天饕友們FB上最熱門的話題。

 

與朋友一道去，沒想到他們提早抵達排到了一號，

等於我不用排隊，有這樣的饕友也太幸運了。

 

(排隊人潮)

 

 

一共點了：
--------------------
鰻骨(NT.80)
鰻魚玉子燒(NT.120)
鰻肝燒(NT.120)
鰻重定食(松，NT.480)
--------------------

 

玉子燒，滑膩與淡雅鰻魚香，還不錯，

鰻魚骨則比預期香酥。

鰻肝烤得微焦與嫩，但太小串了。

鰻重就是所謂大盒裝鰻魚丼飯，今天選中等大小。

台北鰻重名店大多屬先烤再蒸，最後刷醬再烤。

小倉屋的鰻重屬台北少見的純關西燒烤手法。

不過其實一月開幕的日本橋玉井也算純燒烤，

只是使用的是海鰻，而非我們常吃的河鰻。

 

(鰻魚骨)

 

(鰻魚玉子燒)

 

(烤鰻肝)

 

今天有吃到朋友的"鰻蒸"，剛好拿來比較。

簡單一句話，烤比蒸美味，不論香氣或口感都如此。

純燒烤出來的焦香較吸引我，另外醬汁調得不差。

但私心覺得內場訓練嚴重不足，好比散漫的鰻魚切工，

或連鋪白飯都能缺乏最基本的美觀，到處是散落米粒，

甚至還有飯盒邊緣用飯杓壓過去的殘餘飯粒。

朋友更是大發脾氣，聽說他那盒的飯沒什麼醬汁。

外場亦不少問題，上餐時間極不穩定外，還送錯過。

看著可憐的外場想說算了，

認命多吃與多付了一份玉子燒的錢。

 

(小倉屋的鰻重)

 

(附湯)

 

(內場鋪得也太草率了...)

 

整體來說今天用餐不算滿意，

或該說與預期的"百年名店稱號"有極大落差，

希望只是開店時期的混亂現象。
