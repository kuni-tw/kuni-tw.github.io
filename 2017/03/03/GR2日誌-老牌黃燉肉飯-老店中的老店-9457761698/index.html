GR2日誌---老牌黃燉肉飯：老店中的老店

Mar 3, 2017

(知名的黃記老牌燉肉飯)

 

----------------------
老牌黃燉肉飯(黃記老牌燉肉飯)
地址：台北市萬華區漢口街二段25號
電話：(02)2361-0089
時間：10:00~20:00
----------------------


臨時跑去某甜點師傅推薦的老店。

位於西門町漢口街，聽朋友說他吃的是回憶。

菜單在牆壁上，點飯類的話附湯，選蘿蔔或味噌，

若不要湯可減十塊錢。

 

(店門口)

 

(價目表)

 


用餐內容：
----------------------
白菜滷(NT.20)
油豆腐(NT.10)
雞腿(NT.75)
燉肉飯(附湯NT.80)
滷肉飯(NT.20)
----------------------


最近連吃好幾家老店，老牌黃應該屬當中較合胃口的。

聽說創於1948年，是老店中的老店。

進來的客人熟練地跟老闆低聲只講兩三個菜名便坐定。

大多一人一桌，彷彿在自個兒家裡用餐似的。

 

(用餐環境)

 

(大塊先白煮過的燉肉)

 

這裡的白菜滷味道單純，但入味與軟嫩，實在順口。

雞腿不柴，肉質尚可，與油豆腐都是懷念台式老味。

燉肉飯給的其實就是焢肉。

米飯一般，但肉比預期好，瘦肉肌理不會過粗。

肥肉或許偏油，不過底層酸菜夠味適當解膩，

讓我當下多扒幾口白飯。

 

(菜一下子上齊)

 

(白菜滷)

 

(炸雞腿)

 

(油豆腐)

 

(燉肉飯)


臨走前一時興起又叫一碗滷肉飯試試。

明顯使用燉肉鍋裡的碎塊，相較下油膩許多，

失去光彩。

結帳時順便外帶一份酸菜，一包NT.20，

能讓我在家多配一碗白飯。

 

(加點的滷肉飯)

 

(酸菜太好吃了，外帶一包)
