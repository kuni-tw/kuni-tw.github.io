GR2日誌---Villa Ollila再訪：2017年生日饗宴

Feb 9, 2017

(希望年年都能如此開心！)

 

--------------
Villa Ollila
地址：內湖區
電話：0927-340-034
--------------

 

每年生日都要特地挑一家餐廳慶祝，今年亦不例外。

剛好適逢號稱樂沐私廚的Villa Ollila招待所邀請，

順理成章就欣然答應了，機會難得嘛。

 

(又見Continental的仿骨董車)

 

晚間抵達時盡是各高級餐廳總監好友們齊聚一堂，

一段時間未見，大夥兒熱絡擁抱與噓寒問暖一番。

這幾年台北Fine Dining界人事異動劇烈，

光這話題就能聊上好幾個鐘頭。

負責迎賓的是Champagne Perrier Jouet N.V.，

淺黃的明亮光澤，帶著水果與核果香氣。

 

(Champagne Perrier Jouet N.V.)

 

(左邊Sam是舊識了，右邊新外場)

 

(酒杯擺設)

 

(朋友帶來送我的禮物之一)

 

(要喝的酒杯)

 

(盛宴開始)

 

感謝大主廚Dennis為今天發表了全新菜單，

首道開胃菜便讓我驚艷。

蕪菁切得極薄，細心包覆極致鮮美的帝王蟹肉，

一同浸泡於菠菜泥表層，無可挑剔。

 

(餐前麵包)

 

(帝王蟹佐蕪菁薄片與菠菜泥)

 

(可食性植物桌飾)

 

第二道是湯品，搭配一支清酒，入口後嚇了一跳，

甘美純粹尾韻悠長，找來侍酒師端看酒瓶，

原來是岩手縣的南部美人純米大吟釀。

鮑魚切得厚實大方，口感相對滑嫩異常，

昆布湯盡是優雅松露香，味道好，但下得重，

令我有些擔心後面菜色調味的走向。

 

(這支喜歡)

 

(活鮑佐黑松露與昆布湯)

 

第三道配酒是Domaine Emile Balland Les Beaux Jours。

2015年份，有著活潑與魄力的酸度，與鮮美干貝匹配。

小而美的手摘豆仁負責甜美，羊肚菌一旁點綴淡雅與氣質，

看似簡單俐落卻充滿廚藝者的細膩見解與品味。

 

(Domaine Emile Balland Les Beaux Jours)

 

(香煎大干貝佐甜豆仁與羊肚菌)

 

(充滿美感)

 

第四道是西班牙Jamon Serrano煲湯。

現場由侍酒師將Hennessy VSOP斟入湯裡，入口絕妙。

法國干邑與西班牙火腿湯頭的組合竟毫不衝突，

溫潤中帶著來酒香的層次。

唯一可惜是碗中的黑松露片，存在感較缺。

 

(斟入法國干邑) 

 

第五支配酒是Lillet Blanc，草本香氣濃郁。

主角則是嫩鴨肝，外皮微炙帶點煙燻，

佐一旁特製開心果餅再度是讓人心儀的組合。

 

(香煎鴨肝)

 

(Lillet Blanc)

 

(煙燻鴨肝佐漬櫻桃與開心果餅)

 

第六道主廚冬季沙拉，搭配Stag's Leap Chardonnay。

喝過幾支Napa谷鹿躍酒廠的CS紅酒，但白酒卻是首次。

沙拉裡的菜葉種類繁多，連草石蠶的塊莖都有，

現場淋上熱燙帕馬森起司醬，散著核果與香檳茸香味。

 

(Stag's Leap Chardonnay)

 

(主廚冬季沙拉)

 

第七支配酒是國產的Weightstone Musann Blanc，

初飲訝異，全然的荔枝甜香之下入口卻是強勁白葡萄酒，

違和到了一個極致，驚人的品質與亮酸，

台灣造酒技術竟已到這等地步。

搭配的是現流金目鯛，熟度抓得極好。

雖然外皮缺了香酥，但魚肉嫩度鮮美動人。

與首訪相較下，Chef對於熟度掌控又更上一層樓了。

 

(來自埔里的Weightstone Musann Blanc)

 

(現流金目鯛)

 

(精彩熟度)

 

第八道是非常高檔的羊腿，可惜不能透露太多細節，

只能說產自某知名山區。

熟度維持一貫高超，juicy與柔嫩肉質還有細膩馨香。

才想起來，羊排料理向來最鍾愛台北中山區的Popina，

羊腿卻還沒吃過這等品質，肌理不粗不柴，味道雋永。

配酒是Chateau Mas Neuf Chateauneuf-du-Pape，

花香衝鼻，正值最佳試飲期。

 

(一整支羊腿)

 

(Chateau Mas Neuf Chateauneuf-du-Pape)

 

(小羔羊腿佐韭蔥與發酵黑蒜)

 

(肌理不粗不柴，味道雋永)

 

(臨時加開朋友送的Denis Thomas Echezeaux)

 

整餐吃下來最驚豔的是Dennis對於食材的熟度掌控，

除毫無失誤外，我更喜歡他類似走鋼索般的膽大心細。

另外侍酒師Sam表現突出，比2015年首訪更沉穩。

甜點完後是誠品行旅總監與總經理帶來的蛋糕。

年紀大了，許願時沒想太多，只盼年年健健康康，

時時與父母或好友們相聚一堂，別無所求。

 

(草莓佐白味增與椰子Sorbet)

 

(茶點)

 

(伯爵茶)

 

(再開一支朋友送的Lafite-Rothschild 1973)

 

(酒瓶排排站)

 

(感謝你們！)

 

(感謝誠品行旅總監的生日蛋糕！)
